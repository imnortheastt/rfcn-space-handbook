## Tai sao dieu nay quan trong

5G Core co nhieu instance cua moi NF de mo rong va phuc hoi. Khi AMF can SMF, no chon cai nao?

Tieu chi lua chon NF xac dinh phan phoi tai, phuc hoi loi, va dinh tuyen toi uu. Lua chon sai gay diem nong, tang do tre, hoac loi dich vu.

</section>

## Ap dung

**Tom tat tieu chi lua chon:**

| Tieu chi | Loai | Gia tri | Muc dich |
|----------|------|---------|----------|
| Priority | Xep hang | 0-65535 (thap=tot hon) | Chinh/phu |
| Capacity | Trong so | 0-65535 (tuong doi) | Phan phoi tai |
| Load | Dong | 0-100% | Tranh qua tai |
| Locality | Uu tien | Chuoi phan cap | Do tre |
| S-NSSAI | Loc | SST+SD | Dinh tuyen slice |
| TAI | Loc | PLMN+TAC | Phu vung |

**Vi du kham pha NRF:**

```bash
# Kham pha SMF cho slice 1 trong TAI 0001
curl -X GET "http://nrf:8080/nnrf-disc/v1/nf-instances\
?target-nf-type=SMF\
&requester-nf-type=AMF\
&snssais=%5B%7B%22sst%22%3A1%7D%5D\
&tai=%7B%22plmnId%22%3A%7B%22mcc%22%3A%22001%22%2C%22mnc%22%3A%2201%22%7D%2C%22tac%22%3A%220001%22%7D" | jq

# Kiem tra trang thai tai NF
curl -X GET "http://nrf:8080/nnrf-nfm/v1/nf-instances/smf-1-uuid" | jq '.load, .capacity'

# Cap nhat tai NF
curl -X PATCH "http://nrf:8080/nnrf-nfm/v1/nf-instances/smf-1-uuid" \
  -H "Content-Type: application/json-patch+json" \
  -d '[{"op": "replace", "path": "/load", "value": 75}]'
```

**Ma gia thuat toan lua chon:**

```python
def select_nf(discovery_response):
    candidates = discovery_response.nfInstances
    
    # Nhom theo priority
    groups = group_by(candidates, key=lambda nf: nf.priority)
    
    for priority in sorted(groups.keys()):
        group = groups[priority]
        
        # Loc bo qua tai
        available = [nf for nf in group if nf.load < 90]
        if not available:
            continue
        
        # Uu tien cuc bo
        local = [nf for nf in available if nf.locality == my_locality]
        pool = local if local else available
        
        # Ngau nhien theo trong so capacity
        return weighted_choice(pool, weight=lambda nf: nf.capacity)
    
    return None  # Khong co NF phu hop
```

<section class="lesson-pitfalls">

## Cam bay

1. **Tat ca NF cung priority.** Khong co phan biet priority, khong the lam chinh/phu. Dat priority co chu dich.

2. **Capacity khong cap nhat.** Gia tri capacity tinh khong phan anh kha dung thuc. Cap nhat capacity khi co su kien mo rong.

3. **Load cu.** Cap nhat load khong du thuong xuyen. Dat khoang thoi gian heartbeat phu hop (30s-60s).

4. **Khong khop locality.** Consumer va producer dung ten locality khac nhau. Chuan hoa trong toan bo trien khai.

5. **Dich vu khong dang ky.** NF dang ky nhung dich vu thieu trong danh sach nfServices. Consumer khong tim thay.

6. **Cache discovery het han.** Consumer cache ket qua discovery qua validityPeriod. Ton trong TTL.

7. **Khong du phong khi that bai.** NF da chon that bai, consumer khong thu lai voi ung vien tiep theo. Trien khai logic thu lai.

8. **Bo loc qua han che.** Khong NF nao khop tat ca tieu chi. Noi long bo loc tuy chon, giu bo loc bat buoc.
