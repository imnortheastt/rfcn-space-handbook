## Tai sao dieu nay quan trong

NSSF khong chon slice ngau nhien. Co thuat toan xac dinh xem xet nhung gi UE muon, goi cuoc cho phep gi, va nhung gi kha dung tai vi tri hien tai.

Hieu thuat toan nay la can thiet de go loi that bai lua chon slice va cau hinh trien khai da slice.

</section>

## Cach hoat dong

### Thuat toan lua chon

```
Thuat toan Lua chon Chi tiet:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  function selectSlices(request):                        │
│                                                          │
│    requestedNssai = request.requestedNssai              │
│    subscribedNssai = request.subscribedNssai            │
│    tai = request.tai                                    │
│                                                          │
│    allowedNssai = []                                    │
│    rejectedNssai = []                                   │
│                                                          │
│    for each sNssai in requestedNssai:                   │
│                                                          │
│      // Kiem tra goi cuoc                               │
│      if sNssai not in subscribedNssai:                  │
│        rejectedNssai.add({sNssai, cause: "NOT_SUBSCR"}) │
│        continue                                         │
│                                                          │
│      // Kiem tra kha dung TAI                           │
│      taiConfig = getConfiguredNssai(tai)                │
│      if sNssai not in taiConfig:                        │
│        rejectedNssai.add({sNssai, cause: "NOT_AVAIL"})  │
│        continue                                         │
│                                                          │
│      // Kiem tra kha dung AMF                           │
│      amfSet = findAmfSet(sNssai, tai)                   │
│      if amfSet is empty:                                │
│        rejectedNssai.add({sNssai, cause: "NO_SERVING"}) │
│        continue                                         │
│                                                          │
│      // Slice duoc phep                                 │
│      allowedNssai.add({sNssai, amfSet: amfSet})         │
│                                                          │
│    // Ap dung mac dinh neu rong                         │
│    if allowedNssai.isEmpty() and hasDefaultSlice():     │
│      allowedNssai.add(getDefaultSlice(tai))             │
│                                                          │
│    return {allowedNssai, rejectedNssai, targetAmfSet}   │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## Ap dung

**Quy tac uu tien lua chon:**

| Uu tien | Quy tac |
|---------|---------|
| 1 | Slice UE yeu cau khop voi goi cuoc |
| 2 | Slice UE yeu cau kha dung trong TAI |
| 3 | S-NSSAI mac dinh goi cuoc cho TAI |
| 4 | S-NSSAI mac dinh nha mang |

**Vi du cau hinh (NSSF):**

```yaml
# Cau hinh slice NSSF
nssaiConfiguration:
  plmnId:
    mcc: "001"
    mnc: "01"
  taiConfiguration:
    - tai:
        tac: "0001"
      supportedSnssaiList:
        - sst: 1
        - sst: 2
        - sst: 3
      defaultSnssai:
        sst: 1
    - tai:
        tac: "0002"
      supportedSnssaiList:
        - sst: 1
      defaultSnssai:
        sst: 1

amfSetConfiguration:
  - amfSetId: "amf-set-embb"
    supportedSnssaiList:
      - sst: 1
  - amfSetId: "amf-set-urllc"
    supportedSnssaiList:
      - sst: 2
  - amfSetId: "amf-set-all"
    supportedSnssaiList:
      - sst: 1
      - sst: 2
      - sst: 3
```

**Lenh go loi:**

```bash
# Kiem tra quyet dinh lua chon NSSF
kubectl logs -l app=nssf | grep -E "(allowedNssai|rejectedNssai)"

# Xac minh cau hinh TAI
curl http://nssf:8080/internal/config/tai | jq

# Kiem tra lua chon cu the
curl -X POST http://nssf:8080/nnssf-nsselection/v1/network-slice-info \
  -H "Content-Type: application/json" \
  -d '{
    "nfType": "AMF",
    "sliceInfoRequestForRegistration": {
      "requestedNssai": [{"sst": 1}, {"sst": 2}],
      "tai": {"plmnId": {"mcc": "001", "mnc": "01"}, "tac": "0001"}
    }
  }' | jq '.allowedNssai, .rejectedNssaiInTa'
```

<section class="lesson-pitfalls">

## Cam bay

1. **Khong khop TAI trong yeu cau.** NSSF dung TAI de xac dinh kha dung cuc bo. TAI sai cho ket qua sai.

2. **Subscribed NSSAI khong co trong yeu cau.** AMF phai bao gom subscribed NSSAI tu UDM. Du lieu thieu gay tu choi.

3. **Slice mac dinh khong cau hinh.** Neu khong slice yeu cau nao kha dung va khong co mac dinh, UE khong nhan duoc gi.

4. **AMF set khong dang ky.** NSSF biet slice nhung khong biet AMF nao phuc vu. Dang ky kha dung AMF.

5. **SD bi bo qua khi so khop.** SST=1 va SST=1,SD=abc la khac nhau. Can khop chinh xac.

6. **Thieu anh xa NSSAI roaming.** PLMN khach dung gia tri S-NSSAI khac. Cau hinh bang anh xa.

7. **Du lieu goi cuoc cu.** NSSF cache goi cuoc. Thay doi trong UDM khong phan anh. Kiem tra TTL cache.

8. **Du phong mot slice bat ngo.** UE yeu cau nhieu, chi nhan mot. Kiem tra neu cac slice khac bi tu choi hoac AMF khong the phuc vu.
