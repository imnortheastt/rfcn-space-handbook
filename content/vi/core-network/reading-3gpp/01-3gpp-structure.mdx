## Ap dung

**URLs tham chieu nhanh:**

| Tai nguyen | URL |
|------------|-----|
| Tim spec | https://portal.3gpp.org/desktopmodules/Specifications/SpecificationDetails.aspx |
| Chuoi spec | https://www.3gpp.org/specifications/specification-numbering |
| TT Release | https://www.3gpp.org/specifications/releases |
| Ke hoach | https://www.3gpp.org/specifications/work-plan |

**Dieu huong dong lenh:**

```bash
# Tai spec tu 3GPP (sau khi tim URL)
wget "https://www.3gpp.org/ftp/Specs/archive/23_series/23.501/23501-h60.zip"
unzip 23501-h60.zip

# Tim trong spec (sau khi giai nen)
grep -i "registration" TS_23.501_*.docx 2>/dev/null || \
  # Neu docx, dung cong cu thay the
  unzip -p TS_23.501*.docx word/document.xml | grep -i "registration"

# So sanh phien ban (dung diff tren xuat text)
diff ts23501_v16.txt ts23501_v17.txt | head -100
```

**Mau tra cuu pho bien:**

```bash
# Tim spec cho chu de:

1. "Dang ky UE hoat dong the nao?"
   → Bat dau: TS 23.502 (thu tuc)
   → Muc: 4.2.2 Registration

2. "API AMF cho dang ky la gi?"
   → Bat dau: TS 29.518 (dich vu AMF)
   → Tim Namf_Communication

3. "Ban tin NAS cho dang ky la gi?"
   → Bat dau: TS 24.501 (giao thuc NAS)
   → Muc: 8 (dinh nghia ban tin)

4. "Dang ky duoc bao mat the nao?"
   → Bat dau: TS 33.501 (bao mat)
   → Muc: 6 (xac thuc)
```

<section class="lesson-pitfalls">

## Cam bay

1. **Sai release.** Doc spec Rel-15 cho tinh nang Rel-17. Kiem tra release truoc.

2. **Nham lan stage.** Stage 1 (dich vu) ≠ Stage 2 (kien truc) ≠ Stage 3 (giao thuc). Biet ban can gi.

3. **Qua tai viet tat.** 3GPP thich viet tat. Giu Clause 3 mo moi luc.

4. **Tham chieu vong.** Spec A tham chieu B roi tham chieu A. Xay dung ban do tinh than.

5. **Bay "FFS".** "For Further Study" nghia la chua dinh nghia. Khong trien khai cac muc FFS.

6. **Ban sao cuc bo cu.** Dung spec tai xuong tu nhieu nam truoc. Kiem tra cap nhat.

7. **Quy chuan vs thong tin.** Phu luc thuong la thong tin (vi du). Phan chinh la quy chuan.

8. **Thieu ngu canh.** Doc TS 29.xxx khong hieu kien truc TS 23.xxx. Bat dau voi kien truc.
