<section class="lesson-hook">

## Tai sao dieu nay quan trong

Khi AMF can mot SMF, lam sao no biet co nhung SMF instance nao? Dia chi IP nao? Nhung slice nao chung phuc vu?

NRF tra loi dieu nay. Moi NF dang ky voi NRF khi khoi dong, gui heartbeat khi chay, va huy dang ky khi tat. Khong co NRF, kien truc dich vu dong cua 5G se sup do.

</section>

## Ap dung

**Cac truong NF profile quan trong theo loai NF:**

| Loai NF | Truong Profile Quan trong |
|---------|--------------------------|
| SMF | smfInfo (sNssaiSmfInfoList, taiList, dnnList) |
| AMF | amfInfo (amfSetId, amfRegionId, guamiList, taiList) |
| UDM | udmInfo (supiRanges, gpsiRanges) |
| AUSF | ausfInfo (supiRanges, groupId) |
| PCF | pcfInfo (dnnList, supiRanges) |
| UPF | upfInfo (sNssaiUpfInfoList, interfaceUpfInfoList) |

**Debug dang ky:**

```bash
# Kiem tra dang ky NF
curl -k "https://nrf.local:29510/nnrf-nfm/v1/nf-instances/{id}"

# Liet ke tat ca NF da dang ky
curl -k "https://nrf.local:29510/nnrf-nfm/v1/nf-instances"

# Wireshark: dang ky/kham pha
http2.header.path contains "nnrf-nfm" || http2.header.path contains "nnrf-disc"

# Kiem tra loi heartbeat
grep "heartbeat" /var/log/nrf/*.log | grep -i "fail\|timeout"
```

**Thoi gian heartbeat:**

| Tinh huong | Timer Heartbeat | Phuc hoi |
|------------|-----------------|----------|
| Binh thuong | 30-60 giay | — |
| HA cao | 10-20 giay | Failover nhanh |
| Dung sai NRF | 2-3 lan mat | Danh dau NF khong khoe |
| Huy dang ky | DELETE graceful | Xoa ngay lap tuc |

<section class="lesson-pitfalls">

## Cam bay

1. **Timeout heartbeat.** Neu NF crash ma khong huy dang ky, NRF doi heartbeat timeout. Cau hinh timer ngan hon cho HA.

2. **NRF diem loi duy nhat.** Trien khai NRF theo cum HA. Cac NF nen cache NF profile da kham pha.

3. **Profile khong khop.** Neu NF profile khong khop chinh xac tieu chi truy van, se khong duoc kham pha. Kiem tra sNssai, DNN, TAI.

4. **Cache kham pha cu.** Consumer cache NF profile. Neu producer thay doi, cache cu. Dang ky nhan thong bao.

5. **Can bang tai.** NRF tra ve nhieu instance. Consumer phai implement logic chon (round-robin, least-load).

6. **Thu lai dang ky.** Neu dang ky ban dau that bai, NF phai thu lai voi backoff. Dung lam ngap NRF.

7. **Cap nhat profile.** Khi NF thay doi (service moi, IP moi), phai dang ky lai. Dung PUT de thay the hoac PATCH de cap nhat.

8. **Tao UUID.** nfInstanceId phai duy nhat. Dung tao UUID dung, khong phai ID tuan tu.
