## Tai sao dieu nay quan trong

API SBI 5G khong duoc tai lieu hoa bang van ban — chung duoc dinh nghia trong cac dac ta OpenAPI doc duoc bang may. Moi request body, response, va loi deu duoc chi dinh chinh thuc.

Dieu nay co nghia ban co the tao ma client, xac thuc tin nhan, va debug van de tuong thich bang cach doc spec. Hieu OpenAPI trong boi canh 5G la can thiet cho trien khai va xu ly su co.

</section>

## Ap dung

**Tim spec OpenAPI:**

| Nguon | Vi tri |
|-------|--------|
| 3GPP chinh thuc | https://www.3gpp.org/ftp/Specs/archive/ |
| Thu muc API | Chuoi TS 29.5xx (moi NF co spec rieng) |
| Kieu chung | TS 29.571 |
| Dinh dang | File YAML trong archive spec |

**Xac thuc tin nhan:**

```bash
# Tao kieu tu OpenAPI
openapi-generator generate -i TS29502_Nsmf.yaml -g python

# Xac thuc JSON voi schema
ajv validate -s schema.json -d message.json

# Wireshark: xem body JSON
http2.data && json

# In dep JSON tu log
jq '.' /var/log/smf/sbi-trace.json
```

**Mau xac thuc pho bien:**

| Truong | Xac thuc | Regex/Pattern |
|--------|----------|---------------|
| SUPI | imsi-[15 chu so] | `^imsi-[0-9]{15}$` |
| GPSI | msisdn-[chu so] | `^msisdn-[0-9]{5,15}$` |
| DNN | Chuoi ASCII | `^[a-zA-Z0-9.-]+$` |
| S-NSSAI SST | 0-255 | so nguyen |
| S-NSSAI SD | 6 chu so hex | `^[0-9A-Fa-f]{6}$` |

<section class="lesson-pitfalls">

## Cam bay

1. **Khong khop version spec.** Spec OpenAPI phat trien theo cac release 3GPP. Dam bao ban dung dung version (R15, R16, R17).

2. **Xu ly field khong biet.** Mot so trien khai tu choi field JSON khong biet. So khac bo qua. Test interop can than.

3. **Null vs vang mat.** JSON null khac voi field vang mat. Mot so API nghiem ngat ve dieu nay.

4. **Ma hoa Base64.** Du lieu nhi phan phai duoc ma hoa base64 dung. Van de padding pho bien.

5. **Dinh dang DateTime.** Phai la RFC 3339 (ISO 8601). Yeu cau time zone. "Z" cho UTC.

6. **Giai quyet $ref.** Tham chieu cheo file yeu cau tat ca spec duoc tham chieu phai co san.

7. **Parse multipart.** Xu ly boundary kho. Content-Id phai khop chinh xac.

8. **Phan biet hoa thuong enum.** Enum 3GPP thuong la UPPER_CASE. "registered" ≠ "REGISTERED".

</section>
