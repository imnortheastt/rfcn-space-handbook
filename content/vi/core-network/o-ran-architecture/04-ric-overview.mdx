## Tai sao dieu nay quan trong

RAN truyen thong dong va tinh. Sau khi trien khai, toi uu can su tham gia cua vendor va dieu chinh thu cong.

RIC mang den kha nang lap trinh. Ung dung ben thu ba co the quan sat trang thai RAN, chay mo hinh ML, va day quyet dinh — ma khong can cham vao code cua vendor. Day la diem khac biet chinh cua O-RAN.

</section>

│  │                                                  │    │
│  │  REPORT:     E2 Node → RIC (do luong)          │    │

│  Mo hinh Dich vu E2 (E2SM):                             │
│  ┌─────────────────────────────────────────────────┐    │

## Ap dung

**Phan loai xApp:**

| Loai | Vong lap Dieu khien | Vi du |
|------|---------------------|-------|
| Dieu huong luu luong | 100-500 ms | Toi uu HO, can bang tai |
| Quan ly QoS | 200 ms - 1 s | Thuc thi SLA slice |
| Phat hien bat thuong | 500 ms - 1 s | Phat hien nhieu |
| Tiet kiem nang luong | 1+ s | Bat/tat cell, che do ngu |

**Debug RIC:**

```bash
# Trang thai ket noi E2
kubectl -n ricplt exec -it e2term-0 -- e2cli status

# Trang thai subscription xApp
kubectl -n ricxapp logs <xapp-pod> | grep -i subscribe

# Du lieu SDL
kubectl -n ricplt exec -it dbaas-0 -- redis-cli KEYS "*"

# Bang dinh tuyen RMR
kubectl -n ricplt exec -it rtmgr-0 -- rtmgr-cli show routes

# Chinh sach A1
curl http://<a1-mediator>:10000/A1-P/v2/policytypes
```

**Cac interface chinh:**

| Interface | Diem cuoi | Giao thuc | Do tre |
|-----------|-----------|-----------|--------|
| E2 | Near-RT RIC ↔ CU/DU | SCTP | < 10 ms |
| A1 | Non-RT RIC ↔ Near-RT RIC | HTTP/REST | < 1 s |
| O1 | SMO ↔ tat ca node | NETCONF | N/A |
| O2 | SMO ↔ O-Cloud | REST | N/A |

<section class="lesson-pitfalls">

## Cam bay

1. **Xung dot giua cac xApp.** Hai xApp co the gui dieu khien xung dot. Su dung framework quan ly xung dot.

2. **Overhead subscription E2.** Qua nhieu subscription ap do E2 node. Thiet ke do chi tiet subscription hieu qua.

3. **Nhat quan SDL.** Du lieu cu trong SDL dan den quyet dinh sai. Dat TTL phu hop.

4. **Ngan sach do tre.** Quyet dinh xApp + RTT E2 phai nam trong 10ms-1s. Profile can than.

5. **Troi mo hinh ML.** Mo hinh huan luyen tren du lieu cu suy giam. Trien khai huan luyen lien tuc trong Non-RT RIC.

6. **Mat tin nhan RMR.** Message router co the bo khi qua tai. Trien khai thao tac idempotent.

7. **Xung dot chinh sach A1.** Chinh sach xung dot tu Non-RT RIC. Xac thuc tuong thich chinh sach truoc khi day.

8. **Khoi dong lai E2 node.** Subscription E2 mat khi node restart. xApp phai xu ly dang ky lai.
