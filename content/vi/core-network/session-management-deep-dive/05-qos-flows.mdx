## Tai sao dieu nay quan trong

Mot PDU session co the mang nhieu ung dung — duyet web, goi video, gaming, tai file. Moi loai co yeu cau do tre va bang thong khac nhau.

**Luong QoS** cho phep mang xu ly chung khac nhau trong cung mot session. Khong co chung, moi thu deu duoc xu ly best-effort.

</section>

## Truc giac

### Luong QoS la gi

```
Khai niem Luong QoS:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  PDU Session (Dia chi IP: 10.0.0.5)                     │
│  │                                                       │
│  ├── Luong QoS 1 (QFI=1)                                │
│  │   └── Luong mac dinh, best effort                    │
│  │   └── Duyet web, email, nen                          │
│  │                                                       │
│  ├── Luong QoS 2 (QFI=2)                                │
│  │   └── Goi video, GBR                                 │
│  │   └── Dam bao 5 Mbps                                 │
│  │                                                       │
│  └── Luong QoS 3 (QFI=3)                                │
│      └── Gaming, do tre thap                            │
│      └── Muc tieu delay 20ms                            │
│                                                          │
│  Cung IP, xu ly khac nhau theo flow                     │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### QFI vs 5QI

```
Moi quan he QFI va 5QI:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  QFI (QoS Flow Identifier):                             │
│  - Dinh danh cuc bo trong PDU session                   │
│  - Pham vi: 1-63                                        │
│  - SMF gan                                              │
│  - Dung trong header GTP-U de danh dau goi              │
│                                                          │
│  5QI (5G QoS Identifier):                               │
│  - Dinh nghia dac tinh QoS                              │
│  - Gia tri chuan hoa (1-255)                            │
│  - Anh xa den uu tien, do tre, ty le loi               │
│  - Co the la chuan hoa hoac do nha mang dinh nghia      │
│                                                          │
│  Moi quan he:                                           │
│  QFI → tro den → 5QI + cac tham so bo sung              │
│                                                          │
│  Vi du:                                                 │
│  QFI=2 → 5QI=2 (thoai) + GBR=64kbps + MBR=128kbps      │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### GBR vs Non-GBR

```
Cac Loai Luong QoS:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  GBR (Guaranteed Bit Rate):                             │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  - Mang dat truoc bang thong                    │    │
│  │  - Tham so: GBR, MBR, ngan sach delay goi       │    │
│  │  - Kiem soat chap nhan: co the tu choi neu het  │    │
│  │  - Su dung: thoai, video conference, thoi gian  │    │
│  │    thuc                                          │    │
│  │                                                  │    │
│  │  Vi du 5QI: 1 (thoai), 2 (video), 65 (MCPTT)   │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  Non-GBR (Best Effort):                                 │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  - Khong dat truoc bang thong                   │    │
│  │  - Tham so: uu tien, ngan sach delay goi        │    │
│  │  - Luon duoc chap nhan (khong dam bao tai nguyen)│    │
│  │  - Su dung: web, email, streaming, download     │    │
│  │                                                  │    │
│  │  Vi du 5QI: 5 (IMS signaling), 8, 9 (web)      │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  Delay-Critical GBR:                                    │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  - GBR + do tre nghiem ngat                     │    │
│  │  - Tham so: GBR + ngan sach delay goi (cung)    │    │
│  │  - Su dung: V2X, AR/VR, phau thuat tu xa        │    │
│  │                                                  │    │
│  │  Vi du 5QI: 82 (V2X), 83 (dien luc)             │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### Reflective QoS

```
Reflective QoS (RQoS):

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  Van de: UE can quy tac QoS cho uplink                  │
│  Giai phap: Suy ra tu luu luong downlink                │
│                                                          │
│  1. Mang gui goi DL voi QFI=2                           │
│  2. UE thay: src=server:port, dst=ue:port, QFI=2        │
│  3. UE tao quy tac reflective:                          │
│     src=ue:port, dst=server:port → dung QFI=2          │
│  4. Goi UL den cung server duoc QFI=2                   │
│                                                          │
│  Kich hoat qua:                                         │
│  - RQI (Reflective QoS Indication) bit trong GTP-U     │
│  - RQ timer (quy tac ton tai bao lau)                  │
│                                                          │
│  Truong hop:                                            │
│  - Dich vu do mang khoi tao                            │
│  - Khi UE khong biet dia chi server truoc              │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## Ap dung

**Luong QoS theo ung dung:**

| Ung dung | 5QI | Loai | Thiet lap Dien hinh |
|----------|-----|------|---------------------|
| Duyet web | 9 | Non-GBR | Luong mac dinh |
| VoLTE | 1 | GBR | Bearer rieng, 64kbps |
| Goi video | 2 | GBR | 2-10 Mbps |
| Gaming | 3 | GBR | Do tre thap, 10-50ms |
| Live streaming | 6 | Non-GBR | Uu tien hon web |
| V2X | 82 | DC-GBR | Ngan sach 10ms |

**Debug QoS:**

```bash
# Wireshark: loc theo QFI trong GTP-U
gtp.ext.qos_flow_id == 2

# Kiem tra quy tac QoS gui den UE
nas-5gs.sm.qos_rule

# PFCP QER trong thiet lap session
pfcp.qer_id

# Cau hinh DRB gNB
grep "drb-ToAddMod" /var/log/gnb/*.log
```

**Xu ly su co:**

| Van de | Kiem tra |
|--------|----------|
| Video giat | Luong GBR da thiet lap? Du bang thong? |
| Gaming lag | 5QI do tre thap da dung? Lap lich DRB? |
| Thoai reu | Luong QFI=1 hoat dong? Mat goi? |
| Tat ca cham | AMBR mac dinh qua thap? UPF qua tai? |

<section class="lesson-pitfalls">

## Cam bay

1. **Khong khop bo loc goi.** Neu UE gui goi khong khop quy tac QoS nao, no dung luong mac dinh. Thieu bo loc = QoS sai.

2. **Trung uu tien.** Hai quy tac cung uu tien — hanh vi khong xac dinh. Luon gan uu tien duy nhat.

3. **Chap nhan GBR that bai.** Mang co the tu choi luong GBR neu thieu tai nguyen. App phai xu ly giam cap nhe nhang.

4. **Can QFI.** Chi co 63 gia tri QFI moi session. Kho xay ra nhung co the voi app phuc tap.

5. **Anh xa DRB sai.** gNB anh xa luong QoS den DRB. Anh xa te = cac luong yeu cau khac nhau chia se DRB.

6. **Timeout Reflective QoS.** RQ timer het han → UE xoa quy tac → goi UL di vao luong mac dinh. Server phai tiep tuc gui.

7. **AMBR vs toc do flow.** Session AMBR gioi han tat ca flow cong lai. MBR flow don le co the vuot session AMBR nhung tong khong.

8. **Lech policy.** PCF policy noi mot dang, SMF cau hinh khac. Kiem tra quyet dinh SM policy.
