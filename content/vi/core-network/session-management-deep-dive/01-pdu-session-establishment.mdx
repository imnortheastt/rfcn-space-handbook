## Tai sao dieu nay quan trong

Dang ky dua ban len mang. Nhung de thuc su su dung du lieu — luot web, xem video, gui tin nhan — ban can **PDU session**. Day la khi ban co dia chi IP va mang thiet lap duong du lieu.

Moi lan ban mo ung dung can internet, PDU session lam cho no hoat dong.

</section>

<section class="lesson-intuition">

## Truc giac

### PDU session cung cap gi

```
PDU Session = Ket noi Du lieu:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  Truoc PDU Session:                                     │
│  ┌─────────────────────────────────────────────────┐    │
│  │ UE da dang ky (co 5G-GUTI)                      │    │
│  │ Nhung KHONG CO dia chi IP, KHONG CO duong du lieu│   │
│  │ Chi co the lam bao hieu (tin nhan NAS)          │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  Sau PDU Session:                                       │
│  ┌─────────────────────────────────────────────────┐    │
│  │ UE co dia chi IP (vd: 10.0.0.5)                │    │
│  │ Duong du lieu: UE → gNB → UPF → Internet       │    │
│  │ Quy tac QoS duoc ap dung (5QI, AMBR)           │    │
│  │ Co the gui/nhan du lieu nguoi dung              │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### Tham so PDU session

| Tham so | Y nghia | Vi du |
|---------|---------|-------|
| **DNN** | Ten Mang Du lieu (nhu APN) | "internet", "ims" |
| **S-NSSAI** | Dinh danh slice | SST=1 (eMBB) |
| **PDU type** | Phien ban IP | IPv4, IPv6, IPv4v6 |
| **SSC mode** | Lien tuc session | Mode 1, 2, hoac 3 |

<section class="lesson-apply">

## Ap dung

**Cau hinh DNN pho bien:**

| DNN | Muc dich | Cai dat thuong gap |
|-----|----------|-------------------|
| internet | Du lieu chung | IPv4v6, SSC mode 1 |
| ims | Cuoc goi thoai/video | IPv6, 5QI chuyen biet |
| mms | Nhan tin da phuong tien | IPv4, uu tien thap |
| enterprise | Truy cap doanh nghiep | IPv4, tunnel VPN |

**Debug PDU session:**

```bash
# Bo loc Wireshark cho tin nhan PDU session
nas-5gs.sm.message_type == 0xc1 || nas-5gs.sm.message_type == 0xc2

# Kiem tra chon SMF
curl "https://nrf.5gc/nnrf-disc/v1/nf-instances?target-nf-type=SMF&dnn=internet"

# PFCP session tren N4
pfcp.message_type == 1 || pfcp.message_type == 2
```

**Van de thuong gap:**

| Van de | Kiem tra |
|--------|----------|
| Khong tim thay SMF | DNN/NSSAI duoc ho tro? NRF co the lien lac? |
| N4 that bai | UPF co the lien lac? Cong PFCP mo? |
| Khong gan duoc IP | Pool IP da cau hinh? Pool can? |
| Session bi tu choi | Thue bao cho phep DNN? Policy tu choi? |

</section>

<section class="lesson-pitfalls">

## Cam bay

1. **DNN khong trong thue bao.** UE yeu cau DNN="enterprise" nhung thue bao chi cho phep "internet". Session bi tu choi.

2. **NSSAI khong khop.** UE yeu cau slice khong trong Allowed NSSAI. Phai dung slice duoc phep hoac dang ky lai.

3. **Can pool IP.** UPF het dia chi IP. Session that bai cho den khi IP duoc giai phong.

4. **Timeout N4.** SMF khong the lien lac UPF tren giao dien N4. Kiem tra ket noi mang va cong PFCP 8805.

5. **Policy tu choi.** PCF tu choi session dua tren policy (thoi diem trong ngay, gioi han du lieu, v.v.).

6. **SSC mode khong tuong thich.** UE yeu cau SSC mode 3, nhung mang chi ho tro mode 1 cho DNN nay.

7. **Thieu quy tac QoS.** Session thiet lap nhung khong co quy tac QoS → luu luong bi drop hoac chi best-effort.

8. **Session GBR khong co tai nguyen.** Yeu cau bitrate dam bao khi mang tac nghen → tu choi.
