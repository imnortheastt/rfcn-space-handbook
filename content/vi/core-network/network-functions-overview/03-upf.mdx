## Tai sao dieu nay quan trong

Moi byte du lieu nguoi dung — luong video, yeu cau web, goi tin game — chay qua **UPF**.

UPF la noi goi tin thuc su di chuyen. Trong khi AMF va SMF xu ly signaling (control plane), UPF xu ly **data plane**. No la ngua keo lam cho hieu nang 5G tro thanh hien thuc.

</section>

<section class="lesson-intuition">

## Truc giac

### UPF lam gi

```
UPF trong duong du lieu:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│      UE ──► gNB ──────► UPF ──────► Internet            │
│                          │                               │
│                     Du lieu nguoi dung                   │
│                    (duong ham GTP-U)                     │
│                                                          │
│                          │                               │
│                    SMF dieu khien                        │
│                    (PFCP / N4)                           │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

**Diem quan trong:** UPF la "ngu" — no lam theo chi dan SMF. SMF noi "chuyen tiep goi tin khop X den Y voi QoS Z", UPF thuc hien.

### Trach nhiem chinh

| Trach nhiem | Y nghia |
|-------------|---------|
| **Chuyen tiep goi tin** | Route du lieu nguoi dung giua gNB va DN |
| **Ket thuc GTP-U** | Encap/decap duong ham GTP tu RAN |
| **Thuc thi QoS** | Gioi han toc do, danh dau, lap lich |
| **Thuc thi policy** | Ap dung quy tac PCF (chan, chuyen huong) |
| **Bao cao su dung** | Theo doi du lieu de tinh cuoc |
| **Dem** | Luu goi tin cho UE idle (downlink) |
| **NAT** | Dich dia chi cho IPv4 |
| **Nghe lén hop phap** | Sao goi tin cho co quan phap luat |

### Tu 4G sang 5G user plane

```
4G User Plane:
┌─────────────────────────────────────────────────────────┐
│                                                          │
│  eNodeB ──► SGW ──► PGW ──► Internet                    │
│             │        │                                   │
│          (neo)    (neo + policy + NAT)                  │
│                                                          │
│  Hai node, vi tri co dinh                               │
│                                                          │
└─────────────────────────────────────────────────────────┘

5G User Plane:
┌─────────────────────────────────────────────────────────┐
│                                                          │
│  gNB ──► UPF ──► Internet                               │
│           │                                              │
│    (mot chuc nang, vi tri linh hoat)                    │
│                                                          │
│  Mot node, co the o bat cu dau (trung tam, edge, v.v.)  │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

**Khac biet chinh:** UPF hop nhat chuc nang SGW va PGW, co the dat o bat cu dau.

## Cach hoat dong

### Giao dien UPF

```
Giao dien UPF:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│                    ┌─────────┐                          │
│                    │   SMF   │                          │
│                    └────┬────┘                          │
│                         │ N4 (PFCP)                     │
│                         │ Dieu khien                    │
│                         │                                │
│  ┌─────────┐      ┌─────┴─────┐      ┌─────────┐       │
│  │   gNB   │◄────►│    UPF    │◄────►│   DN    │       │
│  └─────────┘  N3  │           │  N6  └─────────┘       │
│              GTP-U │           │  IP                    │
│                    └─────┬─────┘                        │
│                          │ N9                           │
│                          │ GTP-U                        │
│                    ┌─────┴─────┐                        │
│                    │  I-UPF    │ (trung gian)           │
│                    └───────────┘                        │
│                                                          │
└─────────────────────────────────────────────────────────┘

N3: gNB ↔ UPF (du lieu nguoi dung, GTP-U)
N4: SMF ↔ UPF (dieu khien, PFCP)
N6: UPF ↔ Data Network (IP)
N9: UPF ↔ UPF (chuoi, GTP-U)
```

### Pipeline xu ly goi tin

```
Xu ly goi tin UPF:

Uplink (UE → Internet):

gNB                         UPF                         DN
 │                           │                           │
 │  GTP-U encapsulated      │                           │
 │  (ngoai: IP/UDP/GTP)      │                           │
 │  (trong: goi nguoi dung)  │                           │
 │──────────────────────────►│                           │
 │                           │                           │
 │                      ┌────┴────┐                      │
 │                      │ 1. Khop PDR (F-TEID, UE IP)   │
 │                      │ 2. Ap dung QER (gioi han)     │
 │                      │ 3. Thuc hien FAR (chuyen tiep)│
 │                      │ 4. Cap nhat URR (dem byte)    │
 │                      │ 5. NAT neu can                │
 │                      │ 6. Decap GTP, chuyen tiep IP  │
 │                      └────┬────┘                      │
 │                           │                           │
 │                           │  Goi IP                   │
 │                           │──────────────────────────►│
 │                           │                           │


Downlink (Internet → UE):

DN                          UPF                         gNB
 │                           │                           │
 │  Goi IP                  │                           │
 │──────────────────────────►│                           │
 │                           │                           │
 │                      ┌────┴────┐                      │
 │                      │ 1. Khop PDR (UE IP, DN)       │
 │                      │ 2. Ap dung QER (gioi han)     │
 │                      │ 3. Thuc hien FAR (chuyen tiep)│
 │                      │ 4. Cap nhat URR (dem byte)    │
 │                      │ 5. Encap trong GTP-U          │
 │                      │ 6. Chuyen tiep den gNB        │
 │                      └────┬────┘                      │
 │                           │                           │
 │                           │  GTP-U encapsulated      │

 │                           │                           │
```

## Ap dung

**Tham so cau hinh UPF:**

| Tham so | Muc dich | Vi du |
|---------|----------|-------|
| Giao dien N3 | Ket noi gNB | Dia chi IP, pool TEID |
| Giao dien N6 | Ket noi DN | Dia chi IP, tag VLAN |
| Peer N4 | Ket noi SMF | Dia chi IP SMF |
| Pool IP | Dai dia chi UE | 10.0.0.0/8 |
| Kich thuoc buffer | Dem idle | 1000 goi/UE |
| Dai TEID GTP | ID duong ham | 0x10000000-0x1FFFFFFF |

**Chi so dung luong:**

| Chi so | Mo ta |
|--------|-------|
| Thong luong | Gbps (uplink + downlink) |
| Session | PDU session dong thoi |
| PPS | Goi tin moi giay |
| Do tre | Tre xu ly (P50, P99) |
| Su dung buffer | So goi tin dem |

**Xu ly su co:**

| Van de | Kiem tra |
|--------|----------|
| Khong co luong du lieu | PDR/FAR cai dat, duong ham GTP len |
| Do tre cao | Su dung CPU, do sau hang doi |
| Mat goi tin | Gioi han toc do QER, tran buffer |
| QoS sai | Cau hinh QER, danh dau QFI |
| Khong khop tinh cuoc | Nguong URR, bao cao |

<section class="lesson-pitfalls">

## Cam bay

1. **UPF thong minh (sai).** UPF chi thuc hien quy tac. Neu luu luong khong chay, kiem tra lap trinh PFCP cua SMF.

2. **Bo qua quan ly TEID.** ID duong ham phai duy nhat. Trung TEID lam hong session.

3. **UPF trung tam cho use case edge.** App do tre sieu thap can UPF edge. Vi tri trung tam them mili-giay.

4. **Kich co buffer.** Qua nho: mat goi. Qua lon: can kiet memory. Tinh chinh theo mau luu luong.

5. **Phuc tap NAT.** IPv4 NAT can lap ke hoach can than. CGN (Carrier-Grade NAT) them overhead trang thai.

6. **Bo qua PFCP heartbeat.** Mat N4 association nghia la UPF ngung nhan cap nhat. Giam sat trang thai association.

7. **Test hieu nang muon.** Hieu nang data plane kho du doan. Test thong luong va do tre som.

8. **Mot UPF cho tat ca luu luong.** Cac slice khac nhau co the can UPF khac nhau. Lap ke hoach tach luu luong.
