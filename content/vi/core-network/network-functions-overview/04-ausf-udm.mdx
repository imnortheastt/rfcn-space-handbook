<section class="lesson-hook">

## Tai sao dieu nay quan trong

Khi ban lap SIM va ket noi voi mang 5G, hai dieu xay ra: mang xac minh ban la thue bao thuc (**xac thuc**), va lay du lieu thue bao cua ban (**ban la ai va duoc phep lam gi**).

**AUSF** xu ly xac thuc. **UDM** quan ly du lieu thue bao. Cung nhau, chung la nen tang tin tuong cua 5G.

</section>

## Truc giac

### AUSF va UDM trong kien truc

```
Luong xac thuc va du lieu:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│                    ┌─────────┐                          │
│                    │   UDR   │ (database)               │
│                    └────▲────┘                          │
│                         │ Nudr                          │
│                         │                                │
│  ┌─────────┐      ┌─────┴─────┐      ┌─────────┐       │
│  │   AMF   │◄────►│    UDM    │◄────►│  AUSF   │       │
│  └─────────┘ Nudm │           │ Nausf└────▲────┘       │
│       │           │ Quan ly   │           │            │
│       │           │ du lieu TB │           │            │
│       │           └───────────┘           │            │
│       │                                    │            │
│       └────────────────────────────────────┘            │
│              Yeu cau xac thuc                           │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

**Luong:** AMF yeu cau AUSF xac thuc → AUSF hoi UDM lay thong tin xac thuc → UDM lay tu UDR → xac thuc hoan tat.

### Moi chuc nang lam gi

| Chuc nang | Trach nhiem |
|-----------|-------------|
| **AUSF** | Thuc hien 5G-AKA/EAP-AKA', xac thuc danh tinh UE |
| **UDM** | Du lieu thue bao, tao thong tin xac thuc, quan ly dang ky |
| **UDR** | Luu tru database (kho du lieu khong trang thai) |

<section class="lesson-apply">

## Ap dung

**Cau hinh AUSF:**

| Tham so | Muc dich |
|---------|----------|
| PLMN ID | Mang phuc vu |
| UDM peer | Endpoint UDM |
| Dai SUPI | Thue bao phuc vu |
| Dan xuat khoa | Chon thuat toan |

**Cau hinh UDM:**

| Tham so | Muc dich |
|---------|----------|
| PLMN ID | Mang goc |
| Endpoint UDR | Kho du lieu |
| Dai SUPI | Thue bao quan ly |
| NSSAI mac dinh | Cau hinh slice mac dinh |

**Cung cap thue bao:**

```yaml
# Vi du du lieu thue bao UDM
supi: "imsi-452021123456789"
gpsi:
  - "msisdn-+84901234567"
authentication:
  authMethod: "5G_AKA"
  encPermanentKey: "base64-encoded-K"
  encOpcKey: "base64-encoded-OPc"
accessAndMobilitySubscription:
  nssai:
    defaultSingleNssais:
      - sst: 1
      - sst: 2
        sd: "000001"
  ratRestrictions: []
  subscribedUeAmbr:
    uplink: "100 Mbps"
    downlink: "500 Mbps"
sessionManagement:
  - dnn: "internet"
    sscModes:
      defaultSscMode: "SSC_MODE_1"
    pduSessionTypes:
      defaultSessionType: "IPV4"
```

**Xu ly su co:**

| Van de | Kiem tra |
|--------|----------|
| Xac thuc that bai | K/OPc khop giua UE va UDR |
| Dong bo SQN | Thu tuc resync, SQN trong UDR |
| Khong tim thay thue bao | SUPI da cung cap, UDR truy cap duoc |
| Slice sai | NSSAI mac dinh cau hinh |

<section class="lesson-pitfalls">

## Cam bay

1. **AUSF luu du lieu thue bao (sai).** AUSF chi thuc hien xac thuc. UDM/UDR luu du lieu.

2. **Van de dong bo SQN.** Neu SQN UE va mang lech, xac thuc that bai. Trien khai thu tuc resync.

3. **K/OPc khong khop.** Khoa vinh vien phai khop chinh xac giua USIM va UDR. Bat ky khong khop = that bai xac thuc.

4. **Bo qua roaming.** UDM can cau hinh dieu huong roaming (SoR) cho mang di dong.

5. **UDR la diem loi duy nhat.** UDR giu tat ca du lieu thue bao. Lap ke hoach cho du phong va backup.

6. **Cung ma NSSAI.** NSSAI mac dinh/cho phep nen lay tu thue bao, khong cung ma trong AMF/SMF.

7. **Thieu dang ky UECM.** Neu UDM khong biet UE dang ky o dau, dich vu MT that bai.

8. **Timeout xac thuc.** Do tre AUSF/UDM anh huong thoi gian dang ky. Giam sat thoi gian phan hoi SBI.
