## Tai sao dieu nay quan trong

Khi dien thoai cua ban bat nguon, dang ky voi mang, di chuyen giua cac cell, hoac chuyen tu idle sang connected — **AMF** xu ly tat ca.

AMF la **neo control plane** cho 5G. Moi UE noi chuyen voi AMF. Moi gNB ket noi voi AMF. Hieu AMF la hieu nhip dap cua signaling 5G.

</section>

## Truc giac

### AMF lam gi

```
AMF o trung tam:

                         ┌─────────────────┐
                         │      UDM        │
                         │ (du lieu thue bao)│
                         └────────▲────────┘
                                  │ Nudm
                                  │
┌─────────┐    N2/NGAP    ┌───────┴───────┐    Nsmf     ┌─────────┐
│  gNB    │◄─────────────►│      AMF      │◄───────────►│   SMF   │
└─────────┘               │               │             └─────────┘
                          │  (trung tam   │
┌─────────┐    N1/NAS     │  signaling)   │    Nausf    ┌─────────┐
│   UE    │◄─────────────►│               │◄───────────►│  AUSF   │
└─────────┘               └───────┬───────┘             └─────────┘
                                  │ Npcf
                                  │
                         ┌────────▼────────┐
                         │      PCF        │
                         │ (policy truy cap)│
                         └─────────────────┘
```

**AMF la diem lien lac duy nhat cua UE** cho tat ca signaling control plane.

### Trach nhiem chinh

| Trach nhiem | Y nghia |
|-------------|---------|
| **Registration** | Gan ket UE, phoi hop authentication |
| **Quan ly ket noi** | Trang thai Idle/Connected, paging |
| **Quan ly di dong** | Handover, quan ly danh sach TAI |
| **Dieu khien truy cap** | Chon network slice, han che truy cap |
| **Ket thuc NAS** | Signaling NAS end-to-end voi UE |
| **Ket thuc N2** | Signaling control plane voi gNB |

## Ap dung

**Tham so cau hinh AMF:**

| Tham so | Muc dich | Gia tri dien hinh |
|---------|----------|-------------------|
| PLMN ID | PLMN phuc vu | MCC-MNC (v.d., 452-02) |
| AMF Set ID | Nhom pool AMF | 0-1023 |
| AMF Pointer | AMF trong set | 0-63 |
| Danh sach TAI | Tracking area phuc vu | Danh sach TAI |
| NSSAI ho tro | Ho tro slice | To hop SST/SD |
| T3512 | Timer dang ky dinh ky | 54 phut mac dinh |
| T3502 | Timer thu lai dang ky | 12 phut mac dinh |

**Lap ke hoach dung luong AMF:**

| Chi so | Mo ta |
|--------|-------|
| Max UE context | So UE da dang ky |
| Ty le dang ky | Dang ky moi giay |
| Dung luong paging | Paging moi giay |
| Ket noi N2 | So gNB |
| Thong luong SBI | HTTP/2 request moi giay |

**Checklist xu ly su co:**

| Van de | Kiem tra |
|--------|----------|
| Dang ky that bai | AUSF/UDM truy cap duoc, dang ky NRF |
| Paging that bai | Danh sach TAI dung, trang thai N2 gNB |
| Handover that bai | AMF dich truy cap duoc, routing GUAMI |
| Hieu nang cham | CPU/memory, do tre SBI, nghen N2 |
| UE khong lien lac duoc | Trang thai CM, TAI cuoi biet, timer paging |

</section>

<section class="lesson-pitfalls">

## Cam bay

1. **Nham AMF voi MME.** AMF khong xu ly session — do la SMF. Dung mong doi quan ly bearer trong AMF.

2. **Bo qua lap ke hoach TAI.** Thiet ke TAI kem gay ra cap nhat dang ky qua nhieu. Lap ke hoach danh sach TAI theo mau di dong.

3. **Danh gia thap tai paging.** Thiet bi IoT voi du lieu thuong xuyen co the lam qua tai paging. Xem xet chien luoc luon ket noi.

4. **Trien khai AMF don.** Pool AMF voi can bang tai la thiet yeu cho kha nang phuc hoi va scale.

5. **Quen di dong lien AMF.** UE di chuyen giua cac AMF yeu cau chuyen context. Test handover lien AMF.

6. **Cau hinh bao mat NAS sai.** Ho tro thuat toan sai lam hong authentication. Xac nhan kha nang bao mat NAS.

7. **Van de N2 SCTP.** Multi-homing va heartbeat SCTP yeu cau cau hinh mang. Firewall thuong chan SCTP.

8. **Khong khop timer.** T3512 (dang ky dinh ky) nen phu hop voi hanh vi UE mong doi va tai mang.
