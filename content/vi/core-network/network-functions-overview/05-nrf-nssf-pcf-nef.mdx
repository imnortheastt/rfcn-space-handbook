## Tai sao dieu nay quan trong

AMF, SMF, va UPF la "nhan vat chinh" cua 5G. Nhung chung can **ha tang ho tro** de hoat dong.

**NRF** cho phep kham pha dich vu. **NSSF** chon network slice. **PCF** thuc thi policy. **NEF** expose API cho ung dung ben ngoai. Khong co chung, kien truc SBA se sup do.

</section>

## NRF — Network Repository Function

### NRF lam gi

NRF la **dang ky dich vu** cho 5G SBA. Moi NF dang ky dich vu voi NRF; nguoi tieu thu kham pha dich vu bang cach truy van NRF.

```
Dang ky dich vu NRF:

┌─────────────────────────────────────────────────────────┐
│                         NRF                              │
│                                                          │
│  NF da dang ky:                                         │
│  ┌─────────────────────────────────────────────────┐    │
│  │ NF: AMF-001                                      │    │
│  │   Type: AMF                                      │    │
│  │   Services: [Namf_Communication, Namf_Location]  │    │
│  │   Address: https://amf1.5gc.operator.vn         │    │
│  │   PLMN: 452-02                                   │    │
│  │   Slice: [SST=1, SST=2]                         │    │
│  │   TAI: [TAI1, TAI2, TAI3]                       │    │
│  ├─────────────────────────────────────────────────┤    │
│  │ NF: SMF-001                                      │    │
│  │   Type: SMF                                      │    │
│  │   Services: [Nsmf_PDUSession]                   │    │
│  │   DNN: [internet, ims]                          │    │
│  │   ...                                            │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## NSSF — Network Slice Selection Function

### NSSF lam gi

NSSF giup chon **network slice** phu hop cho UE. Cac slice khac nhau phuc vu muc dich khac nhau (eMBB, URLLC, mMTC).

```
Khai niem Network Slicing:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│        Mang 5G vat ly                                   │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  ┌─────────────────────────────────────────┐   │    │
│  │  │ Slice 1 (SST=1): eMBB                   │   │    │
│  │  │ AMF-A, SMF-A, UPF-A                     │   │    │
│  │  └─────────────────────────────────────────┘   │    │
│  │                                                  │    │
│  │  ┌─────────────────────────────────────────┐   │    │
│  │  │ Slice 2 (SST=2): URLLC                  │   │    │
│  │  │ AMF-B, SMF-B, UPF-B (edge)              │   │    │
│  │  └─────────────────────────────────────────┘   │    │
│  │                                                  │    │
│  │  ┌─────────────────────────────────────────┐   │    │
│  │  │ Slice 3 (SST=3): mMTC                   │   │    │
│  │  │ AMF-C, SMF-C, UPF-C                     │   │    │
│  │  └─────────────────────────────────────────┘   │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  NSSF quyet dinh: "UE X duoc Slice 1 + Slice 2"        │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### Dich vu NSSF

```
Dich vu NSSF (Nnssf):

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  Nnssf_NSSelection:                                     │
│  └── NSSelectionGet        (chon slice cho UE)         │
│                                                          │
│  Nnssf_NSSAIAvailability:                               │
│  ├── NSSAIAvailabilityPut  (anh xa TA → slice)         │
│  ├── NSSAIAvailabilityDelete                            │
│  └── NSSAIAvailabilitySubscribe                         │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

│ Registration   │                │
│ NSSAI yeu cau  │                │

│───────────────►│                │
│                │                │
│                │ NSSelectionGet │
│                │ - SUPI         │
│                │ - NSSAI yeu cau│
│                │ - TAI          │
│                │───────────────►│
│                │                │
│                │         ┌──────┴──────┐
│                │         │ Kiem tra:   │
│                │         │ - Thue bao  │
│                │         │ - TA ho tro │
│                │         │ - Policy    │
│                │         └──────┬──────┘

## PCF — Policy Control Function

### PCF lam gi

PCF cung cap **quy tac policy** cho AMF (policy truy cap) va SMF (policy session). No quyet dinh QoS, quy tac tinh cuoc, va xu ly luu luong.

```
PCF trong khung Policy:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│                    ┌─────────┐                          │
│                    │   UDR   │ (du lieu policy)         │
│                    └────▲────┘                          │
│                         │                                │
│                    ┌────┴────┐                          │
│                    │   PCF   │                          │
│                    └────┬────┘                          │
│                         │                                │
│           ┌─────────────┼─────────────┐                 │
│           │             │             │                  │
│           ▼             ▼             ▼                  │
│      ┌─────────┐   ┌─────────┐   ┌─────────┐           │
│      │   AMF   │   │   SMF   │   │   AF    │           │
│      │         │   │         │   │(qua NEF)│           │
│      │ Policy  │   │ Policy  │   │ Yeu cau │           │
│      │ truy cap│   │ session │   │ App     │           │
│      └─────────┘   └─────────┘   └─────────┘           │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### Dich vu PCF

```
Dich vu PCF (Npcf):

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  Npcf_AMPolicyControl:                                  │
│  ├── Create             (policy truy cap cho UE)       │
│  ├── Update                                             │

│                                                          │
│  Npcf_SMPolicyControl:                                  │

│  └── UpdateNotify       (trigger thay doi policy)      │
│                                                          │
│  Npcf_PolicyAuthorization:                              │
│  └── Create             (AF yeu cau QoS)               │
│                                                          │
│  Npcf_BDTPolicyControl:                                 │

## NEF — Network Exposure Function

### NEF lam gi

NEF la **API gateway** giua ung dung ben ngoai va 5G core. No expose kha nang mang mot cach an toan va bao mat.

```
NEF nhu API Gateway:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  The gioi ben ngoai        │      5G Core               │
│                              │                          │
│  ┌─────────┐                │                          │
│  │ IoT App │────┐           │                          │
│  └─────────┘    │           │                          │
│                  │           │                          │
│  ┌─────────┐    │    ┌──────┴─────┐    ┌─────────┐    │
│  │ Video   │────┼───►│    NEF     │───►│   PCF   │    │
│  │ Service │    │    │            │    └─────────┘    │
│  └─────────┘    │    │ - Xac thuc │                    │
│                  │    │ - Dich     │    ┌─────────┐    │
│  ┌─────────┐    │    │ - Uy quyen │───►│   UDM   │    │
│  │ Traffic │────┘    │            │    └─────────┘    │
│  │ Mgmt    │         └──────┬─────┘                    │
│  └─────────┘                │          ┌─────────┐    │
│                              │         │   AMF   │    │
│                              │         └─────────┘    │
│                              │                          │
└─────────────────────────────────────────────────────────┘
```

### Dich vu NEF

```
Dich vu NEF (Nnef):

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  Expose ben ngoai (cho AF):                             │
│  ├── MonitoringEvent       (vi tri UE, kha nang truy cap)│

│  ├── ParameterProvision    (dat tham so UE)            │
│  ├── TrafficInfluence      (route luu luong den edge)  │

│  └── AFSessionWithQoS      (QoS cho app cu the)        │

## Ap dung

**Cau hinh NRF:**

| Tham so | Muc dich |
|---------|----------|
| PLMN ID | Danh tinh mang |
| NRF peer | Cho giao tiep NRF-to-NRF |
| Cau hinh OAuth2 | Uy quyen dua tren token |

**Cau hinh NSSF:**

| Tham so | Muc dich |
|---------|----------|
| S-NSSAI ho tro | Slice kha dung |
| Anh xa TA-NSSAI | Slice nao trong vung nao |
| Thong tin AMF set | AMF dich moi slice |

**Cau hinh PCF:**

| Tham so | Muc dich |
|---------|----------|
| Endpoint UDR | Kho du lieu policy |
| Policy mac dinh | Quy tac du phong |
| Quy tac tinh cuoc | Tinh cuoc online/offline |

**Cau hinh NEF:**

| Tham so | Muc dich |
|---------|----------|
| Danh sach AF cho phep | App ben ngoai duoc phep |
| Gioi han API | Rate limiting |
| Pham vi expose | Kha nang nao duoc expose |

<section class="lesson-pitfalls">

## Cam bay

1. **NRF la diem loi duy nhat.** NRF hong dung discovery. Trien khai instance NRF du phong.

2. **Phuc tap NSSF.** Logic chon slice co the phuc tap. Bat dau don gian, them quy tac dan.

3. **Qua tai PCF.** Moi session truy van PCF. Giam sat va scale phu hop.

4. **Lo hong bao mat NEF.** Expose API mang cho app ben ngoai yeu cau uy quyen nghiem ngat. Kiem tra truy cap AF.

5. **Cache ket qua NRF.** Du lieu discovery cu gay routing den NF chet. Dat TTL phu hop.

6. **Bo qua tinh nhat quan NSSAI.** NSSAI yeu cau, cho phep, va cau hinh phai phu hop. Khong khop gay that bai chon slice.

7. **Split brain PCF-UDR.** Du lieu policy trong UDR va trang thai runtime PCF phai dong bo.

8. **Gioi han toc do NEF.** AF ben ngoai co the lam ngap NEF voi request. Trien khai throttling tu ngay dau.
