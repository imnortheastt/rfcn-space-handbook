**Doc bieu do thu tuc:**

```
Cach Doc MSCs (Bieu do Chuoi Ban tin):

1. Trai sang phai = tien trinh thoi gian
2. Duong doc = cac thuc the (UE, NFs)
3. Mui ten = ban tin/tin hieu
4. So = thu tu buoc
5. Hop = buoc tuy chon/co dieu kien
6. Ghi chu = dieu kien bo sung

Vi du ky hieu:
  ────────→  = yeu cau dong bo
  - - - - →  = khong dong bo/tuy chon
  [dieu kien] = buoc tuy chon
```

**Tham chieu cheo:**

```
TS 23.502 noi "UE gui Registration Request"
  → TS 24.501 dinh nghia dinh dang Registration Request
  → TS 33.501 dinh nghia bao mat cho dang ky
  → TS 38.331 dinh nghia thu tuc RRC cho truyen NAS
```

<section class="lesson-pitfalls">

## Cam bay

1. **Chi doc van ban.** Bieu do MSC mang thong tin quan trong. Nghien cuu chung.

2. **Bo qua dieu kien truoc.** Moi thu tuc co gia dinh. Vi pham chung = hanh vi khong xac dinh.

3. **Bo qua buoc tuy chon.** Buoc trong ngoac [] co the bat buoc trong kich ban cua ban.

4. **Nham lan thu tuc.** Dang ky ≠ Ket noi ≠ Phien PDU. Biet su khac biet.

5. **Bo lo duong loi.** Luong thanh cong hien thi truoc. Xu ly loi thuong o clause con rieng.

6. **Gia dinh mot phien ban.** Thu tuc phat trien qua cac release. Kiem tra release muc tieu.

7. **Khong theo hoat dong dich vu.** Goi Nxxx_Service tham chieu TS 29.xxx de biet chi tiet.

8. **Quen trang thai.** Thu tuc phu thuoc vao trang thai UE/NF hien tai. Theo doi may trang thai.
