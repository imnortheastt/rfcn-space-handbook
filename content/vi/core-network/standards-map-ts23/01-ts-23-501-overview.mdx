## Tai sao dieu nay quan trong

TS 23.501 la "kinh thanh" cua kien truc 5G. Moi NF, moi giao dien, moi khai niem deu bat nguon tu tai lieu nay. Hieu cau truc cua no nghia la hieu 5G.

**Tra cuu pho bien:**

```bash
# AMF lam gi?
# → Clause 6.2.1 (mo ta AMF)

# Giao dien N4 la gi?
# → Clause 4.2.4 (diem tham chieu) + TS 29.244

# Cat mang hoat dong the nao?
# → Clause 5.8 (cat mang)

# Co nhung tham so QoS nao?
# → Clause 5.4 (mo hinh QoS) + Clause 5.7 (bang 5QI)
```

**Tham chieu cheo:**

```
TS 23.501 noi "X xay ra"
  → TS 23.502 cho thay THE NAO X xay ra (thu tuc)
  → TS 29.xxx cho thay dinh dang ban tin cho X
  → TS 33.501 cho thay bao mat cho X
```

<section class="lesson-pitfalls">

## Cam bay

1. **Nham voi TS 23.502.** 23.501 = kien truc (tinh). 23.502 = thu tuc (dong). Can ca hai.

2. **Thieu goc nhin dua tren dich vu.** Spec co HAI bieu dien. SBI la trien khai thuc te.

3. **Bo qua phu luc.** Phu luc co vi du va lam ro. Chung la thong tin nhung co gia tri.

4. **Khong khop phien ban.** Rel-15, 16, 17, 18 co tinh nang khac nhau. Kiem tra release muc tieu.

5. **Gia dinh day du.** 23.501 la muc cao. Chi tiet giao thuc nam trong spec khac.

6. **Bo qua Clause 3.** Viet tat day dac. Giu dinh nghia san sang.

7. **Khong theo tham chieu.** Trích dan "[N]" chi den phan doc bat buoc.

8. **Coi tat ca NFs nhu nhau.** AMF, SMF, UPF la bat buoc. Cac NF khac co the tuy chon.
