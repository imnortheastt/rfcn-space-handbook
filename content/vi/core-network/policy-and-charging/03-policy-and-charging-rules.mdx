## Tai sao dieu nay quan trong

Quy tac PCC la cac nguyen tu cua thuc thi policy. Moi quy tac noi: "voi luu luong khop bo loc nay, ap dung QoS nay, tinh cuoc theo cach nay, va cong mo hay dong."

Hieu quy tac PCC giup ban debug tai sao goi video bi throttle hoac tai sao luu luong nhat dinh khong bi tinh cuoc.

</section>

### Quy tac PCC chua gi

```
Cau truc Quy tac PCC:

┌─────────────────────────────────────────────────────────┐

│                                                          │
│  ┌─────────────────────────────────────────────────┐    │
│  │ Ten Quy tac: "video_streaming_rule"             │    │
│  │                                                  │    │
│  │ Khop Luu luong (Bo loc SDF):                    │    │
│  │ ├── Protocol: UDP                               │    │
│  │ ├── IP xa: 203.0.113.0/24                       │    │
│  │ ├── Port xa: 1935, 8080                         │    │
│  │ └── Huong: hai chieu                            │    │
│  │                                                  │    │

│  │ ├── 5QI: 6 (video streaming)                   │    │

│  │ Tinh cuoc:                                      │    │
```

### Giai thich bo loc SDF

```
Bo loc Service Data Flow:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  Bo loc SDF = Bo Phan loai Goi                          │
│                                                          │
│  Khop 5-tuple:                                          │
│  ├── IP nguon / prefix                                 │
│  ├── IP dich / prefix                                  │
│  ├── Port nguon / pham vi                              │
│  ├── Port dich / pham vi                               │
│  └── Protocol (TCP, UDP, any)                          │
│                                                          │
│  Huong:                                                 │
│  ├── Uplink: UE → mang                                 │
│  ├── Downlink: mang → UE                               │
│  └── Hai chieu: ca hai                                 │
│                                                          │
│  Vi du:                                                 │
│  ┌─────────────────────────────────────────────────┐    │
│  │ Bao hieu VoLTE:                                 │    │
│  │   permit out ip from any to 198.51.100.10 5060 │    │
│  │                                                  │    │
│  │ Xem video:                                      │    │
│  │   permit in udp from 203.0.113.0/24 1935 to any│    │
│  │                                                  │    │
│  │ Tat ca luu luong (mac dinh):                    │    │
│  │   permit out ip from any to any                │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## Cach hoat dong

### Luong quy tac: PCF → SMF → UPF

```
Cai dat Quy tac PCC:

PCF                    SMF                    UPF
│                       │                       │
│ Quyet dinh SM Policy │                       │
│ (quy tac PCC)        │                       │
│──────────────────────►│                       │
│                       │                       │
│                       │ Phan tich quy tac PCC│
│                       │ Tao quy tac QoS      │
│                       │ Tao quy tac tinh cuoc│
│                       │                       │
│                       │ PFCP Session Mod     │
│                       │ - PDR (phat hien)    │
│                       │ - QER (thuc thi QoS) │
│                       │ - URR (bao cao su dung)
│                       │──────────────────────►│
│                       │                       │
│                       │                       │ UPF cai dat:
│                       │                       │ - Bo loc goi
│                       │                       │ - Gioi han toc do
│                       │                       │ - Bo dem su dung
│                       │                       │
│                       │ PFCP Mod Response    │
│                       │◄──────────────────────│
│                       │                       │
│                       │ NAS: quy tac QoS den │
│                       │ UE (cho danh dau UL) │
│                       │───────────────────────────► UE
│                       │                       │
```

### Anh xa quy tac PCC sang PFCP

```
Dich Quy tac PCC → Quy tac PFCP:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  Thanh phan Quy tac PCC    →    Quy tac PFCP            │
│  ─────────────────────────────────────────────────────  │
│                                                          │
│  Bo loc SDF                →    PDR (Packet Detection)  │
│  ├── Dia chi IP/port            ├── PDI (Packet        │
│  ├── Protocol                   │   Detection Info)    │
│  └── Huong                      └── SDF Filter IE      │
│                                                          │
│  Tham so QoS               →    QER (QoS Enforcement)   │
│  ├── 5QI                        ├── Trang thai Cong    │
│  ├── MBR                        ├── MBR                │
│  ├── GBR                        ├── GBR                │
│  └── ARP                        └── QFI                │
│                                                          │
│  Tham so Tinh cuoc         →    URR (Usage Reporting)   │
│  ├── Nhom Rating                ├── Phuong phap Do     │
│  ├── Muc Bao cao                ├── Trigger Bao cao    │
│  └── Online/Offline             └── Nguong Volume      │
│                                                          │
│  Cong / Redirect           →    FAR (Forwarding Action) │
│  ├── Mo/Dong                    ├── Apply Action       │
│  └── URL Redirect               └── Redirect Info      │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

<section class="lesson-apply">

## Ap dung

**Mau quy tac PCC pho bien:**

| Dich vu | Bo loc SDF | 5QI | Nhom Rating |
|---------|------------|-----|-------------|
| Du lieu mac dinh | any | 9 | 100 |
| Bao hieu VoLTE | UDP 5060-5061 | 5 | - |
| Media VoLTE | UDP 16384-32767 | 1 | 300 |
| Xem video | TCP 443, UDP 1935 | 6 | 200 |
| Gaming | UDP 3478-3479 | 3 | 100 |
| VPN doanh nghiep | IP den 10.0.0.0/8 | 8 | 400 |

**Debug quy tac PCC:**

```bash
# Wireshark: SM policy voi quy tac PCC
http2.header.path contains "sm-policies" && http2.data

# Cai dat PFCP PDR/QER
pfcp.create_pdr || pfcp.create_qer

# Kiem tra uu tien quy tac
grep "precedence" /var/log/smf/*.log

# Xac minh quy tac UPF
upf_cli show pdr
upf_cli show qer
```

**Vong doi quy tac:**

| Su kien | Hanh dong Quy tac PCC |
|---------|------------------------|
| Thiet lap session | Cai dat quy tac mac dinh |
| AF yeu cau tai nguyen | Them quy tac rieng |
| Cap nhat policy | Sua doi quy tac |
| AF giai phong tai nguyen | Xoa quy tac rieng |
| Giai phong session | Xoa tat ca quy tac |

<section class="lesson-pitfalls">

## Cam bay

1. **Bo loc SDF chong cheo.** Hai quy tac khop cung luu luong — uu tien quyet dinh. Dam bao uu tien dung.

2. **Thieu quy tac mac dinh.** Neu khong quy tac nao khop, luu luong co the bi drop. Luon co quy tac catch-all.

3. **Trung nhom rating.** Cung nhom rating cho cac dich vu khac nhau = billing sai. Giu cac nhom tach biet.

4. **Khoang trong uu tien.** De khoang giua gia tri uu tien cho quy tac tuong lai (dung 100, 200, 300 khong phai 1, 2, 3).

5. **Hai chieu vs mot chieu.** UL va DL co the can bo loc SDF rieng. Kiem tra huong can than.

6. **Khong khop ARP.** Allocation and Retention Priority anh huong preemption. Uu tien cao hon co the preempt thap hon.

7. **Thoi gian quy tac dong.** Cai dat quy tac AF mat thoi gian. Media co the bat dau truoc khi quy tac hoat dong.

8. **Bo qua trang thai cong.** Neu cong dong, luu luong bi drop ngay ca khi QoS da cau hinh. Kiem tra trang thai cong.
