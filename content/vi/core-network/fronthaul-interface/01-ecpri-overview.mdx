## Tai sao dieu nay quan trong

CPRI cu la diem-den-diem, toc do bit khong doi, va doc quyen theo vendor. No khong the mo rong cho 5G massive MIMO ma khong can bang thong qua cao.

eCPRI dua tren goi, ho tro nen, va chay tren Ethernet chuan. No la nen tang cua O-RAN Open Fronthaul.

</section>

## Truc giac

### CPRI vs eCPRI

```
Tien hoa tu CPRI sang eCPRI:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  CPRI (Common Public Radio Interface):                  │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  Truyen tai: Toc do bit khong doi (giong TDM)   │    │
│  │  Vat ly:     Cap quang rieng, optic doc quyen   │    │
│  │  Split:      Option 8 (PHY-RF)                  │    │
│  │  Du lieu:    Mau IQ mien thoi gian              │    │
│  │  BW:         Co dinh, ty le voi so antenna      │    │
│  │              24.3 Gbps cho 100 MHz, 8 antenna   │    │
│  │  Vendor:     Doc quyen, don vendor              │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  eCPRI (enhanced CPRI):                                 │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  Truyen tai: Dua tren goi (Ethernet/IP)         │    │
│  │  Vat ly:     Optic Ethernet chuan               │    │
│  │  Split:      Linh hoat (Options 6-8, 7-2x)      │    │
│  │  Du lieu:    IQ mien tan so (nen duoc)          │    │
│  │  BW:         Bien doi, co the nen               │    │
│  │              ~10 Gbps nen cho cung cau hinh     │    │
│  │  Vendor:     Dac ta mo                          │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  Uu diem chinh:                                         │
│  - He sinh thai Ethernet chuan (switch, optic)         │
│  - Co the ghep kenh thong ke                           │
│  - Nen giam bang thong 50-70%                          │
│  - Phan tach chuc nang linh hoat                       │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## Cach hoat dong

### Header eCPRI

```
Header Chung eCPRI (8 bytes):

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  Byte 0: Phien ban Giao thuc + Reserved + C             │
│  ┌─────────────────────────────────────────────────┐    │
│  │ Bits 7-4: Phien ban (0x01 = V1.x, 0x02 = V2.0)  │    │
│  │ Bits 3-1: Reserved (000)                         │    │
│  │ Bit 0:    Chi bao noi tiep                       │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  Byte 1: Loai Tin nhan                                  │
│  ┌─────────────────────────────────────────────────┐    │
│  │ 0x00: Du lieu IQ                                │    │
│  │ 0x01: Chuoi Bit                                 │    │
│  │ 0x02: Du lieu Dieu khien Thoi gian thuc        │    │
│  │ 0x03: Truyen Du lieu Chung                     │    │
│  │ 0x04: Truy cap Bo nho Xa                       │    │
│  │ 0x05: Do Do tre Mot chieu                      │    │
│  │ 0x06: Reset Xa                                  │    │
│  │ 0x07: Chi bao Su kien                          │    │
│  │ 0x08-0x3F: Du tru                              │    │
│  │ 0x40-0xFF: Rieng vendor                        │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  Bytes 2-3: Kich thuoc Payload (big-endian)             │
│  ┌─────────────────────────────────────────────────┐    │
│  │ Kich thuoc payload tinh bang byte (khong header)│    │
│  │ Max: 65535 bytes                                │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  Bytes 4-5: RTC_ID / PC_ID (dinh danh ngu canh)        │
│  ┌─────────────────────────────────────────────────┐    │
│  │ Dung de nhan dien stream/flow                   │    │
│  │ PC_ID cho du lieu IQ: eAxC ID (extended Antenna │    │
│  │                       Carrier ID)               │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  Bytes 6-7: SEQ_ID (dinh danh thu tu)                  │
│  ┌─────────────────────────────────────────────────┐    │
│  │ Bits 15-8: So thu tu (0-255, quay vong)         │    │
│  │ Bits 7-0:  Sub-sequence (cho phan manh)         │    │
│  │ E bit (bit 7 cua sub-seq): Ket thuc tin nhan    │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

│  │                                                  │    │

│  │  └─────────┴─────────┴─────────┴─────────┘      │    │
│  │                                                  │    │

│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │

│                                                          │

## Ap dung

**So sanh eCPRI vs CPRI:**

| Khia canh | CPRI | eCPRI |
|-----------|------|-------|
| Truyen tai | TDM | Goi (Ethernet) |
| Hieu qua BW | Co dinh | Bien doi, nen |
| Do tre | Xac dinh | Gan xac dinh |
| Switch | Doc quyen | Ethernet chuan |
| Da vendor | Khong | Co |
| Tuy chon split | Chi Option 8 | Linh hoat |

**Bat eCPRI:**

```bash
# Bo loc Wireshark cho eCPRI
eth.type == 0xaefe

# Bat tcpdump
tcpdump -i eth0 ether proto 0xaefe -w ecpri.pcap

# Loc theo loai tin nhan (du lieu IQ)
ecpri.msg_type == 0

# Loc theo eAxC ID
ecpri.pc_id == 0x0000

# Kiem tra loi so thu tu
tshark -r ecpri.pcap -Y "ecpri" -T fields -e ecpri.seq_id | uniq -u
```

**Kich thuoc khung:**

| Kich ban | Khung Ethernet | Ghi chu |
|----------|----------------|---------|
| Chuan | 1518 bytes | Du lieu IQ han che |
| Jumbo | 9000 bytes | Uu tien cho FH |
| Max eCPRI | 65535 payload | It dung |

<section class="lesson-pitfalls">

## Cam bay

1. **Sai EtherType.** eCPRI dung 0xAEFE. Switch cau hinh sai co the bo hoac dinh tuyen sai.

2. **Loi so thu tu.** Chi ra mat goi. Giam sat tinh lien tuc SEQ_ID.

3. **Khong khop eAxC.** DU va RU phai thong nhat anh xa eAxC. Anh xa sai gui du lieu den sai antenna.

4. **Overhead phan manh.** Tin nhan phan manh them do tre. Kich thuoc tin nhan de tranh phan manh.

5. **Phan tich noi tiep.** Mot so trien khai khong xu ly dung C bit. Xac minh tuong thich.

6. **Uu tien VLAN.** eCPRI can uu tien cao. Cau hinh VLAN sai gay drop.

7. **Khong khop MTU.** Tat ca switch phai ho tro khung jumbo. Mot lien ket MTU 1500-byte pha vo luong.

8. **Loai rieng vendor.** Loai tin nhan 0x40-0xFF do vendor dinh nghia. Co the gay van de interop.

</section>
