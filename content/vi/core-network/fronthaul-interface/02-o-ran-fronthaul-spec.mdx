## Tai sao dieu nay quan trong

eCPRI dinh nghia transport. O-RAN WG4 dinh nghia gi ben trong — cau truc section, tham so timing, dinh dang nen, va interface quan ly.

Hieu dac ta fronthaul O-RAN la can thiet cho tuong thich da vendor. Day la noi cac chi tiet interop ton tai.

</section>

## Cach hoat dong

### Cau truc section C-Plane

```
Dinh dang Section C-Plane O-RAN:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  Header Lop Ung dung:                                   │
│  ┌─────────────────────────────────────────────────┐    │
│  │ dataDirection (1 bit): 0=UL, 1=DL               │    │
│  │ payloadVersion (3 bits): 001 cho hien tai       │    │
│  │ filterIndex (4 bits): Nhan dien tap bo loc      │    │
│  │ frameId (8 bits): So khung radio mod 256        │    │
│  │ subframeId (4 bits): 0-9                        │    │
│  │ slotId (6 bits): Slot trong subframe            │    │
│  │ startSymbolId (6 bits): Symbol bat dau          │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  Header Section (moi section):                          │
│  ┌─────────────────────────────────────────────────┐    │
│  │ sectionId (12 bits): Nhan dien section          │    │
│  │ rb (1 bit): Chi bao resource block              │    │
│  │ symInc (1 bit): Che do tang symbol              │    │
│  │ startPrbc (10 bits): PRB bat dau                │    │
│  │ numPrbc (8 bits): So PRB (0=tat ca)             │    │
│  │ reMask (12 bits): RE nao trong PRB              │    │
│  │ numSymbol (4 bits): So symbol                   │    │
│  │ ef (1 bit): Co mo rong                          │    │
│  │ beamId (15 bits): Dinh danh beam                │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  Mo rong Section (tuy chon, ef=1):                      │
│  ┌─────────────────────────────────────────────────┐    │
│  │ extType 1: Trong so beamforming                 │    │
│  │ extType 2: Tham so nen BFW                      │    │
│  │ extType 3: Offset thoi gian/tan so              │    │
│  │ extType 4: Tham so nen dieu che                 │    │
│  │ extType 5: Chi so codebook                      │    │
│  │ ...them cac loai mo rong...                     │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### Dinh dang IQ U-Plane

```
Dinh dang Du lieu IQ U-Plane O-RAN:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  Header Lop Ung dung:                                   │
│  ┌─────────────────────────────────────────────────┐    │
│  │ dataDirection, payloadVersion, filterIndex      │    │
│  │ frameId, subframeId, slotId, symbolId           │    │
│  │ (Giong header C-Plane)                          │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  Header Section:                                        │
│  ┌─────────────────────────────────────────────────┐    │
│  │ sectionId: Phai khop section C-Plane            │    │
│  │ rb, symInc, startPrbc, numPrbc, reMask          │    │
│  │                                                  │    │
│  │ udCompHdr (neu dung nen):                       │    │
│  │ - udIqWidth: Do rong bit IQ (v.d., 9 bits)      │    │
│  │ - udCompMeth: Phuong phap nen                   │    │
│  │   0: Khong nen                                  │    │
│  │   1: Block floating point                       │    │
│  │   2: Block scaling                              │    │
│  │   3: µ-law                                      │    │
│  │   4: Nen dieu che                               │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  Du lieu Mau IQ (moi PRB):                              │
│  ┌─────────────────────────────────────────────────┐    │
│  │ [exponent (4 bits neu BFP)]                     │    │
│  │ [I0, Q0], [I1, Q1], ..., [I11, Q11]            │    │
│  │ (12 subcarrier moi PRB)                         │    │
│  │                                                  │    │
│  │ Dinh dang BFP (vi du 9-bit):                    │    │
│  │ ┌─────────────────────────────────────────────┐ │    │
│  │ │ Exp │ I0 (9b) │ Q0 (9b) │ ... │ I11 │ Q11 │ │    │
│  │ │ 4b  │         │         │     │      │     │ │    │
│  │ └─────────────────────────────────────────────┘ │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## Ap dung

**Tai lieu O-RAN WG4 chinh:**

| Tai lieu | Noi dung |
|----------|----------|
| O-RAN.WG4.CUS.0-v09.00 | Dac ta C/U/S-Plane |
| O-RAN.WG4.MP.0-v09.00 | Dac ta M-Plane |
| O-RAN.WG4.IOT | Spec kiem tra interop |
| O-RAN.WG4.CONF | Spec kiem tra phu hop |

**Thao tac NETCONF:**

```bash
# Lay kha nang RU
netconf-cli get --filter "<o-ran-module-cap/>"

# Cau hinh carrier
netconf-cli edit-config --target running <<EOF
<config>
  <user-plane-configuration xmlns="urn:o-ran:uplane-conf:1.0">
    <tx-array-carriers>
      <name>cc0</name>
      <absolute-frequency-center>3600000</absolute-frequency-center>
      <channel-bandwidth>100000</channel-bandwidth>
      <active>true</active>
    </tx-array-carriers>
  </user-plane-configuration>
</config>
EOF

# Kiem tra tham so timing
netconf-cli get --filter "<delay-management/>"

# Xem canh bao
netconf-cli get --filter "<alarms/>"
```

<section class="lesson-pitfalls">

## Cam bay

1. **Khong khop Section ID.** U-Plane sectionId phai khop C-Plane. Khong khop gay mat du lieu.

2. **Khong tuong thich tham so timing.** DU va RU phai thong nhat gia tri T2a/Ta3. Thuong luong qua M-Plane.

3. **Khong khop nen.** udIqWidth va udCompMeth phai khop cau hinh. Cai dat sai lam hong du lieu IQ.

4. **Phien ban mo hinh YANG.** Cac phien ban spec O-RAN khac co YANG khac. Xac minh tuong thich.

5. **Ket noi M-Plane.** Kien truc phan cap vs ket hop phai khop trien khai. Mo hinh sai pha vo quan ly.

6. **Phan tich mo rong.** Mo rong tuy chon (ef=1) phai phan tich dung. Mo rong khong biet nen bo qua.

7. **Khong khop profile PTP.** O-RAN yeu cau G.8275.1. Profile sai (v.d., G.8265.1) cho do chinh xac sai.

8. **Troi timing symbol.** Clock RU phai dong bo. Troi gay lech ranh gioi symbol.
