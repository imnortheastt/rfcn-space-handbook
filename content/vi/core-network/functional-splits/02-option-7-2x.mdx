## Tai sao dieu nay quan trong

Option 7-2x la lua chon ky thuat dinh nghia cua O-RAN. No xac dinh gi di qua fronthaul, RU phai lam gi, va cach cac vendor tuong tac.

Hieu 7-2x chi tiet la can thiet cho trien khai O-RAN — no anh huong den lap ke hoach dung luong, lua chon thiet bi, va xu ly su co.

### Cau truc tin nhan eCPRI

```
Dinh dang Khung eCPRI:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  Khung Ethernet:                                        │
│  ┌────────┬────────┬─────────────────────────────────┐  │
│  │ Dst MAC│ Src MAC│ EtherType (0xAEFE = eCPRI)      │  │
│  └────────┴────────┴─────────────────────────────────┘  │
│                                                          │
│  Header eCPRI (8 bytes):                                │
│  ┌─────────────────────────────────────────────────┐    │
│  │ Byte 0: Phien ban giao thuc (0x10)              │    │
│  │ Byte 1: Loai tin nhan                           │    │
│  │         0x00: Du lieu IQ (U-Plane)              │    │
│  │         0x02: Dieu khien thoi gian thuc (C-Plane)│   │
│  │ Bytes 2-3: Kich thuoc payload                   │    │
│  │ Bytes 4-5: RTC_ID / PC_ID                       │    │
│  │ Bytes 6-7: SEQ_ID (so thu tu)                   │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  Header Section O-RAN (cho du lieu IQ):                 │
│  ┌─────────────────────────────────────────────────┐    │
│  │ - Huong du lieu (UL/DL)                         │    │
│  │ - Phien ban payload                             │    │
│  │ - Chi so bo loc                                 │    │
│  │ - Frame ID, subframe ID, slot ID, symbol ID     │    │
│  │ - Section ID                                    │    │
│  │ - PRB bat dau, so PRB                           │    │
│  │ - Mat na RE (RE nao trong PRB)                  │    │
│  │ - Header nen (neu nen)                          │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  Mau IQ:                                                │
│  ┌─────────────────────────────────────────────────┐    │
│  │ [I0, Q0], [I1, Q1], ... moi RE, moi PRB         │    │
│  │ Do rong bit: 9-16 bit moi I/Q (cau hinh duoc)   │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## Ap dung

**Vi du cau hinh:**

| Tham so | Cat-A (4T4R) | Cat-B (64T64R) |
|---------|--------------|----------------|
| Streams | 4 (moi antenna) | 8 (moi layer) |
| BW (100 MHz, BFP) | ~10 Gbps | ~25 Gbps |
| Beamforming RU | Khong | Co |
| Trong so beam | N/A | Moi PRB hoac codebook |

**Debug fronthaul:**

```bash
# Bat luu luong eCPRI
tcpdump -i eth0 ether proto 0xaefe -w fronthaul.pcap

# Kiem tra vi pham timing
grep "late_packet\|timing_error" /var/log/du/fronthaul.log

# O-RAN M-Plane: kiem tra trang thai RU
netconf-cli get --filter "<o-ran-delay-management/>"

# Xac minh cai dat nen
netconf-cli get --filter "<o-ran-compression-factors/>"

# Phan tich section header (bo loc Wireshark)
# oran.c_section.type == 1
```

**Vi du ngan sach timing:**

```
Xu ly DU:         100 µs
Switch Ethernet:   10 µs (moi hop)
Do tre cap quang:   5 µs/km
Xu ly RU:          50 µs

Cho cap quang 10 km + 2 switch:
Tong do tre = 100 + 20 + 50 + 50 = 220 µs

Ta3_max = 131 µs → VAN DE: can duong ngan hon hoac DU edge
```

<section class="lesson-pitfalls">

## Cam bay

1. **Khong khop Cat-A/B.** DU va RU phai thong nhat category. Cau hinh sai pha vo beamforming hoan toan.

2. **Nham loai section.** Dung sai loai section cho PRACH hoac SRS gay loi phat hien.

3. **Vi pham cua so timing.** Goi C-Plane hoac U-Plane tre bi bo im lang. Khong truyen lai.

4. **Thuong luong nen.** DU va RU phai thong nhat nen. Khong khop lam hong du lieu IQ.

5. **Uu tien VLAN.** C-Plane can uu tien cao hon U-Plane. QoS sai bo tin dieu khien.

6. **Loi so thu tu.** SEQ_ID bi mat chi ra mat goi. Giam sat chat luong fronthaul.

7. **Timing trong so beam.** Cho Cat-B, trong so beam phai den truoc U-Plane. Lech gay sai beam.

8. **Thang hang ranh gioi symbol.** Du lieu IQ phai thang hang voi ranh gioi symbol. Troi gay nhieu lien symbol.
