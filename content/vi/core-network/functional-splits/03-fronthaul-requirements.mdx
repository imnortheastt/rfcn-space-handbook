## Tai sao dieu nay quan trong

Fronthaul O-RAN khong phai Ethernet binh thuong. No mang mau IQ thoi gian thuc voi rang buoc timing micro giay. Lo cua so, goi bi bo.

Hieu yeu cau fronthaul la can thiet cho thiet ke mang transport. Du tru thieu gay mat goi. Du tru nhieu lang phi dung luong va ngan sach.

</section>

## Truc giac

### Fronthaul vs backhaul

```

│                                                          │
│  Fronthaul (DU ↔ RU):                                   │

│  │ Do tre: < 100-250 µs mot chieu                  │    │

│  │ Dong bo: ±1.5 µs pha (TDD)                      │    │
│  │                                                  │    │
│  │ Che do loi: Symbol bi bo                        │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  Midhaul (CU ↔ DU):                                     │
│  ┌─────────────────────────────────────────────────┐    │

│  │                                                  │    │
```

### Phan tich bang thong

```
Tinh Bang thong Fronthaul (Option 7-2x):

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  Cong thuc:                                             │
│  BW = Fs × Nsymbol × Nprb × Nre × Nant × Nbits × (1+OH)│
│                                                          │
│  Trong do:                                              │
│  - Fs: Toc do subframe (1000/s)                        │
│  - Nsymbol: Symbol moi subframe (14 × 2 cho 30kHz)     │
│  - Nprb: PRB (273 cho 100 MHz)                         │
│  - Nre: Subcarrier moi PRB (12)                        │
│  - Nant: Cong antenna                                  │
│  - Nbits: Bit moi mau IQ (32 raw, 18 BFP)              │
│  - OH: Overhead (~10-20%)                              │
│                                                          │
│  Vi du (100 MHz, 64T64R, Cat-B voi BFP):               │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  Moi layer: 1000 × 28 × 273 × 12 × 18 = 1.65 Gbps│   │
│  │  8 layers (Cat-B): 8 × 1.65 = 13.2 Gbps         │    │
│  │  Voi C-Plane + overhead: ~25 Gbps               │    │
│  │                                                  │    │
│  │  Cat-A (64 streams): ~50+ Gbps                  │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  So sanh theo cau hinh:                                 │
│  ┌─────────────────────────────────────────────────┐    │
│  │ Cau hinh    │ BW Raw      │ Nen (BFP)           │    │
│  │ ────────────┼─────────────┼─────────────────────│    │
│  │ 4T4R        │ ~10 Gbps    │ ~6 Gbps             │    │
│  │ 32T32R Cat-A│ ~80 Gbps    │ ~45 Gbps            │    │
│  │ 64T64R Cat-A│ ~160 Gbps   │ ~90 Gbps            │    │
│  │ 64T64R Cat-B│ ~40 Gbps    │ ~25 Gbps            │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## Ap dung

**Tom tat yeu cau:**

| Tham so | Yeu cau | Ghi chu |
|---------|---------|---------|
| Bang thong | 10-100 Gbps | Moi RU, tuy MIMO |
| Do tre | < 100-250 µs | Mot chieu, cua so Ta3 |
| Jitter | < 10 µs | PDV |
| Mat goi | < 10^-6 | Thuc te bang khong |
| Dong bo tan so | ±50 ppb | SyncE |
| Dong bo pha (TDD) | ±1.5 µs | PTP G.8275.1 |

**Lap ke hoach transport:**

```bash
# Tinh do tre
fiber_km=5
switch_hops=2
switch_delay_us=10

total_network_us=$((fiber_km * 5 + switch_hops * switch_delay_us))
echo "Do tre mang: ${total_network_us} µs"

# Bang thong moi sector
mimo_config="64T64R"
bw_mhz=100
cat_b_layers=8
bfp_bits=18

# Moi layer: 1000 × 28 × 273 × 12 × 18 bits/s
per_layer_gbps=1.65
total_gbps=$((cat_b_layers * per_layer_gbps * 12 / 10))  # voi 20% overhead
echo "BW Fronthaul: ~${total_gbps} Gbps"
```

**Yeu cau switch:**

| Tinh nang | Tai sao |
|-----------|---------|
| Cong 25G/100G | Bang thong fronthaul |
| Cut-through | Giam do tre |
| PTP BC | Dong bo pha |
| SyncE | Dong bo tan so |
| 802.1Qbv (TSN) | Do tre xac dinh |
| 802.1Qbu | Preemption khung |

<section class="lesson-pitfalls">

## Cam bay

1. **Vuot ngan sach do tre.** Cong tat ca thanh phan. Them mot hop switch co the pha vo timing.

2. **PTP khong dau cuoi.** Moi switch phai nhan biet PTP. Mot switch transparent pha hu do chinh xac pha.

3. **Dut chuoi SyncE.** Tat ca switch can SyncE. Mot switch khong SyncE pha vo chuoi tan so.

4. **Du ky qua muc.** Ghep kenh thong ke that bai cho fronthaul. Phai du tru dung luong dinh.

5. **Cau hinh QoS sai.** C-Plane phai co uu tien cao hon. Anh xa PCP sai bo tin dieu khien.

6. **Bat ngo do dai cap quang.** Duong ngam dai hon tam nhin. Khao sat duong thuc te.

7. **Thieu dinh hinh luu luong.** Dau ra DU bung gay tran buffer switch. Dinh hinh tai egress.

8. **Phuc tap du phong.** Chuyen doi phai < 50 ms. Can duong bao ve tinh toan truoc.
