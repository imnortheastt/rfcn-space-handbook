## Tai sao dieu nay quan trong

Near-RT RIC xu ly quyet dinh nhanh. Non-RT RIC xu ly lop chien luoc — huan luyen mo hinh ML, tao chinh sach, va quan ly vong doi RAN.

Non-RT RIC la mot phan cua SMO va cung cap tri tue huong dan hanh vi Near-RT RIC. Hieu ca hai la can thiet cho trien khai O-RAN hoan chinh.

## Ap dung

**Cac loai chinh sach A1:**

| Loai Chinh sach | Muc dich | Pham vi |
|-----------------|----------|---------|
| QoS | Muc tieu throughput/do tre | Cell, slice, UE |
| Traffic Steering | Quy tac can bang tai | Cell, TA |
| Resource | Huong dan cap phat PRB | Cell, slice |
| Energy Saving | Quan ly nang luong | Cell, site |

**Cac thanh phan OSC Non-RT RIC:**

```bash
# Trien khai OSC Non-RT RIC
helm install nonrtric oran-sc/nonrtric

# Kiem tra rApp manager
kubectl get pods -n nonrtric | grep rappmanager

# Dang ky loai chinh sach A1
curl -X PUT http://a1-pms:8081/policytype \
  -H "Content-Type: application/json" \
  -d '{
    "policyTypeId": "ORAN_QoS_1.0",
    "policySchema": {...}
  }'

# Tao chinh sach A1
curl -X PUT http://a1-pms:8081/policy \
  -H "Content-Type: application/json" \
  -d '{
    "policyId": "qos-001",
    "policyTypeId": "ORAN_QoS_1.0",
    "ricId": "ric-1",
    "policyData": {
      "scope": {"cellId": "cell-001"},
      "qosObjectives": {"minThroughput": 100000000}
    }
  }'

# Kiem tra trang thai chinh sach
curl http://a1-pms:8081/policy/qos-001/status
```

**Phat trien rApp:**

```python
# Vi du rApp: Du doan luu luong
from nonrtric_sdk import RAppClient

client = RAppClient()

# Lay du lieu PM lich su
pm_data = client.get_pm_history(
    cell_id="cell-001",
    metrics=["prb_util", "throughput"],
    hours=168  # 1 tuan
)

# Huan luyen mo hinh
model = train_traffic_prediction(pm_data)

# Trien khai den Near-RT RIC
client.deploy_model(
    model_id="traffic-pred-v1",
    model=model,
    target_xapp="traffic-steering-xapp"
)

# Tao chinh sach A1 dua tren du doan
if predicted_congestion():
    client.create_policy(
        policy_type="ORAN_TrafficSteering_1.0",
        policy_data={"action": "balance_load"}
    )
```

<section class="lesson-pitfalls">

## Cam bay

1. **Chinh sach A1 khong duoc thuc thi.** Near-RT RIC nhan chinh sach nhung xApp bo qua. Kiem tra xu ly A1 cua xApp.

2. **Mo hinh ML cu.** Mo hinh huan luyen tren du lieu cu khong phu hop dieu kien hien tai. Lap lich huan luyen lai thuong xuyen.

3. **Xung dot chinh sach.** Nhieu rApps tao chinh sach xung dot. Su dung phat hien xung dot chinh sach.

4. **Tre pipeline du lieu.** Du lieu PM den qua cham cho quyet dinh kip thoi. Toi uu pipeline thu thap du lieu.

5. **Mo hinh qua lon.** Near-RT RIC khong the tai mo hinh trong rang buoc bo nho. Nen hoac don gian mo hinh.

6. **Thieu phan hoi.** Khong co so lieu ve hieu nang mo hinh sau trien khai. Trien khai giam sat.

7. **Khong khop phien ban R1 API.** rApp su dung R1 API cu. Kiem tra tuong thich voi phien ban Non-RT RIC.

8. **Du lieu lam giau cu.** Du lieu ben ngoai (du doan) khong cap nhat thuong xuyen. Kiem tra nha cung cap lam giau.
