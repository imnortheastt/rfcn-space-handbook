## Tai sao dieu nay quan trong

xApps la cac dong co tri tue cua Near-RT RIC. Chung tieu thu du lieu RAN, ap dung thuat toan (bao gom ML), va phat hanh dong dieu khien — tat ca trong vong dieu khien 10ms den 1s.

Hieu phat trien xApp la can thiet de xay dung cac giai phap tu dong hoa va toi uu O-RAN.

## Ap dung

**Cac tuy chon xApp SDK:**

| Ngon ngu | SDK | Ghi chu |
|----------|-----|---------|

| Python | ric-plt/xapp-frame-py | Than thien voi ML |
| C/C++ | ric-plt/xapp-frame-cpp | Yeu cau hieu nang cao |

**Vi du MLB xApp (Go):**

```go
package main

import (
    "gerrit.o-ran-sc.org/r/ric-plt/xapp-frame/pkg/xapp"
)

type MLBXapp struct {
    threshold int
}

func (m *MLBXapp) Consume(msg *xapp.RMRParams) error {
    switch msg.Mtype {
    case xapp.RIC_INDICATION:
        return m.handleIndication(msg)
    case xapp.A1_POLICY_REQ:
        return m.handlePolicy(msg)
    }
    return nil
}

func (m *MLBXapp) handleIndication(msg *xapp.RMRParams) error {
    // Giai ma KPM indication
    kpm := DecodeKpmIndication(msg.Payload)
    
    // Kiem tra tai cell
    for _, cell := range kpm.Cells {
        if cell.PrbUtilDL > m.threshold {
            // Tim cell dich de can bang tai
            target := m.findLeastLoadedNeighbor(cell)
            if target != nil {
                m.triggerHandover(cell, target)
            }
        }
    }
    return nil
}

func main() {
    mlb := &MLBXapp{threshold: 70}
    xapp.SetMessageCallback(mlb)
    xapp.Run()
}
```

<section class="lesson-pitfalls">

## Cam bay

1. **Chan callback RMR.** Xu ly dai trong callback chan hang doi ban tin. Dung async/goroutines.

2. **Thieu don dep dang ky.** xApp crash khong huy dang ky. Subscription manager xu ly mo coi nhung kiem tra trang thai.

3. **Mau truy cap SDL.** Doc nho thuong xuyen lam cham. Doc theo lo, cache cuc bo.

4. **Tan suat dieu khien qua cao.** Gui qua nhieu ban tin dieu khien gay nghen E2. Trien khai gioi han tan suat.

5. **Loi ma hoa E2SM.** Khong khop ma hoa ASN.1 giua xApp va E2 node. Xac minh tuong thich phien ban E2SM.

6. **Cau hinh khong tai lai.** xApp khong nhan thay doi cau hinh. Trien khai theo doi cau hinh hoac khoi dong lai.

7. **Ro ri bo nho.** Payload indication lon khong giai phong. Profiling su dung bo nho, dac biet cho xApp ML.

8. **Xung dot voi xApps khac.** Nhieu xApps dieu khien cung tham so. Su dung dich vu giam thieu xung dot.
