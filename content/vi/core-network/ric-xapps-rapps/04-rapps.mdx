## Tai sao dieu nay quan trong

rApps la cac dong co tri tue cua Non-RT RIC. Chung hoat dong tren thang thoi gian >1 giay, thuc hien phan tich, huan luyen mo hinh ML, va tao chinh sach huong dan xApps.

Hieu phat trien rApp cho phep xay dung "bo nao" toi uu hanh vi RAN o cap do chien luoc.

## Ap dung

**Ngan xep phat trien rApp:**

| Thanh phan | Tuy chon |
|------------|----------|
| Ngon ngu | Python (ML), Go, Java |
| Framework ML | TensorFlow, PyTorch, scikit-learn |
| Dinh dang Mo hinh | ONNX (di dong), SavedModel |
| Container | Docker, Kubernetes |
| Truy cap Du lieu | Kafka, REST APIs |

**Vi du rApp du doan luu luong (Python):**

```python
from rapp_sdk import RAppFramework, R1Client
import pandas as pd
from sklearn.ensemble import GradientBoostingRegressor
import onnx

class TrafficPredictionRApp:
    def __init__(self, config):
        self.r1 = R1Client()
        self.model = None
        self.config = config
    
    def start(self):
        # Dang ky du lieu PM
        self.r1.subscribe_data(
            data_type="pm",
            filter={"counters": ["PRBUsageDL", "PRBUsageUL"]},
            callback=self.on_data
        )
        
        # Dang ky nhu nha san xuat EI
        self.r1.register_ei_producer(
            info_type="traffic-prediction",
            schema=self.prediction_schema
        )
    
    def on_data(self, data):
        # Tich luy du lieu de huan luyen
        self.data_buffer.append(data)
        
        if len(self.data_buffer) >= self.config['min_data_points']:
            self.train_model()
    
    def train_model(self):
        df = pd.DataFrame(self.data_buffer)
        X = self.extract_features(df)
        y = df['target_load']
        
        self.model = GradientBoostingRegressor()
        self.model.fit(X, y)
        
        # Xuat sang ONNX va trien khai
        onnx_model = self.to_onnx(self.model)
        self.r1.upload_model(
            model_id="traffic-pred-v2",
            artifact=onnx_model
        )
    
    def generate_predictions(self):
        predictions = self.model.predict(self.current_features)
        
        # Xuat ban nhu thong tin lam giau
        self.r1.publish_ei(
            info_type="traffic-prediction",
            data={"predictions": predictions.tolist()}
        )

if __name__ == "__main__":
    app = RAppFramework()
    rapp = TrafficPredictionRApp(app.config)
    app.run(rapp)
```

<section class="lesson-pitfalls">

## Cam bay

1. **Huan luyen tren du lieu cu.** Du lieu PM tu nhieu tuan truoc co the khong phan anh mang hien tai. Trien khai kiem tra do moi du lieu.

2. **Xung dot chinh sach.** Nhieu rApps tao chinh sach xung dot. Su dung dich vu phoi hop chinh sach.

3. **Troi mo hinh khong phat hien.** Du doan suy giam theo thoi gian. Trien khai giam sat va kich hoat huan luyen lai.

4. **Qua tai A1.** Qua nhieu cap nhat chinh sach lam bao hoa giao dien A1. Gom nhom thay doi chinh sach.

5. **Do tre EI qua cao.** Thong tin lam giau den sau quyet dinh xApp. Tinh toan truoc va cache du doan.

6. **Dich vu R1 khong kha dung.** Dich vu Non-RT RIC khoi dong lai. Trien khai thu lai voi backoff.

7. **Trien khai mo hinh lon.** Mo hinh ONNX qua lon cho Near-RT RIC. Luong tu hoa hoac cat tia mo hinh.

8. **Khong khop schema.** Schema chinh sach thay doi giua cac phien ban. Quan ly phien ban loai chinh sach can than.
