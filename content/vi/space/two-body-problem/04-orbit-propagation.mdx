## Ap dung

**Bo lan truyen hai vat day du:**

```python
import numpy as np

def propagate_kepler(coe, t0, t, mu=398600.4418):
    """
    Lan truyen quy dao hai vat
    
    coe: dict voi a, e, i, RAAN, omega, M0 (goc o radian)
    t0: thoi gian ky nguyen (giay)
    t: thoi gian muc tieu (giay)
    mu: tham so hap dan (km³/s²)
    
    Tra ve: r_eci, v_eci (vi tri va van toc trong ECI)
    """
    a = coe['a']
    e = coe['e']
    i = coe['i']
    RAAN = coe['RAAN']
    omega = coe['omega']
    M0 = coe['M0']
    
    # Chuyen dong trung binh
    n = np.sqrt(mu / a**3)
    
    # Di thuong trung binh tai thoi gian t
    dt = t - t0
    M = M0 + n * dt
    M = M % (2 * np.pi)
    
    # Giai phuong trinh Kepler
    E = M  # Uoc luong ban dau
    for _ in range(50):
        E_new = E - (E - e*np.sin(E) - M) / (1 - e*np.cos(E))
        if abs(E_new - E) < 1e-12:
            break
        E = E_new
    
    # Di thuong thuc
    nu = 2 * np.arctan2(
        np.sqrt(1+e) * np.sin(E/2),
        np.sqrt(1-e) * np.cos(E/2)
    )
    
    # Vi tri trong he perifocal
    p = a * (1 - e**2)
    r = p / (1 + e * np.cos(nu))
    r_pqw = np.array([r*np.cos(nu), r*np.sin(nu), 0])
    
    # Van toc trong he perifocal
    v_pqw = np.sqrt(mu/p) * np.array([-np.sin(nu), e+np.cos(nu), 0])
    
    # Ma tran xoay
    def Rz(angle):
        c, s = np.cos(angle), np.sin(angle)
        return np.array([[c, -s, 0], [s, c, 0], [0, 0, 1]])
    
    def Rx(angle):
        c, s = np.cos(angle), np.sin(angle)
        return np.array([[1, 0, 0], [0, c, -s], [0, s, c]])
    
    # Xoay ket hop: PQW -> ECI
    R = Rz(-RAAN) @ Rx(-i) @ Rz(-omega)
    
    r_eci = R @ r_pqw
    v_eci = R @ v_pqw
    
    return r_eci, v_eci

# Vi du: Quy dao giong ISS
coe = {
    'a': 6778.0,       # km
    'e': 0.0006,
    'i': np.radians(51.6),
    'RAAN': np.radians(45.0),
    'omega': np.radians(30.0),
    'M0': np.radians(0.0)
}

r, v = propagate_kepler(coe, t0=0, t=3600)  # 1 gio sau
print(f"Vi tri: {r} km")
print(f"Van toc: {v} km/s")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Nhat quan don vi.** μ o km³/s² can a o km, cho v o km/s.

2. **Quy uoc goc.** radian cho tinh toan, do cho hien thi.

3. **Hoi tu Kepler.** Do lech tam cao can nhieu vong lap hoac uoc luong ban dau tot hon.

4. **Loi goc phan tu.** Dung atan2, khong phai atan, cho chuyen doi di thuong.

5. **Thu tu xoay.** Thu tu nhan ma tran quan trong: R_z(-Ω) × R_x(-i) × R_z(-ω).

6. **Han che hai vat.** Do chinh xac giam nhanh khong co nhieu loan.

7. **He thoi gian.** Dam bao thang thoi gian nhat quan (UTC, TT, etc.).

8. **He toa do.** ECI vs ECEF vs he khac—biet ban dang dung gi.
