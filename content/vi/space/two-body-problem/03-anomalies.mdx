## Tai sao dieu nay quan trong

Ve tinh khong di chuyen voi toc do khong doi trong quy dao ellipse—chung tang toc tai can diem va giam toc tai vien diem. Ba di thuong (thuc, lech tam, trung binh) la cac cach khac nhau de theo doi vi tri xu ly toc do thay doi nay. Hieu chung la thiet yeu cho lan truyen quy dao va tinh thoi gian.

### Phuong trinh Kepler

```
Phuong trinh Trung tam cua Co hoc Quy dao:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  PHUONG TRINH KEPLER:                                   │
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                    │  │
│  │         M = E - e × sin(E)                        │  │
│  │                                                    │  │
│  │  trong do:                                        │  │
│  │    M = di thuong trung binh (rad)                 │  │
│  │    E = di thuong lech tam (rad)                   │  │
│  │    e = do lech tam                                │  │
│  │                                                    │  │
│  └───────────────────────────────────────────────────┘  │
│                                                          │
│  VAN DE:                                                │
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                    │  │
│  │  Cho M (tu thoi gian), tim E                      │  │
│  │  → Phuong trinh sieu viet, khong co dang dong     │  │
│  │  → Phai giai lap!                                 │  │
│  │                                                    │  │
│  └───────────────────────────────────────────────────┘  │
│                                                          │
│  GIAI PHAP NEWTON-RAPHSON:                              │
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                    │  │
│  │  f(E) = E - e×sin(E) - M = 0                      │  │
│  │  f'(E) = 1 - e×cos(E)                             │  │
│  │                                                    │  │
│  │  E_{n+1} = E_n - f(E_n)/f'(E_n)                   │  │
│  │          = E_n - (E_n - e×sin(E_n) - M)           │  │
│  │                  ─────────────────────            │  │
│  │                    (1 - e×cos(E_n))               │  │
│  │                                                    │  │
│  │  Uoc luong ban dau: E₀ = M (tot cho e < 0.8)      │  │
│  │  Hoi tu trong 3-5 vong lap thuong                 │  │
│  │                                                    │  │
│  └───────────────────────────────────────────────────┘  │
```

## Ap dung

**Giai phuong trinh Kepler:**

```python
import numpy as np

def solve_kepler(M, e, tol=1e-10, max_iter=50):
    """
    Giai phuong trinh Kepler: M = E - e*sin(E)
    Tra ve di thuong lech tam E tinh bang radian
    """
    # Uoc luong ban dau
    if e < 0.8:
        E = M
    else:
        E = np.pi
    
    # Lap Newton-Raphson
    for _ in range(max_iter):
        f = E - e * np.sin(E) - M
        f_prime = 1 - e * np.cos(E)
        E_new = E - f / f_prime
        
        if abs(E_new - E) < tol:
            return E_new
        E = E_new
    
    return E  # Co the chua hoi tu

def E_to_nu(E, e):
    """Chuyen doi thuong lech tam sang di thuong thuc"""
    nu = 2 * np.arctan2(
        np.sqrt(1 + e) * np.sin(E / 2),
        np.sqrt(1 - e) * np.cos(E / 2)
    )
    return nu

def propagate_orbit(M0, n, dt, e):
    """
    Lan truyen quy dao theo thoi gian dt
    M0: di thuong trung binh ban dau (rad)
    n: chuyen dong trung binh (rad/s)
    dt: buoc thoi gian (s)
    e: do lech tam
    Tra ve di thuong thuc (rad)
    """
    M = M0 + n * dt
    M = M % (2 * np.pi)  # Giu trong [0, 2π]
    E = solve_kepler(M, e)
    nu = E_to_nu(E, e)
    return nu

# Vi du: GTO voi e = 0.73
e = 0.73
M = np.radians(90)  # Di thuong trung binh = 90°
E = solve_kepler(M, e)
nu = E_to_nu(E, e)
print(f"M = {np.degrees(M):.2f}°")
print(f"E = {np.degrees(E):.2f}°")
print(f"ν = {np.degrees(nu):.2f}°")
```

**Tham chieu nhanh:**

| Dai luong | Cong thuc |
|-----------|-----------|
| Chuyen dong trung binh | n = √(μ/a³) |
| Di thuong trung binh | M = n(t - t₀) |
| Phuong trinh Kepler | M = E - e·sin(E) |
| Ban kinh | r = a(1 - e·cos(E)) |

<section class="lesson-pitfalls">

## Cam bay

1. **Don vi goc.** Phuong trinh Kepler dung radian. Chuyen doi do can than.

2. **Mo ho goc phan tu.** Dung atan2() cho dung goc phan tu khi chuyen doi di thuong.

3. **Do lech tam cao.** Newton-Raphson co the can uoc luong ban dau tot hon cho e > 0.8.

4. **Parabolic/hyperbolic.** Phuong trinh Kepler chi cho ellipse. Dung dang khac cho e ≥ 1.

5. **Wrap goc.** Giu M trong [0, 2π] hoac [-π, π] de nhat quan.

6. **Sai so.** Dat sai so hoi tu phu hop cho bo giai Kepler.

7. **Quy uoc dau.** Di thuong tang theo huong chuyen dong (nguoc chieu kim dong ho khi nhin tu bac).

8. **Tham chieu thoi gian.** M = 0 khi qua can diem. Theo doi t₀ can than.
