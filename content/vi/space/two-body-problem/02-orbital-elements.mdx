## Tai sao dieu nay quan trong

De mo ta quy dao cua mot ve tinh, ban can chinh xac sau con so. Cac Phan tu Quy dao Co dien (COEs) nay cho ban biet hinh dang quy dao, cach no nghieng trong khong gian, va ve tinh o dau tai moi thoi diem. Moi TLE, moi thiet ke nhiem vu, va moi he thong theo doi deu su dung cac phan tu nay.

</section>

## Ap dung

**Vi du mo ta quy dao:**

| Quy dao | a (km) | e | i (°) | Ω (°) | ω (°) | ν (°) |
|---------|--------|---|-------|-------|-------|-------|
| ISS | 6,778 | 0.0006 | 51.6 | thay doi | thay doi | thay doi |
| GEO | 42,164 | 0.0001 | 0.05 | N/A | N/A | thay doi |
| GPS | 26,560 | 0.01 | 55 | thay doi | thay doi | thay doi |
| Molniya | 26,600 | 0.74 | 63.4 | thay doi | 270 | thay doi |

**Chuyen doi vector trang thai sang COEs:**

```python
import numpy as np

def rv_to_coe(r_vec, v_vec, mu=398600.4418):
    """
    Chuyen doi vi tri/van toc sang phan tu quy dao
    r_vec: vector vi tri (km)
    v_vec: vector van toc (km/s)
    mu: tham so hap dan (km³/s²)
    """
    r = np.linalg.norm(r_vec)
    v = np.linalg.norm(v_vec)
    
    # Momen dong luong
    h_vec = np.cross(r_vec, v_vec)
    h = np.linalg.norm(h_vec)
    
    # Vector nut
    K = np.array([0, 0, 1])
    n_vec = np.cross(K, h_vec)
    n = np.linalg.norm(n_vec)
    
    # Vector do lech tam
    e_vec = ((v**2 - mu/r)*r_vec - np.dot(r_vec, v_vec)*v_vec) / mu
    e = np.linalg.norm(e_vec)
    
    # Ban truc lon
    energy = v**2/2 - mu/r
    a = -mu / (2*energy)
    
    # Do nghieng
    i = np.arccos(h_vec[2]/h)
    
    # RAAN
    Omega = np.arccos(n_vec[0]/n) if n > 1e-10 else 0
    if n_vec[1] < 0:
        Omega = 2*np.pi - Omega
    
    # Goc can diem
    omega = np.arccos(np.dot(n_vec, e_vec)/(n*e)) if n > 1e-10 and e > 1e-10 else 0
    if e_vec[2] < 0:
        omega = 2*np.pi - omega
    
    # Di thuong thuc
    nu = np.arccos(np.dot(e_vec, r_vec)/(e*r)) if e > 1e-10 else 0
    if np.dot(r_vec, v_vec) < 0:
        nu = 2*np.pi - nu
    
    return {
        'a': a, 'e': e, 
        'i': np.degrees(i),
        'RAAN': np.degrees(Omega),
        'omega': np.degrees(omega),
        'nu': np.degrees(nu)
    }
```

## Cam bay

1. **Phan tu khong xac dinh.** Cho quy dao tron (e=0) hoac xich dao (i=0), mot so phan tu khong xac dinh.

2. **Pham vi goc.** Goc co the la 0-360° hoac -180° den 180°. Kiem tra quy uoc.

3. **Di thuong trung binh vs thuc.** M (trung binh) thay doi tuyen tinh theo thoi gian. ν (thuc) thi khong.

4. **He toa do.** COEs thuong o ECI (J2000). Cac he khac ton tai.

5. **Nhieu loan thay doi phan tu.** J2 lam Ω va ω troi. a va e thay doi do can khi.

6. **Dinh dang TLE.** TLEs dung phan tu trung binh, khong phai phan tu giao dong.

7. **Don vi.** a thuong la km, goc la do hoac radian. Nhat quan.

8. **Ky nguyen quan trong.** Phan tu co gia tri tai thoi diem cu the. Chung thay doi theo nhieu loan.
