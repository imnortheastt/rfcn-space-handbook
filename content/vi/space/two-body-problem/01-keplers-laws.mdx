## Tai sao dieu nay quan trong

Dinh luat Kepler la nen tang cua co hoc quy dao. Duoc cong bo vao dau nhung nam 1600, chung mo ta cach ve tinh, hanh tinh, va tau vu tru di chuyen trong khong gian. Moi du doan quy dao, moi thao tac ve tinh, va moi quy dao lien hanh tinh deu xay dung tren ba dinh luat thanh lich nay.

### Van toc tai moi diem

```
Phuong trinh Vis-Viva (van toc tu vi tri):

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  v² = μ × (2/r - 1/a)                                   │
│                                                          │
│  trong do:                                              │
│    v = van toc tai ban kinh r                           │
│    μ = tham so hap dan (GM)                             │
│    r = ban kinh hien tai tu tam                         │
│    a = ban truc lon                                     │
│                                                          │
│  TRUONG HOP DAC BIET:                                   │
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                    │  │
│  │  Quy dao tron (a = r):                            │  │
│  │  v² = μ/r  →  v = √(μ/r)                          │  │
│  │                                                    │  │
│  │  Can diem (r = rp):                               │  │
│  │  vp = √(μ × (2/rp - 1/a))  [van toc cuc dai]      │  │
│  │                                                    │  │
│  │  Vien diem (r = ra):                              │  │
│  │  va = √(μ × (2/ra - 1/a))  [van toc cuc tieu]     │  │
│  │                                                    │  │
│  │  Van toc thoat ly (a → ∞):                        │  │
│  │  vesc = √(2μ/r)                                   │  │
│  │                                                    │  │
│  └───────────────────────────────────────────────────┘  │
│                                                          │
│  VI DU: LEO tai do cao 400 km                           │
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                    │  │
│  │  r = 6378 + 400 = 6778 km                         │  │
│  │  v = √(398600/6778) = 7.67 km/s                   │  │
│  │                                                    │  │
│  │  vesc = √(2 × 398600/6778) = 10.85 km/s           │  │
│  │                                                    │  │
│  │  Δv de thoat = 10.85 - 7.67 = 3.18 km/s           │  │
│  │                                                    │  │
│  └───────────────────────────────────────────────────┘  │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## Ap dung

**Tinh toan nhanh:**

```python
import math

# Hang so
mu_earth = 398600.4418  # km³/s² (Trai Dat)
R_earth = 6378.137      # km (ban kinh Trai Dat)

def orbital_period(a_km):
    """Chu ky tinh bang giay tu ban truc lon tinh bang km"""
    return 2 * math.pi * math.sqrt(a_km**3 / mu_earth)

def circular_velocity(r_km):
    """Van toc tron tai ban kinh r_km"""
    return math.sqrt(mu_earth / r_km)

def escape_velocity(r_km):
    """Van toc thoat tai ban kinh r_km"""
    return math.sqrt(2 * mu_earth / r_km)

# Vi du
leo_alt = 400  # km
leo_r = R_earth + leo_alt

print(f"Chu ky LEO: {orbital_period(leo_r)/60:.1f} phut")
print(f"Van toc LEO: {circular_velocity(leo_r):.2f} km/s")
print(f"Van toc thoat: {escape_velocity(leo_r):.2f} km/s")
```

**Gia tri tham chieu pho bien:**

| Quy dao | Do cao (km) | Ban truc lon (km) | Chu ky | Van toc (km/s) |

| ISS | 400 | 6,778 | 92.6 phut | 7.67 |
| GPS | 20,180 | 26,560 | 12 gio | 3.87 |
| GEO | 35,786 | 42,164 | 24 gio | 3.07 |

<section class="lesson-pitfalls">

## Cam bay

1. **Nham lan ban kinh va do cao.** Luon dung ban kinh tu tam Trai Dat, khong phai do cao tren be mat.

2. **Don vi khong khop.** μ o km³/s² can khoang cach o km, van toc o km/s.

3. **Quen ngay thien van vs ngay mat troi.** GEO dung ngay thien van (86164s), khong phai ngay mat troi (86400s).

4. **Gia dinh quy dao tron.** Hau het quy dao la ellipse. Dung vis-viva cho truong hop chung.

5. **Bo qua nhieu loan.** Dinh luat Kepler la hai vat ly tuong. Quy dao thuc co J2, can khi, etc.

6. **Quy uoc dau.** Huong momen dong luong quan trong cho bai toan 3D.

7. **Thoat vs quy dao.** Van toc thoat bang √2 lan van toc tron tai cung ban kinh.

8. **Chu ky chi phu thuoc a.** Chu ky doc lap voi do lech tam cho cung ban truc lon.
