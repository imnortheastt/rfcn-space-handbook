## Tai sao dieu nay quan trong

Bay tu Trai Dat den Sao Hoa lien quan den ba truong hap dan: cua Trai Dat gan diem xuat phat, cua Mat Troi qua khong gian lien hanh tinh, va cua Sao Hoa tai diem den. Giai dong thoi ca ba rat ton kem tinh toan. Phuong phap va conic chia bai toan thanh cac bai toan hai vat the de quan ly, ghep lai voi nhau tai cac bien. Day la cach cac nhiem vu Apollo duoc thiet ke va van la nen tang cho lap ke hoach nhiem vu ngay nay.

## Ap dung

**Tinh toan va conic:**

```python
import numpy as np

# Tham so hap dan (km³/s²)
MU_SUN = 1.327e11
MU_EARTH = 398600
MU_MARS = 42828

# Ban kinh quy dao (km)
R_EARTH_ORBIT = 149.6e6  # 1 AU
R_MARS_ORBIT = 227.9e6   # 1.524 AU
R_EARTH = 6378
R_MARS = 3390

def sphere_of_influence(a, m_planet, m_sun):
    """Tinh ban kinh cau anh huong."""
    return a * (m_planet / m_sun) ** 0.4

def hohmann_v_inf(r1, r2, mu_sun):
    """
    Tinh V_inf tai xuat phat va den cho chuyen doi Hohmann.
    
    Tra ve: (v_inf_xuat_phat, v_inf_den)
    """
    # Ban truc lon quy dao chuyen doi
    a_transfer = (r1 + r2) / 2
    
    # Van toc
    v_planet1 = np.sqrt(mu_sun / r1)  # Van toc quy dao Trai Dat
    v_planet2 = np.sqrt(mu_sun / r2)  # Van toc quy dao Sao Hoa
    
    # Van toc tau tren chuyen doi tai moi dau
    v_sc_depart = np.sqrt(mu_sun * (2/r1 - 1/a_transfer))
    v_sc_arrive = np.sqrt(mu_sun * (2/r2 - 1/a_transfer))
    
    # V_vo cuc (du hyperbolic)
    v_inf_depart = abs(v_sc_depart - v_planet1)
    v_inf_arrive = abs(v_planet2 - v_sc_arrive)
    
    return v_inf_depart, v_inf_arrive

def departure_dv(v_inf, r_park, mu_planet):
    """
    Tinh Δv cho phun lien hanh tinh tu quy dao dau.
    """
    v_circ = np.sqrt(mu_planet / r_park)
    v_periapsis = np.sqrt(v_inf**2 + 2 * mu_planet / r_park)
    return v_periapsis - v_circ

def arrival_dv(v_inf, r_capture, mu_planet, capture_type='circular'):
    """
    Tinh Δv cho nhap quy dao.
    
    capture_type: 'circular' hoac 'elliptical'
    """
    v_p_hyp = np.sqrt(v_inf**2 + 2 * mu_planet / r_capture)
    
    if capture_type == 'circular':
        v_p_capture = np.sqrt(mu_planet / r_capture)
    else:
        # Voi ellipse, can chi dinh vien diem
        v_p_capture = np.sqrt(mu_planet / r_capture)
    
    return v_p_hyp - v_p_capture

# Vi du: Chuyen doi Hohmann Trai Dat den Sao Hoa
print("Chuyen doi Hohmann Trai Dat-Sao Hoa (Va Conic):")
print("=" * 55)

# Tinh SOI
soi_earth = sphere_of_influence(R_EARTH_ORBIT, MU_EARTH, MU_SUN)
soi_mars = sphere_of_influence(R_MARS_ORBIT, MU_MARS, MU_SUN)
print(f"SOI Trai Dat: {soi_earth/1e6:.2f} trieu km")
print(f"SOI Sao Hoa: {soi_mars/1e6:.2f} trieu km")

# Chuyen doi nhat tam
v_inf_earth, v_inf_mars = hohmann_v_inf(R_EARTH_ORBIT, R_MARS_ORBIT, MU_SUN)
print(f"\nGiai doan nhat tam:")
print(f"  V∞ xuat phat Trai Dat: {v_inf_earth:.2f} km/s")
print(f"  V∞ den Sao Hoa: {v_inf_mars:.2f} km/s")

# Xuat phat tu quy dao dau 200 km
r_park = R_EARTH + 200
dv_depart = departure_dv(v_inf_earth, r_park, MU_EARTH)
print(f"\nXuat phat (quy dao dau 200 km):")
print(f"  Δv cho TPI: {dv_depart:.2f} km/s")

# Den quy dao Sao Hoa 400 km
r_capture = R_MARS + 400
dv_arrive = arrival_dv(v_inf_mars, r_capture, MU_MARS)
print(f"\nDen (quy dao Sao Hoa 400 km):")
print(f"  Δv cho MOI: {dv_arrive:.2f} km/s")

# Tong Δv nhiem vu
total_dv = dv_depart + dv_arrive
print(f"\nTong Δv nhiem vu: {total_dv:.2f} km/s")
```

**So sanh nhiem vu:**

| Loai nhiem vu | V∞ xuat phat | V∞ den | Tong Δv |
|---------------|--------------|--------|---------|
| TD-Sao Hoa Hohmann | 2.9 km/s | 2.6 km/s | ~5.6 km/s |
| TD-Sao Kim Hohmann | 2.5 km/s | 2.7 km/s | ~5.0 km/s |
| TD-Sao Moc | 8.8 km/s | 5.6 km/s | ~9+ km/s |

<section class="lesson-pitfalls">

## Cam bay

1. **SOI la tuy y.** Bien khong phai duong sac net—la noi ta chon de "va." Ket qua nhay cam voi vi tri chinh xac.

2. **Bo qua doi mat phang.** Hanh tinh khong dong phang; nhiem vu thuc can them Δv cho do nghieng.

3. **Huong V∞ quan trong.** Tam giac van toc phu thuoc hinh hoc tai xuat phat/den.

4. **Cua so phong.** Va conic gia dinh hinh hoc ly tuong; nhiem vu thuc bi rang buoc boi chu ky hoi hop.

5. **Khong nhieu loan.** Bo qua cac hanh tinh khac, ap suat buc xa mat troi, trong truong khong cau.

6. **Sai so va.** Gian doan tai bien SOI gay sai so nho so voi quy dao thuc.

7. **Thoi gian.** Tinh thoi gian bay can can than hon—khong tinh chuyen dong hanh tinh trong chuyen bay.
