## Tai sao dieu nay quan trong

Cac nhiem vu Sao Hoa phong khoang 26 thang mot lan. Co hoi Sao Moc den moi 13 thang. Lo cua so, va ban doi hang nam cho lan tiep theo—hoac dot nhieu nhien lieu hon. Hieu chu ky hoi hop va hinh hoc hanh tinh la thiet yeu cho lap ke hoach nhiem vu, phan bo ngan sach, va hieu tai sao mot so nhiem vu bi tre hang nam.

## Ap dung

**Tinh toan chu ky hoi hop:**

```python
import numpy as np

def synodic_period(t_earth_years, t_planet_years):
    """
    Tinh chu ky hoi hop giua Trai Dat va hanh tinh khac.
    
    Tra ve: chu ky hoi hop tinh bang nam Trai Dat
    """
    if t_planet_years > t_earth_years:
        # Hanh tinh ngoai
        return 1 / (1/t_earth_years - 1/t_planet_years)
    else:
        # Hanh tinh trong
        return 1 / (1/t_planet_years - 1/t_earth_years)

def days_to_months(days):
    return days / 30.44

# Chu ky quy dao (nam)
ORBITAL_PERIODS = {
    'Sao Thuy': 0.241,
    'Sao Kim': 0.615,
    'Trai Dat': 1.0,
    'Sao Hoa': 1.881,
    'Sao Moc': 11.86,
    'Sao Tho': 29.46,
    'Thien Vuong': 84.01,
    'Hai Vuong': 164.8,
}

print("Chu ky Hoi hop (Tan suat Cua so Phong):")
print("=" * 55)

for planet, t_planet in ORBITAL_PERIODS.items():
    if planet == 'Trai Dat':
        continue
    
    t_syn = synodic_period(1.0, t_planet)
    days = t_syn * 365.25
    months = days_to_months(days)
    
    print(f"{planet:12} Quy dao: {t_planet*365.25:7.0f} ngay  "
          f"Hoi hop: {days:5.0f} ngay ({months:.1f} thang)")

# Vi du: Cua so phong Sao Hoa tiep theo
print("\n" + "=" * 55)
print("Cua so Phong Sao Hoa (xap xi):")
print("-" * 55)

# Chu ky hoi hop Sao Hoa ~780 ngay = 26 thang
base_year = 2024
mars_synodic_months = 26

for i in range(5):
    year = base_year + (i * mars_synodic_months) // 12
    month = (7 + i * mars_synodic_months) % 12 + 1  # Bat dau T7/2024
    print(f"  Cua so {i+1}: ~{year} (xap xi)")

# Goc pha cho chuyen doi Hohmann
def hohmann_phase_angle(r1_au, r2_au):
    """
    Tinh goc pha can thiet cho chuyen doi Hohmann.
    
    Tra ve: goc pha tinh bang do
    """
    # Thoi gian chuyen doi
    a_transfer = (r1_au + r2_au) / 2
    t_transfer = np.pi * np.sqrt(a_transfer**3)  # don vi T_TD = 2π
    
    # Hanh tinh di chuyen trong chuyen doi
    omega_planet = 2 * np.pi / np.sqrt(r2_au**3)  # van toc goc
    planet_travel = omega_planet * t_transfer  # radian
    
    # Goc pha: hanh tinh phai truoc bay nhieu
    phase = np.pi - planet_travel  # muon hanh tinh tai 180° khi ta den
    return np.degrees(phase)

print("\n" + "=" * 55)
print("Goc Pha Chuyen doi Hohmann:")
print("-" * 55)

destinations = [
    ('Sao Hoa', 1.524),
    ('Sao Kim', 0.723),
    ('Sao Moc', 5.203),
]

for name, r_au in destinations:
    phase = hohmann_phase_angle(1.0, r_au)
    print(f"  {name:10} Goc pha: {phase:+.1f}° "
          f"({'truoc' if phase > 0 else 'sau'})")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Nham lan xung doi voi phong.** Xung doi la tiep can gan nhat, khong phai thoi diem phong toi uu.

2. **Bo qua do rong cua so.** Som/muon trong cua so can nhieu nhien lieu hon; lap ke hoach du tru.

3. **Hang ngay vs chu ky.** Chu ky hoi hop xac dinh thang nao; quay hang ngay xac dinh gio nao.

4. **Quy dao khong phai Hohmann.** Quy dao nhanh hon co goc pha khac, cua so khac.

5. **Do phuc tap ho tro hap dan.** Quy dao nhieu bay ngang co cua so rang buoc hon nhieu.

6. **Han che dia diem phong.** Mot so dia diem khong the dat duoc moi do nghieng quy dao.

7. **Thoi tiet va tre.** Phong thuc te bi lui; can nhieu lan thu trong cua so.
