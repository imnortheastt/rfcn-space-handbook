## Tai sao dieu nay quan trong

Cac nha thiet ke nhiem vu khong chi chon ngay phong tu lich. Ho giai hang ngan bai toan Lambert qua luoi ngay xuat phat va den, sau do ve ket qua. Cac duong dang ket qua tron giong nhu mieng thit lon—do do co ten. Cac bieu do nay cho thay ngay nao cung cap chuyen doi nang luong thap, chuyen di mat bao lau, va dieu gi xay ra neu ban lo cua so.

## Ap dung

**Tao porkchop don gian:**

```python
import numpy as np

# Hang so
MU_SUN = 1.327e11  # km³/s²
AU = 149.6e6  # km

def planet_position(t_days, a_au, period_days, lon0=0):
    """
    Vi tri quy dao tron don gian.
    Lap ke hoach nhiem vu thuc dung lich thien van (SPICE/JPL).
    """
    mean_motion = 2 * np.pi / period_days
    lon = lon0 + mean_motion * t_days
    r = a_au * AU
    return np.array([r * np.cos(lon), r * np.sin(lon)])

def lambert_c3_estimate(r1, r2, tof_days):
    """
    Uoc tinh C3 don gian dung vis-viva.
    Bo giai Lambert thuc phuc tap hon.
    """
    r1_mag = np.linalg.norm(r1)
    r2_mag = np.linalg.norm(r2)
    
    # Ban truc lon chuyen doi (don gian)
    c = np.linalg.norm(r2 - r1)  # day cung
    s = (r1_mag + r2_mag + c) / 2  # nua chu vi
    
    # Xap xi ban truc lon chuyen doi
    tof_sec = tof_days * 86400
    a_transfer = (s / 2) * (1 + 0.1 * (tof_sec / 1e7)**0.5)  # xap xi tho
    
    # Van toc xuat phat (vis-viva)
    v_transfer = np.sqrt(MU_SUN * (2/r1_mag - 1/a_transfer))
    
    # Van toc Trai Dat
    v_earth = np.sqrt(MU_SUN / r1_mag)
    
    # C3 (don gian: gia dinh xuat phat tiep tuyen)
    delta_v = abs(v_transfer - v_earth)
    c3 = delta_v ** 2
    
    return c3

def generate_porkchop_data(launch_start, launch_span, arrival_start, arrival_span,
                           resolution=10):
    """
    Tao luoi C3 cho bieu do porkchop.
    """
    launch_days = np.arange(0, launch_span, resolution)
    arrival_days = np.arange(0, arrival_span, resolution)
    
    c3_grid = np.zeros((len(arrival_days), len(launch_days)))
    
    for i, arr_d in enumerate(arrival_days):
        for j, lau_d in enumerate(launch_days):
            tof = (arrival_start + arr_d) - (launch_start + lau_d)
            
            if tof < 100 or tof > 400:  # Pham vi TOF hop le cho Sao Hoa
                c3_grid[i, j] = np.nan
                continue
            
            # Lay vi tri
            r_earth = planet_position(launch_start + lau_d, 1.0, 365.25)
            r_mars = planet_position(arrival_start + arr_d, 1.524, 687, lon0=np.pi/4)
            
            # Tinh C3
            c3_grid[i, j] = lambert_c3_estimate(r_earth, r_mars, tof)
    
    return launch_days, arrival_days, c3_grid

# Vi du: porkchop khai niem Trai Dat-Sao Hoa
print("Tao Du lieu Bieu do Porkchop:")
print("=" * 50)

# Trinh dien don gian
launch_days, arrival_days, c3 = generate_porkchop_data(
    launch_start=0, launch_span=180,
    arrival_start=150, arrival_span=180,
    resolution=20
)

# Tim toi thieu
valid_c3 = c3[~np.isnan(c3)]
if len(valid_c3) > 0:
    min_c3 = np.nanmin(c3)
    min_idx = np.where(c3 == min_c3)
    print(f"C3 toi thieu: {min_c3:.1f} km²/s²")
    print(f"Tai ngay phong ~{launch_days[min_idx[1][0]]}")
    print(f"Tai ngay den ~{arrival_days[min_idx[0][0]]}")
    print(f"TOF: ~{arrival_days[min_idx[0][0]] + 150 - launch_days[min_idx[1][0]]:.0f} ngay")

print("\nLuu y: Bieu do porkchop thuc dung bo giai Lambert day du")
print("va du lieu lich thien van hanh tinh chinh xac.")
```

## Cam bay

1. **C3 khong phai tong Δv.** C3 chi bao gom xuat phat; Δv den la rieng.

2. **Gia dinh quy dao tron.** Hanh tinh thuc co quy dao ellipse; do lech tam quan trong.

3. **Doi mat phang.** Bieu do 2D bo qua chi phi thay doi do nghieng.

4. **Nhieu loi giai.** Bai toan Lambert co the co nhieu loi giai; bieu do chi hien thi mot nhanh.

5. **Khong phai tat ca diem kha thi.** C3 thap khong dam bao nhiem vu thanh cong—xem xet V∞ den.

6. **Chu ky hoi hop.** Co hoi lap lai, nhung khong giong het (quy dao khong hoan toan tron/dong phang).

7. **Don gian hoa.** Thiet ke nhiem vu thuc bao gom nhieu rang buoc hon (thong tin lien lac, nang luong, nhiet).
