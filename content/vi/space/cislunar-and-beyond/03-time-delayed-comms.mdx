## Tai sao dieu nay quan trong

Tai Mat Trang, tin hieu vo tuyen mat 1.3 giay de den Trai dat. Hoi mot cau, doi 2.6 giay de co cau tra loi. Tai Sao Hoa, do tro thanh 3-22 phut mot chieu. Dieu khien nhiem vu thoi gian thuc truyen thong khong hoat dong—ban khong the dieu khien xe tu hanh qua chuong ngai khi phan hoi den sau vai phut. Thong tin co do tre thay doi can ban cach nhiem vu van hanh, cach giao thuc duoc thiet ke, va muc do tu chu ma tau vu tru phai co. Hieu cac rang buoc nay la thiet yeu cho bat ky ai xay dung he thong cislunar hoac khong gian sau.

## Ap dung

**Tinh toan thoi gian anh sang:**

```python
import numpy as np

# Toc do anh sang
C_KM_S = 299792.458  # km/s

def thoi_gian_anh_sang_giay(khoang_cach_km):
    """Tinh thoi gian anh sang mot chieu bang giay."""
    return khoang_cach_km / C_KM_S

def dinh_dang_thoi_gian(giay):
    """Dinh dang thoi gian de doc."""
    if giay < 1:
        return f"{giay * 1000:.1f} ms"
    elif giay < 60:
        return f"{giay:.2f} giay"
    elif giay < 3600:
        return f"{giay / 60:.1f} phut"
    else:
        return f"{giay / 3600:.1f} gio"

# Dinh nghia cac diem den
diem_den = {
    "ISS (LEO)": 400,
    "Ve tinh GEO": 36000,
    "Mat Trang (tb)": 384400,
    "Mat Trang (can)": 356500,
    "Mat Trang (vien)": 406700,
    "Sao Hoa (gan)": 56e6,
    "Sao Hoa (tb)": 225e6,
    "Sao Hoa (xa)": 400e6,
    "Sao Moc (tb)": 630e6,
    "Sao Tho (tb)": 1.3e9,
    "Sao Hai Vuong (tb)": 4.4e9,
    "Voyager 1 (2024)": 24e9,
}

print("Phan tich Do tre Thoi gian Anh sang")
print("=" * 60)

print("\n1. Thoi gian Anh sang Mot chieu va Khu hoi")
print("-" * 60)

print(f"  {'Diem den':<22} {'K.cach':<14} {'Mot chieu':<12} {'RTT':<12}")
print("-" * 60)

for ten, kc_km in diem_den.items():
    owlt = thoi_gian_anh_sang_giay(kc_km)
    rtt = 2 * owlt
    kc_str = f"{kc_km:,.0f} km" if kc_km < 1e6 else f"{kc_km/1e6:.0f}Tr km"
    print(f"  {ten:<22} {kc_str:<14} {dinh_dang_thoi_gian(owlt):<12} {dinh_dang_thoi_gian(rtt):<12}")

# Ham y van hanh
print("\n2. Che do Van hanh theo Do tre")
print("-" * 60)

che_do = [
    (0.5, "Dieu khien thoi gian thuc", "ISS, GEO"),
    (3, "Tu chu co giam sat", "Mat Trang"),
    (30*60, "Tu chu hoan toan", "Sao Hoa"),
    (3600, "Tu chu sau", "Hanh tinh ngoai"),
]

print(f"  {'Nguong RTT':<16} {'Che do':<24} {'Vi du':<20}")
print("-" * 60)

for nguong, che_do, vi_du in che_do:
    print(f"  < {dinh_dang_thoi_gian(nguong):<13} {che_do:<24} {vi_du:<20}")

# Phan tich kha thi TCP
print("\n3. Kha thi TCP theo Lien ket")
print("-" * 60)

tcp_rto_mac_dinh = 3  # giay (mac dinh dien hinh)
tcp_rto_toi_da = 120  # giay (toi da co the chinh)

print(f"  RTO TCP mac dinh: {tcp_rto_mac_dinh} giay")
print(f"  RTO toi da co the chinh: {tcp_rto_toi_da} giay")
print()

for ten, kc_km in [("Mat Trang", 384400), ("Sao Hoa gan", 56e6), ("Sao Hoa xa", 400e6)]:
    rtt = 2 * thoi_gian_anh_sang_giay(kc_km)
    hoat_dong_md = "Co" if rtt < tcp_rto_mac_dinh else "Khong"
    hoat_dong_chinh = "Co" if rtt < tcp_rto_toi_da else "Khong"
    print(f"  {ten:<15} RTT={dinh_dang_thoi_gian(rtt):<10} M.dinh:{hoat_dong_md:<6} Chinh:{hoat_dong_chinh}")

# Dong thoi gian lenh-phan hoi
print("\n4. Dong thoi gian Vong lap Lenh EVA Mat Trang")
print("-" * 60)

su_kien = [
    (0.0, "Phi hanh gia truyen tin nhan"),
    (1.3, "Houston nhan tin nhan"),
    (1.3, "Houston xu ly (phan ung nguoi ~2-5 giay)"),
    (4.0, "Houston truyen phan hoi"),
    (5.3, "Phi hanh gia nhan phan hoi"),
]

print("  Dong thoi gian cho mot trao doi giong noi:")
print()
for t, sk in su_kien:
    print(f"  T+{t:.1f}s: {sk}")

print(f"\n  Tong thoi gian vong lap: ~5-6 giay")
print(f"  So voi: do tre goi dien thoai ~0.1-0.3 giay")

# Ham y truyen du lieu
print("\n5. Thuc te Truyen File o Khoang cach")
print("-" * 60)

kich_thuoc_file_mb = 100  # File 100 MB
toc_do_lien_ket_mbps = 10  # Lien ket 10 Mbps

thoi_gian_truyen = kich_thuoc_file_mb * 8 / toc_do_lien_ket_mbps  # giay

for ten, kc_km in [("Mat Trang", 384400), ("Sao Hoa tb", 225e6)]:
    owlt = thoi_gian_anh_sang_giay(kc_km)
    tong_thoi_gian = thoi_gian_truyen + owlt
    hieu_suat = thoi_gian_truyen / tong_thoi_gian * 100
    
    print(f"  {ten}:")
    print(f"    Thoi gian anh sang: {dinh_dang_thoi_gian(owlt)}")
    print(f"    Thoi gian truyen (100 MB @ 10 Mbps): {thoi_gian_truyen:.0f} giay")
    print(f"    Byte dau den sau: {dinh_dang_thoi_gian(owlt)}")
    print(f"    Byte cuoi den sau: {dinh_dang_thoi_gian(tong_thoi_gian)}")
    print()
```

<section class="lesson-pitfalls">

## Cam bay

1. **Dung TCP khong chinh.** Timeout mac dinh gay gui lai gia. Dung DTN hoac TCP da chinh ky.

2. **Thiet ke cho dieu khien thoi gian thuc tai Mat Trang.** 2.6 giay RTT co nghia phi hanh gia can quyen huy tu dong.

3. **Bo qua khoang cach Sao Hoa thay doi.** Do tre Sao Hoa tu 3-22 phut mot chieu. Thiet ke cho truong hop xau nhat.

4. **Quen lan truyen qua chuyen tiep.** Moi buoc nhay them do tre. Gateway → chuyen tiep → Trai dat them 1+ giay.

5. **Giao thuc giong noi hoi thoai.** Trao doi "Nhan" lang phi thoi gian. Dung luot co cau truc.

6. **Go loi tuong tac.** Khong the buoc qua code tai Sao Hoa. Can ghi log mo rong + xu ly loi tu dong.
