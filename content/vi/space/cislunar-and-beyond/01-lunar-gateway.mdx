## Tai sao dieu nay quan trong

Lunar Gateway la tram khong gian dau tien cua nhan loai vuot qua quy dao Trai dat thap—mot tien don thuong truc quay quanh Mat Trang se phuc vu nhu diem dung cho cac nhiem vu be mat Mat Trang va cuoi cung la Sao Hoa. Cho ky su RF, Gateway trinh bay cac thach thuc doc dao: quy dao Halo Gan Thang thay doi tam nhin Trai dat lien tuc, ngan sach lien ket trai dai 400,000 km, va kien truc thong tin phai ho tro van hanh co nguoi, robot, va tai trong khoa hoc. Hieu thiet ke Gateway cho thay thong tin cislunar khac voi cac he thong GEO/LEO quen thuoc nhu the nao.

## Ap dung

**Uoc luong ngan sach lien ket Gateway:**

```python
import numpy as np

def fspl_db(khoang_cach_km, tan_so_ghz):
    """Ton hao Duong truyen Khong gian Tu do tinh dB."""
    return 20 * np.log10(khoang_cach_km) + 20 * np.log10(tan_so_ghz) + 92.45

def do_loi_parabol_dbi(duong_kinh_m, tan_so_ghz, hieu_suat=0.55):
    """Do loi anten parabol tinh dBi."""
    buoc_song_m = 0.3 / tan_so_ghz
    do_loi_tuyen_tinh = hieu_suat * (np.pi * duong_kinh_m / buoc_song_m) ** 2
    return 10 * np.log10(do_loi_tuyen_tinh)

def ngan_sach_lien_ket_cn0(eirp_dbw, khoang_cach_km, tan_so_ghz, g_tren_t_dbk, ton_hao_khac_db=0):
    """Tinh C/N0 cho lien ket khong gian."""
    k_dbw = -228.6
    ton_hao_dt = fspl_db(khoang_cach_km, tan_so_ghz)
    cn0 = eirp_dbw - ton_hao_dt - ton_hao_khac_db + g_tren_t_dbk - k_dbw
    return cn0, ton_hao_dt

# Tham so Gateway
print("Phan tich Ngan sach Lien ket Lunar Gateway")
print("=" * 60)

# Khoang cach
trai_dat_mat_trang_tb = 384400  # km
nrho_can_diem = 3000  # km tu Mat Trang
nrho_vien_diem = 70000  # km tu Mat Trang

# Khoang cach Gateway tu Trai dat (xap xi)
gateway_min = trai_dat_mat_trang_tb - nrho_vien_diem
gateway_max = trai_dat_mat_trang_tb + nrho_vien_diem
gateway_tb = trai_dat_mat_trang_tb

# Bang tan so
ka_band = 32  # GHz (lien ket nguoc)

# Tham so DSN
dsn_34m_do_loi = do_loi_parabol_dbi(34, ka_band, 0.60)
dsn_70m_do_loi = do_loi_parabol_dbi(70, ka_band, 0.55)
dsn_g_tren_t = 52  # dB/K

# Tham so Gateway
gateway_dia = 2.0  # m
gateway_do_loi = do_loi_parabol_dbi(gateway_dia, ka_band)
gateway_cong_suat_phat = 20  # W
gateway_cong_suat_phat_dbw = 10 * np.log10(gateway_cong_suat_phat)
gateway_eirp = gateway_cong_suat_phat_dbw + gateway_do_loi

print("\n1. So sanh Khoang cach")
print("-" * 60)

khoang_cach = [
    ("ISS (LEO)", 400),
    ("Ve tinh GEO", 36000),
    ("Gateway (min)", gateway_min),
    ("Gateway (tb)", gateway_tb),
    ("Gateway (max)", gateway_max),
]

print(f"  {'Lien ket':<20} {'K.cach (km)':<15} {'FSPL @ 32 GHz':<15}")
print("-" * 60)

for ten, kc in khoang_cach:
    fspl = fspl_db(kc, ka_band)
    print(f"  {ten:<20} {kc:<15,} {fspl:.1f} dB")

# Tham so phat Gateway
print("\n2. He thong Phat Gateway")
print("-" * 60)

print(f"  Duong kinh dia:    {gateway_dia} m")
print(f"  Tan so:            {ka_band} GHz")
print(f"  Do loi anten:      {gateway_do_loi:.1f} dBi")
print(f"  Cong suat phat:    {gateway_cong_suat_phat} W ({gateway_cong_suat_phat_dbw:.1f} dBW)")
print(f"  EIRP:              {gateway_eirp:.1f} dBW")

# Tham so thu DSN
print("\n3. Tram mat dat DSN")
print("-" * 60)

print(f"  Anten BWG 34m:")
print(f"    Do loi @ {ka_band} GHz: {dsn_34m_do_loi:.1f} dBi")
print(f"    G/T:            {dsn_g_tren_t} dB/K")
print(f"")
print(f"  Anten 70m:")
print(f"    Do loi @ {ka_band} GHz: {dsn_70m_do_loi:.1f} dBi")

# Ngan sach lien ket nguoc (Gateway → Trai dat)
print("\n4. Ngan sach Lien ket Nguoc (Gateway → DSN)")
print("-" * 60)

ton_hao_khac = 2.0  # dB

for ten, kc in [("K.cach min", gateway_min), ("K.cach tb", gateway_tb), ("K.cach max", gateway_max)]:
    cn0, fspl = ngan_sach_lien_ket_cn0(gateway_eirp, kc, ka_band, dsn_g_tren_t, ton_hao_khac)
    print(f"  {ten}: C/N₀ = {cn0:.1f} dBHz (FSPL = {fspl:.1f} dB)")

# Kha nang toc do du lieu
print("\n5. Toc do Du lieu Dat duoc")
print("-" * 60)

ebn0_can = 3.0  # dB (QPSK ty le 1/2)

for ten, kc in [("K.cach min", gateway_min), ("K.cach max", gateway_max)]:
    cn0, _ = ngan_sach_lien_ket_cn0(gateway_eirp, kc, ka_band, dsn_g_tren_t, ton_hao_khac)
    rb_bps = 10 ** ((cn0 - ebn0_can) / 10)
    rb_mbps = rb_bps / 1e6
    print(f"  {ten}: Toi da {rb_mbps:.0f} Mbps")

# So sanh voi GEO
print("\n6. So sanh voi GEO Thuong mai")
print("-" * 60)

geo_kc = 36000
geo_eirp = 52  # dBW
geo_gt = 27  # dB/K

cn0_gateway, _ = ngan_sach_lien_ket_cn0(gateway_eirp, gateway_tb, ka_band, dsn_g_tren_t, ton_hao_khac)
cn0_geo, _ = ngan_sach_lien_ket_cn0(geo_eirp, geo_kc, ka_band, geo_gt, 1.0)

print(f"  GEO thuong mai: C/N₀ = {cn0_geo:.1f} dBHz")
print(f"  Gateway:        C/N₀ = {cn0_gateway:.1f} dBHz")
print(f"  Chenh lech:     {cn0_gateway - cn0_geo:.1f} dB")
print(f"")
print(f"  Gateway hoat dong vi DSN G/T ({dsn_g_tren_t} dB/K)")
print(f"  cao hon nhieu so voi thuong mai ({geo_gt} dB/K).")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Gia dinh khoang cach khong doi.** NRHO thay doi tu 314,000 den 454,000 km tu Trai dat. Lap ngan sach cho truong hop xau nhat.

2. **Bo qua che khuat Mat Trang.** ~14% quy dao khong co duong nhin Trai dat. Lap ke hoach cho luu va chuyen tiep.

3. **Ky vong toc do du lieu GEO.** 21 dB ton hao duong truyen them dong nghia thong luong thap hon nhieu neu khong co DSN.

4. **Quen do tre anh sang.** Vong 1.0-1.5 giay dong nghia khong co dieu khien thoi gian thuc.

5. **Tu duy don chuoi.** Gateway can S-band du phong; Ka-band co the hong.

6. **Nham lan chu ky quy dao.** Quy dao 6.5 ngay dong nghia lap lich khac voi GEO hoac LEO.
