## Tai sao dieu nay quan trong

Cac chom sao bang rong LEO nhu Starlink hoat dong tai Ka-band tu do cao 550 km. Huong dan nay xay dung ngan sach duong xuong hoan chinh tung dong, cho thay moi tham so dong gop nhu the nao vao C/N₀ cuoi cung va tai sao LEO tai Ka-band co the cung cap hang tram Mbps den anten phang nho.

## Ap dung

**Tinh toan ngan sach lien ket hoan chinh:**

```python
import numpy as np

def leo_ka_link_budget():
    """
    Ngan sach duong xuong LEO Ka-band hoan chinh (lop Starlink).
    """
    # Hinh hoc
    altitude_km = 550
    elevation_deg = 40
    earth_radius_km = 6371
    
    # Tinh tam nghieng
    el_rad = np.radians(elevation_deg)
    r_sat = earth_radius_km + altitude_km
    slant_range_km = np.sqrt(
        r_sat**2 - (earth_radius_km * np.cos(el_rad))**2
    ) - earth_radius_km * np.sin(el_rad)
    
    # Tan so
    freq_ghz = 19.7
    wavelength_m = 0.3 / freq_ghz  # c/f
    
    # Ton hao khong gian tu do
    fspl_db = 92.45 + 20 * np.log10(freq_ghz * slant_range_km)
    
    # Tham so ve tinh
    p_tx_dbw = 10.0  # 10W moi bup
    g_tx_dbi = 38.0  # khuech dai bup spot
    eirp_dbw = p_tx_dbw + g_tx_dbi
    
    # Ton hao them
    atm_loss_db = 0.5  # khi quyen tai 40° el
    pointing_sat_db = 0.5
    pointing_user_db = 0.5
    pol_loss_db = 0.3
    impl_loss_db = 0.5
    total_add_loss_db = (atm_loss_db + pointing_sat_db + 
                         pointing_user_db + pol_loss_db + impl_loss_db)
    
    # Thiet bi dau cuoi
    g_rx_dbi = 34.0  # mang pha ~50 cm tai Ka
    t_sys_k = 150.0
    g_t_dbk = g_rx_dbi - 10 * np.log10(t_sys_k)
    
    # Hang so Boltzmann
    k_dbw = -228.6
    
    # Ngan sach lien ket
    total_loss_db = fspl_db + total_add_loss_db
    c_n0_dbhz = eirp_dbw - total_loss_db + g_t_dbk - k_dbw
    
    # Dung luong cho kenh 250 MHz
    bandwidth_hz = 250e6
    c_n_db = c_n0_dbhz - 10 * np.log10(bandwidth_hz)
    snr_linear = 10 ** (c_n_db / 10)
    shannon_bps = bandwidth_hz * np.log2(1 + snr_linear)
    practical_bps = 0.8 * shannon_bps  # ~80% voi ma tot
    
    # In ket qua
    print("Ngan sach Lien ket Duong xuong LEO Ka-band")
    print("=" * 60)
    print(f"\n--- HINH HOC ---")
    print(f"Do cao:          {altitude_km} km")
    print(f"Goc nang:        {elevation_deg}°")
    print(f"Tam nghieng:     {slant_range_km:.1f} km")
    
    print(f"\n--- PHAT (Ve tinh) ---")
    print(f"Cong suat phat:  {p_tx_dbw:.1f} dBW")
    print(f"Khuech dai anten phat: {g_tx_dbi:.1f} dBi")
    print(f"EIRP:            {eirp_dbw:.1f} dBW")
    
    print(f"\n--- TON HAO DUONG TRUYEN ---")
    print(f"Tan so:          {freq_ghz} GHz")
    print(f"Ton hao KG tu do: {fspl_db:.1f} dB")
    print(f"Ton hao them:    {total_add_loss_db:.1f} dB")
    print(f"Tong ton hao:    {total_loss_db:.1f} dB")
    
    print(f"\n--- THU (Thiet bi Dau cuoi) ---")
    print(f"Khuech dai anten thu: {g_rx_dbi:.1f} dBi")
    print(f"Nhiet do he thong: {t_sys_k:.0f} K")
    print(f"G/T:             {g_t_dbk:.1f} dB/K")
    
    print(f"\n--- KET QUA ---")
    print(f"C/N₀:            {c_n0_dbhz:.1f} dBHz")
    print(f"C/N (250 MHz):   {c_n_db:.1f} dB")
    print(f"Gioi han Shannon: {shannon_bps/1e9:.2f} Gbps")
    print(f"Toc do thuc te:  {practical_bps/1e9:.2f} Gbps")
    
    # Phan tich du tru
    print(f"\n--- PHAN TICH DU TRU ---")
    required_cn_db = 15.0  # dien hinh cho 32APSK
    margin_db = c_n_db - required_cn_db
    print(f"C/N yeu cau:     {required_cn_db:.1f} dB (32APSK)")
    print(f"Du tru:          {margin_db:.1f} dB")
    
    return c_n0_dbhz, c_n_db, practical_bps

# Chay ngan sach
c_n0, c_n, rate = leo_ka_link_budget()

# Anh huong suy hao mua
print("\n" + "=" * 60)
print("ANH HUONG SUY HAO MUA (Ka-band)")
print("-" * 60)
rain_fades = [0, 3, 6, 10]  # dB
for rain in rain_fades:
    c_n_rain = c_n - rain
    snr = 10 ** (c_n_rain / 10)
    rate_rain = 0.8 * 250e6 * np.log2(1 + snr)
    print(f"Suy mua {rain:2d} dB: C/N = {c_n_rain:.1f} dB, "
          f"Toc do = {rate_rain/1e6:.0f} Mbps")
```

## Cam bay

1. **Dung do cao thay vi tam nghieng.** Tai goc nang thap, tam nghieng >> do cao.

2. **Bo qua mua tai Ka-band.** Suy 10+ dB co the xay ra; phai tinh du tru.

3. **Gia dinh goc nang tinh.** Ve tinh LEO di chuyen; goc nang thay doi trong qua.

4. **Quen Doppler.** Dich ±45 kHz tai Ka-band tu chuyen dong LEO.

5. **G/T tai goc nang thap.** Nhiet do bau troi tang khi nhin gan chan troi.

6. **Phat bien bup.** Nguoi dung o bien bup spot thay EIRP thap hon 2-3 dB.

7. **Dung luong chia se.** Dung luong moi bup chia cho tat ca nguoi dung trong bup do.
