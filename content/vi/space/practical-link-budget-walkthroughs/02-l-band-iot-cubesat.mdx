## Tai sao dieu nay quan trong

IoT ve tinh ket noi cam bien o vung xa xoi—phao dai duong, thiet bi theo doi dong vat hoang da, giam sat nong nghiep—noi mang mat dat khong co. Thach thuc: thiet bi mat dat chay pin voi milliwatt cong suat phai dong lien ket den CubeSat cach 600 km. Huong dan nay cho thay cach L-band va dang song bang hep lam dieu do kha thi.

## Ap dung

**Tinh toan ngan sach lien ket IoT hoan chinh:**

```python
import numpy as np

def iot_cubesat_link_budget():
    """
    Ngan sach duong len IoT L-band (cam bien → CubeSat).
    """
    # Hinh hoc
    altitude_km = 600
    elevation_deg = 45
    earth_radius_km = 6371
    
    el_rad = np.radians(elevation_deg)
    r_sat = earth_radius_km + altitude_km
    slant_range_km = np.sqrt(
        r_sat**2 - (earth_radius_km * np.cos(el_rad))**2
    ) - earth_radius_km * np.sin(el_rad)
    
    # Tan so
    freq_mhz = 401
    freq_ghz = freq_mhz / 1000
    
    # Ton hao khong gian tu do
    fspl_db = 92.45 + 20 * np.log10(freq_ghz * slant_range_km)
    
    # Cam bien IoT (phat)
    p_tx_mw = 100
    p_tx_dbw = 10 * np.log10(p_tx_mw / 1000)  # -10 dBW
    g_tx_dbi = 0.0  # anten roi
    cable_loss_db = 1.0
    eirp_dbw = p_tx_dbw + g_tx_dbi - cable_loss_db
    
    # Ton hao them
    atm_loss_db = 0.5
    iono_loss_db = 1.0
    pol_loss_db = 0.5
    total_add_loss_db = atm_loss_db + iono_loss_db + pol_loss_db
    
    # May thu CubeSat
    g_rx_dbi = 6.0  # helix trien khai
    t_sys_k = 500.0  # nhin Trai dat am
    g_t_dbk = g_rx_dbi - 10 * np.log10(t_sys_k)
    
    # Ngan sach lien ket
    total_loss_db = fspl_db + total_add_loss_db
    k_dbw = -228.6
    c_n0_dbhz = eirp_dbw - total_loss_db + g_t_dbk - k_dbw
    
    print("Ngan sach Lien ket Duong len IoT L-band")
    print("=" * 60)
    print(f"\n--- HINH HOC ---")
    print(f"Do cao:          {altitude_km} km")
    print(f"Goc nang:        {elevation_deg}°")
    print(f"Tam nghieng:     {slant_range_km:.1f} km")
    
    print(f"\n--- PHAT (Cam bien IoT) ---")
    print(f"Cong suat phat:  {p_tx_mw} mW = {p_tx_dbw:.1f} dBW")
    print(f"Khuech dai anten: {g_tx_dbi:.1f} dBi (roi)")
    print(f"Ton hao cap:     -{cable_loss_db} dB")
    print(f"EIRP:            {eirp_dbw:.1f} dBW")
    
    print(f"\n--- DUONG TRUYEN ---")
    print(f"Tan so:          {freq_mhz} MHz")
    print(f"Ton hao KG tu do: {fspl_db:.1f} dB")
    print(f"Ton hao them:    {total_add_loss_db:.1f} dB")
    print(f"Tong ton hao:    {total_loss_db:.1f} dB")
    
    print(f"\n--- THU (CubeSat) ---")
    print(f"Khuech dai anten: {g_rx_dbi:.1f} dBi")
    print(f"Nhiet do HT:     {t_sys_k:.0f} K")
    print(f"G/T:             {g_t_dbk:.1f} dB/K")
    
    print(f"\n--- KET QUA ---")
    print(f"C/N₀:            {c_n0_dbhz:.1f} dBHz")
    
    # Cac tuy chon toc do du lieu
    print(f"\n--- PHAN TICH TOC DO DU LIEU ---")
    data_rates = [100, 400, 1200, 9600]  # bps
    for rate in data_rates:
        bw_hz = 1.2 * rate
        c_n_db = c_n0_dbhz - 10 * np.log10(bw_hz)
        required_cn = 10
        margin = c_n_db - required_cn
        status = "✓" if margin > 3 else "⚠" if margin > 0 else "✗"
        print(f"  {rate:5d} bps: C/N = {c_n_db:.1f} dB, "
              f"du tru = {margin:.1f} dB {status}")
    
    # Thoi gian tin nhan
    print(f"\n--- THOI GIAN TIN NHAN ---")
    msg_bytes = 24
    msg_bits = msg_bytes * 8
    best_rate = 400
    tx_time = msg_bits / best_rate
    print(f"Kich thuoc TN:   {msg_bytes} byte ({msg_bits} bit)")
    print(f"Tai {best_rate} bps:      {tx_time:.2f} giay phat")
    
    # Cua so lien lac
    pass_time_min = 10
    messages_per_pass = int(pass_time_min * 60 / (tx_time + 1))
    print(f"Thoi gian qua:   ~{pass_time_min} phut")
    print(f"Tin nhan/qua:    ~{messages_per_pass} (voi thoi gian bao ve)")
    
    return c_n0_dbhz

c_n0 = iot_cubesat_link_budget()

# Danh doi cong suat vs tam
print("\n" + "=" * 60)
print("DANH DOI CONG SUAT vs TAM")
print("-" * 60)
powers_mw = [50, 100, 250, 500, 1000]
for p in powers_mw:
    eirp = 10 * np.log10(p/1000) + 0 - 1
    c_n0_var = eirp - 147.6 + (-21) + 228.6
    margin_400bps = c_n0_var - 27 - 10
    print(f"  {p:4d} mW: EIRP = {eirp:.1f} dBW, "
          f"C/N₀ = {c_n0_var:.1f} dBHz, "
          f"du tru@400bps = {margin_400bps:.1f} dB")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Bat ngo G/T CubeSat.** -20 dB/K la pho bien—dung gia dinh hieu suat tram mat dat.

2. **Doppler tai LEO.** ±10 kHz tai L-band tu chuyen dong CubeSat; may thu phai theo doi.

3. **Cua so lien lac ngan.** ~10 phut qua, khong phai luc nao cung tren dau—len ke hoach thoi gian tin nhan.

4. **Anh huong tang dien ly.** Quay Faraday, nhap nhay—te hon o tan so thap.

5. **Rang buoc pho.** Chi co phan bo hep cho sat-IoT; can phoi hop.

6. **Tuoi tho pin.** 100 mW × 1 giay = 100 mJ moi tin nhan; nhan voi tin nhan/ngay.

7. **Xac suat va cham.** Nhieu cam bien, mot ve tinh—giao thuc MAC thiet yeu.
