## Tai sao dieu nay quan trong

Truyen thong khong gian sau day ky thuat RF den gioi han. Tai khoang cach Sao Hoa (225 trieu km), may phat 20W tren tau vu tru tao ra tin hieu den Trai dat yeu hon san nhieu nhiet—nhung chung ta van giai ma duoc. Huong dan nay cho thay cach dia 70 met, bo khuech dai lanh dong, va ma hoa cuc manh lam dieu do kha thi.

## Ap dung

**Ngan sach lien ket khong gian sau hoan chinh:**

```python
import numpy as np

def deep_space_link_budget():
    """
    Ngan sach lien ket duong xuong X-band khong gian sau (khoang cach Sao Hoa).
    """
    # Khoang cach
    distance_au = 1.5  # Trung binh Trai dat-Sao Hoa
    distance_km = distance_au * 149597870.7  # km moi AU
    light_time_s = distance_km / 299792.458
    light_time_min = light_time_s / 60
    
    # Tan so duong xuong
    freq_ghz_dl = 8.4  # Duong xuong X-band
    
    # Ton hao khong gian tu do
    fspl_dl = 92.45 + 20 * np.log10(freq_ghz_dl * distance_km)
    
    # Tau vu tru (phat)
    p_tx_w = 35
    p_tx_dbw = 10 * np.log10(p_tx_w)
    g_tx_dbi = 40.0  # HGA 2.5m
    circuit_loss = 0.5
    pointing_loss_sc = 0.5
    eirp_dbw = p_tx_dbw + g_tx_dbi - circuit_loss - pointing_loss_sc
    
    # Ton hao duong truyen
    atm_loss = 0.1
    total_loss = fspl_dl + atm_loss
    
    # DSN 70m (thu)
    g_rx_dbi = 74.0
    efficiency_loss = 0.5
    pointing_loss_dsn = 0.1
    t_sys_k = 25.0  # Maser lanh dong
    g_t_dbk = g_rx_dbi - efficiency_loss - pointing_loss_dsn - 10 * np.log10(t_sys_k)
    
    # Ngan sach lien ket
    k_dbw = -228.6
    c_n0_dbhz = eirp_dbw - total_loss + g_t_dbk - k_dbw
    
    print("Ngan sach Duong xuong X-band Khong gian Sau")
    print("=" * 65)
    print(f"\n--- HINH HOC ---")
    print(f"Khoang cach:     {distance_au} AU = {distance_km/1e6:.1f} trieu km")
    print(f"Thoi gian AS:    {light_time_min:.1f} phut mot chieu")
    
    print(f"\n--- PHAT (Tau vu tru) ---")
    print(f"Cong suat phat:  {p_tx_w} W = {p_tx_dbw:.1f} dBW")
    print(f"Khuech dai anten: {g_tx_dbi:.1f} dBi (HGA 2.5m)")
    print(f"Ton hao:         {circuit_loss + pointing_loss_sc:.1f} dB")
    print(f"EIRP:            {eirp_dbw:.1f} dBW")
    
    print(f"\n--- DUONG TRUYEN ---")
    print(f"Tan so:          {freq_ghz_dl} GHz")
    print(f"Ton hao KG tu do: {fspl_dl:.1f} dB")
    print(f"Khi quyen:       {atm_loss} dB")
    print(f"Tong ton hao:    {total_loss:.1f} dB")
    
    print(f"\n--- THU (DSN 70m) ---")
    print(f"Khuech dai anten: {g_rx_dbi:.1f} dBi")
    print(f"Nhiet do HT:     {t_sys_k:.0f} K (lanh dong)")
    print(f"G/T:             {g_t_dbk:.1f} dB/K")
    
    print(f"\n--- KET QUA ---")
    print(f"C/N₀:            {c_n0_dbhz:.1f} dBHz")
    
    # Tinh toan toc do du lieu
    print(f"\n--- PHAN TICH TOC DO DU LIEU ---")
    eb_n0_required = {
        'BPSK khong ma': 10.5,
        'Chap (r=1/2)': 5.0,
        'Ma turbo (r=1/3)': 1.0,
        'LDPC (r=1/2)': 1.5
    }
    
    for coding, eb_n0 in eb_n0_required.items():
        max_rate = 10 ** ((c_n0_dbhz - eb_n0) / 10)
        print(f"  {coding:25} Eb/N₀={eb_n0} dB → {max_rate/1000:.1f} kbps")
    
    # Do nhay khoang cach
    print(f"\n--- DO NHAY KHOANG CACH ---")
    distances_au = [0.5, 1.0, 1.5, 2.0, 2.5]
    for d_au in distances_au:
        d_km = d_au * 149597870.7
        fspl = 92.45 + 20 * np.log10(freq_ghz_dl * d_km)
        c_n0 = eirp_dbw - fspl - atm_loss + g_t_dbk - k_dbw
        rate = 10 ** ((c_n0 - 1.0) / 10)  # Ma turbo
        print(f"  {d_au:.1f} AU: Ton hao DT = {fspl:.1f} dB, "
              f"C/N₀ = {c_n0:.1f} dBHz, Toc do = {rate/1000:.1f} kbps")
    
    return c_n0_dbhz

c_n0 = deep_space_link_budget()

# So sanh anten
print("\n" + "=" * 65)
print("SO SANH ANTEN DSN")
print("-" * 65)
antennas = [
    ('DSN-70', 70, 74),
    ('DSN-34', 34, 68),
    ('DSN-26', 26, 65),
    ('ESA 35m', 35, 68.5),
]
base_cn0 = 62.8  # Voi 70m
for name, diam, gain in antennas:
    delta = gain - 74  # vs 70m
    cn0 = base_cn0 + delta
    rate = 10 ** ((cn0 - 1.0) / 10)
    print(f"  {name:12} ({diam}m): G={gain} dBi, "
          f"C/N₀={cn0:.1f} dBHz, Toc do={rate/1000:.1f} kbps")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Bien thien khoang cach.** Sao Hoa dao dong 0.4-2.7 AU; tinh cho truong hop xau nhat (giao hoi).

2. **Giao hoi Mat Troi.** Khi Sao Hoa o sau Mat Troi, lien ket bi chan ~2 tuan.

3. **Doppler dang ke.** Van toc tuong doi ±30 km/s = dich tan so lon.

4. **Chi huong then chot.** Sai 0.01° tren HGA = ton hao dang ke; dung thiet bi theo doi sao.

5. **Thoi gian anh sang mot chieu.** 4-24 phut; khong the dieu khien tuong tac.

6. **Lich DSN.** Dia 70m qua tai; len ke hoach truoc hang thang.

7. **Che do khan cap.** Phai hoat dong voi omni khuech dai thap; thiet ke truyen thong che do an toan.
