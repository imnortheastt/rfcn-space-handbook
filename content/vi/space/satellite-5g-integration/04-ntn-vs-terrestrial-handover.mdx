## Tai sao dieu nay quan trong

Tuong lai cua ket noi di dong la ket hop: 5G mat dat trong thanh pho, ve tinh NTN o vung nong thon va tren bien. Hieu cach handover giua cac he thong nay hoat dong—va tai sao no khac biet co ban voi handover mat dat-den-mat dat—la can thiet de thiet ke vung phu toan cau lien mach. Cac khac biet bat nguon tu vat ly, khong chi la lua chon trien khai.

## Ap dung

**So sanh thoi gian handover:**

```python
from dataclasses import dataclass
from enum import Enum

class AccessType(Enum):
    TERRESTRIAL = "mat_dat"
    NTN_LEO = "ntn_leo"
    NTN_GEO = "ntn_geo"

@dataclass
class HandoverTiming:
    measurement_report_ms: float
    ho_preparation_ms: float
    ho_execution_ms: float
    service_interruption_ms: float

def get_handover_timing(source: AccessType, target: AccessType) -> HandoverTiming:
    """
    Uoc tinh thoi gian handover giua cac loai truy cap.
    """
    timings = {
        # Noi bo mat dat
        (AccessType.TERRESTRIAL, AccessType.TERRESTRIAL): HandoverTiming(
            measurement_report_ms=5,
            ho_preparation_ms=50,
            ho_execution_ms=30,
            service_interruption_ms=20
        ),
        # Mat dat den LEO NTN
        (AccessType.TERRESTRIAL, AccessType.NTN_LEO): HandoverTiming(
            measurement_report_ms=5,
            ho_preparation_ms=150,  # Khoi dong am GNSS + tinh TA
            ho_execution_ms=100,    # RTT cao hon cho RACH
            service_interruption_ms=100
        ),
        # LEO NTN den Mat dat
        (AccessType.NTN_LEO, AccessType.TERRESTRIAL): HandoverTiming(
            measurement_report_ms=25,  # RTT cao hon cho bao cao do
            ho_preparation_ms=80,
            ho_execution_ms=30,
            service_interruption_ms=50
        ),
        # Noi bo NTN (ve tinh den ve tinh)
        (AccessType.NTN_LEO, AccessType.NTN_LEO): HandoverTiming(
            measurement_report_ms=0,   # Kich hoat theo thoi gian, khong bao cao
            ho_preparation_ms=100,     # Cau hinh truoc qua CHO
            ho_execution_ms=50,
            service_interruption_ms=30
        ),
        # LEO den GEO (hiem nhung co the)
        (AccessType.NTN_LEO, AccessType.NTN_GEO): HandoverTiming(
            measurement_report_ms=25,
            ho_preparation_ms=300,
            ho_execution_ms=600,  # RTT GEO ~600ms
            service_interruption_ms=300
        ),
    }
    
    return timings.get(
        (source, target),
        HandoverTiming(100, 200, 200, 200)  # Du phong mac dinh
    )

def estimate_call_quality_impact(
    source: AccessType,
    target: AccessType,
    service: str
) -> dict:
    """
    Uoc tinh tac dong chat luong trong qua trinh handover.
    """
    timing = get_handover_timing(source, target)
    total_ho_time = (
        timing.measurement_report_ms + 
        timing.ho_preparation_ms + 
        timing.ho_execution_ms
    )
    
    # Tac dong theo dich vu
    impacts = {
        'thoai': {
            'gian_doan_chap_nhan_ms': 200,
            'bo_dem_jitter_ms': 80,
            'tac_dong_chat_luong': 'khong' if timing.service_interruption_ms < 200 else 'khoang trong nghe thay'
        },
        'video': {
            'gian_doan_chap_nhan_ms': 500,
            'bo_dem_ms': 2000,
            'tac_dong_chat_luong': 'khong' if timing.service_interruption_ms < 500 else 'dem lai'
        },
        'web': {
            'gian_doan_chap_nhan_ms': 1000,
            'tcp_timeout_ms': 3000,
            'tac_dong_chat_luong': 'khong' if timing.service_interruption_ms < 1000 else 'thu lai'
        }
    }
    
    return {
        'nguon': source.value,
        'dich': target.value,
        'tong_thoi_gian_ho_ms': total_ho_time,
        'gian_doan_dich_vu_ms': timing.service_interruption_ms,
        'dich_vu': service,
        **impacts.get(service, {})
    }

# Vi du: So sanh cac kich ban handover
print("So sanh Thoi gian Handover")
print("=" * 60)

scenarios = [
    (AccessType.TERRESTRIAL, AccessType.TERRESTRIAL, "thoai"),
    (AccessType.TERRESTRIAL, AccessType.NTN_LEO, "thoai"),
    (AccessType.NTN_LEO, AccessType.TERRESTRIAL, "thoai"),
    (AccessType.NTN_LEO, AccessType.NTN_LEO, "thoai"),
]

for source, target, service in scenarios:
    result = estimate_call_quality_impact(source, target, service)
    print(f"\n{result['nguon']} → {result['dich']}:")
    print(f"  Tong thoi gian HO: {result['tong_thoi_gian_ho_ms']} ms")
    print(f"  Gian doan: {result['gian_doan_dich_vu_ms']} ms")
    print(f"  Tac dong len {service}: {result['tac_dong_chat_luong']}")

# So sanh do tre dien hinh
print("\n" + "=" * 60)
print("So sanh Do tre Dau-Cuoi:")
print("-" * 60)
latencies = [
    ("5G Mat dat", "10-20 ms"),
    ("LEO NTN (truc tiep)", "25-50 ms"),
    ("LEO NTN (bent-pipe)", "40-80 ms"),
    ("GEO NTN", "550-650 ms"),
]
for system, latency in latencies:
    print(f"  {system:25} {latency}")
```

## Cam bay

1. **Khoi dong lanh GNSS.** Handover den NTN co the that bai neu UE khong duy tri dinh vi GNSS.

2. **Khong khop QoS.** Mot so 5QI khong the thoa man qua NTN; dich vu co the suy giam.

3. **Do phuc tap ket noi kep.** DC giua mat dat va NTN co thach thuc timing.

4. **Lap lich khoang cach do.** UE can khoang cach de do NTN trong khi o mat dat.

5. **Phoi hop chinh sach.** Thoa thuan roaming phai bao gom ca truy cap mat dat va NTN.

6. **Kha nang UE.** Khong phai tat ca UE 5G ho tro NTN; can trao doi kha nang.

7. **Dich vu khan cap.** Vi tri E911/eCall co the bi anh huong trong vung phu NTN.
