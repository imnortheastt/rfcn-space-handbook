## Tai sao dieu nay quan trong

Mang Khong Mat dat 5G (NTN) cho phep dien thoai thong minh tieu chuan ket noi truc tiep den ve tinh LEO. Nhung vat ly la khac: do cao 600 km co nghia la do tre 4-8 ms va chuyen dong ve tinh nhanh. Hieu cach 3GPP dieu chinh quy trinh dang ky—voi phat song ephemeris, timing advance, va cell di dong—tiet lo ky thuat lam cho "dien thoai den ve tinh" hoat dong.

## Ap dung

**Tinh toan timing advance:**

```python
import numpy as np

def calculate_ntn_timing_advance(
    ue_lat_deg, ue_lon_deg, ue_alt_m,
    sat_x_km, sat_y_km, sat_z_km,
    feeder_delay_ms=0
):
    """
    Tinh timing advance cho 5G NTN.
    
    Tra ve: TA theo mili giay
    """
    # Chuyen vi tri UE sang ECEF
    earth_radius_km = 6371
    lat_rad = np.radians(ue_lat_deg)
    lon_rad = np.radians(ue_lon_deg)
    r_ue = earth_radius_km + ue_alt_m / 1000
    
    ue_x = r_ue * np.cos(lat_rad) * np.cos(lon_rad)
    ue_y = r_ue * np.cos(lat_rad) * np.sin(lon_rad)
    ue_z = r_ue * np.sin(lat_rad)
    
    # Khoang cach den ve tinh
    d_km = np.sqrt(
        (sat_x_km - ue_x)**2 + 
        (sat_y_km - ue_y)**2 + 
        (sat_z_km - ue_z)**2
    )
    
    # Do tre lan truyen
    c_km_s = 299792.458
    prop_delay_ms = d_km / c_km_s * 1000
    
    # Thoi gian khu hoi (lien ket dich vu)
    rtt_service_ms = 2 * prop_delay_ms
    
    # Tong RTT bao gom feeder link (cho trong suot)
    rtt_total_ms = rtt_service_ms + 2 * feeder_delay_ms
    
    return {
        'distance_km': d_km,
        'one_way_delay_ms': prop_delay_ms,
        'rtt_service_ms': rtt_service_ms,
        'rtt_total_ms': rtt_total_ms,
        'ta_ms': rtt_total_ms
    }

def calculate_doppler(
    freq_ghz, ue_lat_deg, ue_lon_deg,
    sat_vx_km_s, sat_vy_km_s, sat_vz_km_s,
    sat_x_km, sat_y_km, sat_z_km
):
    """
    Tinh dich Doppler cho lien ket NTN.
    """
    # Vi tri UE (don gian hoa, gia dinh tren be mat)
    earth_radius_km = 6371
    lat_rad = np.radians(ue_lat_deg)
    lon_rad = np.radians(ue_lon_deg)
    
    ue_x = earth_radius_km * np.cos(lat_rad) * np.cos(lon_rad)
    ue_y = earth_radius_km * np.cos(lat_rad) * np.sin(lon_rad)
    ue_z = earth_radius_km * np.sin(lat_rad)
    
    # Vector don vi tu UE den ve tinh
    dx = sat_x_km - ue_x
    dy = sat_y_km - ue_y
    dz = sat_z_km - ue_z
    d = np.sqrt(dx**2 + dy**2 + dz**2)
    ux, uy, uz = dx/d, dy/d, dz/d
    
    # Van toc huong tam (tich vo huong cua van toc va vector don vi)
    v_radial = sat_vx_km_s * ux + sat_vy_km_s * uy + sat_vz_km_s * uz
    
    # Dich Doppler
    c_km_s = 299792.458
    doppler_hz = freq_ghz * 1e9 * v_radial / c_km_s
    
    return {
        'radial_velocity_km_s': v_radial,
        'doppler_hz': doppler_hz,
        'doppler_khz': doppler_hz / 1000
    }

# Vi du: Ve tinh LEO tai do cao 600 km
print("Vi du Timing Advance 5G NTN")
print("=" * 55)

# Vi tri ve tinh (don gian hoa: truc tiep nam cua xich dao)
sat_altitude_km = 600
sat_x = 0
sat_y = 6371 + sat_altitude_km  # Tren xich dao, kinh do 0°
sat_z = 0

# UE tai cac vi do khac nhau
ue_positions = [
    (0, 0, 0, "Ngay ben duoi"),
    (10, 0, 0, "Cach 10°"),
    (30, 0, 0, "Cach 30°"),
    (50, 0, 0, "Cach 50° (goc nang thap)"),
]

for lat, lon, alt, desc in ue_positions:
    result = calculate_ntn_timing_advance(lat, lon, alt, sat_x, sat_y, sat_z)
    print(f"\n{desc}:")
    print(f"  Khoang cach: {result['distance_km']:.1f} km")
    print(f"  Do tre mot chieu: {result['one_way_delay_ms']:.2f} ms")
    print(f"  RTT (dich vu): {result['rtt_service_ms']:.2f} ms")

# Voi payload trong suot (cong them feeder link)
print("\n" + "=" * 55)
print("Voi Payload Trong suot (do tre feeder link = 10 ms):")
result = calculate_ntn_timing_advance(30, 0, 0, sat_x, sat_y, sat_z, feeder_delay_ms=10)
print(f"  Tong RTT: {result['rtt_total_ms']:.2f} ms")
print(f"  TA can thiet: {result['ta_ms']:.2f} ms")

# Vi du Doppler
print("\n" + "=" * 55)
print("Vi du Dich Doppler (2 GHz, ve tinh di chuyen)")
# Ve tinh di chuyen ve phia dong voi van toc quy dao
v_orbital = 7.5  # km/s cho do cao 600 km
doppler_result = calculate_doppler(
    2.0,  # 2 GHz (S-band)
    30, 0,  # Vi tri UE
    v_orbital, 0, 0,  # Van toc ve tinh (huong dong)
    sat_x, sat_y, sat_z
)
print(f"  Van toc huong tam: {doppler_result['radial_velocity_km_s']:.2f} km/s")
print(f"  Dich Doppler: {doppler_result['doppler_khz']:.1f} kHz")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Yeu cau GNSS.** UE phai co dinh vi GNSS de tinh TA; khong GNSS = khong NTN.

2. **Hieu luc ephemeris.** Ephemeris cu di; UE phai theo doi thoi gian va yeu cau cap nhat.

3. **Trong suot vs tai tao.** Tong do tre rat khac; kien truc quan trong.

4. **Du Doppler.** Bu truoc khong hoan hao; gNB xu ly phan du.

5. **Do phuc tap cell co dinh Trai dat.** Yeu cau lai bup song; tang chi phi ve tinh.

6. **Mo rong timer.** Tat ca timer NR (T300, T310, v.v.) phai mo rong cho NTN.

7. **Tieu thu dien.** GNSS va theo doi Doppler lien tuc lam can pin UE.
