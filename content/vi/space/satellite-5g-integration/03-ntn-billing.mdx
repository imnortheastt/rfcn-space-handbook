## Tai sao dieu nay quan trong

Khi dien thoai thong minh ket noi qua Starlink trong luc chuyen vung o nuoc khac, ai tra tien cho ai? 5G NTN gioi thieu tinh cuoc da ben phuc tap: nha khai thac ve tinh, MNO mat dat, mang tham, va nha cung cap dich vu deu can duoc boi thuong. Hieu kien truc tinh cuoc NTN la can thiet cho cac nha khai thac xay dung mang lai mat dat-ve tinh.

## Ap dung

**Tinh toan tinh cuoc NTN:**

```python
from dataclasses import dataclass
from datetime import datetime, timedelta
from typing import Optional
import json

@dataclass
class NTNCDRecord:
    """Ban ghi Du lieu Tinh cuoc NTN."""
    record_id: str
    supi: str
    session_start: datetime
    session_end: datetime
    data_volume_ul_kb: int
    data_volume_dl_kb: int
    satellite_id: str
    ntn_cell_id: str
    access_type: str  # NR-NTN-LEO, NR-NTN-GEO, v.v.
    service_area: str
    qos_class: int
    home_plmn: str
    visited_plmn: Optional[str] = None

def rate_ntn_session(cdr: NTNCDRecord, rate_plan: dict) -> dict:
    """Tinh phi cho phien NTN."""
    
    total_kb = cdr.data_volume_ul_kb + cdr.data_volume_dl_kb
    total_mb = total_kb / 1024
    
    # Gia co ban
    base_rate = rate_plan.get('base_rate_per_mb', 0.01)
    base_charge = total_mb * base_rate
    
    # Phu phi NTN theo loai truy cap
    ntn_surcharges = {
        'NR-NTN-LEO': 0.005,  # LEO co phu phi thap hon (do tre thap hon)
        'NR-NTN-MEO': 0.008,
        'NR-NTN-GEO': 0.010,  # GEO cao hon (dung luong khan hiem)
    }
    ntn_surcharge = ntn_surcharges.get(cdr.access_type, 0.010)
    ntn_charge = total_mb * ntn_surcharge
    
    # He so gio cao diem
    hour = cdr.session_start.hour
    peak_hours = rate_plan.get('peak_hours', [8, 9, 10, 17, 18, 19, 20])
    peak_multiplier = 1.5 if hour in peak_hours else 1.0
    
    # Phi QoS
    qos_premiums = {
        1: 0.0,    # No luc tot nhat
        2: 0.10,   # Thoai dam thoai
        3: 0.05,   # Video streaming
        4: 0.02,   # Tuong tac
    }
    qos_premium = qos_premiums.get(cdr.qos_class, 0.0)
    
    # Tinh tong
    subtotal = (base_charge + ntn_charge) * peak_multiplier
    total_charge = subtotal * (1 + qos_premium)
    
    return {
        'record_id': cdr.record_id,
        'total_mb': round(total_mb, 2),
        'base_charge': round(base_charge, 4),
        'ntn_surcharge': round(ntn_charge, 4),
        'peak_multiplier': peak_multiplier,
        'qos_premium_pct': qos_premium * 100,
        'total_charge': round(total_charge, 4),
        'currency': rate_plan.get('currency', 'USD')
    }

def generate_tap_record(cdr: NTNCDRecord, iot_rate: float) -> dict:
    """Tao ban ghi TAP cho thanh toan chuyen vung."""
    
    total_kb = cdr.data_volume_ul_kb + cdr.data_volume_dl_kb
    duration = (cdr.session_end - cdr.session_start).total_seconds()
    
    charge = (total_kb / 1024) * iot_rate
    tax = charge * 0.10  # Vi du thue 10%
    
    return {
        'tap_version': '3.12',
        'sender': cdr.visited_plmn or 'NTNOP001',
        'recipient': cdr.home_plmn,
        'file_sequence': 1,
        'call_records': [{
            'imsi': cdr.supi.replace('imsi-', ''),
            'call_type': 'GPRS',
            'gprs_service_used': {
                'data_volume_incoming_kb': cdr.data_volume_dl_kb,
                'data_volume_outgoing_kb': cdr.data_volume_ul_kb,
                'charging_id': cdr.record_id,
                'pdp_type': 'IPv4',
            },
            'call_timestamp': cdr.session_start.isoformat(),
            'duration_seconds': int(duration),
            'charge': {
                'amount': round(charge, 4),
                'currency': 'USD',
                'exchange_rate': 1.0,
            },
            'tax': round(tax, 4),
            # Mo rong NTN (de xuat)
            'ntn_info': {
                'satellite_id': cdr.satellite_id,
                'access_type': cdr.access_type,
                'service_area': cdr.service_area,
            }
        }],
        'total_charge': round(charge + tax, 4)
    }

def calculate_wholesale_cost(
    capacity_mbps: float,
    hours_used: float,
    wholesale_rate_per_mbps_hour: float
) -> dict:
    """Tinh chi phi dung luong ve tinh ban si."""
    
    capacity_cost = capacity_mbps * hours_used * wholesale_rate_per_mbps_hour
    
    # Chi phi co dinh (gateway, phan mat dat)

        'capacity_cost': round(capacity_cost, 2),
        'ground_segment_cost': round(ground_segment_cost, 2),
        'total_wholesale_cost': round(total, 2)
    }
```

<section class="lesson-pitfalls">

## Cam bay

1. **Do tre anh huong tinh cuoc truc tuyen.** Kiem tra tin dung mat lau hon qua ve tinh; dung cap han muc lon hon.

2. **Anh xa ID ve tinh.** CDR phai xac dinh dung ve tinh nao da phuc vu phien de phan bo chi phi chinh xac.

3. **Ranh gioi buc xa phu song.** Phien co the trai qua nhieu buc xa; xac dinh buc xa nao de tinh cuoc.

4. **Chuyen giao giua cac nha khai thac.** Neu nguoi dung chuyen giao tu mat dat sang NTN giua phien, chia CDR chinh xac.

5. **Tien te va thue.** NTN vuot bien gioi; xac dinh thue ap dung va quy doi tien te.

6. **Het so du tra truoc.** O vung xa khong co mat dat du phong, dam bao ha cap dich vu on hoa.

7. **Bien loi ban si vs ban le.** MNO phai dinh gia dich vu NTN de trang trai chi phi ban si trong khi van canh tranh.
