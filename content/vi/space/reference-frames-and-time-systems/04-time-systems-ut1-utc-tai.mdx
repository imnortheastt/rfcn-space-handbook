## Tai sao dieu nay quan trong

Van hanh khong gian yeu cau do chinh xac thoi gian duoi mili-giay. Nhung ban dung dong ho nao? May tinh cua ban noi 12:00:00 UTC, may thu GPS noi 12:00:18 thoi gian GPS, va viec quay cua Trai Dat noi la 12:00:00.3 UT1. Day khong phai loi — chung la cac he thoi gian khac nhau duoc toi uu cho cac muc dich khac nhau. UTC giu thoi gian dan su thang hang voi Mat Troi. TAI dap deu cho vat ly. Thoi gian GPS khong bao gio co giay nhay cho dan duong lien tuc. UT1 theo doi viec quay thuc te cua Trai Dat de chi huong anten. Nham chung khien cac luot bay ve tinh bi lo, du doan va cham sai, va ve duong di mat dat bi troi.

## Ap dung

**Chuyen doi he thoi gian:**

```python
from datetime import datetime, timedelta
import numpy as np

# Giay nhay hien tai (TAI - UTC)
GIAY_NHAY = 37  # Tinh tu 2017, kiem tra IERS de cap nhat

# Do lech co dinh
TAI_TRU_GPS = 19      # giay (co dinh tu epoch GPS 1980)
TT_TRU_TAI = 32.184   # giay (co dinh theo dinh nghia)
EPOCH_GPS = datetime(1980, 1, 6, 0, 0, 0)

def utc_sang_tai(utc_datetime):
    """Chuyen UTC sang TAI."""
    return utc_datetime + timedelta(seconds=GIAY_NHAY)

def tai_sang_utc(tai_datetime):
    """Chuyen TAI sang UTC."""
    return tai_datetime - timedelta(seconds=GIAY_NHAY)

def utc_sang_gps(utc_datetime):
    """Chuyen UTC sang thoi gian GPS."""
    gps_tru_utc = GIAY_NHAY - TAI_TRU_GPS  # Hien tai 18
    return utc_datetime + timedelta(seconds=gps_tru_utc)

def gps_sang_utc(gps_datetime):
    """Chuyen thoi gian GPS sang UTC."""
    gps_tru_utc = GIAY_NHAY - TAI_TRU_GPS
    return gps_datetime - timedelta(seconds=gps_tru_utc)

def utc_sang_tt(utc_datetime):
    """Chuyen UTC sang Terrestrial Time (TT)."""
    return utc_datetime + timedelta(seconds=GIAY_NHAY + TT_TRU_TAI)

def tuan_giay_gps(utc_datetime):
    """Chuyen UTC sang so tuan GPS va giay trong tuan."""
    thoi_gian_gps = utc_sang_gps(utc_datetime)
    delta = thoi_gian_gps - EPOCH_GPS
    tong_giay = delta.total_seconds()
    tuan = int(tong_giay // (7 * 86400))
    giay_trong_tuan = tong_giay % (7 * 86400)
    return tuan, giay_trong_tuan

def ngay_julian(utc_datetime):
    """Tinh Ngay Julian tu datetime UTC."""
    nam = utc_datetime.year
    thang = utc_datetime.month
    ngay = utc_datetime.day
    gio = utc_datetime.hour + utc_datetime.minute/60 + utc_datetime.second/3600
    
    if thang <= 2:
        nam -= 1
        thang += 12
    
    A = int(nam / 100)
    B = 2 - A + int(A / 4)
    
    jd = int(365.25 * (nam + 4716)) + int(30.6001 * (thang + 1)) + ngay + gio/24 + B - 1524.5
    return jd

def gmst_do(jd_ut1):
    """Tinh Greenwich Mean Sidereal Time theo do tu JD(UT1)."""
    T = (jd_ut1 - 2451545.0) / 36525
    gmst = 280.46061837 + 360.98564736629 * (jd_ut1 - 2451545.0) + \
           0.000387933 * T**2 - T**3 / 38710000
    return gmst % 360

print("Tham khao He Thoi gian")
print("=" * 70)

# Thoi gian vi du
utc_hien_tai = datetime(2026, 6, 3, 12, 0, 0)

print(f"\n1. Chuyen doi Thang Thoi gian (tham chieu: {utc_hien_tai} UTC)")
print("-" * 70)

tai_hien_tai = utc_sang_tai(utc_hien_tai)
gps_hien_tai = utc_sang_gps(utc_hien_tai)
tt_hien_tai = utc_sang_tt(utc_hien_tai)

print(f"  UTC: {utc_hien_tai}")
print(f"  TAI: {tai_hien_tai} (UTC + {GIAY_NHAY}s)")
print(f"  GPS: {gps_hien_tai} (UTC + {GIAY_NHAY - TAI_TRU_GPS}s)")
print(f"  TT:  {tt_hien_tai} (UTC + {GIAY_NHAY + TT_TRU_TAI:.3f}s)")

# Tuan va giay GPS
print(f"\n2. Dinh dang Tuan GPS")

print(f"  Tuan GPS: {tuan}")

print(f"  Ngay trong Tuan: {int(gtuan // 86400)} ({['CN','T2','T3','T4','T5','T6','T7'][int(gtuan // 86400)]})")
```

| GPS tu UTC | GPS = UTC + 18s |
| GPS tu TAI | GPS = TAI - 19s |
| TT tu TAI | TT = TAI + 32.184s |
| UT1 tu UTC | UT1 = UTC + ΔUT1 (kiem tra IERS) |

</section>

<section class="lesson-pitfalls">

## Cam bay

1. **Dung UTC cho thoi gian thien van.** GST phai duoc tinh tu UT1. UTC gay sai vi tri toi 400m.

2. **Gia dinh giay nhay co the du doan.** Chung duoc thong bao truoc ~6 thang. Dung ma cung gia tri tuong lai.

3. **Tron dau thoi gian GPS va UTC.** Sai 18 giay gay sai khoang cach 5.4 trieu met o toc do anh sang.

4. **Bo qua thong bao giay nhay.** Phan mem crash vao ngay giay nhay (loi Linux 2012, mat dien hang khong 2015).

5. **Dung TT cho thoi gian dan su.** TT danh cho ephemeris. No vuot truoc UTC 69+ giay.

6. **Quen GPS week rollover.** Tuan GPS la 10 bit (0-1023). Da tran vao 1999 va 2019. Tiep theo: 2038.
