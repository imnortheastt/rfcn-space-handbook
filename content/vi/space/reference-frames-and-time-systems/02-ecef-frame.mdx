## Tai sao dieu nay quan trong

Tram mat dat khong di chuyen. Thanh pho khong quay quanh quy dao. Khi ban can chi anten vao ve tinh, tinh vung phu, hoac xac dinh tau vu tru co dang bay qua mot quoc gia cu the hay khong, ban can toa do quay theo Trai Dat. ECEF (Tam Trai Dat - Gan Trai Dat) la he quy chieu do — he toa do noi vi tri mat dat co toa do khong doi. GPS, Google Maps, va moi radar mat dat deu su dung ECEF (cu the la WGS84). Hieu ECEF la thiet yeu cho ve duong di mat dat ve tinh, phan tich vung phu, va bat ky van hanh nao lien quan den co so ha tang gan mat dat.

## Ap dung

**Lam viec voi toa do ECEF:**

```python
import numpy as np

# Tham so ellipsoid WGS84
WGS84_A = 6378137.0           # Ban truc lon (m)
WGS84_F = 1 / 298.257223563   # Do det
WGS84_B = WGS84_A * (1 - WGS84_F)  # Ban truc nho
WGS84_E2 = 2 * WGS84_F - WGS84_F**2  # Do lech tam binh phuong
OMEGA_TRAI_DAT = 7.292115e-5  # Toc do quay Trai Dat (rad/s)

def trac_dia_sang_ecef(lat_do, lon_do, do_cao_m):
    """Chuyen doi trac dia (lat, lon, alt) sang ECEF (X, Y, Z)."""
    lat = np.radians(lat_do)
    lon = np.radians(lon_do)
    
    sin_lat = np.sin(lat)
    cos_lat = np.cos(lat)
    sin_lon = np.sin(lon)
    cos_lon = np.cos(lon)
    
    # Ban kinh cong trong mat phang thang dung goc
    N = WGS84_A / np.sqrt(1 - WGS84_E2 * sin_lat**2)
    
    X = (N + do_cao_m) * cos_lat * cos_lon
    Y = (N + do_cao_m) * cos_lat * sin_lon
    Z = (N * (1 - WGS84_E2) + do_cao_m) * sin_lat
    
    return np.array([X, Y, Z])

def ecef_sang_trac_dia(X, Y, Z, sai_so=1e-12, lap_toi_da=10):
    """Chuyen doi ECEF (X, Y, Z) sang trac dia (lat, lon, alt) dung phuong phap Bowring."""
    lon = np.arctan2(Y, X)
    
    p = np.sqrt(X**2 + Y**2)
    
    # Uoc luong ban dau dung phuong phap Bowring
    lat = np.arctan2(Z, p * (1 - WGS84_E2))
    
    for _ in range(lap_toi_da):
        sin_lat = np.sin(lat)
        N = WGS84_A / np.sqrt(1 - WGS84_E2 * sin_lat**2)
        lat_moi = np.arctan2(Z + WGS84_E2 * N * sin_lat, p)
        
        if abs(lat_moi - lat) < sai_so:
            break
        lat = lat_moi
    
    sin_lat = np.sin(lat)
    cos_lat = np.cos(lat)
    N = WGS84_A / np.sqrt(1 - WGS84_E2 * sin_lat**2)
    
    if abs(cos_lat) > 1e-10:
        do_cao = p / cos_lat - N
    else:
        do_cao = abs(Z) - WGS84_B
    
    return np.degrees(lat), np.degrees(lon), do_cao

def gmst_rad(jd):
    """Tinh Greenwich Mean Sidereal Time (radian)."""
    T = (jd - 2451545.0) / 36525
    gmst_do = 280.46061837 + 360.98564736629 * (jd - 2451545.0) + \
               0.000387933 * T**2 - T**3 / 38710000
    return np.radians(gmst_do % 360)

def eci_sang_ecef(r_eci, v_eci, jd):
    """Chuyen doi trang thai ECI sang trang thai ECEF."""
    theta = gmst_rad(jd)
    cos_t = np.cos(theta)
    sin_t = np.sin(theta)
    
    # Ma tran quay
    R = np.array([
        [cos_t, sin_t, 0],
        [-sin_t, cos_t, 0],
        [0, 0, 1]
    ])
    
    r_ecef = R @ r_eci
    
    # Van toc bao gom anh huong quay Trai Dat
    omega = np.array([0, 0, OMEGA_TRAI_DAT])
    v_ecef = R @ v_eci - np.cross(omega, r_ecef)
    
    return r_ecef, v_ecef

def goc_nang_phuong_vi_ve_tinh(sat_ecef, tram_lat, tram_lon, tram_cao):
    """Tinh goc nang va phuong vi ve tinh tu tram mat dat."""
    tram_ecef = trac_dia_sang_ecef(tram_lat, tram_lon, tram_cao)
    
    # Vector tu tram den ve tinh
    delta = sat_ecef - tram_ecef
    
    # Chuyen doi ENU (Dong-Bac-Len) cuc bo
    lat = np.radians(tram_lat)
    lon = np.radians(tram_lon)
    
    sin_lat = np.sin(lat)
    cos_lat = np.cos(lat)
    sin_lon = np.sin(lon)
    cos_lon = np.cos(lon)
    
    # Ma tran quay ECEF sang ENU
    R = np.array([
        [-sin_lon, cos_lon, 0],
        [-sin_lat*cos_lon, -sin_lat*sin_lon, cos_lat],
        [cos_lat*cos_lon, cos_lat*sin_lon, sin_lat]
    ])
    
    enu = R @ delta
    E, N, U = enu
    
    # Goc nang va phuong vi
    kc_ngang = np.sqrt(E**2 + N**2)
    goc_nang = np.degrees(np.arctan2(U, kc_ngang))
    phuong_vi = np.degrees(np.arctan2(E, N)) % 360
    
    return goc_nang, phuong_vi
```

<section class="lesson-pitfalls">

## Cam bay

1. **Vi do trac dia vs tam dia.** Chung chenh lech toi 0.19° o vi do 45°. GPS dung vi do trac dia.

2. **Tham chieu do cao.** Do cao WGS84 la tren ellipsoid, khong phai muc nuoc bien trung binh (geoid). Chenh lech toi ±100m.

3. **Quen quay Trai Dat cho van toc.** Van toc ECEF bao gom quay Trai Dat. Chuyen doi van toc ECI sang ECEF can so hang ω×r.

4. **Tron cac datum.** WGS84, NAD83, ITRF khac nhau cm-m. Tron chung lam hong do chinh xac.

5. **Gia dinh Trai Dat hinh cau.** Dung R_trai_dat = 6371 km thay vi ellipsoid WGS84 gay sai so cap km o vi do cao.

6. **ECEF cho lan truyen quy dao.** Khong bao gio lan truyen quy dao trong ECEF — luc ao lam phuong trinh phuc tap. Dung ECI, roi chuyen doi.
