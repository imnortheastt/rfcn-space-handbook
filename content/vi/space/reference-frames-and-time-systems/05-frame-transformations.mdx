## Tai sao dieu nay quan trong

Vector trang thai ve tinh den tu bo truyen quy dao trong he ECI. Anten mat dat cua ban nam tai toa do trac dia co dinh trong he ECEF. De chi huong anten, ban can goc phuong vi va goc nang trong he topocentric cuc bo. Ba he quy chieu nay — ECI, ECEF, topocentric — moi he phuc vu mot muc dich, va ban can di chuyen giua chung lien tuc. Bien doi sai va anten cua ban chi vao bau troi trong, ve duong di mat dat troi qua cac luc dia, hoac sang loc va cham bo sot mot cuoc gap go.

### Bien doi ECI sang ECEF

```

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  DON GIAN HOA (tot den ~1 km):                                     │
│                                                                      │
│    r_ECEF = R_z(θ_GMST) × r_ECI                                    │
│                                                                      │
│  trong do θ_GMST = Greenwich Mean Sidereal Time (tu UT1)           │
│                                                                      │
│  ─────────────────────────────────────────────────────────────────  │
│                                                                      │
│  BIEN DOI DAY DU (chinh xac duoi met):                             │
│                                                                      │
│    r_ECEF = W × R × N × P × r_ECI                                  │
│                                                                      │
│  trong do:                                                          │
│    P = Ma tran tue sai (truc Trai Dat tue sai qua 26,000 nam)     │

│    R = Ma tran quay Trai Dat (GMST + phuong trinh xuan phan)      │

│                                                                      │

│                                                                      │
│  BIEN DOI VAN TOC:                                                  │
│                                                                      │
│    v_ECEF = R × v_ECI - ω_E × r_ECEF                               │
│                                                                      │
│  trong do ω_E = [0, 0, 7.292115×10⁻⁵] rad/s (toc do quay Trai Dat)│

│                                                                      │
```

### ECEF sang topocentric (ENU)

```
ECEF → ENU: He Quan sat Cuc bo

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  BUOC 1: Tinh vi tri quan sat trong ECEF                           │
│                                                                      │
│    Cho quan sat vien tai trac dia (φ, λ, h):                       │
│                                                                      │
│    N = a / √(1 - e² sin²φ)                                         │
│    X_obs = (N + h) cos(φ) cos(λ)                                   │
│    Y_obs = (N + h) cos(φ) sin(λ)                                   │
│    Z_obs = (N(1-e²) + h) sin(φ)                                    │
│                                                                      │
│  BUOC 2: Tinh vi tri tuong doi                                     │
│                                                                      │
│    Δr_ECEF = r_muc_tieu_ECEF - r_quan_sat_ECEF                     │
│                                                                      │
│  BUOC 3: Quay sang ENU                                             │
│                                                                      │
│    r_ENU = R_ECEF_sang_ENU × Δr_ECEF                               │
│                                                                      │
│              ┌ -sin(λ)        cos(λ)        0      ┐               │
│    R_ENU =   │ -sin(φ)cos(λ) -sin(φ)sin(λ)  cos(φ) │               │
│              └  cos(φ)cos(λ)  cos(φ)sin(λ)  sin(φ) ┘               │
│                                                                      │
│    trong do φ = vi do quan sat, λ = kinh do quan sat               │
│                                                                      │
│  KET QUA: r_ENU = [E, N, U] (dong, bac, len tinh bang met)        │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### ENU sang goc phuong vi/goc nang

```
ENU → Goc phuong vi/Goc nang/Tam xa

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  Cho r_ENU = [E, N, U]:                                            │
│                                                                      │
│  TAM XA:                                                            │
│    ρ = √(E² + N² + U²)                                             │
│                                                                      │
│  GOC NANG:                                                          │
│    El = arcsin(U / ρ)                  [pham vi: -90° den +90°]    │
│                                                                      │
│    hoac tuong duong:                                                │
│    El = arctan(U / √(E² + N²))                                     │
│                                                                      │
│  GOC PHUONG VI (do theo chieu kim dong ho tu Bac):                 │
│    Az = arctan2(E, N)                  [pham vi: 0° den 360°]      │
│                                                                      │
│    LUU Y: arctan2(y, x) cho goc tu +x huong +y                     │
│          Day x=N, y=E → goc tu Bac huong Dong                      │
│                                                                      │
│  NGUOC LAI (Az/El/Tam xa → ENU):                                   │
│    E = ρ × cos(El) × sin(Az)                                       │
│    N = ρ × cos(El) × cos(Az)                                       │
│    U = ρ × sin(El)                                                 │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## Ap dung

**Duong ong bien doi hoan chinh:**

```python
import numpy as np
from datetime import datetime

def ma_tran_quay_z(theta):
    """Ma tran quay quanh truc Z (radian)."""
    c, s = np.cos(theta), np.sin(theta)
    return np.array([
        [ c, s, 0],
        [-s, c, 0],
        [ 0, 0, 1]
    ])

def gmst_rad(jd_ut1):
    """Greenwich Mean Sidereal Time theo radian tu JD(UT1)."""
    T = (jd_ut1 - 2451545.0) / 36525
    gmst_deg = 280.46061837 + 360.98564736629 * (jd_ut1 - 2451545.0) + \
               0.000387933 * T**2 - T**3 / 38710000
    return np.deg2rad(gmst_deg % 360)

def ngay_julian(dt):
    """Tinh Ngay Julian tu datetime."""
    nam, thang, ngay = dt.year, dt.month, dt.day
    gio = dt.hour + dt.minute/60 + dt.second/3600
    if thang <= 2:
        nam -= 1
        thang += 12
    A = int(nam / 100)
    B = 2 - A + int(A / 4)
    return int(365.25 * (nam + 4716)) + int(30.6001 * (thang + 1)) + ngay + gio/24 + B - 1524.5

def eci_sang_ecef(r_eci, v_eci, dt):
    """
    Bien doi ECI sang ECEF (don gian hoa, chinh xac ~1km).
    
    Args:
        r_eci: Vi tri [x, y, z] trong ECI (met)
        v_eci: Van toc [vx, vy, vz] trong ECI (m/s)
        dt: datetime (UTC, dung xap xi UT1)
    
    Returns:
        r_ecef, v_ecef: Vi tri va van toc trong ECEF
    """
    jd = ngay_julian(dt)
    theta = gmst_rad(jd)
    R = ma_tran_quay_z(theta)
    
    r_ecef = R @ r_eci
    
    omega_trai_dat = 7.292115e-5  # rad/s
    omega_vec = np.array([0, 0, omega_trai_dat])
    v_ecef = R @ v_eci - np.cross(omega_vec, r_ecef)
    
    return r_ecef, v_ecef

def ecef_sang_eci(r_ecef, v_ecef, dt):
    """Bien doi ECEF sang ECI (don gian hoa)."""
    jd = ngay_julian(dt)
    theta = gmst_rad(jd)
    R = ma_tran_quay_z(theta)
    R_nghich = R.T
    
    omega_trai_dat = 7.292115e-5
    omega_vec = np.array([0, 0, omega_trai_dat])
    
    r_eci = R_nghich @ r_ecef
    v_eci = R_nghich @ (v_ecef + np.cross(omega_vec, r_ecef))
    
    return r_eci, v_eci

def trac_dia_sang_ecef(vi_do_deg, kinh_do_deg, cao_do_m):
    """
    Chuyen trac dia (vi do, kinh do, cao do) sang ECEF (x, y, z).
    
    Args:
        vi_do_deg: Vi do theo do
        kinh_do_deg: Kinh do theo do
        cao_do_m: Cao do tren ellipsoid WGS84 theo met
    
    Returns:
        [x, y, z] trong ECEF (met)
    """
    a = 6378137.0
    f = 1/298.257223563
    e2 = 2*f - f**2
    
    phi = np.deg2rad(vi_do_deg)
    lam = np.deg2rad(kinh_do_deg)
    
    N = a / np.sqrt(1 - e2 * np.sin(phi)**2)
    
    x = (N + cao_do_m) * np.cos(phi) * np.cos(lam)
    y = (N + cao_do_m) * np.cos(phi) * np.sin(lam)
    z = (N * (1 - e2) + cao_do_m) * np.sin(phi)
    
    return np.array([x, y, z])

def ecef_sang_enu(r_muc_tieu_ecef, vi_do_qs_deg, kinh_do_qs_deg, cao_do_qs_m):
    """
    Bien doi vi tri ECEF sang ENU tuong doi voi quan sat vien.
    
    Args:
        r_muc_tieu_ecef: Vi tri muc tieu trong ECEF (met)
        vi_do_qs_deg, kinh_do_qs_deg: Toa do trac dia quan sat vien
        cao_do_qs_m: Cao do quan sat vien tren WGS84 (met)
    
    Returns:
        [E, N, U]: Toa do Dong, Bac, Len (met)
    """
    r_qs_ecef = trac_dia_sang_ecef(vi_do_qs_deg, kinh_do_qs_deg, cao_do_qs_m)
    dr = r_muc_tieu_ecef - r_qs_ecef
    
    phi = np.deg2rad(vi_do_qs_deg)
    lam = np.deg2rad(kinh_do_qs_deg)
    
    R_enu = np.array([
        [-np.sin(lam),              np.cos(lam),              0           ],
        [-np.sin(phi)*np.cos(lam), -np.sin(phi)*np.sin(lam),  np.cos(phi)],
        [ np.cos(phi)*np.cos(lam),  np.cos(phi)*np.sin(lam),  np.sin(phi)]
    ])
    
    return R_enu @ dr

def enu_sang_azel(enu):
    """
    Chuyen ENU sang Goc phuong vi/Goc nang/Tam xa.
    
    Args:
        enu: [E, N, U] theo met
    
    Returns:
        az_deg: Goc phuong vi (0=Bac, 90=Dong, theo chieu kim dong ho)
        el_deg: Goc nang (0=chan troi, 90=thien dinh)
        tam_xa_m: Tam xa nghieng theo met
    """
    E, N, U = enu
    
    tam_xa_m = np.sqrt(E**2 + N**2 + U**2)
    el_rad = np.arcsin(U / tam_xa_m)
    az_rad = np.arctan2(E, N)
    
    az_deg = np.rad2deg(az_rad) % 360
    el_deg = np.rad2deg(el_rad)
    
    return az_deg, el_deg, tam_xa_m

def eci_sang_azel(r_eci, dt, vi_do_qs, kinh_do_qs, cao_do_qs):
    """
    Duong ong day du: ECI → ECEF → ENU → Az/El.
    """
    v_eci = np.array([0, 0, 0])
    r_ecef, _ = eci_sang_ecef(r_eci, v_eci, dt)
    enu = ecef_sang_enu(r_ecef, vi_do_qs, kinh_do_qs, cao_do_qs)
    return enu_sang_azel(enu)
```

## Cam bay

1. **Dung UTC thay vi UT1 cho GMST.** Gay sai so toi 400m. Lay ΔUT1 tu IERS cho cong viec chinh xac.

2. **Sai huong quay.** R × v quay vector; Rᵀ × v quay he quy chieu. Biet ro ban can cai nao.

3. **Quen bien doi van toc.** v_ECEF ≠ R × v_ECI. Ban phai tru so hang ω × r.

4. **Quy uoc goc phuong vi khong khop.** Mot so he thong dung 0°=Dong, mot so dung 0°=Bac. Luon xac minh.

5. **Tron ENU va NED.** U chi len; D chi xuong. Sai dau trong thanh phan thang dung.

6. **Bo qua chuyen dong cuc.** Voi do chinh xac 1 met, x_p va y_p quan trong. Lay tu ban tin IERS.
