## Tai sao dieu nay quan trong

Moi vi tri va van toc tau vu tru phai duoc bieu dien trong mot he toa do nao do. Chon sai, va lan truyen quy dao cua ban se troi mot cach tham lang, tranh va cham se that bai, va chi huong anten se truot muc tieu. ECI (Quan tinh tam Trai Dat) la he quy chieu nen tang cho co hoc quy dao — no khong quay theo Trai Dat, nen dinh luat Newton ap dung truc tiep ma khong can luc ao. Hieu ECI la dieu kien tien quyet de doc TLE, tinh toan cac yeu to quy dao, va chuyen doi giua nhieu he quy chieu duoc su dung trong van hanh khong gian.

## Ap dung

**Lam viec voi toa do ECI:**

```python
import numpy as np

# Tham so Trai Dat
MU_TRAI_DAT = 398600.4418  # km³/s² (tham so hap dan)
R_TRAI_DAT = 6378.137      # km (ban kinh xich dao)
OMEGA_TRAI_DAT = 7.292115e-5  # rad/s (toc do quay)

def trang_thai_sang_yeu_to_quy_dao(r, v, mu=MU_TRAI_DAT):
    """Chuyen vector trang thai ECI sang yeu to quy dao co dien."""
    r_mag = np.linalg.norm(r)
    v_mag = np.linalg.norm(v)
    
    # Momen dong luong rieng
    h = np.cross(r, v)
    h_mag = np.linalg.norm(h)
    
    # Vector nut (chi den nut len)
    k = np.array([0, 0, 1])
    n = np.cross(k, h)
    n_mag = np.linalg.norm(n)
    
    # Vector do lech tam
    e_vec = ((v_mag**2 - mu/r_mag) * r - np.dot(r, v) * v) / mu
    e = np.linalg.norm(e_vec)
    
    # Ban truc lon
    nang_luong = v_mag**2 / 2 - mu / r_mag
    if abs(e - 1.0) > 1e-10:
        a = -mu / (2 * nang_luong)
    else:
        a = float('inf')  # Parabol
    
    # Do nghieng
    i = np.arccos(h[2] / h_mag)
    
    # RAAN (Kinh do thang giao cua nut len)
    if n_mag > 1e-10:
        raan = np.arccos(n[0] / n_mag)
        if n[1] < 0:
            raan = 2 * np.pi - raan
    else:
        raan = 0.0
    
    # Tham so can diem
    if n_mag > 1e-10 and e > 1e-10:
        omega = np.arccos(np.dot(n, e_vec) / (n_mag * e))
        if e_vec[2] < 0:
            omega = 2 * np.pi - omega
    else:
        omega = 0.0
    
    # Di thuong thuc
    if e > 1e-10:
        nu = np.arccos(np.dot(e_vec, r) / (e * r_mag))
        if np.dot(r, v) < 0:
            nu = 2 * np.pi - nu
    else:
        nu = 0.0
    
    return {
        'a': a,          # Ban truc lon (km)
        'e': e,          # Do lech tam
        'i': np.degrees(i),     # Do nghieng (do)
        'raan': np.degrees(raan),  # RAAN (do)
        'omega': np.degrees(omega), # Tham so can diem (do)
        'nu': np.degrees(nu)    # Di thuong thuc (do)
    }

def chu_ky_quy_dao(a, mu=MU_TRAI_DAT):
    """Tinh chu ky quy dao theo phut."""
    return 2 * np.pi * np.sqrt(a**3 / mu) / 60

def greenwich_sidereal_time(jd):
    """Tinh Greenwich Mean Sidereal Time (radian)."""
    T = (jd - 2451545.0) / 36525  # The ky Julian tu J2000
    gmst_do = 280.46061837 + 360.98564736629 * (jd - 2451545.0) + \
               0.000387933 * T**2 - T**3 / 38710000
    return np.radians(gmst_do % 360)

def eci_sang_ecef(r_eci, jd):
    """Chuyen vi tri ECI sang ECEF."""
    gst = greenwich_sidereal_time(jd)
    cos_gst = np.cos(gst)
    sin_gst = np.sin(gst)
    
    R = np.array([
        [cos_gst, sin_gst, 0],
        [-sin_gst, cos_gst, 0],
        [0, 0, 1]
    ])
    
    return R @ r_eci

print("Vi du He Quy chieu ECI")
print("=" * 70)

# Vi du: Quy dao kieu ISS
print("\n1. Vector Trang thai ISS (J2000 ECI)")
print("-" * 70)

r_eci = np.array([-2436.45, -2436.45, 6891.04])  # km
v_eci = np.array([-5.654, 4.830, 2.312])         # km/s

print(f"  Vi tri: [{r_eci[0]:.2f}, {r_eci[1]:.2f}, {r_eci[2]:.2f}] km")
print(f"  Van toc: [{v_eci[0]:.3f}, {v_eci[1]:.3f}, {v_eci[2]:.3f}] km/s")

r_mag = np.linalg.norm(r_eci)
v_mag = np.linalg.norm(v_eci)
do_cao = r_mag - R_TRAI_DAT

print(f"\n  Ban kinh: {r_mag:.2f} km")
print(f"  Do cao: {do_cao:.2f} km")
print(f"  Toc do: {v_mag:.3f} km/s")

# Chuyen sang yeu to quy dao
print("\n2. Yeu to Quy dao Co dien")
print("-" * 70)

yeu_to = trang_thai_sang_yeu_to_quy_dao(r_eci, v_eci)
chu_ky = chu_ky_quy_dao(yeu_to['a'])

print(f"  Ban truc lon (a): {yeu_to['a']:.2f} km")
print(f"  Do lech tam (e): {yeu_to['e']:.6f}")
print(f"  Do nghieng (i): {yeu_to['i']:.2f}°")
print(f"  RAAN (Ω): {yeu_to['raan']:.2f}°")
print(f"  Tham so can diem (ω): {yeu_to['omega']:.2f}°")
print(f"  Di thuong thuc (ν): {yeu_to['nu']:.2f}°")
print(f"  Chu ky quy dao: {chu_ky:.2f} phut")

# Chuyen doi ECI sang ECEF
print("\n3. Chuyen doi ECI sang ECEF")
print("-" * 70)

jd = 2451545.0 + 26 * 365.25 + 154  # JD gan dung cho 2026-06-03
r_ecef = eci_sang_ecef(r_eci, jd)

print(f"  Julian Date: {jd:.1f}")
print(f"  GST: {np.degrees(greenwich_sidereal_time(jd)):.2f}°")
print(f"\n  ECI:  [{r_eci[0]:.2f}, {r_eci[1]:.2f}, {r_eci[2]:.2f}] km")
print(f"  ECEF: [{r_ecef[0]:.2f}, {r_ecef[1]:.2f}, {r_ecef[2]:.2f}] km")

# So sanh he quy chieu
print("\n4. Tai sao Chon He Quy chieu Quan trong")
print("-" * 70)

print("  Vi du chenh lech he quy chieu (gia tri dien hinh):")
print()
print(f"  {'So sanh':<25} {'Lech Goc':<18} {'Sai so Vi tri (LEO)'}")
print("-" * 70)
print(f"  {'J2000 vs GCRF':<25} {'~0.02 arcsec':<18} {'~0.1 m'}")
print(f"  {'J2000 vs TEME':<25} {'~20 arcsec':<18} {'~600 m'}")
print(f"  {'J2000 vs B1950':<25} {'~0.5°':<18} {'~50 km'}")

print("\n  QUAN TRONG: Khop he voi bo lan truyen!")
print("    - TLE/SGP4 → He TEME")
print("    - Ephemeris chinh xac → J2000/GCRF")
print("    - GPS → WGS84 (la ECEF)")

# Truc quan tue sai
print("\n5. Tue sai Truc Trai Dat")
print("-" * 70)

toc_do_tue_sai = 50.3  # arcsec/nam
nam = [0, 10, 50, 100, 1000]

print("  Nam tu J2000.0 │ Goc tue sai │ Dich chuyen vi tri (GEO)")
print("-" * 70)
for y in nam:
    goc = toc_do_tue_sai * y / 3600  # do
    dich_chuyen = 42164 * np.sin(np.radians(goc))  # km tai GEO
    print(f"  {y:>14} │ {goc:>10.4f}° │ {dich_chuyen:>14.1f} km")

print("\n  Khong chi ro epoch, vi tri tro nen mo ho theo thoi gian!")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Tron cac bien the ECI.** J2000, TEME, TOD deu la "ECI" nhung khac nhau tu arcsec den arcmin. Luon chi ro loai nao.

2. **Quen epoch.** ECI khong co epoch la mo ho. Luon ghep toa do voi thoi gian.

3. **Dung ECI cho tram mat dat.** Anten mat dat khong di chuyen trong ECEF, khong phai ECI. Chuyen doi truoc khi tinh chi huong.

4. **Sai khop he TLE.** TLE o trong TEME. Chuyen dau ra SGP4 sang J2000 khong dung cach chuyen doi se mat do chinh xac.

5. **Bo qua chuyen dong cuc.** Cho do chinh xac duoi met, tinh ca tham so dinh huong Trai Dat (EOP).

6. **Gia dinh quan tinh hoan hao.** ECI tue sai cham. Cho phan tich nhieu thap ky, dung GCRF hoac ap dung mo hinh tue sai.
