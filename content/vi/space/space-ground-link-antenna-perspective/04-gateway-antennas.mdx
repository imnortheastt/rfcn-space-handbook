## Tai sao dieu nay quan trong

Tram mat dat cong la xuong song cua truyen thong ve tinh—cac trai anten khong lo tong hop luu luong tu hang trieu thiet bi dau cuoi nguoi dung. Du ban dang thiet ke chom sao HTS, lap ke hoach dung luong phan mat dat, hay hieu kien truc mang ve tinh, cong la noi khong gian gap co so ha tang mat dat. Mot cong don co the xu ly thong luong 10-100+ Gbps.

## Ap dung

**Tinh toan ngan sach lien ket va dung luong cong:**

```python
import numpy as np

def cong_eirp_dbw(cong_suat_w, tang_ich_anten_dbi, ton_hao_db=2.0):
    """
    Tinh EIRP cong.
    """
    cong_suat_dbw = 10 * np.log10(cong_suat_w)
    return cong_suat_dbw + tang_ich_anten_dbi - ton_hao_db

def cong_gt_dbk(tang_ich_anten_dbi, nhiet_do_he_thong_k, ton_hao_db=1.0):
    """
    Tinh G/T cong.
    """
    nhiet_do_nhieu_db = 10 * np.log10(nhiet_do_he_thong_k)
    return tang_ich_anten_dbi - ton_hao_db - nhiet_do_nhieu_db

def tang_ich_parabol_dbi(duong_kinh_m, tan_so_ghz, hieu_suat=0.65):
    """
    Tinh tang ich anten parabol.
    """
    buoc_song = 0.3 / tan_so_ghz  # met
    dien_tich = np.pi * (duong_kinh_m / 2) ** 2
    tang_ich = hieu_suat * (4 * np.pi * dien_tich) / (buoc_song ** 2)
    return 10 * np.log10(tang_ich)

def dung_luong_cong_gbps(bang_thong_mhz, hieu_suat_pho, so_anten, duty_cycle=1.0):
    """
    Tinh tong dung luong thong luong cong.
    """
    dung_luong_moi_anten = bang_thong_mhz * hieu_suat_pho / 1000  # Gbps
    return dung_luong_moi_anten * so_anten * duty_cycle

def san_sang_da_dang(san_sang_mot_vi_tri, so_vi_tri, tuong_quan=0.1):
    """
    Uoc tinh do san sang voi da dang cong.
    """
    khong_san_sang_mot = 1 - san_sang_mot_vi_tri
    khong_san_sang_ket_hop = khong_san_sang_mot ** so_vi_tri * (1 + tuong_quan * (so_vi_tri - 1))
    return 1 - khong_san_sang_ket_hop

# Tinh toan vi du
print("Phan tich Tram Mat dat Cong")
print("=" * 60)

# Tang ich anten vs tan so
print("\n1. Tang ich Anten theo Duong kinh va Tan so")
print("-" * 60)
print(f"  {'Duong kinh':>10} {'Ku (12 GHz)':>14} {'Ka (20 GHz)':>14} {'V (50 GHz)':>14}")

for duong_kinh in [5, 7, 9, 11, 13]:
    tang_ich_ku = tang_ich_parabol_dbi(duong_kinh, 12)
    tang_ich_ka = tang_ich_parabol_dbi(duong_kinh, 20)
    tang_ich_v = tang_ich_parabol_dbi(duong_kinh, 50)
    print(f"  {duong_kinh:>8}m {tang_ich_ku:>12.1f} dBi {tang_ich_ka:>12.1f} dBi {tang_ich_v:>12.1f} dBi")

# EIRP va G/T cho cong dien hinh
print("\n2. Thong so Ngan sach Lien ket Cong")
print("-" * 60)
cau_hinh = [
    ("9m Ka-band", 9, 20, 400, 80),
    ("11m Ka-band", 11, 20, 600, 75),
    ("13m Ka-band", 13, 20, 1000, 70),
    ("9m V-band", 9, 50, 200, 150),
]

print(f"  {'Cau hinh':<15} {'Tang ich':>10} {'EIRP':>10} {'G/T':>10}")
for ten, dk, ts, cs, nd in cau_hinh:
    tang_ich = tang_ich_parabol_dbi(dk, ts)
    eirp = cong_eirp_dbw(cs, tang_ich)
    gt = cong_gt_dbk(tang_ich, nd)
    print(f"  {ten:<15} {tang_ich:>8.1f} dBi {eirp:>8.1f} dBW {gt:>8.1f} dB/K")

# Tinh toan dung luong
print("\n3. Dung luong Thong luong Cong")
print("-" * 60)
tinh_huong = [
    ("GEO HTS", 2000, 4.5, 8, 1.0),
    ("Cong LEO", 500, 3.5, 4, 0.4),
    ("Cong MEO", 1000, 4.0, 6, 0.6),
]

print(f"  {'Tinh huong':<15} {'BT':>10} {'HSP':>8} {'Anten':>10} {'Dung luong':>12}")
print(f"  {'':<15} {'(MHz)':>10} {'(bps/Hz)':>8} {'':>10} {'(Gbps)':>12}")

for ten, bt, hsp, so_ant, duty in tinh_huong:
    dl = dung_luong_cong_gbps(bt, hsp, so_ant, duty)
    print(f"  {ten:<15} {bt:>10} {hsp:>8.1f} {so_ant:>10} {dl:>12.1f}")

# Do san sang da dang vi tri
print("\n4. Do San sang Da dang Cong")
print("-" * 60)
san_sang_mot = 0.995

print(f"  Do san sang mot vi tri: {san_sang_mot*100:.2f}%")
print(f"  {'Vi tri':>8} {'Tuong quan':>12} {'San sang ket hop':>16}")

for vi_tri in [1, 2, 3, 4]:
    for tq in [0.0, 0.1, 0.3]:
        ss = san_sang_da_dang(san_sang_mot, vi_tri, tq)
        print(f"  {vi_tri:>8} {tq:>12.1f} {ss*100:>15.4f}%")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Danh gia thap suy hao mua.** Cong V-band co the thay suy hao 30+ dB; lap ngan sach bien va da dang can than.

2. **Phu thuoc cap quang.** Cong khong co cap quang = dung luong mac ket. Xac minh nhieu duong cap quang da dang.

3. **Can nhieu den/tu he thong khac.** Can phoi hop ITU; gioi han mat do EIRP cong ap dung.

4. **Dien va lam mat.** HPA lon tao ra nhiet dang ke; lap ke hoach cong suat 2-3× cho lam mat.

5. **Do phuc tap theo doi LEO.** Cong NGSO can nhieu anten theo doi va logic chuyen giao nhanh.

6. **Tre cap phep.** Cap phep cong co the mat 6-18 thang; bat dau som.

7. **Mo rong vs du phong.** Du phong N+M (vd: 8+2) can thiet cho bao tri khong ngat.
