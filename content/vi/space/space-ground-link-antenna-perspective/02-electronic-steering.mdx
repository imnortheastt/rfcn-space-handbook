## Tai sao dieu nay quan trong

Lai dien tu la thu lam cho anten ve tinh panel phang co the. Thay vi xoay dia vat ly, chung ta tinh toan dich pha dung cho moi phan tu anten de chi chum bat cu dau chung ta muon—trong micro giay. Hieu thuat toan tao chum, cong nghe bo dich pha, va danh doi giua cac phuong phap tuong tu va so la can thiet de thiet ke cac thiet bi dau cuoi ve tinh va tram goc the he tiep theo.

## Ap dung

**Trien khai thuat toan tao chum:**

```python
import numpy as np

def tinh_vector_lai(vi_tri_phan_tu, theta_deg, phi_deg, tan_so_ghz):
    """
    Tinh vector lai cho hinh hoc mang tuy y.
    
    Args:
        vi_tri_phan_tu: Mang Nx3 cua vi tri (x,y,z) tinh bang met
        theta_deg: Goc nang tu broadside (0=len)
        phi_deg: Goc phuong vi (0=truc x)
        tan_so_ghz: Tan so hoat dong
    
    Returns:
        Vector lai phuc (dich pha)
    """
    c = 299792458
    buoc_song = c / (tan_so_ghz * 1e9)
    k = 2 * np.pi / buoc_song
    
    theta = np.radians(theta_deg)
    phi = np.radians(phi_deg)
    
    # Vector don vi theo huong mong muon
    u = np.array([
        np.sin(theta) * np.cos(phi),
        np.sin(theta) * np.sin(phi),
        np.cos(theta)
    ])
    
    # Dich pha cho moi phan tu
    pha = -k * np.dot(vi_tri_phan_tu, u)
    
    return np.exp(1j * pha)

def tao_ula(n_phan_tu, khoang_cach_buoc_song, buoc_song):
    """Tao vi tri mang tuyen tinh dong deu."""
    vi_tri = np.zeros((n_phan_tu, 3))
    vi_tri[:, 0] = np.arange(n_phan_tu) * khoang_cach_buoc_song * buoc_song
    return vi_tri

def luong_tu_hoa_pha(pha, bits):
    """Luong tu hoa pha theo do phan giai cho truoc."""
    muc = 2 ** bits
    buoc = 360 / muc
    return np.round(pha / buoc) * buoc

# Vi du: Tao chum mang tuyen tinh 8 phan tu
print("Vi du Lai Chum Dien tu")
print("=" * 60)

tan_so = 12.0  # GHz
buoc_song = 299792458 / (tan_so * 1e9)
n_phan_tu = 8
khoang_cach = 0.5  # buoc song

# Tao mang
vi_tri = tao_ula(n_phan_tu, khoang_cach, buoc_song)

print(f"\nCau hinh Mang:")
print(f"  Phan tu: {n_phan_tu}")
print(f"  Tan so: {tan_so} GHz")
print(f"  Khoang cach: {khoang_cach}λ = {khoang_cach * buoc_song * 1000:.2f} mm")

# Tinh vector lai cho cac goc khac nhau
print(f"\n1. Dich Pha cho cac Goc Lai Khac nhau")
print("-" * 60)
print(f"  {'Goc':>8} | {'Dich pha (do)':>45}")
print("-" * 60)

for theta in [0, 15, 30, 45]:
    sv = tinh_vector_lai(vi_tri, theta, 0, tan_so)
    pha = np.angle(sv, deg=True)
    pha_str = " ".join([f"{p:6.1f}" for p in pha])
    print(f"  {theta:>6}° | {pha_str}")

# Anh huong do phan giai bo dich pha
print(f"\n2. Anh huong Do phan giai Bo Dich Pha")
print("-" * 60)

theta_muc_tieu = 30
sv_ly_tuong = tinh_vector_lai(vi_tri, theta_muc_tieu, 0, tan_so)
pha_ly_tuong = np.angle(sv_ly_tuong, deg=True) % 360

print(f"  Goc muc tieu: {theta_muc_tieu}°")
print(f"  {'Bits':>6} {'Muc':>8} {'Sai so max':>12} {'Sai chi huong':>15}")

for bits in [2, 3, 4, 5, 6]:
    pha_luong_tu = luong_tu_hoa_pha(pha_ly_tuong, bits)
    sai_so_max = np.max(np.abs(pha_ly_tuong - pha_luong_tu))
    sai_chi_huong = sai_so_max / 30
    print(f"  {bits:>6} {2**bits:>8} {sai_so_max:>10.1f}° {sai_chi_huong:>13.2f}°")

# Yeu cau toc do cap nhat theo doi LEO
print(f"\n3. Tan suat Cap nhat Theo doi LEO")
print("-" * 60)

do_cao_ve_tinh_km = 550
van_toc_quy_dao_km_s = 7.5
toc_do_goc_deg_s = np.degrees(van_toc_quy_dao_km_s / do_cao_ve_tinh_km)

for bw in [0.5, 1.0, 2.0, 5.0]:
    chu_ky_cap_nhat_ms = (bw / toc_do_goc_deg_s) * 1000
    print(f"  Do rong chum {bw:>4.1f}°: cap nhat moi {chu_ky_cap_nhat_ms:>6.0f} ms")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Do phan giai bo dich pha.** Toi thieu 4-bit (16 muc) de chi chum tot; 6-bit cho thuy phu thap.

2. **Thuy luong tu.** Do phan giai pha tho tao thuy phu them lang phi cong suat.

3. **Troi hieu chuan.** Thay doi nhiet do co the dich pha vai do; can hieu chuan lien tuc.

4. **Ghep tuong ho.** Cac phan tu anh huong lan nhau; bo qua va chum chi sai.

5. **Do tre xu ly.** Tao chum so can DSP nhanh; do tre quan trong cho theo doi LEO.

6. **Tieu thu dien.** Tao chum so ngon dien; quan ly nhiet quan trong.

7. **Phuc tap vs do tin cay.** Cang so = cang nhieu tinh toan, cang nhieu diem loi.
