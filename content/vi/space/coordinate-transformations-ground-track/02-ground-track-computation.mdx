## Tai sao dieu nay quan trong

Vet mat dat la duong ve boi diem duoi ve tinh tren be mat Trai dat—diem ngay ben duoi ve tinh. Hieu vet mat dat cho ban biet vung nao ve tinh co the quan sat, khi nao no se di qua tram mat dat, va tai sao mot so quy dao cung cap do phu toan cau trong khi nhung quy dao khac thi khong. Day la cau noi giua co hoc quy dao va lap ke hoach nhiem vu thuc te.

## Ap dung

**Tinh toan vet mat dat:**

```python
import numpy as np
from datetime import datetime, timedelta

# Hang so WGS84
WGS84_A = 6378137.0
WGS84_E2 = 0.00669437999014
MU = 3.986004418e14  # m³/s²

def gmst_rad(dt):
    """Gio Sao Trung binh Greenwich tinh bang radian."""
    jd = (dt - datetime(2000, 1, 1, 12, 0, 0)).total_seconds() / 86400.0 + 2451545.0
    T = (jd - 2451545.0) / 36525.0
    gmst_do = 280.46061837 + 360.98564736629 * (jd - 2451545.0) + 0.000387933 * T**2
    return np.radians(gmst_do % 360.0)

def kepler_sang_eci(a, e, i, raan, argp, nu):
    """Chuyen cac yeu to Kepler sang vi tri ECI."""
    r = a * (1 - e**2) / (1 + e * np.cos(nu))
    
    x_qd = r * np.cos(nu)
    y_qd = r * np.sin(nu)
    
    cos_raan, sin_raan = np.cos(raan), np.sin(raan)
    cos_argp, sin_argp = np.cos(argp), np.sin(argp)
    cos_i, sin_i = np.cos(i), np.sin(i)
    
    x = (cos_raan * cos_argp - sin_raan * sin_argp * cos_i) * x_qd + \
        (-cos_raan * sin_argp - sin_raan * cos_argp * cos_i) * y_qd
    y = (sin_raan * cos_argp + cos_raan * sin_argp * cos_i) * x_qd + \
        (-sin_raan * sin_argp + cos_raan * cos_argp * cos_i) * y_qd
    z = (sin_argp * sin_i) * x_qd + (cos_argp * sin_i) * y_qd
    
    return np.array([x, y, z])

def lan_truyen_kepler(a, e, i, raan, argp, M0, t_tu):
    """Lan truyen Kepler don gian (khong nhieu loan)."""
    n = np.sqrt(MU / a**3)
    M = M0 + n * t_tu
    
    E = M
    for _ in range(10):
        E = E - (E - e * np.sin(E) - M) / (1 - e * np.cos(E))
    
    nu = 2 * np.arctan2(np.sqrt(1 + e) * np.sin(E / 2),
                         np.sqrt(1 - e) * np.cos(E / 2))
    
    return kepler_sang_eci(a, e, i, raan, argp, nu)

def eci_sang_tracdia(r_eci, dt):
    """Chuyen ECI sang toa do trac dia."""
    theta = gmst_rad(dt)
    c, s = np.cos(theta), np.sin(theta)
    x_ecef = c * r_eci[0] + s * r_eci[1]
    y_ecef = -s * r_eci[0] + c * r_eci[1]
    z_ecef = r_eci[2]
    
    lon = np.arctan2(y_ecef, x_ecef)
    p = np.sqrt(x_ecef**2 + y_ecef**2)
    lat = np.arctan2(z_ecef, p)
    
    for _ in range(5):
        N = WGS84_A / np.sqrt(1 - WGS84_E2 * np.sin(lat)**2)
        lat = np.arctan2(z_ecef + WGS84_E2 * N * np.sin(lat), p)
    
    return np.degrees(lon), np.degrees(lat)

def tinh_vet_mat_dat(a, e, i, raan, argp, M0, epoch, gio, dt_phut=1):
    """Tinh vet mat dat cho cac yeu to quy dao cho truoc."""
    vet = []
    n_diem = int(gio * 60 / dt_phut)
    
    for k in range(n_diem + 1):
        t_tu = k * dt_phut * 60
        thoi_gian_hien_tai = epoch + timedelta(seconds=t_tu)
        
        r_eci = lan_truyen_kepler(a, e, i, raan, argp, M0, t_tu)
        lon, lat = eci_sang_tracdia(r_eci, thoi_gian_hien_tai)
        
        vet.append((lon, lat))
    
    return vet

# Vi du: Quy dao giong ISS
print("Tinh toan Vet mat dat")
print("=" * 60)

a_iss = 6778e3  # do cao 400 km
e_iss = 0.0001
i_iss = np.radians(51.6)
raan_iss = np.radians(0)
argp_iss = np.radians(0)
M0_iss = np.radians(0)

epoch = datetime(2026, 6, 3, 0, 0, 0)
thoi_luong = 3  # gio

vet = tinh_vet_mat_dat(a_iss, e_iss, i_iss, raan_iss, argp_iss, M0_iss, epoch, thoi_luong, dt_phut=2)

print(f"\nQuy dao giong ISS (a={a_iss/1e3:.0f} km, i={np.degrees(i_iss):.1f}°)")
print(f"Thoi luong: {thoi_luong} gio ({len(vet)} diem)")
print(f"\nCac diem mau (moi 30 phut):")
print(f"{'Thoi gian':<12} {'Kinh do':<12} {'Vi do':<12}")
print("-" * 40)

for k in range(0, len(vet), 15):
    lon, lat = vet[k]
    t_phut = k * 2
    print(f"{t_phut:<12} {lon:>10.2f}° {lat:>10.2f}°")

T = 2 * np.pi * np.sqrt(a_iss**3 / MU)
print(f"\nChu ky quy dao: {T/60:.1f} phut")
print(f"Trai dat quay trong mot quy dao: {360 * T / 86164.1:.1f}° dich ve tay")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Quen quay Trai dat.** Vi tri ECI vo nghia neu khong chuyen doi GMST.

2. **Wrap-around kinh do.** Dung ve duong tu +179° den -179°; xu ly diem gian doan.

3. **Dung ngay mat troi.** Vet mat dat lap lai voi ngay sao (86164 s), khong phai ngay mat troi (86400 s).

4. **Bo qua nhieu loan.** J2 gay troi RAAN ~0.5-10°/ngay cho LEO; anh huong vet dai han.

5. **Gia dinh Trai dat cau.** Chap nhan cho truc quan nhung gay sai so ~20 km trong SSP.

6. **Nham lan do cao va ban kinh.** Do cao la tren be mat; ban kinh quy dao bao gom ban kinh Trai dat.
