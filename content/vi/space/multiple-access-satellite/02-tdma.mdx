## Tai sao dieu nay quan trong

FDMA lang phi bang thong khi luu luong theo cum — moi nguoi dung chiem phan tan so cua ho ngay ca khi im lang. TDMA khac phuc dieu nay: tat ca nguoi dung chia se toan bo bang thong transponder nhung thay phien truyen trong cac khe thoi gian duoc gan. Khi ban khong noi, nguoi khac dung khe cua ban. Ket qua: su dung transponder gan 100% thay vi 10%. Day la ly do TDMA thong tri truyen thong ve tinh hien dai, tu trung ke Intelsat den bang rong DVB-RCS2.

## Ap dung

**May tinh khung va dung luong TDMA:**

```python
import numpy as np

def hieu_qua_khung_tdma(thoi_gian_khung_us, so_cum, thoi_gian_bao_ve_us,
                        bit_preamble, toc_do_ky_hieu_msps):
    """
    Tinh hieu qua khung TDMA.
    
    Args:
        thoi_gian_khung_us: Tong thoi gian khung (micro giay)
        so_cum: So cum luu luong moi khung
        thoi_gian_bao_ve_us: Thoi gian bao ve moi cum (micro giay)
        bit_preamble: Bit preamble + UW moi cum
        toc_do_ky_hieu_msps: Toc do ky hieu (Msps)
    
    Returns:
        Dict voi cac chi so hieu qua
    """
    overhead_bao_ve_us = so_cum * thoi_gian_bao_ve_us
    
    thoi_gian_preamble_us = bit_preamble / toc_do_ky_hieu_msps
    overhead_preamble_us = so_cum * thoi_gian_preamble_us
    
    tong_overhead_us = overhead_bao_ve_us + overhead_preamble_us
    
    thoi_gian_luu_luong_us = thoi_gian_khung_us - tong_overhead_us
    hieu_qua = thoi_gian_luu_luong_us / thoi_gian_khung_us
    
    return {
        "thoi_gian_khung_us": thoi_gian_khung_us,
        "overhead_bao_ve_us": overhead_bao_ve_us,
        "overhead_preamble_us": overhead_preamble_us,
        "thoi_gian_luu_luong_us": thoi_gian_luu_luong_us,
        "hieu_qua": hieu_qua
    }

def dung_luong_tdma(bw_transponder_mhz, toc_do_ky_hieu_msps, bit_moi_ky_hieu,
                    ty_le_ma, hieu_qua_khung):
    """
    Tinh thong luong he thong TDMA.
    """
    toc_do_tho_mbps = toc_do_ky_hieu_msps * bit_moi_ky_hieu
    
    toc_do_ma_hoa_mbps = toc_do_tho_mbps * ty_le_ma
    
    toc_do_thuc_mbps = toc_do_ma_hoa_mbps * hieu_qua_khung
    
    return {
        "toc_do_tho_mbps": toc_do_tho_mbps,
        "toc_do_ma_hoa_mbps": toc_do_ma_hoa_mbps,
        "toc_do_thuc_mbps": toc_do_thuc_mbps
    }

def tinh_som_dinh_thoi(khoang_cach_tram_km, cao_do_vt_km=35786):
    """
    Tinh thoi gian truyen som can cho dong bo TDMA.
    """
    c = 299792.458  # km/s
    
    tre_khu_hoi_ms = 2 * (cao_do_vt_km + khoang_cach_tram_km) / c * 1000
    
    tre_tham_chieu_ms = 2 * cao_do_vt_km / c * 1000
    
    truyen_som_us = (tre_khu_hoi_ms - tre_tham_chieu_ms) * 1000
    
    return {
        "tre_khu_hoi_ms": tre_khu_hoi_ms,
        "truyen_som_us": truyen_som_us
    }

def so_sanh_fdma_tdma(so_nguoi_dung, he_so_hoat_dong, bw_transponder_mhz):
    """
    So sanh dung luong FDMA vs TDMA cho luu luong theo cum.
    """
    bw_fdma_moi_nd = bw_transponder_mhz / so_nguoi_dung
    bw_hieu_qua_fdma = bw_fdma_moi_nd * he_so_hoat_dong
    hieu_qua_tong_fdma = he_so_hoat_dong
    
    overhead_tdma = 0.05  # 5% dien hinh
    hieu_qua_tdma = 1 - overhead_tdma
    
    ty_so_dung_luong = hieu_qua_tdma / hieu_qua_tong_fdma
    
    return {
        "hieu_qua_fdma": hieu_qua_tong_fdma,
        "hieu_qua_tdma": hieu_qua_tdma,
        "ty_so_loi_the_tdma": ty_so_dung_luong
    }

print("Cong cu Phan tich TDMA")
print("=" * 70)

# Phan tich hieu qua khung
```

<section class="lesson-pitfalls">

## Cam bay

1. **Bo qua do phuc tap dinh thoi.** TDMA yeu cau dong bo chinh xac micro giay. Moi tram can truyen som duy nhat dua tren khoang cach. GPS giup rat nhieu.

2. **Danh gia thap overhead preamble.** Moi cum can khoi phuc song mang, khoi phuc dinh thoi bit, va tu duy nhat. Cho cum ngan, overhead nay chiem uu the.

3. **Quen hieu ung suy hao mua.** Mua suy giam tin hieu VA thay doi tre lan truyen. TDMA phai dieu chinh dinh thoi dong.

4. **Gia dinh TDMA luon thang.** Cho luu luong lien tuc (phat video), su don gian cua FDMA co the thich hop hon. TDMA thang cho luu luong theo cum, tuong tac.

5. **Thoi gian bao ve qua ngan.** Khong chac chan dinh thoi, troi dao dong, va Doppler deu cong lai. Thoi gian bao ve 1-5 μs la dien hinh cho GEO.
