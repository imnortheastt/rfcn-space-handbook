## Tai sao dieu nay quan trong

MF-TDMA la cong cu chinh cua he thong VSAT hien dai. No ket hop FDMA (nhieu song mang tan so) voi TDMA (khe thoi gian tren moi song mang). Ket qua: he thong quan ly mang co the gan dong bat ky thiet bi dau cuoi nao vao bat ky khe nao tren bat ky song mang nao, phu hop dung luong voi nhu cau theo thoi gian thuc. Day la ly do ket noi Viasat hoac HughesNet cua ban co the tang toc cao khi can nhung chia se tai nguyen hieu qua qua hang trieu thue bao. DVB-RCS2, tieu chuan toan cau, duoc xay dung tren MF-TDMA.

import numpy as np

def dung_luong_khung_mf_tdma(so_song_mang, bw_song_mang_mhz, toc_do_ky_hieu_msps,
                              bit_moi_ky_hieu, ty_le_ma, khe_moi_khung,
                              ky_hieu_preamble, ky_hieu_bao_ve):
    """
    Tinh dung luong khung MF-TDMA.
    """
    thoi_gian_khe_us = 1000000 / toc_do_ky_hieu_msps / khe_moi_khung
    
    ky_hieu_tai_trong = int(thoi_gian_khe_us * toc_do_ky_hieu_msps) - ky_hieu_preamble - ky_hieu_bao_ve
    
    bit_moi_khe = ky_hieu_tai_trong * bit_moi_ky_hieu * ty_le_ma
    byte_moi_khe = bit_moi_khe / 8
    
    tong_khe = so_song_mang * khe_moi_khung
    
    return {
        "so_song_mang": so_song_mang,
        "khe_moi_khung": khe_moi_khung,
        "tong_khe": tong_khe,
        "byte_moi_khe": int(byte_moi_khe),
        "ky_hieu_tai_trong": ky_hieu_tai_trong
    }

def phan_bo_dung_luong(tong_khe, dau_cuoi):
    """
    Mo phong phan bo dung luong MF-TDMA.
    
    Args:
        tong_khe: Tong khe moi sieu khung
        dau_cuoi: Danh sach dict {id, khe_cra, khe_rbdc, khe_vbdc}
    
    Returns:
        Ket qua phan bo
    """
    tong_cra = sum(t.get("khe_cra", 0) for t in dau_cuoi)
    tong_rbdc = sum(t.get("khe_rbdc", 0) for t in dau_cuoi)
    tong_vbdc = sum(t.get("khe_vbdc", 0) for t in dau_cuoi)
    
    da_cam_ket = tong_cra + tong_rbdc + tong_vbdc
    fca_san_co = max(0, tong_khe - da_cam_ket)
    
    dc_hoat_dong = len([t for t in dau_cuoi if 
                       t.get("khe_cra", 0) + t.get("khe_rbdc", 0) + t.get("khe_vbdc", 0) > 0])
    
    fca_moi_dc = fca_san_co // max(1, dc_hoat_dong) if dc_hoat_dong > 0 else 0
    
    return {
        "tong_khe": tong_khe,
        "khe_cra": tong_cra,
        "khe_rbdc": tong_rbdc,
        "khe_vbdc": tong_vbdc,
        "khe_fca": fca_san_co,
        "su_dung": da_cam_ket / tong_khe,
        "fca_moi_dc": fca_moi_dc
    }

def thong_luong_dau_cuoi(cra_kbps, rbdc_kbps, vbdc_burst_kb, fca_kbps,
                         cir_kbps, mir_kbps):
    """
    Tinh thong luong dau cuoi voi rang buoc QoS.
    """
    tong = cra_kbps + rbdc_kbps + fca_kbps
    
    dam_bao = min(cra_kbps, cir_kbps)
    
    dinh = min(tong, mir_kbps)
    
    return {
        "dam_bao_kbps": dam_bao,
        "dien_hinh_kbps": cra_kbps + rbdc_kbps,
        "dinh_kbps": dinh,
        "vbdc_burst_kb": vbdc_burst_kb
    }

| Tong khe | So_song_mang × Khe_moi_song_mang |
| Dung luong khe | (Ky_hieu_khe - Preamble - Bao_ve) × Bit/ky_hieu × Ty_le_ma / 8 |
| Dam bao CIR | min(Toc_do_CRA, Toc_do_Cam_ket) |
| Toc do dinh | min(CRA + RBDC + FCA, MIR) |
| Su dung | (CRA + RBDC + VBDC) / Tong_khe |

</section>

<section class="lesson-pitfalls">

## Cam bay

1. **Bo qua tre lap lich.** MF-TDMA them 2-3 sieu khung tre cho yeu cau dung luong. Tre goi dau tien co the cao hon SCPC 100-200 ms.

2. **Dang ky qua nhieu CRA.** CRA duoc dam bao — neu ban hua nhieu CRA hon dung luong he thong, ban se that bai tai tai dinh. Giu tong CRA < 70% dung luong.

3. **Ky vong burst tuc thi.** Yeu cau VBDC/RBDC mat 2-3 sieu khung de dap ung. Luu luong web theo cum cam nhan tre nay.

4. **Quen bien do suy hao mua.** He thong MF-TDMA dung ma hoa/dieu che thich ung. Mua to giam khe moi dau cuoi — danh ngan sach cho che do suy giam.

5. **Danh gia thap overhead preamble.** Khe nho co phan tram preamble cao hon. Khe 64-byte voi preamble 96 ky hieu mat ~20% cho overhead.

</section>
