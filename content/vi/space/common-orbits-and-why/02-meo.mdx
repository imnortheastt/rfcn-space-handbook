## Tai sao dieu nay quan trong

Quy dao Trai dat Trung binh (MEO) la "vung vang" cho dan duong va mot so vien thong: cao hon LEO de vung phu tot hon, thap hon GEO de do tre chap nhan duoc. GPS, Galileo, GLONASS, va O3b mPOWER deu hoat dong trong MEO. Hieu vat ly MEO giai thich tai sao chom sao dan duong chi can ~30 ve tinh trong khi bang rong LEO can hang ngan.

## Ap dung

**Tinh toan quy dao MEO:**

```python
import numpy as np

# Hang so
MU_EARTH = 398600.4418  # km³/s²
R_EARTH = 6371  # km
C = 299792.458  # km/s

def meo_parameters(altitude_km):
    """Tinh cac tham so quy dao MEO."""
    a = R_EARTH + altitude_km
    
    # Chu ky quy dao
    period_s = 2 * np.pi * np.sqrt(a**3 / MU_EARTH)
    period_hr = period_s / 3600
    
    # Van toc
    velocity = np.sqrt(MU_EARTH / a)
    
    # Vung phu (goc nadir cho goc nang toi thieu 5°)
    min_elev = 5
    rho = np.arcsin(R_EARTH * np.cos(np.radians(min_elev)) / a)
    nadir_angle = 90 - min_elev - np.degrees(rho)
    coverage_diameter = 2 * R_EARTH * np.radians(nadir_angle)
    
    return {
        'altitude_km': altitude_km,
        'period_hours': period_hr,
        'velocity_km_s': velocity,
        'coverage_diameter_km': coverage_diameter
    }

def gnss_visibility(altitude_km, min_elevation_deg=5):
    """
    Tinh so ve tinh GNSS nhin thay tu mot diem.
    Mo hinh hinh hoc don gian.
    """
    a = R_EARTH + altitude_km
    
    # Goc trung tam Trai dat nhin thay tu ve tinh
    rho = np.arcsin(R_EARTH * np.cos(np.radians(min_elevation_deg)) / a)
    earth_angle = np.pi/2 - np.radians(min_elevation_deg) - rho
    
    # Phan be mat Trai dat nhin thay
    coverage_fraction = (1 - np.cos(earth_angle)) / 2
    
    # Voi chom sao 24 ve tinh phan bo deu
    # Ve tinh nhin thay du kien ≈ 24 × phan vung phu × he so hinh hoc
    # He so hinh hoc tinh den phan bo khong deu
    geometry_factor = 1.5  # thuc nghiem
    expected_visible = 24 * coverage_fraction * geometry_factor
    
    return {
        'coverage_fraction': coverage_fraction,
        'earth_angle_deg': np.degrees(earth_angle),
        'expected_visible_sats': expected_visible
    }

def meo_latency(altitude_km, elevation_deg=30):
    """Tinh do tre lien ket MEO."""
    h = altitude_km
    eps = np.radians(elevation_deg)
    
    # Tam xien
    rho = np.arcsin(R_EARTH * np.cos(eps) / (R_EARTH + h))
    slant_range = (R_EARTH + h) * np.sin(np.pi/2 - eps - rho) / np.cos(eps)
    
    # Do tre
    one_way_ms = slant_range / C * 1000
    
    return {
        'slant_range_km': slant_range,
        'one_way_ms': one_way_ms,
        'round_trip_ms': 2 * one_way_ms
    }

# So sanh cac he thong MEO
print("So sanh He thong MEO")
print("=" * 65)

systems = [
    ("O3b mPOWER", 8062),
    ("GLONASS", 19130),
    ("GPS", 20180),
    ("Galileo", 23222),
]

for name, alt in systems:
    params = meo_parameters(alt)
    latency = meo_latency(alt, 30)
    
    print(f"\n{name} ({alt} km):")
    print(f"  Chu ky: {params['period_hours']:.2f} gio")
    print(f"  Van toc: {params['velocity_km_s']:.2f} km/s")
    print(f"  Duong kinh vung phu: {params['coverage_diameter_km']:.0f} km")
    print(f"  RTT (goc nang 30°): {latency['round_trip_ms']:.1f} ms")

# Phan tich tam nhin GPS
print("\n" + "=" * 65)
print("Phan tich Tam nhin GPS:")
print("-" * 65)

vis = gnss_visibility(20180, 5)
print(f"  Phan vung phu moi ve tinh: {vis['coverage_fraction']*100:.1f}%")
print(f"  Goc Trai dat nhin thay: {vis['earth_angle_deg']:.1f}°")
print(f"  Ve tinh nhin thay du kien: {vis['expected_visible_sats']:.1f}")
print(f"  (Can 4 de dinh vi, thuong thay 6-12)")

# So sanh kich thuoc chom sao
print("\n" + "=" * 65)
print("So sanh Kich thuoc Chom sao:")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Tiep xuc Van Allen.** Ve tinh MEO can tang cuong buc xa; them khoi luong va chi phi.

2. **Ngo nhan vung khe.** 8,000 km cua O3b la buc xa thap hon nhung van can bao ve.

3. **Han che xich dao.** O3b khong the phuc vu vung cuc; khong phai vung phu toan cau thuc su.

4. **Cong huong 12 gio.** Chu ky 11:58 cua GPS duoc chon dac biet de vet mat dat lap lai.

5. **Do phuc tap theo doi.** Khac voi GEO, MEO can anten theo doi; lai duoc hoac mang pha.

6. **Manh vo ton tai lau.** Manh vo MEO khong suy giam tu nhien; ton tai hang ngan nam.

7. **Chi phi phong.** Do cao cao hon = can nhieu delta-v hon = dat hon de dat toi so voi LEO.
