## Tai sao dieu nay quan trong

Quy dao Trai dat Thap (LEO) la noi hanh dong dien ra: Starlink, OneWeb, Kuiper, va hau het cac ve tinh quan sat Trai dat hoat dong o day. LEO cung cap do tre thap (25-50 ms khu hoi) va link budget de quan ly, nhung doi hoi cac chom sao lon de phu toan cau vi moi ve tinh chi nhin thay mot phan nho cua Trai dat. Hieu vat ly LEO la nen tang cho ky thuat truyen thong ve tinh.

## Ap dung

**Tinh toan quy dao LEO:**

```python
import numpy as np

# Hang so
MU_EARTH = 398600.4418  # km³/s² - tham so hap dan Trai dat
R_EARTH = 6371  # km - ban kinh Trai dat
C = 299792.458  # km/s - toc do anh sang

def leo_orbital_parameters(altitude_km):
    """
    Tinh cac tham so quy dao LEO.
    """
    a = R_EARTH + altitude_km  # ban truc lon
    
    # Chu ky quy dao (dinh luat Kepler thu ba)
    period_s = 2 * np.pi * np.sqrt(a**3 / MU_EARTH)
    period_min = period_s / 60
    
    # Van toc quy dao
    velocity_km_s = np.sqrt(MU_EARTH / a)
    
    # Van toc vet mat dat (phep chieu len be mat Trai dat)
    ground_velocity = velocity_km_s * R_EARTH / a
    
    return {
        'altitude_km': altitude_km,
        'semi_major_axis_km': a,
        'period_min': period_min,
        'orbits_per_day': 1440 / period_min,
        'velocity_km_s': velocity_km_s,
        'ground_velocity_km_s': ground_velocity
    }

def coverage_diameter(altitude_km, min_elevation_deg):
    """
    Tinh duong kinh vung phu tren be mat Trai dat.
    """
    h = altitude_km
    eps = np.radians(min_elevation_deg)
    
    # Goc trung tam tu ve tinh den mep vung phu
    # Su dung hinh hoc cau
    rho = np.arcsin(R_EARTH * np.cos(eps) / (R_EARTH + h))
    lambda_0 = np.pi/2 - eps - rho
    
    # Ban kinh va duong kinh vung phu
    coverage_radius_km = R_EARTH * lambda_0
    coverage_diameter_km = 2 * coverage_radius_km
    
    # Dien tich vung phu
    coverage_area_km2 = 2 * np.pi * R_EARTH**2 * (1 - np.cos(lambda_0))
    
    return {
        'coverage_diameter_km': coverage_diameter_km,
        'coverage_area_km2': coverage_area_km2,
        'central_angle_deg': np.degrees(lambda_0)
    }

def link_latency(altitude_km, elevation_deg):
    """
    Tinh do tre mot chieu phan khong gian.
    """
    h = altitude_km
    eps = np.radians(elevation_deg)
    
    # Tinh toan tam xien
    rho = np.arcsin(R_EARTH * np.cos(eps) / (R_EARTH + h))
    slant_range = (R_EARTH + h) * np.sin(np.pi/2 - eps - rho) / np.cos(eps)
    
    # Do tre mot chieu
    delay_ms = slant_range / C * 1000
    
    return {
        'slant_range_km': slant_range,
        'one_way_delay_ms': delay_ms,
        'round_trip_ms': 2 * delay_ms
    }

def min_constellation_size(altitude_km, min_elevation_deg, overlap_factor=1.5):
    """
    Uoc tinh kich thuoc chom sao toi thieu cho vung phu toan cau.
    """
    earth_surface_km2 = 4 * np.pi * R_EARTH**2
    cov = coverage_diameter(altitude_km, min_elevation_deg)
    
    # Ve tinh toi thieu (khong chong lap)
    min_sats = earth_surface_km2 / cov['coverage_area_km2']
    
    # Thuc te voi chong lap
    practical_sats = min_sats * overlap_factor
    
    return {
        'earth_surface_km2': earth_surface_km2,
        'coverage_per_sat_km2': cov['coverage_area_km2'],
        'theoretical_minimum': int(np.ceil(min_sats)),
        'practical_estimate': int(np.ceil(practical_sats))
    }

# Vi du: So sanh cac do cao LEO
print("So sanh Quy dao LEO")
print("=" * 60)

altitudes = [400, 550, 1200]
for alt in altitudes:
    params = leo_orbital_parameters(alt)
    cov = coverage_diameter(alt, 25)
    lat = link_latency(alt, 45)
    
    print(f"\nDo cao: {alt} km")
    print(f"  Chu ky: {params['period_min']:.1f} phut ({params['orbits_per_day']:.1f} vong/ngay)")
    print(f"  Van toc: {params['velocity_km_s']:.2f} km/s")
    print(f"  Duong kinh vung phu (goc nang 25°): {cov['coverage_diameter_km']:.0f} km")
    print(f"  RTT (goc nang 45°): {lat['round_trip_ms']:.1f} ms")

# Kich thuoc chom sao
print("\n" + "=" * 60)
print("Kich thuoc Chom sao (goc nang toi thieu 25°):")
print("-" * 60)

for alt in [550, 1200]:
    size = min_constellation_size(alt, 25)
    print(f"\nDo cao {alt} km:")
    print(f"  Toi thieu ly thuyet: {size['theoretical_minimum']} ve tinh")
    print(f"  Thuc te (chong lap 1.5×): {size['practical_estimate']} ve tinh")

# So sanh chom sao thuc te
print("\n" + "=" * 60)
print("Cac Chom sao LEO Thuc te:")
print("-" * 60)
constellations = [
    ("Starlink Gen1", 550, 53, 1584),
    ("Starlink Gen2", 340, 43, 7500),
    ("OneWeb", 1200, 87.9, 648),
    ("Kuiper", 630, 51.9, 3236),
]
for name, alt, inc, sats in constellations:
    print(f"  {name:15} {alt:4} km, {inc:5.1f}°, {sats:5} ve tinh")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Dai Van Allen.** Tranh 1,000-2,000 km cho cac nhiem vu co nguoi; buc xa dat dinh o do.

2. **Manh vo quy dao.** Cac do cao pho bien (500-600 km) dong duc; rui ro va cham tang.

3. **Hoat dong Mat troi.** Luc can khi quyen tang trong cuc dai Mat troi; ve tinh roi nhanh hon.

4. **Dong bo Mat troi vs nghieng.** SSO (98°) cho quan sat Trai dat; nghieng thap hon cho vien thong.

5. **Nhat thuc.** Ve tinh LEO vao bong Trai dat moi vong; chu ky nhiet pin.

6. **Theo doi Doppler.** Terminal nguoi dung phai xu ly dich ±50 kHz; phuc tap hoa dieu che.

7. **Toc do handover.** Moi 60-90 giay cho LEO vs khong bao gio cho GEO; do phuc tap giao thuc.
