## Tai sao dieu nay quan trong

Quy dao dia tinh (GEO) da cach mang hoa vien thong: dat mot ve tinh o 35,786 km, va no xuat hien tinh tu Trai dat. Dia co dinh huong vao no mai mai—khong can theo doi. Nhan thuc duy nhat nay da cho phep phat song TV toan cau, mang VSAT, va ve tinh thoi tiet. Hieu GEO so voi GSO, khe quy dao, va duy tri tram la can thiet cho ky thuat satcom.

## Ap dung

**Tinh toan quy dao GEO:**

```python
import numpy as np

# Hang so
MU_EARTH = 398600.4418  # km³/s²
R_EARTH = 6371  # km
C = 299792.458  # km/s
SIDEREAL_DAY_S = 86164.1  # giay

def geo_altitude():
    """Tinh do cao GEO tu nguyen ly dau tien."""
    # Voi quy dao dong bo: T = 2π√(a³/μ)
    # Giai a: a = (μ × T² / 4π²)^(1/3)
    
    a_cubed = MU_EARTH * SIDEREAL_DAY_S**2 / (4 * np.pi**2)
    a = a_cubed ** (1/3)  # Ban truc lon (tu tam Trai dat)
    altitude = a - R_EARTH
    
    return {
        'semi_major_axis_km': a,
        'altitude_km': altitude,
        'orbital_period_hours': SIDEREAL_DAY_S / 3600
    }

def geo_coverage(min_elevation_deg=5):
    """Tinh hinh hoc vung phu GEO."""
    h = 35786  # km
    eps = np.radians(min_elevation_deg)
    
    # Vi do toi da nhin thay
    # Tai chan troi (goc nang 0°), vi do max ≈ arccos(R/(R+h))
    max_lat_theoretical = np.degrees(np.arccos(R_EARTH / (R_EARTH + h)))
    
    # Voi rang buoc goc nang toi thieu
    rho = np.arcsin(R_EARTH * np.cos(eps) / (R_EARTH + h))
    max_lat_practical = 90 - np.degrees(eps) - np.degrees(rho)
    
    # Pham vi kinh do vung phu (xap xi cho diem nadir)
    coverage_half_angle = max_lat_practical
    
    # Phan Trai dat nhin thay
    coverage_fraction = (1 - np.cos(np.radians(max_lat_practical))) / 2
    
    return {
        'max_latitude_theoretical': max_lat_theoretical,
        'max_latitude_at_min_elev': max_lat_practical,
        'coverage_fraction': coverage_fraction,
        'coverage_percent': coverage_fraction * 100
    }

def geo_latency(elevation_deg=30):
    """Tinh do tre lien ket GEO."""
    h = 35786  # km
    eps = np.radians(elevation_deg)
    
    # Tam xien
    rho = np.arcsin(R_EARTH * np.cos(eps) / (R_EARTH + h))
    slant_range = (R_EARTH + h) * np.sin(np.pi/2 - eps - rho) / np.cos(eps)
    
    # Do tre
    one_way_ms = slant_range / C * 1000
    
    return {
        'slant_range_km': slant_range,
        'one_way_ms': one_way_ms,
        'round_trip_ms': 2 * one_way_ms,
        'user_to_user_ms': 4 * one_way_ms  # Len + xuong + len + xuong
    }

def station_keeping_budget():
    """Uoc tinh yeu cau delta-V duy tri tram GEO."""
    # Gia tri dien hinh cho duy tri tram GEO
    return {
        'north_south_m_s_year': 50,  # Kiem soat do nghieng
        'east_west_m_s_year': 3,     # Kiem soat kinh do
        'eccentricity_m_s_year': 2,  # Kiem soat do lech tam
        'total_m_s_year': 55,
        'typical_lifetime_years': 15,
        'total_delta_v_m_s': 55 * 15
    }

# Tinh tham so GEO
print("Phan tich Quy dao GEO")
print("=" * 60)

geo = geo_altitude()
print(f"\nDo cao GEO (tinh toan):")
print(f"  Ban truc lon: {geo['semi_major_axis_km']:.1f} km")
print(f"  Do cao: {geo['altitude_km']:.1f} km")
print(f"  Chu ky quy dao: {geo['orbital_period_hours']:.3f} gio")

# Phan tich vung phu
print("\n" + "-" * 60)
print("Phan tich Vung phu:")
coverage = geo_coverage(5)
print(f"  Vi do max (ly thuyet): {coverage['max_latitude_theoretical']:.1f}°")
print(f"  Vi do max (goc nang min 5°): {coverage['max_latitude_at_min_elev']:.1f}°")
print(f"  Vung phu moi ve tinh: {coverage['coverage_percent']:.1f}%")
print(f"  Ve tinh cho toan cau: 3 (ngoai tru cuc)")

# Phan tich do tre
print("\n" + "-" * 60)
print("Phan tich Do tre:")
for elev in [90, 45, 10, 5]:
    lat = geo_latency(elev)
    print(f"  Tai goc nang {elev}°:")
    print(f"    Tam xien: {lat['slant_range_km']:.0f} km")
    print(f"    Mot chieu: {lat['one_way_ms']:.1f} ms")
    print(f"    RTT (hub-terminal): {lat['round_trip_ms']:.1f} ms")

# Duy tri tram
print("\n" + "-" * 60)
print("Ngan sach Duy tri Tram:")
sk = station_keeping_budget()
print(f"  Kiem soat B-N: {sk['north_south_m_s_year']} m/s/nam")
print(f"  Kiem soat D-T: {sk['east_west_m_s_year']} m/s/nam")
print(f"  Tong: {sk['total_m_s_year']} m/s/nam")
print(f"  Vong doi 15 nam: {sk['total_delta_v_m_s']} m/s tong")
```

<section class="lesson-pitfalls">

## Cam bay

1. **GEO ≠ GSO.** GEO la tap con cua GSO; GSO nghieng can anten theo doi.

2. **Khong phu vung cuc.** Tren ~81° vi do, ve tinh GEO o duoi chan troi.

3. **Khan hiem khe.** Khe GEO chinh bi gioi han; phoi hop ITU mat nhieu nam.

4. **Hinh phat do tre.** RTT 550+ ms lam GEO khong phu hop cho ung dung tuong tac.

5. **Nhien lieu duy tri tram.** Dong tac B-N tieu thu 90% nhien lieu; gioi han tuoi tho ve tinh.

6. **Mua nhat thuc.** Hai lan moi nam, ve tinh GEO vao bong Trai dat trong ~72 phut.

7. **Quy dao nghia dia.** Ve tinh het vong doi phai di chuyen len; rac ton tai mai mai.
