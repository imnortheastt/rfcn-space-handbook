## Tai sao dieu nay quan trong

Mot so vi tri trong khong gian cho phep vat the nho quay quanh Mat troi voi cung chu ky nhu Trai dat—mac du o khoang cach khac nhau. Cac diem Lagrange nay phat sinh tu su can bang cua luc hap dan va luc ly tam trong he Mat troi-Trai dat. JWST nam tai L2 (1.5 trieu km tu Trai dat), ve tinh thoi tiet khong gian quan sat tu L1, va L4/L5 giu cac dam may tieu hanh tinh. Hieu cac diem nay mo ra canh cua thiet ke nhiem vu khong gian sau.

## Ap dung

**Tinh toan diem Lagrange:**

```python
import numpy as np

# Hang so
AU = 149597870.7  # km
M_SUN = 1.989e30  # kg
M_EARTH = 5.972e24  # kg
G = 6.674e-11  # m³/(kg·s²)

def lagrange_l1_l2_distance():
    """Tinh khoang cach L1 va L2 tu Trai dat."""
    # Ti le khoi luong
    mu = M_EARTH / (M_SUN + M_EARTH)
    
    # Xap xi hinh cau Hill cho L1, L2
    # r_L ≈ R × (μ/3)^(1/3)
    r_hill = AU * (mu / 3) ** (1/3)
    
    # Chinh xac hon: giai phuong trinh bac ba
    # Xap xi: r_L1 ≈ R × (μ/3)^(1/3) × (1 - μ/3 - μ²/9)
    # r_L2 ≈ R × (μ/3)^(1/3) × (1 + μ/3 - μ²/9)
    
    correction_l1 = 1 - mu/3 - mu**2/9
    correction_l2 = 1 + mu/3 - mu**2/9
    
    r_l1 = r_hill * correction_l1
    r_l2 = r_hill * correction_l2
    
    return {
        'mass_ratio_mu': mu,
        'hill_sphere_km': r_hill,
        'l1_distance_km': r_l1,
        'l2_distance_km': r_l2,
        'l1_distance_lunar': r_l1 / 384400,
        'l2_distance_lunar': r_l2 / 384400
    }

def l4_l5_positions():
    """Tinh vi tri L4 va L5."""
    # L4 va L5 tao tam giac deu voi Mat troi va Trai dat
    # Truoc va sau Trai dat 60° trong quy dao
    
    r_earth = AU
    angle_l4 = 60  # do truoc
    angle_l5 = -60  # do sau (hoac 300°)
    
    # Trong he toa do quy dao Trai dat
    l4_x = r_earth * np.cos(np.radians(angle_l4))
    l4_y = r_earth * np.sin(np.radians(angle_l4))
    
    l5_x = r_earth * np.cos(np.radians(angle_l5))
    l5_y = r_earth * np.sin(np.radians(angle_l5))
    
    # Khoang cach tu Trai dat
    dist_l4 = np.sqrt((l4_x - r_earth)**2 + l4_y**2)
    dist_l5 = np.sqrt((l5_x - r_earth)**2 + l5_y**2)
    
    return {
        'l4_x_au': l4_x / AU,
        'l4_y_au': l4_y / AU,
        'l5_x_au': l5_x / AU,
        'l5_y_au': l5_y / AU,
        'distance_from_earth_km': dist_l4,
        'distance_from_earth_au': dist_l4 / AU
    }

def stability_timescale():
    """Tinh thang thoi gian khong on dinh tai L2."""
    # Dong luc hoc tuyen tinh hoa gan L2 cho phan ky ham mu
    # Thoi gian dac trung τ ≈ T_quy_dao / (2π × √(9μ/2))
    
    mu = M_EARTH / (M_SUN + M_EARTH)
    T_orbit_days = 365.25
    
    # Thoi gian e-folding xap xi
    tau = T_orbit_days / (2 * np.pi * np.sqrt(9 * mu / 2))
    
    return {
        'e_folding_time_days': tau,
        'doubling_time_days': tau * np.log(2),
        'station_keeping_interval_weeks': tau / 7 * 0.5  # Than trong
    }

def transfer_delta_v():
    """Tinh delta-V cho chuyen tiep tu Trai dat den L2."""
    # Tu LEO (400 km) den quy dao halo L2
    
    r_leo = 6371 + 400  # km
    v_leo = np.sqrt(G * M_EARTH / (r_leo * 1000)) / 1000  # km/s
    
    # Van toc thoat tu LEO
    v_escape = v_leo * np.sqrt(2)
    
    # Chuyen tiep L2 it hon chuyen tiep Mat trang
    # vi L2 o ranh gioi anh huong hap dan Trai dat
    
    # Delta-V phun xuyên L2
    v_tli_l2 = 3.2  # km/s (thuc nghiem tu cac nhiem vu)
    
    # Dot phu khoi hanh tu LEO
    delta_v_departure = v_tli_l2
    
    # Hieu chinh giua duong
    delta_v_corrections = 0.05  # km/s
    
    # Nhap L2
    delta_v_insertion = 0.02  # km/s (vao quy dao halo)
    
    return {
        'leo_velocity_km_s': v_leo,
        'escape_velocity_km_s': v_escape,
        'departure_delta_v_km_s': delta_v_departure,
        'corrections_km_s': delta_v_corrections,
        'insertion_km_s': delta_v_insertion,
        'total_km_s': delta_v_departure + delta_v_corrections + delta_v_insertion
    }

# L4/L5
```

<section class="lesson-pitfalls">

## Cam bay

1. **L1/L2/L3 khong on dinh.** Vat the troi xa theo ham mu; duy tri tram la bat buoc.

2. **"Tai L2" khong co nghia o dung diem.** JWST quay quanh L2 trong quy dao halo, khong phai tai vi tri chinh xac.

3. **Do tre truyen thong.** L2 cach ~5 giay anh sang tu Trai dat; khong dieu khien thoi gian thuc.

4. **Khong the cuu ho.** L2 xa gap 4 lan Mat trang; nhiem vu sua chua khong kha thi.

5. **Rui ro che khuat trong quy dao Lissajous.** Mot so duong di cat duong Mat troi-Trai dat, gay mat dien.

6. **On dinh L4/L5 yeu cau ti le khoi luong.** Chi hoat dong neu ti le khoi luong chinh/phu > 25.

7. **Van can nhien lieu.** Ngay ca L4/L5 on dinh cung co nhieu dong tu cac hanh tinh khac can hieu chinh thoi gian.
