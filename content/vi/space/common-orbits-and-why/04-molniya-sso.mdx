## Tai sao dieu nay quan trong

GEO khong the nhin thay vung cuc. Nga da giai quyet van de nay voi quy dao Molniya—duong di elip cao "treo" tren vi do cao trong nhieu gio. Trong khi do, ve tinh quan sat Trai dat can dieu kien chieu sang nhat quan, dat duoc thong qua quy dao dong bo Mat troi (SSO). Cac quy dao dac biet nay lap day khoang trong ma quy dao tron xich dao khong the.

## Ap dung

**Tinh toan quy dao Molniya va SSO:**

```python
import numpy as np

# Hang so
MU_EARTH = 398600.4418  # km³/s²
R_EARTH = 6378.137  # km (xich dao)
J2 = 1.08263e-3
EARTH_ROTATION_DEG_DAY = 360.9856  # ngay thien van
SUN_MOTION_DEG_DAY = 0.9856  # Trai dat quanh Mat troi

def molniya_parameters():
    """Tinh tham so quy dao Molniya."""
    # Chu ky 12 gio
    period_s = 12 * 3600
    
    # Ban truc lon tu dinh luat Kepler thu ba
    a = (MU_EARTH * (period_s / (2 * np.pi))**2) ** (1/3)
    
    # Do nghieng toi han (khong quay apsidal)
    i_critical = np.degrees(np.arcsin(np.sqrt(0.8)))
    
    # Tham so Molniya dien hinh
    perigee = 500  # km
    r_perigee = R_EARTH + perigee
    
    # Do lech tam tu a va r_perigee
    # r_perigee = a(1-e) → e = 1 - r_perigee/a
    e = 1 - r_perigee / a
    
    # Vien diem
    r_apogee = a * (1 + e)
    apogee = r_apogee - R_EARTH
    
    # Van toc
    v_perigee = np.sqrt(MU_EARTH * (2/r_perigee - 1/a))
    v_apogee = np.sqrt(MU_EARTH * (2/r_apogee - 1/a))
    
    return {
        'period_hours': 12,
        'semi_major_axis_km': a,
        'eccentricity': e,
        'inclination_deg': i_critical,
        'perigee_km': perigee,
        'apogee_km': apogee,
        'v_perigee_km_s': v_perigee,
        'v_apogee_km_s': v_apogee,
        'dwell_time_hours': 8  # xap xi tren 60°B
    }

def sso_inclination(altitude_km):
    """Tinh do nghieng can thiet cho quy dao dong bo mat troi."""
    a = R_EARTH + altitude_km
    
    # Chuyen dong trung binh (rad/s)
    n = np.sqrt(MU_EARTH / a**3)
    
    # Toc do RAAN can thiet (rad/s)
    omega_dot_required = np.radians(SUN_MOTION_DEG_DAY) / 86400
    
    # Tu cong thuc di chuyen RAAN (quy dao tron, e=0):
    # Ω̇ = -3/2 × J2 × (Re/a)² × n × cos(i)
    # Giai cos(i)
    cos_i = -omega_dot_required / (1.5 * J2 * (R_EARTH/a)**2 * n)
    
    if abs(cos_i) > 1:
        return None  # Khong co giai phap SSO tai do cao nay
    
    i = np.degrees(np.arccos(cos_i))
    
    # Chu ky
    period_min = 2 * np.pi / n / 60
    
    return {
        'altitude_km': altitude_km,
        'inclination_deg': i,
        'period_min': period_min,
        'is_retrograde': i > 90
    }

def sso_local_time_drift(altitude_km, inclination_error_deg):
    """Tinh troi gio dia phuong tu sai so do nghieng."""
    # SSO danh dinh
    nominal = sso_inclination(altitude_km)
    if nominal is None:
        return None
    
    a = R_EARTH + altitude_km
    n = np.sqrt(MU_EARTH / a**3)
    
    # Toc do RAAN thuc te voi sai so
    i_actual = np.radians(nominal['inclination_deg'] + inclination_error_deg)
    omega_dot_actual = -1.5 * J2 * (R_EARTH/a)**2 * n * np.cos(i_actual)
    
    # Toc do can thiet
    omega_dot_required = np.radians(SUN_MOTION_DEG_DAY) / 86400
    
    # Troi
    drift_rad_s = omega_dot_actual - omega_dot_required
    drift_deg_day = np.degrees(drift_rad_s) * 86400
    
    # Troi gio dia phuong (360° = 24 gio)
    local_time_drift_min_day = drift_deg_day / 360 * 24 * 60
    
    return {
        'inclination_error_deg': inclination_error_deg,
        'raan_drift_deg_day': drift_deg_day,
        'local_time_drift_min_day': local_time_drift_min_day
    }

# Tinh va hien thi
print("Phan tich Quy dao Molniya")
print("=" * 60)

mol = molniya_parameters()
print(f"\nTham so Quy dao Molniya:")
print(f"  Chu ky: {mol['period_hours']} gio")
print(f"  Ban truc lon: {mol['semi_major_axis_km']:.0f} km")
print(f"  Do lech tam: {mol['eccentricity']:.3f}")
print(f"  Do nghieng toi han: {mol['inclination_deg']:.2f}°")
print(f"  Can diem: {mol['perigee_km']:.0f} km")
print(f"  Vien diem: {mol['apogee_km']:.0f} km")
print(f"  Van toc tai can diem: {mol['v_perigee_km_s']:.2f} km/s")
print(f"  Van toc tai vien diem: {mol['v_apogee_km_s']:.2f} km/s")
print(f"  Thoi gian luu tren 60°B: ~{mol['dwell_time_hours']} gio")

print("\n" + "=" * 60)
print("Phan tich Quy dao Dong bo Mat troi")
print("-" * 60)

for alt in [400, 600, 800, 1000, 1200]:
    sso = sso_inclination(alt)
    if sso:
        print(f"  {alt:4d} km → i = {sso['inclination_deg']:.2f}°, "
              f"T = {sso['period_min']:.1f} phut "
              f"{'(nghich hanh)' if sso['is_retrograde'] else ''}")
    else:
        print(f"  {alt:4d} km → Khong co giai phap SSO")

# Phan tich troi
print("\n" + "-" * 60)
print("Troi Gio dia phuong SSO (tai 700 km, moi 0.1° sai so do nghieng):")
drift = sso_local_time_drift(700, 0.1)
if drift:
    print(f"  Troi RAAN: {drift['raan_drift_deg_day']:.4f}°/ngay")
    print(f"  Troi gio dia phuong: {drift['local_time_drift_min_day']:.2f} phut/ngay")
    print(f"  Troi hang nam: {drift['local_time_drift_min_day'] * 365:.0f} phut/nam")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Do nghieng toi han khong phai tuy chon.** Tai bat ky goc nao khac, vien diem Molniya troi—quy dao tro nen vo dung trong vai thang.

2. **Can diem Molniya nam trong dai Van Allen.** Ve tinh chiu buc xa cao hai lan moi quy dao; gioi han tuoi tho linh kien.

3. **SSO chi hoat dong duoi ~1500 km.** Tren do, hieu ung J2 qua yeu de dat toc do di chuyen can thiet.

4. **Do nghieng SSO phai duoc duy tri.** Sai so 0.1° gay ~10 phut troi gio dia phuong moi nam—can duy tri tram.

5. **Quy dao nghich hanh kho phong hon.** SSO yeu cau phong nguoc chieu quay Trai dat—mat ~0.4 km/s.

6. **Molniya 12 gio co vet mat dat lap lai hang ngay.** Chuyen giao giua ve tinh xay ra cung gio moi ngay.

7. **SSO khong co nghia la luon sang.** SSO binh minh-hoang hon luon bay doc duong ranh gioi sang/toi—huu ich cho SAR, khong phai anh quang hoc.
