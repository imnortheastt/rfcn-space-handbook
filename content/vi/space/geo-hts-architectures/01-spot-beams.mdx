## Tai sao dieu nay quan trong

Ve tinh GEO truyen thong phu song toan bo luc dia voi mot bup song rong—dung luong thap, giai phap do tre cao. Ve tinh HTS su dung hang chuc den hang tram bup song diem de tap trung cong suat va tai su dung pho, mang lai dung luong gap 10-100 lan tu cung vi tri quy dao.

## Ap dung

**Tinh toan bup song diem:**

```python
import numpy as np

def spot_beam_diameter(altitude_km, beam_angle_deg, off_nadir_deg=0):
    """
    Tinh duong kinh dau chan bup song diem
    
    altitude_km: do cao ve tinh
    beam_angle_deg: do rong bup nua cong suat (do)
    off_nadir_deg: goc chi huong tu thien de
    
    Tra ve: duong kinh tinh bang km (tuong duong thien de)
    """
    h = altitude_km
    theta = np.radians(beam_angle_deg)
    
    # Bup song thien de
    diameter = 2 * h * np.tan(theta / 2)
    
    # Hieu chinh lech thien de (don gian hoa)
    if off_nadir_deg > 0:
        off_nadir = np.radians(off_nadir_deg)
        diameter *= 1 / np.cos(off_nadir)
    
    return diameter

def antenna_beamwidth(frequency_ghz, diameter_m):
    """
    Uoc tinh do rong bup song anten
    
    Tra ve: do rong bup nua cong suat tinh bang do
    """
    c = 3e8  # m/s
    wavelength = c / (frequency_ghz * 1e9)
    theta_rad = 70 * wavelength / diameter_m
    return np.degrees(theta_rad)

def frequency_reuse_capacity(total_bw_mhz, num_colors, num_beams):
    """
    Tinh bang thong hieu qua voi tai su dung tan so
    
    Tra ve: bang thong hieu qua tinh bang MHz
    """
    bw_per_beam = total_bw_mhz / num_colors
    return bw_per_beam * num_beams

def hts_capacity_estimate(bw_per_beam_mhz, spectral_eff, num_beams):
    """
    Uoc tinh tong dung luong HTS
    
    spectral_eff: bits/s/Hz (thuong 2-5 cho HTS)
    
    Tra ve: dung luong tinh bang Gbps
    """
    capacity_per_beam = bw_per_beam_mhz * spectral_eff / 1000  # Gbps
    return capacity_per_beam * num_beams

# Vi du: HTS Ka-band
print("Thiet ke Bup song Diem HTS Ka-Band:")
print("-" * 50)

freq = 20  # GHz (Ka downlink)
ant_dia = 2.5  # met

beamwidth = antenna_beamwidth(freq, ant_dia)
print(f"Duong kinh anten: {ant_dia} m")
print(f"Do rong bup Ka-band ({freq} GHz): {beamwidth:.2f}°")

diameter = spot_beam_diameter(35786, beamwidth)
print(f"Duong kinh bup song diem (thien de): {diameter:.0f} km")

# Tinh dung luong
total_bw = 500  # MHz kha dung
num_colors = 4
num_beams = 100

eff_bw = frequency_reuse_capacity(total_bw, num_colors, num_beams)
print(f"\nTai su dung tan so:")
print(f"  {total_bw} MHz / {num_colors} mau × {num_beams} bup = {eff_bw/1000:.1f} GHz hieu qua")

capacity = hts_capacity_estimate(total_bw/num_colors, 3.5, num_beams)
print(f"  Dung luong @ 3.5 bits/Hz: {capacity:.0f} Gbps tong")
```

## Cam bay

1. **Hieu suat bien bup.** Nguoi dung tai bien bup thay cong suat it hon 3dB; can du tru thiet ke.

2. **Nhieu bup ke.** Bup cung tan so co the nhieu neu qua gan.

3. **Suy hao mua.** Bup song diem Ka-band chiu suy hao mua dang ke.

4. **Do chinh xac chi huong bup.** Sai so chi huong nho di chuyen bup song hang tram km.

5. **Mat can bang luu luong.** Mot so bup co the tac nghen trong khi bup khac roi.

6. **Do phuc tap anten.** Nhieu bup hon = nhieu bo chieu hon, guong lon hon, chi phi cao hon.

7. **Thiet ke nhiet.** Bo khuech dai cong suat cao trong tai huu ich chat = thach thuc nhiet.
