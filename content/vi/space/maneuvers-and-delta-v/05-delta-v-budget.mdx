## Tai sao dieu nay quan trong

Ngan sach delta-v la bao cao tai chinh cua du hanh vu tru—no cho ban biet nhiem vu co kha thi khong va no se keo dai bao lau. Moi thao tac, tu nang quy dao den tranh va cham, deu rut tu tai khoan huu han nay. Bo sot mot muc va ban se mac ket ve tinh. Uoc tinh qua du tru va ban lang phi suc chua tai trong. Lap ngan sach chinh xac phan biet nhiem vu thanh cong voi that bai dat do.

## Ap dung

**May tinh ngan sach delta-v:**

```python
import numpy as np

def tsiolkovsky(delta_v_m_s, isp_s, khoi_luong_kho_kg):
    """
    Tinh khoi luong nhien lieu su dung phuong trinh Tsiolkovsky.
    """
    g0 = 9.81  # m/s²
    v_thoat = isp_s * g0  # m/s
    
    ti_le_khoi_luong = np.exp(delta_v_m_s / v_thoat)
    nhien_lieu = khoi_luong_kho_kg * (ti_le_khoi_luong - 1)
    khoi_luong_uot = khoi_luong_kho_kg + nhien_lieu
    phan_nhien_lieu = nhien_lieu / khoi_luong_uot
    
    return {
        "ti_le_khoi_luong": ti_le_khoi_luong,
        "nhien_lieu_kg": nhien_lieu,
        "khoi_luong_uot_kg": khoi_luong_uot,
        "phan_nhien_lieu": phan_nhien_lieu,
    }

def tsiolkovsky_nguoc(nhien_lieu_kg, isp_s, khoi_luong_kho_kg):
    """Tinh delta-v kha dung tu khoi luong nhien lieu."""
    g0 = 9.81
    v_thoat = isp_s * g0
    
    khoi_luong_uot = khoi_luong_kho_kg + nhien_lieu_kg
    ti_le_khoi_luong = khoi_luong_uot / khoi_luong_kho_kg
    delta_v = v_thoat * np.log(ti_le_khoi_luong)
    
    return delta_v

def xay_dung_ngan_sach(muc, du_tru_phan_tram=10):
    """
    Xay dung ngan sach delta-v tu cac muc.
    """
    tong_phu = sum(dv for _, dv in muc)
    du_tru = tong_phu * du_tru_phan_tram / 100
    tong = tong_phu + du_tru
    
    return {
        "muc": muc,
        "tong_phu": tong_phu,
        "du_tru_phan_tram": du_tru_phan_tram,
        "du_tru": du_tru,
        "tong": tong,
    }

print("Phan tich Ngan sach Delta-V")
print("=" * 60)
```

<section class="lesson-pitfalls">

## Cam bay

1. **Quen xuong quy dao.** Yeu cau phap quy; phai de danh nhien lieu hoac dung phuong phap thu dong.

2. **Danh gia thap SK B-N.** 50 m/s/nam chiem uu the ngan sach GEO; can nhac van hanh quy dao nghieng.

3. **Bo qua tranh va cham.** Moi truong manh rac xau di; lap ngan sach 1-5 m/s/nam.

4. **Dung het du tru som.** Thao tac khong du kien xay ra; bao toan du tru xuyen suot nhiem vu.

5. **Sai Isp cho van hanh.** Isp EP thay doi theo cong suat; su dung gia tri van hanh thuc te.

6. **Quen xa dong luong.** Banh da phan ung bao hoa; can dong co de giai bao hoa.

7. **Tan man phuong tien phong.** Giao GTO thay doi; lap ngan sach cho truong hop phong xau nhat.
