## Tai sao dieu nay quan trong

DVB-S2 hoat dong trong 1 dB cua gioi han Shannon—toc do du lieu toi da ly thuyet cho ty so tin hieu tren nhieu cho truoc. Thanh tuu dang chu y nay den tu ma LDPC (Kiem tra Chan le Mat do Thap), thay the ma Reed-Solomon/Convolutional noi tiep cua DVB-S. Ket hop voi ma BCH ben ngoai de bat loi con lai, so do ma hoa nay cho phep lien ket ve tinh hoat dong o SNR ma he thong cu se that bai hoan toan. Hieu LDPC va BCH la thiet yeu cho bat ky ai lam viec voi modem ve tinh hien dai hoac thiet ke ngan sach lien ket.

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  DO THI TANNER: Do thi hai phan cua bit va kiem tra                │
│                                                                      │

│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  Tong tat ca tin nhan cho moi bit                     │         │
│  │  LLR > 0 → bit = 0                                   │         │
│  │  LLR < 0 → bit = 1                                   │         │
│  │                                                        │         │
│  │  Neu k.tra van that bai sau toi da vong:             │         │
│  │  → Loi khong sua duoc (hiem, BCH co the van sua)     │         │
```

## Ap dung

**Phan tich tham so LDPC/BCH:**

```python

# Tham so FEC DVB-S2
MA_DVB_S2 = {
    "thuong": {
        "Nldpc": 64800,
        "ty_le": {
            "1/4":  {"Kldpc": 16200,  "t_bch": 12, "Kbch": 16008},

            "3/5":  {"Kldpc": 38880,  "t_bch": 12, "Kbch": 38688},
            "2/3":  {"Kldpc": 43200,  "t_bch": 10, "Kbch": 43040},
            "3/4":  {"Kldpc": 48600,  "t_bch": 12, "Kbch": 48408},
            "4/5":  {"Kldpc": 51840,  "t_bch": 12, "Kbch": 51648},
            "5/6":  {"Kldpc": 54000,  "t_bch": 10, "Kbch": 53840},
            "8/9":  {"Kldpc": 57600,  "t_bch": 8,  "Kbch": 57472},
            "9/10": {"Kldpc": 58320,  "t_bch": 8,  "Kbch": 58192},
        }
    },
    "ngan": {
        "Nldpc": 16200,
        "ty_le": {
            "1/4":  {"Kldpc": 4050,   "t_bch": 12, "Kbch": 3792},
            "1/3":  {"Kldpc": 5400,   "t_bch": 12, "Kbch": 5232},
            "1/2":  {"Kldpc": 8100,   "t_bch": 12, "Kbch": 7932},
            "2/3":  {"Kldpc": 10800,  "t_bch": 12, "Kbch": 10632},
            "3/4":  {"Kldpc": 12150,  "t_bch": 12, "Kbch": 11982},
            "5/6":  {"Kldpc": 13500,  "t_bch": 12, "Kbch": 13332},
        }
    }
}

# Es/N0 yeu cau cho van hanh gan nhu khong loi (QEF)
ES_N0_YEU_CAU = {
    "1/4": -2.35, "1/3": -1.24, "2/5": -0.30, "1/2": 1.00,
    "3/5": 2.23, "2/3": 3.10, "3/4": 4.03, "4/5": 4.68,
    "5/6": 5.18, "8/9": 6.20, "9/10": 6.42
}

def gioi_han_shannon(ty_le):
    """Gioi han Shannon cho kenh AWGN dau vao nhi phan (Eb/N0 theo dB)."""
    return 10 * np.log10((2 ** (2 * ty_le) - 1) / (2 * ty_le))

print("Phan tich Ma LDPC/BCH DVB-S2")
print("=" * 70)

print("\n1. Tham so Ma (Khung Thuong)")
print("-" * 70)

print(f"  {'T.le':<6} {'Kbch':<8} {'Kldpc':<8} {'Nldpc':<8} {'BCH t':<7} {'Chan le LDPC':<12}")
print("-" * 70)

for ty_le, tham_so in MA_DVB_S2["thuong"]["ty_le"].items():
    Nldpc = MA_DVB_S2["thuong"]["Nldpc"]
    chan_le_ldpc = Nldpc - tham_so["Kldpc"]
    print(f"  {ty_le:<6} {tham_so['Kbch']:<8} {tham_so['Kldpc']:<8} {Nldpc:<8} {tham_so['t_bch']:<7} {chan_le_ldpc:<12}")

print("\n2. Phan tich Hieu suat Ma")
print("-" * 70)

print(f"  {'T.le':<6} {'Bit t.tin':<10} {'Tong bit':<12} {'Overhead %':<12} {'T.le Thuc te':<12}")
print("-" * 70)

for ty_le, tham_so in MA_DVB_S2["thuong"]["ty_le"].items():
    Nldpc = MA_DVB_S2["thuong"]["Nldpc"]
    bit_thong_tin = tham_so["Kbch"] - 80  # Tru BBHeader
    overhead = (Nldpc - bit_thong_tin) / bit_thong_tin * 100
    ty_le_thuc = bit_thong_tin / Nldpc
    print(f"  {ty_le:<6} {bit_thong_tin:<10} {Nldpc:<12} {overhead:<12.1f} {ty_le_thuc:<12.3f}")

print("\n3. Khoang cach den Gioi han Shannon")
print("-" * 70)

print(f"  {'T.le':<6} {'Es/N0 y.cau':<12} {'Shannon':<10} {'K.cach (dB)':<10} {'Do loi ma':<12}")
print("-" * 70)

qpsk_khong_ma = 9.6  # dB cho BER = 10^-5

for ty_le, es_n0 in ES_N0_YEU_CAU.items():
    ty_le_so = eval(ty_le)
    shannon = gioi_han_shannon(ty_le_so)
    khoang_cach = es_n0 - shannon
    do_loi_ma = qpsk_khong_ma - es_n0
    print(f"  {ty_le:<6} {es_n0:<12.2f} {shannon:<10.2f} {khoang_cach:<10.2f} {do_loi_ma:<12.1f}")

print("\n4. Do phuc tap Bo giai ma LDPC")
print("-" * 70)

vong_lap_dien_hinh = 25
vong_lap_toi_da = 50

for loai_khung in ["thuong", "ngan"]:
    Nldpc = MA_DVB_S2[loai_khung]["Nldpc"]
    
    # Uoc tinh: ~10 phep toan moi bit moi vong
    phep_toan_moi_khung = Nldpc * vong_lap_dien_hinh * 10
    
    # O 30 Msps voi QPSK (2 bit/ky hieu)
    toc_do_ky_hieu = 30e6
    bit_moi_ky_hieu = 2
    khung_moi_giay = toc_do_ky_hieu * bit_moi_ky_hieu / Nldpc
    
    gops = phep_toan_moi_khung * khung_moi_giay / 1e9
    
    print(f"  Khung {loai_khung} ({Nldpc} bit):")
    print(f"    Khung/giay o 30 Msps QPSK: {khung_moi_giay:.1f}")
    print(f"    GOPS uoc tinh: {gops:.1f}")
    print()

print("5. Kha nang Sua loi BCH")
print("-" * 70)

print(f"  {'T.le':<6} {'BCH t':<8} {'Bit chan le BCH':<16} {'San loi':<15}")
print("-" * 70)

for ty_le, tham_so in MA_DVB_S2["thuong"]["ty_le"].items():
    t = tham_so["t_bch"]
    bit_chan_le = tham_so["Kldpc"] - tham_so["Kbch"]
    san = f"< 10^-{7 + t//2}"
    print(f"  {ty_le:<6} {t:<8} {bit_chan_le:<16} {san:<15}")

print("\n6. Phan tich Do tre")
print("-" * 70)

toc_do_ky_hieu_msps = 30

for loai_khung in ["thuong", "ngan"]:
    Nldpc = MA_DVB_S2[loai_khung]["Nldpc"]
    
    # Do tre ma hoa (toan khung)
    ky_hieu_ma_hoa = Nldpc / 2  # QPSK
    thoi_gian_ma_hoa_us = ky_hieu_ma_hoa / toc_do_ky_hieu_msps
    
    # Do tre giai ma (vong lap)
    thoi_gian_giai_ma_us = thoi_gian_ma_hoa_us + vong_lap_dien_hinh * 10
    
    print(f"  Khung {loai_khung}:")
    print(f"    Thoi luong khung: {thoi_gian_ma_hoa_us:.0f} µs")
    print(f"    Do tre ma hoa: ~{thoi_gian_ma_hoa_us:.0f} µs (mot khung)")
    print(f"    Do tre giai ma: ~{thoi_gian_giai_ma_us:.0f} µs ({vong_lap_dien_hinh} vong)")
    print()
```

<section class="lesson-pitfalls">

## Cam bay

1. **Bo qua overhead BCH.** BCH them 128-192 bit; tinh vao tinh toan ty le.

2. **So vong lap co dinh.** Ket thuc som tiet kiem 30-50% dien. Kiem tra modem co ho tro.

3. **Khung ngan kem hieu qua.** Khung ngan co hieu nang kem ~0.3 dB so voi khung thuong cung ty le.

4. **San loi LDPC.** Khong co BCH, LDPC dat san o ~10⁻⁷ BER. BCH la thiet yeu cho van hanh QEF.

5. **Thong luong bo giai ma.** LDPC can ~1-2 GOPS o 30 Msps. Xac minh FPGA/ASIC xu ly duoc.

6. **Gia dinh xen kenh.** Bo xen kenh bit DVB-S2 khac theo dieu che. Dung trao doi bo xen QPSK/8PSK.

</section>
