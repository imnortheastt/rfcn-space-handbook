## Tai sao dieu nay quan trong

Chon MODCOD la quyet dinh quan trong nhat trong thiet ke lien ket ve tinh. Chon sai, va ban hoac lang phi mot nua dung luong transponder (qua than trong) hoac chiu gian doan lien tuc (qua mao hiem). DVB-S2 cung cap 28 MODCOD; DVB-S2X them 88 nua. Moi to hop dieu che va ty le ma co nguong SNR cu the va hieu suat pho rieng. Bai hoc nay cung cap bang tra cuu day du va truc giac de chon MODCOD dung cho bat ky lien ket nao.

### Quy uoc dat ten MODCOD

```
Hieu Ten MODCOD:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  MODCOD = Dieu che + Ty le ma                                       │
│                                                                      │
│  Vi du: "16APSK 3/4"                                                │
│          │      │                                                    │
│          │      └── Ty le ma: 3/4 bit la du lieu huu ich           │
│          │          (1/4 la chan le cho sua loi)                    │
│          │                                                           │
│          └── Dieu che: Khoa Dich Pha-Bien do 16 diem               │
│              (4 bit moi ky hieu)                                     │
│                                                                      │
│  BIT MOI KY HIEU THEO DIEU CHE:                                     │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  Dieu che   │ Bit/Ky hieu │ Diem Chom sao            │         │
│  │  ───────────┼─────────────┼─────────────────────      │         │
│  │  QPSK       │ 2           │ 4                        │         │
│  │  8PSK       │ 3           │ 8                        │         │
│  │  16APSK     │ 4           │ 16                       │         │
│  │  32APSK     │ 5           │ 32                       │         │
│  │  64APSK     │ 6           │ 64   (chi S2X)          │         │
│  │  128APSK    │ 7           │ 128  (chi S2X)          │         │
│  │  256APSK    │ 8           │ 256  (chi S2X)          │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  CONG THUC HIEU SUAT PHO:                                           │
│                                                                      │
│  η = (bit moi ky hieu) × (ty le ma)                                │
│                                                                      │
│  Vi du: 16APSK 3/4                                                  │
│  η = 4 × (3/4) = 3.0 bps/Hz (ly thuyet)                            │
│  Thuc te: 2.97 bps/Hz (tinh ca overhead)                           │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Luong chon MODCOD

```
Cach Chon MODCOD:

┌─────────────────────────────────────────────────────────────────────┐

│  │                                                        │         │
```

```python
def chon_modcod(es_n0_db, margin_db=1.0, modcods=MODCODS_S2):
    """Chon MODCOD tot nhat cho Es/N0 cho truoc."""
    co_san = es_n0_db - margin_db
    ung_cu = [mc for mc in modcods if mc["nguong"] <= co_san]
    if not ung_cu:
        return None
    return max(ung_cu, key=lambda x: x["hs"])

    """Tinh toc do du lieu theo Mbps."""
    return toc_do_ky_hieu_msps * modcod["hs"]

bang_thong_transponder_mhz = 36
rolloff = 0.20
toc_do_ky_hieu = bang_thong_transponder_mhz / (1 + rolloff)

print(f"  Transponder: {bang_thong_transponder_mhz} MHz, Roll-off: {rolloff*100:.0f}%")
print(f"  Toc do ky hieu: {toc_do_ky_hieu:.2f} Msps")

print(f"  {'MODCOD':<14} {'Hieu suat':<12} {'Toc do D.lieu':<14} {'Ghi chu':<20}")

vi_du_modcods = ["QPSK 1/2", "QPSK 3/4", "8PSK 2/3", "16APSK 3/4", "32APSK 5/6"]

for ten in vi_du_modcods:

    toc_do_du_lieu = tinh_toc_do_du_lieu(toc_do_ky_hieu, mc)
    ghi_chu = f"Can >{mc['nguong']:.1f} dB"

# So sanh dieu che
print("\n4. So sanh Ho Dieu che")
print("-" * 70)

dieu_che = ["QPSK", "8PSK", "16APSK", "32APSK"]

for dc in dieu_che:
    ho_dc = [mc for mc in MODCODS_S2 if mc["dieu_che"] == dc]
    nguong_min = min(mc["nguong"] for mc in ho_dc)
    nguong_max = max(mc["nguong"] for mc in ho_dc)
    hs_min = min(mc["hs"] for mc in ho_dc)
    hs_max = max(mc["hs"] for mc in ho_dc)
    
    print(f"  {dc}:")
    print(f"    Dai hieu suat: {hs_min:.2f} - {hs_max:.2f} bps/Hz")
    print(f"    Dai SNR: {nguong_min:.1f} - {nguong_max:.1f} dB")
    print()

# Dai ACM

print("-" * 70)

acm_min = min(mc["nguong"] for mc in MODCODS_S2)
acm_max = max(mc["nguong"] for mc in MODCODS_S2)
```

<section class="lesson-pitfalls">

## Cam bay

1. **Nham C/N va Es/N0.** Chung khac nhau 10·log(B/Rs). Dung dung don vi khi tra cuu MODCOD.

2. **Quen ton that trien khai.** Modem thuc te kem ~0.5-1 dB so voi nguong bang.

3. **Bo qua phat khung ngan.** Khung ngan kem ~0.2-0.5 dB so voi khung thuong cung ty le.

4. **Tron bang DVB-S2 va S2X.** MODCOD S2X yeu cau thiet bi tuong thich S2X.

5. **Lui HPA cho APSK bac cao.** 32APSK can lui dau ra 2-3 dB, giam EIRP hieu dung.

6. **MODCOD tinh trong dieu kien thay doi.** Dung ACM cho lien ket co suy giam mua.
