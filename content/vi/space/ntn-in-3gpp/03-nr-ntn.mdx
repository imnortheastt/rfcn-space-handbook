## Tai sao dieu nay quan trong

NR-NTN la noi 5G dua tren ve tinh tro thanh hien thuc. Release 17 (2022) lay cac nghien cuu (TR 38.811, TR 38.821) va chuyen chung thanh dac ta quy pham. Bay gio cac nha san xuat chipset nhu Qualcomm, MediaTek, va Skylo co the che tao modem NR-NTN hoat dong voi bat ky mang ve tinh tuan thu nao. Day la tieu chuan dang sau T-Mobile/Starlink direct-to-cell, AST SpaceMobile, va bang rong hang hai/hang khong the he tiep theo.

### Co che timing advance

```
Timing Advance NR-NTN (Quy pham):

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  CONG THUC:                                                         │
│                                                                      │
│  N_TA = N_TA,common + N_TA,offset + N_TA,UE                        │
│                                                                      │
│  Trong do:                                                          │
│    N_TA,common = ta-Common tu SIB19 (don vi T_c × 64)             │
│    N_TA,offset = UE tinh tu GNSS + ephemeris                       │
│    N_TA,UE = dieu chinh TA NR chuan tu MAC-CE                     │
│                                                                      │
│  DO PHAN GIAI THOI GIAN:                                           │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                                 │ │
│  │  T_c = 1 / (480,000 × 4096) = 0.509 ns                        │ │
│  │  T_c × 64 = 32.55 ns                                          │ │
│  │  Pham vi N_TA,common: 0 den 2^20 - 1 (len den 34 ms)          │ │
│  │                                                                 │ │
│  │  So sanh voi NR mat dat:                                       │ │
│  │    Pham vi N_TA: 0 den 3846 × 16 (len den ~2 ms)              │ │
│  │                                                                 │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                      │
│  TINH TOAN TA CUA UE:                                              │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                                 │ │
│  │  1. Lay vi tri ECEF ve tinh [x_s, y_s, z_s] tu SIB19          │ │
│  │  2. Lay vi tri ECEF UE [x_u, y_u, z_u] tu GNSS                │ │
│  │  3. Lay vi tri ECEF tham chieu [x_r, y_r, z_r] tu SIB19       │ │
│  │                                                                 │ │
│  │  4. Tinh khoang cach nghieng:                                  │ │
│  │     d_UE = sqrt((x_s-x_u)² + (y_s-y_u)² + (z_s-z_u)²)         │ │
│  │     d_ref = sqrt((x_s-x_r)² + (y_s-y_r)² + (z_s-z_r)²)        │ │
│  │                                                                 │ │
│  │  5. TA_offset = 2 × (d_UE - d_ref) / c                        │ │
│  │     (he so 2 cho khu hoi trong che do trong suot)              │ │
│  │                                                                 │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## Ap dung

**May tinh ngan sach lien ket va timing NR-NTN:**

```python
import math

def ngan_sach_lien_ket_nr_ntn(tan_so_ghz, do_cao_ve_tinh_km, goc_nang_do, 
                               ue_eirp_dbm, ve_tinh_gt_db, bang_thong_mhz):
    """
    Tinh ngan sach lien ket NR-NTN cho duong len.
    """
    ban_kinh_trai_dat = 6371
    el_rad = math.radians(goc_nang_do)
    
    khoang_cach_nghieng = (-ban_kinh_trai_dat * math.sin(el_rad) + 
                           math.sqrt((ban_kinh_trai_dat * math.sin(el_rad))**2 + 
                                    2 * ban_kinh_trai_dat * do_cao_ve_tinh_km + do_cao_ve_tinh_km**2))
    
    fspl = 32.45 + 20 * math.log10(tan_so_ghz * 1000) + 20 * math.log10(khoang_cach_nghieng)
    
    suy_hao_khi_quyen = 0.5 if tan_so_ghz < 10 else 2.0
    
    cong_suat_nhan = ue_eirp_dbm - fspl - suy_hao_khi_quyen + ve_tinh_gt_db
    
    he_so_nhieu = 3.0
    k_boltz = -228.6
    nhieu_nhiet = k_boltz + 10 * math.log10(bang_thong_mhz * 1e6) + he_so_nhieu
    
    snr = cong_suat_nhan - nhieu_nhiet
    
    return {
        "khoang_cach_nghieng_km": khoang_cach_nghieng,
        "fspl_db": fspl,
        "suy_hao_khi_quyen_db": suy_hao_khi_quyen,
        "cong_suat_nhan_dbm": cong_suat_nhan,
        "nhieu_nhiet_dbm": nhieu_nhiet,
        "snr_db": snr
    }

def uoc_tinh_thong_luong_nr_ntn(snr_db, bang_thong_mhz, harq_enabled):
    """
    Uoc tinh thong luong NR-NTN dua tren SNR.
    """
    if snr_db < -3:
        se = 0.15
        dieu_che = "QPSK 1/8"
    elif snr_db < 0:
        se = 0.4
        dieu_che = "QPSK 1/3"
    elif snr_db < 3:
        se = 0.8
        dieu_che = "QPSK 1/2"
    elif snr_db < 6:

    elif snr_db < 10:
        se = 2.0
        dieu_che = "16QAM 1/2"
    else:
        se = 3.0
        dieu_che = "16QAM 3/4"
    
    overhead = 0.75 if harq_enabled else 0.70
    
    thong_luong_mbps = bang_thong_mhz * se * overhead
    
    return {
        "snr_db": snr_db,
        "hieu_qua_pho": se,
        "dieu_che": dieu_che,
        "thong_luong_mbps": thong_luong_mbps
    }
```

## Cam bay

1. **Quen lien ket feeder cho trong suot.** Duong UE-gateway co HAI nhay khong gian. Ngan sach lien ket phai bao gom ca hai.

2. **Su dung sai do phan giai TA.** NR-NTN su dung don vi T_c × 64 cho N_TA,common, khong phai T_c × 16 NR chuan.

3. **Bo qua rang buoc bang.** Bang S (n256) chi co 30 MHz tong. Dung lap ke hoach cho song mang 100 MHz.

4. **Gia dinh HARQ luon hoat dong.** Trien khai GEO thuong vo hieu hoa HARQ. Lap ke hoach cho truyen lai lop RLC.

5. **Danh gia thap han che UL.** EIRP smartphone (23 dBm) + anten omni → UL la nut that. DL co the nhanh gap 10 lan.
