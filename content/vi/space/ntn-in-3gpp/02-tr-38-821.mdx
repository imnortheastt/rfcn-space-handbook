## Tai sao dieu nay quan trong

TR 38.821 la noi ly thuyet gap ky thuat. Trong khi TR 38.811 xac dinh cac thach thuc cua 5G dua tren ve tinh, TR 38.821 dinh nghia cac giai phap thuc te: cach mo rong timing advance cho khoang cach 600+ km, cach sua doi HARQ cho khu hoi 500+ ms, va cach xu ly handover beam khi ve tinh bay ngang qua bau troi. Tai lieu nay dat nen mong cho cac dac ta NTN quy pham trong Release 17.

## Ap dung

**May tinh bu timing va Doppler TR 38.821:**

```python
import math

def tinh_thanh_phan_ta(ue_lat, ue_lon, sat_lat, sat_lon, sat_alt_km, 
                       ref_lat, ref_lon):
    """
    Tinh TA-Common va TA-Offset theo TR 38.821.
    """
    def khoang_cach_nghieng(lat1, lon1, lat2, lon2, alt):
        R = 6371
        lat1, lon1, lat2, lon2 = map(math.radians, [lat1, lon1, lat2, lon2])
        
        dlat = lat2 - lat1
        dlon = lon2 - lon1
        a = math.sin(dlat/2)**2 + math.cos(lat1) * math.cos(lat2) * math.sin(dlon/2)**2
        c = 2 * math.asin(math.sqrt(a))
        kc_mat_dat = R * c
        
        nghieng = math.sqrt(kc_mat_dat**2 + alt**2 + 2 * R * alt)
        
        return nghieng
    
    c = 299792.458
    
    ref_nghieng = khoang_cach_nghieng(ref_lat, ref_lon, sat_lat, sat_lon, sat_alt_km)
    ta_common_ms = (2 * ref_nghieng / c) * 1000
    
    ue_nghieng = khoang_cach_nghieng(ue_lat, ue_lon, sat_lat, sat_lon, sat_alt_km)
    ta_offset_ms = (2 * (ue_nghieng - ref_nghieng) / c) * 1000
    
    tong_ta_ms = ta_common_ms + ta_offset_ms
    
    return {
        "ref_nghieng_km": ref_nghieng,
        "ue_nghieng_km": ue_nghieng,
        "ta_common_ms": ta_common_ms,
        "ta_offset_ms": ta_offset_ms,
        "tong_ta_ms": tong_ta_ms
    }

def tinh_bu_doppler(ue_lat, ue_lon, van_toc_ve_tinh_km_s, 
                    huong_ve_tinh_do, tan_so_ghz):
    """
    Tinh dich Doppler va bu can thiet.
    """
    c = 299792.458
    
    doppler_hz = (van_toc_ve_tinh_km_s / c) * tan_so_ghz * 1e9 * math.cos(math.radians(30))
    
    bu_hz = -doppler_hz
    
    return {
        "doppler_hz": doppler_hz,
        "bu_hz": bu_hz,
        "doppler_khz": doppler_hz / 1000,
        "tuong_doi_15khz_scs": abs(doppler_hz) / 15000 * 100,
        "tuong_doi_30khz_scs": abs(doppler_hz) / 30000 * 100
    }

def tinh_tien_trinh_harq(rtt_ms, thoi_luong_slot_ms):
    """
    Tinh yeu cau tien trinh HARQ cho NTN.
    """
    slot_dang_bay = math.ceil(rtt_ms / thoi_luong_slot_ms)
    
    tien_trinh_toi_thieu = slot_dang_bay + 1
    
    tien_trinh_chuan = 16
    tien_trinh_mo_rong = 32
    
    if tien_trinh_toi_thieu <= tien_trinh_chuan:
        khuyen_nghi = tien_trinh_chuan
        che_do = "NR Chuan"
    elif tien_trinh_toi_thieu <= tien_trinh_mo_rong:
        khuyen_nghi = tien_trinh_mo_rong
        che_do = "NTN Mo rong"
    else:
        khuyen_nghi = None
        che_do = "Vo hieu hoa HARQ"
    
    return {
        "rtt_ms": rtt_ms,
        "slot_dang_bay": slot_dang_bay,
        "tien_trinh_toi_thieu": tien_trinh_toi_thieu,
        "tien_trinh_khuyen_nghi": khuyen_nghi,
        "che_do": che_do
    }

print("Phan tich Giai phap NTN TR 38.821")
print("=" * 70)

print("\n1. Phan giai Timing Advance (LEO-600)")
print("-" * 70)

ta = tinh_thanh_phan_ta(
    ue_lat=40.0, ue_lon=-74.0,
    sat_lat=45.0, sat_lon=-70.0, sat_alt_km=600,
    ref_lat=42.0, ref_lon=-72.0
)

print(f"""
  Diem tham chieu (tam beam):
    Vi tri: 42.0°N, 72.0°W
    Khoang cach nghieng: {ta['ref_nghieng_km']:.1f} km
    
  Vi tri UE:
    Vi tri: 40.0°N, 74.0°W  
    Khoang cach nghieng: {ta['ue_nghieng_km']:.1f} km
    
  Thanh phan Timing Advance:
    TA-Common (phat):     {ta['ta_common_ms']:.2f} ms
    TA-Offset (UE tinh):  {ta['ta_offset_ms']:.2f} ms

""")

print("\n2. Yeu cau Tien trinh HARQ")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Gia dinh UE co GNSS hoan hao.** UE trong nha hoac hem pho thanh thi co the co GNSS kem. Co che du phong ton tai nhung giam hieu suat.

2. **Bo qua ephemeris cu.** Vi tri ve tinh thay doi — ephemeris cu gay loi timing/Doppler. SIB19 phai duoc lam moi thuong xuyen.

3. **Quen timing lien ket feeder.** Cho tai trong trong suot, tre gateway-den-ve tinh cong vao tong TA. Bao gom ca hai nhay.

4. **Su dung sai dau Doppler.** Doppler duong khi ve tinh toi gan, am khi xa di. Bu truoc phai dung dau nguoc lai.

5. **Danh gia thap tan suat HO.** Chom sao LEO co the yeu cau 50+ handover moi gio. Overhead tin hieu la dang ke.
