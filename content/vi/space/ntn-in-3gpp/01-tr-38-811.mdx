## Tai sao dieu nay quan trong

TR 38.811 la noi tham vong ve tinh cua 3GPP tro thanh hien thuc. Phat hanh nam 2019, Bao cao Ky thuat nay dinh nghia mo hinh kenh, kich ban trien khai, va thach thuc ky thuat cho viec tich hop mang khong mat dat (NTN) vao 5G NR. Moi dac ta NTN tiep theo deu xay dung tren nen tang 38.811. Neu ban muon hieu tai sao NTN hoat dong theo cach no hoat dong — cong thuc timing advance, yeu cau bu Doppler, sua doi HARQ — hay bat dau tu day.

## Ap dung

**May tinh ngan sach lien ket va tre TR 38.811:**

```python
import math

def suy_hao_khong_gian_tu_do(tan_so_ghz, khoang_cach_km):
    """
    Tinh suy hao khong gian tu do theo dB.
    FSPL = 32.45 + 20*log10(f_MHz) + 20*log10(d_km)
    """
    tan_so_mhz = tan_so_ghz * 1000
    fspl = 32.45 + 20 * math.log10(tan_so_mhz) + 20 * math.log10(khoang_cach_km)
    return fspl

def suy_hao_duong_truyen_ntn(tan_so_ghz, khoang_cach_km, moi_truong, goc_nang_do):
    """
    Tinh tong suy hao duong truyen NTN theo TR 38.811.
    """
    fspl = suy_hao_khong_gian_tu_do(tan_so_ghz, khoang_cach_km)
    
    bang_clutter = {
        "do_thi_day_dac": 23,
        "do_thi": 17,
        "ngoai_o": 10,
        "nong_thon": 4
    }
    clutter = bang_clutter.get(moi_truong, 10)
    
    if tan_so_ghz < 10:
        khi_quyen = 0.3
    else:
        khi_quyen = 2 + 0.1 * (tan_so_ghz - 10)
    
    bong = 4.0
    
    he_so_nang = max(0, (90 - goc_nang_do) / 90)
    clutter_dieu_chinh = clutter * he_so_nang
    
    tong = fspl + clutter_dieu_chinh + khi_quyen + bong
    
    return {
        "fspl_db": fspl,
        "clutter_db": clutter_dieu_chinh,
        "khi_quyen_db": khi_quyen,
        "bong_db": bong,
        "tong_db": tong
    }

def dich_doppler(tan_so_ghz, van_toc_ve_tinh_km_s, goc_nang_do):
    """
    Tinh dich Doppler toi da.
    """
    c = 299792.458
    theta = math.radians(goc_nang_do)
    
    doppler_toi_da = (van_toc_ve_tinh_km_s / c) * tan_so_ghz * 1e9 * math.cos(theta)
    
    return doppler_toi_da

def tre_lan_truyen(do_cao_km, goc_nang_do):
    """
    Tinh tre lan truyen mot chieu.
    """
    ban_kinh_trai_dat = 6371
    el_rad = math.radians(goc_nang_do)
    
    khoang_cach_nghieng = (-ban_kinh_trai_dat * math.sin(el_rad) + 
                           math.sqrt((ban_kinh_trai_dat * math.sin(el_rad))**2 + 
                                    2 * ban_kinh_trai_dat * do_cao_km + do_cao_km**2))
    
    c = 299792.458
    tre_ms = (khoang_cach_nghieng / c) * 1000
    
    return khoang_cach_nghieng, tre_ms

def yeu_cau_timing_advance(do_cao_km, goc_nang_do, loai_tai_trong="trong_suot"):
    """
    Tinh timing advance can thiet cho NTN.
    """
    khoang_cach_nghieng, mot_chieu_ms = tre_lan_truyen(do_cao_km, goc_nang_do)
    
    if loai_tai_trong == "trong_suot":
        rtt = mot_chieu_ms * 4
    else:
        rtt = mot_chieu_ms * 2
    
    ta_can_thiet = rtt / 2
    
    ta_chuan_toi_da = 2.0
    
    return {
        "khoang_cach_nghieng_km": khoang_cach_nghieng,
        "tre_mot_chieu_ms": mot_chieu_ms,
        "khu_hoi_ms": rtt,
        "ta_can_thiet_ms": ta_can_thiet,
        "ta_chuan_toi_da_ms": ta_chuan_toi_da,
        "vuot_chuan": ta_can_thiet > ta_chuan_toi_da
    }

print("Phan tich NTN TR 38.811")
print("=" * 70)

kich_ban = [
    ("LEO-600", 600, 7.5, 30),
    ("LEO-1200", 1200, 7.35, 30),
    ("MEO-8000", 8000, 4.9, 20),
    ("GEO", 35786, 3.07, 10),
]

print("\n1. Phan tich Tre Lan truyen")
print("-" * 70)
print(f"  {'Kich ban':<12} {'Do cao (km)':>10} {'Nghieng (km)':>12} {'1-chieu (ms)':>12} {'RTT (ms)':>10}")
print(f"  {'-'*58}")

for ten, cao, vel, el in kich_ban:
    ta = yeu_cau_timing_advance(cao, el, "trong_suot")
    print(f"  {ten:<12} {cao:>10} {ta['khoang_cach_nghieng_km']:>12.0f} {ta['tre_mot_chieu_ms']:>12.1f} {ta['khu_hoi_ms']:>10.1f}")

print("\n2. Phan tich Dich Doppler (bang S: 2 GHz)")
print("-" * 70)
print(f"  {'Kich ban':<12} {'Van toc':>12} {'Goc nang':>8} {'Doppler toi da':>14}")
print(f"  {'-'*48}")

for ten, cao, vel, el in kich_ban:
    doppler = dich_doppler(2.0, vel, el)
    print(f"  {ten:<12} {vel:>10.1f}km/s {el:>6}° {doppler:>12.0f} Hz")

print("""

THONG TIN CHINH tu TR 38.811:
5G NR chuan khong the hoat dong cho NTN ma khong sua doi:
• Timing advance (can bu truoc dua tren GNSS)
• HARQ (can nhieu tien trinh hon hoac vo hieu hoa)
• Doppler (can bu truoc hoac SCS rong hon)
• Handover (cell LEO di chuyen nhanh → handover thuong xuyen)
```

|---------|-----------|

<section class="lesson-pitfalls">

## Cam bay

1. **Su dung mo hinh kenh mat dat.** Mo hinh kenh TR 38.811 bao gom LOS, clutter, va fading phu thuoc goc nang — mo hinh mat dat khong ap dung.

2. **Bo qua lien ket feeder.** Cho ve tinh trong suot, duong UE-gateway di qua hai nhay — dung quen lien ket gateway-den-ve tinh.

3. **Gia dinh Doppler tinh.** Doppler LEO thay doi nhanh trong mot luot bay. Gia tri o chan troi khac voi tren dinh 100%.

4. **Nham lan TR va TS.** TR 38.811 la nghien cuu (thong tin). Dac ta quy pham nam trong TS 38.300 va TS 38.101.

5. **Quen suy hao mua.** NTN bang Ka chiu suy hao mua nghiem trong (10+ dB). Bang S ben vung hon nhung han che bang thong.
