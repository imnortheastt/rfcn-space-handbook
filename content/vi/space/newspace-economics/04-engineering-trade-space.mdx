## Tai sao dieu nay quan trong

Hang khong vu tru truyen thong toi uu cho hieu nang: kha nang toi da, rui ro toi thieu, mac ke chi phi. NewSpace lat nguoc dieu nay: toi uu cho chi phi, chap nhan hieu nang "du tot". Ve tinh GEO co the dung bo xu ly chiu buc xa $200k voi thoi gian su dung dam bao 15 nam. Starlink dung chip o to $5, chap nhan ty le hong 2%, va thay the ve tinh chet. Ca hai deu hop ly — cho cac mo hinh kinh doanh khac nhau. Hieu phan tich khong gian danh doi cho ban biet cach dua ra nhung quyet dinh nay mot cach co he thong, khong phai bang cam tinh.

```python
import numpy as np

def diem_trong_so(diem, trong_so):
    """
    Tinh diem co trong so cho nghien cuu danh doi.
    
    Args:
        diem: Dict {tieu_chi: diem} (thang 1-5)
        trong_so: Dict {tieu_chi: trong_so} (tong bang 1.0)
    
    Returns:
        Diem tong co trong so
    """
    tong = sum(diem[c] * trong_so[c] for c in diem)
    return tong

def chay_nghien_cuu_danh_doi(lua_chon, tieu_chi, trong_so):
    """
    Chay nghien cuu danh doi hoan chinh.
    
    Args:
        lua_chon: Dict {ten_lua_chon: {tieu_chi: diem}}
        tieu_chi: Danh sach ten tieu chi
        trong_so: Dict {tieu_chi: trong_so}
    
    Returns:
        Danh sach xep hang (ten_lua_chon, diem)
    """
    ket_qua = []
    for ten, diem in lua_chon.items():
        tong = diem_trong_so(diem, trong_so)
        ket_qua.append((ten, tong))
    
    return sorted(ket_qua, key=lambda x: -x[1])

def phan_tich_do_nhay(lua_chon, tieu_chi, trong_so_goc, tieu_chi_thay_doi, pham_vi):
    """
    Thu nghiem xep hang thay doi khi mot trong so thay doi.
    """
    ket_qua = []
    for gia_tri_ts in pham_vi:
        trong_so = trong_so_goc.copy()
        trong_so[tieu_chi_thay_doi] = gia_tri_ts
        con_lai = 1 - gia_tri_ts
        tong_khac = sum(trong_so_goc[c] for c in tieu_chi if c != tieu_chi_thay_doi)
        
        for c in tieu_chi:
            if c != tieu_chi_thay_doi:
                trong_so[c] = trong_so_goc[c] * con_lai / tong_khac
        
        xep_hang = chay_nghien_cuu_danh_doi(lua_chon, tieu_chi, trong_so)
        thang = xep_hang[0][0]
        ket_qua.append((gia_tri_ts, thang))
    
    return ket_qua

def kiem_tra_ngan_sach_khoi_luong(phan_bo, thuc_te, bien_do_yeu_cau=0.10):
    """
    Kiem tra trang thai ngan sach khoi luong.
    """
    ket_qua = []
    for he_thong in phan_bo:
        alloc = phan_bo[he_thong]
        actual = thuc_te.get(he_thong, 0)
        bien_do = (alloc - actual) / alloc
        trang_thai = "OK" if bien_do >= bien_do_yeu_cau else "CANH BAO"
        ket_qua.append({
            "he_thong": he_thong,
            "phan_bo": alloc,
            "thuc_te": actual,
            "bien_do_phan_tram": bien_do * 100,
            "trang_thai": trang_thai
        })
    return ket_qua

def phan_bo_thiet_ke_theo_chi_phi(chi_phi_muc_tieu, phan_tram_he_thong):
    """
    Phan bo ngan sach chi phi cho he thong con.
    """
    return {ht: chi_phi_muc_tieu * pt for ht, pt in phan_tram_he_thong.items()}
```

<section class="lesson-pitfalls">

## Cam bay

1. **Trong so khong noi ro.** Moi quyet dinh co trong so an. Lam ro rang chung hoac ban se tranh luan ve diem khi bat dong ve uu tien.

2. **Benh 3 diem.** Cham moi thu 3/5 lam tat ca lua chon bang nhau. Buoc ban than dung 1 va 5.

3. **Bo qua tuong tac.** Khoi luong anh huong cong suat anh huong chi phi. Danh gia o cap he thong, khong chi he thong con.

4. **Toi uu qua diem gia tri.** Tiet kiem $1k tren ve tinh $300k khong quan trong. Tap trung vao yeu to lon.

5. **Quen tien do.** Thiet ke hoan hao giao hang cham 2 nam thua thiet ke tot giao ngay.

6. **Tinh lai moi lan lap.** Lam nghien cuu danh doi mot lan, tai lieu hoa, di tiep. Dung toi uu lai hang ngay.
