## Khai niem Co ban

### Tai sao Ve tinh Sieu nho?

Ve tinh truyen thong co chi phi tren $100M va mat 5-10 nam phat trien. Ve tinh sieu nho (10-100 kg) mang lai:
- Thoi gian phat trien: 2-4 nam
- Chi phi: $5-15M (bao gom phong)
- Dao tao cong nghe: Kinh nghiem thuc hanh cho ky su
- Chap nhan rui ro: Mat mat co the chap nhan cho trinh dien cong nghe

## Phan tich Quy dao va Vung Phu

### Tinh toan Quy dao Dong bo Mat troi

```python
import math

def microdragon_orbit_analysis():
    """
    Analyze MicroDragon's sun-synchronous orbit parameters.
    """
    # Orbital elements
    altitude_km = 511
    inclination_deg = 97.31
    ltdn_hours = 10.5  # 10:30 local time descending node
    
    # Earth parameters
    R_earth_km = 6371
    mu_earth = 398600.4418  # km^3/s^2
    J2 = 1.08263e-3
    
    # Calculate orbital period
    a_km = R_earth_km + altitude_km  # Semi-major axis
    period_sec = 2 * math.pi * math.sqrt(a_km**3 / mu_earth)
    period_min = period_sec / 60
    
    # Calculate orbital velocity
    v_km_s = math.sqrt(mu_earth / a_km)
    
    # Calculate ground track velocity
    ground_track_km_s = v_km_s * math.cos(math.radians(inclination_deg))
    
    # Nodal precession rate for SSO
    # dΩ/dt = -3/2 * J2 * (R_e/a)^2 * n * cos(i)
    n = math.sqrt(mu_earth / a_km**3)  # Mean motion (rad/s)
    precession_rad_s = -1.5 * J2 * (R_earth_km/a_km)**2 * n * math.cos(math.radians(inclination_deg))
    precession_deg_day = math.degrees(precession_rad_s) * 86400
    
    # SSO requires precession = 360°/365.25 days = 0.9856°/day
    required_precession = 360 / 365.25
    
    # Revisit time (approximate)
    orbits_per_day = 86400 / period_sec
    
    print("=== Phan tich Quy dao MicroDragon ===")
    print(f"Do cao:             {altitude_km:.1f} km")
    print(f"Do nghieng:         {inclination_deg:.2f}°")
    print(f"Ban truc lon:       {a_km:.1f} km")
    print(f"Chu ky quy dao:     {period_min:.2f} phut ({period_sec:.1f} s)")
    print(f"Van toc quy dao:    {v_km_s:.3f} km/s")
    print(f"So vong/ngay:       {orbits_per_day:.2f}")
    print(f"")
    print(f"=== Kiem tra Dong bo Mat troi ===")
    print(f"Tien dong nut:      {precession_deg_day:.4f}°/ngay")
    print(f"Yeu cau (SSO):      {required_precession:.4f}°/ngay")
    print(f"Chenh lech:         {abs(precession_deg_day - required_precession):.4f}°/ngay")
    print(f"LTDN:               {ltdn_hours:.1f} gio (10:30 dia phuong)")
    
    # Ground track repeat
    # For 511 km, approximately 14-15 day repeat cycle
    print(f"")
    print(f"=== Vung phu ===")
    print(f"Do rong dai quet:   98 km")
    print(f"GSD:                78 m")
    
    return period_min, orbits_per_day

if __name__ == "__main__":
    microdragon_orbit_analysis()
```

### Thuat toan Truy xuat Chat luong Nuoc

```python
import numpy as np

def chlorophyll_a_estimation(band2_blue, band4_green, band5_red):
    """
    Estimate chlorophyll-a concentration from MicroDragon imagery.
    Uses Ocean Color Index (OCI) algorithm variant.
    
    Args:
        band2_blue: Blue band reflectance (433-453 nm)
        band4_green: Green band reflectance (545-565 nm)  
        band5_red: Red band reflectance (655-675 nm)
    
    Returns:
        Chlorophyll-a concentration in mg/m^3
    """
    # Band ratio approach (similar to OC3M algorithm)
    # R = log10(max(Rrs_blue/Rrs_green, Rrs_blue/Rrs_red))
    
    ratio_bg = band2_blue / band4_green
    ratio_br = band2_blue / band5_red
    
    R = np.log10(np.maximum(ratio_bg, ratio_br))
    
    # Polynomial coefficients (empirical, regional calibration needed)
    # Chl-a = 10^(a0 + a1*R + a2*R^2 + a3*R^3 + a4*R^4)
    a = [0.3272, -2.9940, 2.7218, -1.2259, -0.5683]
    
    chl_log = a[0] + a[1]*R + a[2]*R**2 + a[3]*R**3 + a[4]*R**4
    chl_a = 10 ** chl_log
    
    return chl_a

def turbidity_estimation(band3_cyan, band6_nir):
    """
    Estimate turbidity (NTU) from suspended sediment.
    
    Args:
        band3_cyan: Cyan band reflectance (480-500 nm)
        band6_nir: NIR band reflectance (765-785 nm)
    
    Returns:
        Turbidity in NTU
    """
    # Empirical relationship for coastal waters
    # Higher NIR/Cyan ratio indicates more suspended sediment
    ratio = band6_nir / band3_cyan
    
    # Linear model (needs regional calibration)
    turbidity_ntu = 50 * ratio + 5
    
    return turbidity_ntu

# Example usage
if __name__ == "__main__":
    # Simulated reflectance values (typical coastal water)
    b2 = 0.012  # Blue
    b3 = 0.015  # Cyan
    b4 = 0.018  # Green
    b5 = 0.008  # Red
    b6 = 0.004  # NIR
    
    chl = chlorophyll_a_estimation(b2, b4, b5)
    turb = turbidity_estimation(b3, b6)
    
    print(f"Uoc tinh Chlorophyll-a: {chl:.2f} mg/m³")
    print(f"Uoc tinh Do duc: {turb:.1f} NTU")
```

<section class="lesson-pitfalls">

## Loi Thuong gap

### 1. Nham lan MicroDragon voi NanoDragon
- **MicroDragon** (2019): 50 kg, chup anh da pho, VNSC phat trien
- **NanoDragon** (2021): 4 kg, thu tin hieu AIS, VNSC phat trien
Nhiem vu khac nhau, kha nang khac nhau.

### 2. Bo qua Hieu chinh Khi quyen
Anh MicroDragon tho can hieu chinh khi quyen truoc khi truy xuat chat luong nuoc. Do sau quang hoc aerosol tai vung bien Viet Nam co the gay sai so phan xa 20-40%.

### 3. Do phu May tai Nhiet doi
Viet Nam co 60-80% may phu trong mua mua (thang 5-10). Cua so chup anh hieu qua bi han che, can chien luoc tong hop nhieu ngay.

### 4. Hieu chinh Hinh hoc
GSD 78m yeu cau dinh vi chinh xac. Sai so chi huong va goc chup lech co the gay lech 1-2 pixel (80-160m) neu khong co diem khong che mat dat.

### 5. Troi Hieu chuan
Hieu chuan buc xa tren quy dao suy giam theo thoi gian. MicroDragon su dung hieu chuan uy nhiem qua so sanh voi cac vi tri tham chieu (sa mac, dai duong sau).

## Cau noi den Bai hoc Tiep theo

MicroDragon cho thay nang luc phat trien ve tinh dang tang cua Viet Nam nhung hoat dong doc lap. Bai hoc tiep theo xem xet **ha tang tram mat dat** tren khap Dong Nam A cho phep van hanh ve tinh, thu nhan du lieu va hop tac khu vuc, bao gom tram Hoa Lac cua Viet Nam noi nhan du lieu MicroDragon.

**Ket noi:**
- Tai san quy dao (VINASAT, MicroDragon) can ha tang mat dat
- Cac tram khu vuc cho phep chia se du lieu va du phong nhiem vu
- Thanh phan mat dat thuong chiem 50%+ tong chi phi nhiem vu
