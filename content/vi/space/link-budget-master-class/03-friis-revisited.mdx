## Tai sao dieu nay quan trong

Phuong trinh truyen dan Friis la xuong song cua tat ca ngan sach lien ket—no cho ban biet bao nhieu cong suat den may thu voi nhung gi da phat. Cho cac lien ket ve tinh, noi khoang cach do bang hang ngan km va moi dB deu quan trong, hieu Friis la thiet yeu. No giai thich tai sao lien ket GEO kho hon LEO, tai sao Ka-band can nhieu EIRP hon C-band, va cach kich thuoc anten danh doi voi cong suat phat.

## Ap dung

**Tinh toan ngan sach lien ket:**

```python
import numpy as np

def fspl_db(khoang_cach_km, tan_so_ghz):
    """
    Tinh Ton hao Duong truyen Khong gian Tu do tinh bang dB.
    
    FSPL = 20 log(d_km) + 20 log(f_GHz) + 92.45
    """
    return 20 * np.log10(khoang_cach_km) + 20 * np.log10(tan_so_ghz) + 92.45

def do_loi_parabol_dbi(duong_kinh_m, tan_so_ghz, hieu_suat=0.55):
    """Tinh do loi anten parabol tinh bang dBi."""
    buoc_song_m = 0.3 / tan_so_ghz
    do_loi_tuyen_tinh = hieu_suat * (np.pi * duong_kinh_m / buoc_song_m) ** 2
    return 10 * np.log10(do_loi_tuyen_tinh)

def ngan_sach_lien_ket(eirp_dbw, khoang_cach_km, tan_so_ghz, g_tren_t_dbk, ton_hao_khac_db=0):
    """
    Tinh C/N0 cho lien ket ve tinh.
    """
    k_dbw = -228.6  # Hang so Boltzmann tinh bang dBW/K/Hz
    ton_hao_duong_truyen = fspl_db(khoang_cach_km, tan_so_ghz)
    
    cn0 = eirp_dbw - ton_hao_duong_truyen - ton_hao_khac_db + g_tren_t_dbk - k_dbw
    
    return cn0, ton_hao_duong_truyen

def ebn0_can_thiet_sang_cn0(ebn0_db, toc_do_du_lieu_bps):
    """Chuyen yeu cau Eb/N0 sang C/N0."""
    return ebn0_db + 10 * np.log10(toc_do_du_lieu_bps)

# Vi du tinh toan
print("Phuong trinh Friis & Ngan sach Lien ket")
print("=" * 60)

# So sanh FSPL
print("\n1. So sanh Ton hao Duong truyen Khong gian Tu do")
print("-" * 60)

quy_dao = [
    ("LEO (550 km)", 600),
    ("MEO (GPS)", 22000),
    ("GEO", 36000),
    ("Mat Trang", 384000),
]

tan_so = [4, 12, 30]

print(f"  {'Quy dao':<20}", end="")
for f in tan_so:
    print(f"{f} GHz".center(12), end="")
print()
print("-" * 60)

for ten, kc in quy_dao:
    print(f"  {ten:<20}", end="")
    for f in tan_so:
        ton_hao = fspl_db(kc, f)
        print(f"{ton_hao:.1f} dB".center(12), end="")
    print()

# Ngan sach lien ket day du
print("\n2. Ngan sach Duong xuong GEO Ku-band")
print("-" * 60)

# Tham so
vt_eirp = 52  # dBW
khoang_cach = 36000  # km
ts = 12  # GHz
md_gt = 27  # dB/K
ton_hao_kq = 0.5  # dB
ton_hao_ch = 0.5  # dB
tong_ton_hao_khac = ton_hao_kq + ton_hao_ch

cn0, fspl = ngan_sach_lien_ket(vt_eirp, khoang_cach, ts, md_gt, tong_ton_hao_khac)

print(f"  EIRP Ve tinh:          {vt_eirp:+.1f} dBW")
print(f"  Ton hao KG tu do:      {-fspl:.1f} dB")
print(f"  Ton hao khi quyen:     {-ton_hao_kq:.1f} dB")
print(f"  Ton hao chi huong:     {-ton_hao_ch:.1f} dB")
print(f"  G/T tram mat dat:      {md_gt:+.1f} dB/K")
print(f"  Hang so Boltzmann:     +228.6 dBW/K/Hz")
print("-" * 60)
print(f"  C/N₀:                  {cn0:.1f} dBHz")

# Chuyen sang Eb/N0 cho toc do du lieu
toc_do_dl = 10e6  # 10 Mbps
ebn0_can = 8  # dB (dien hinh cho QPSK voi ma hoa)
cn0_can = ebn0_can_thiet_sang_cn0(ebn0_can, toc_do_dl)
du_tru = cn0 - cn0_can

print(f"\n  Cho toc do du lieu {toc_do_dl/1e6:.0f} Mbps:")
print(f"  Eb/N₀ can thiet:       {ebn0_can:.1f} dB")
print(f"  C/N₀ can thiet:        {cn0_can:.1f} dBHz")
print(f"  Du tru lien ket:       {du_tru:.1f} dB")

# So sanh LEO
print("\n3. So sanh LEO vs GEO (cung EIRP)")
print("-" * 60)

leo_kc = 600  # km
geo_kc = 36000  # km

for ten, kc in [("LEO", leo_kc), ("GEO", geo_kc)]:
    cn0, fspl = ngan_sach_lien_ket(vt_eirp, kc, ts, md_gt, tong_ton_hao_khac)
    print(f"  {ten}: FSPL = {fspl:.1f} dB, C/N₀ = {cn0:.1f} dBHz")

loi_the_leo = fspl_db(geo_kc, ts) - fspl_db(leo_kc, ts)
print(f"\n  Loi the LEO: {loi_the_leo:.1f} dB it ton hao duong truyen hon")

# So sanh khe vat ly
print("\n4. Kich thuoc Dia Co dinh theo Tan so")
print("-" * 60)

kich_thuoc_dia = 3.0  # m
khoang_cach = 36000  # km

print(f"  Dia {kich_thuoc_dia}m tai khoang cach GEO ({khoang_cach} km):")
print(f"")
print(f"  {'Tan so (GHz)':<12} {'Do loi (dBi)':<12} {'FSPL (dB)':<12} {'Rong (dB)':<12}")
print("-" * 60)

for f in [4, 12, 30]:
    do_loi = do_loi_parabol_dbi(kich_thuoc_dia, f)
    ton_hao = fspl_db(khoang_cach, f)
    rong = do_loi - ton_hao
    print(f"  {f:<12} {do_loi:<12.1f} {ton_hao:<12.1f} {rong:<12.1f}")

print("\n  Luu y: Rong tuong tu qua cac tan so (dia triet tieu FSPL)!")
print("  Tan so cao hon mat vi khi quyen/mua, khong phai FSPL.")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Quen dung km va GHz.** Hang so 92.45 gia dinh km va GHz. Tron don vi → ket qua sai.

2. **Do loi FSPL cho kho khan Ka-band.** Van de thuc su la hap thu khi quyen va suy giam mua, khong phai FSPL.

3. **Dung khoang cach thien de cho LEO.** Khoang cach nghieng thay doi theo goc nang; dung khoang cach duong thuc.

4. **Bo qua goc nang.** Goc nang thap hon = duong dai hon + nhieu khi quyen hon + G/T te hon.

5. **Nham lan tuyen tinh vs dB.** Friis la phep nhan; ngan sach lien ket la phep cong tinh bang dB.

6. **Quen Boltzmann.** Khi dung G/T, ban can -228.6 dBW/K/Hz de co C/N₀.
