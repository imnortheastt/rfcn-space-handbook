<section class="lesson-hook">

## Tai sao dieu nay quan trong

EIRP la con so duy nhat dac trung cho kha nang may phat cua ban—bao nhieu cong suat no thuc su dua vao huong lien ket. Day la diem bat dau cua moi phep tinh ngan sach lien ket. Du ban dang thiet ke bo phat dap ve tinh, duong len tram mat dat, hay dau cuoi di dong, EIRP quyet dinh lieu tin hieu cua ban co den duoc may thu tren nguong nhieu hay khong.

</section>

### EIRP la gi?

```
Khai niem EIRP:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  EIRP = "Can bao nhieu cong suat tu anten dang huong de            │

│                                                                      │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  ANTEN DANG HUONG (tham chieu ly thuyet):             │         │
│  │                                                        │         │
│  │           ↑ ↗                                         │         │
│  │         ↖ ● → buc xa deu theo moi huong              │         │
│  │           ↓ ↘                                         │         │
│  │                                                        │         │
│  │  Cong suat trai deu tren toan mat cau 4π steradian    │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  ANTEN DINH HUONG (anten thuc):                       │         │
│  │                                                        │         │
│  │           ════════●════════→ tap trung cong suat      │         │
│  │                               vao mot huong           │         │
│  │                                                        │         │
```

### Tai sao dBW va dBi?

```
Don vi Decibel trong Ngan sach Lien ket:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  DON VI CONG SUAT:                                                  │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  dBW = 10 log₁₀(P / 1 W)                             │         │
│  │                                                        │         │
│  │  Vi du:                                               │         │
│  │  • 1 W = 0 dBW                                        │         │
│  │  • 10 W = 10 dBW                                      │         │
│  │  • 100 W = 20 dBW                                     │         │
│  │  • 1 kW = 30 dBW                                      │         │
│  │  • 1 mW = -30 dBW = 0 dBm                            │         │
│  │                                                        │         │
│  │  dBm = 10 log₁₀(P / 1 mW) = dBW + 30                │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  DON VI DO LOI:                                                     │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  dBi = 10 log₁₀(G / G_dang_huong)                    │         │
│  │                                                        │         │
│  │  "dBi" nghia la do loi tuong doi voi anten dang huong │         │
│  │                                                        │         │
│  │  Vi du:                                               │         │
│  │  • Anten dang huong: 0 dBi                           │         │
│  │  • Dipole: 2.15 dBi                                   │         │
│  │  • Dia 1m @ 12 GHz: ~38 dBi                          │         │
│  │  • Dia 10m @ 12 GHz: ~58 dBi                         │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  TAI SAO dB?                                                        │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  • Phep nhan → Phep cong (toan de hon)               │         │
│  │  • Dai dong rong (mW den kW trong cung thang do)     │         │
│  │  • Tieu chuan nganh cho ky thuat RF                  │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## Cach hoat dong

### Chuoi tinh EIRP

```
Tu Bo khuech dai den EIRP:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  ┌───────┐    ┌───────┐    ┌───────┐    ┌───────────┐              │
│  │  HPA  │───→│ Cap   │───→│ Bo    │───→│  Anten    │═══→ EIRP    │
│  │       │    │ Ton hao│    │ tiep  │    │  Do loi   │              │
│  └───────┘    └───────┘    └───────┘    └───────────┘              │
│   P_hpa       L_cap        L_tiep         G_anten                   │
│                                                                      │
│  EIRP = P_hpa - L_cap - L_tiep + G_anten                           │
│                                                                      │
│  VI DU:                                                             │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  Dau ra HPA:        100 W = 20 dBW                   │         │
│  │  Ton hao cap:       -1 dB                             │         │
│  │  Bo tiep/dan song:  -0.5 dB                          │         │
│  │  Do loi anten:      +45 dBi                          │         │
│  │  ────────────────────────────                         │         │
│  │  EIRP:              63.5 dBW                         │         │
│  │                                                        │         │
│  │  Tuong duong 2.2 MW tu anten dang huong!             │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Cong thuc do loi anten

```
Do loi Anten Parabol:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  G = η × (π D / λ)²                                                │
│                                                                      │
│  Tinh dBi: G = 10 log₁₀(η) + 20 log₁₀(π D / λ)                    │
│                                                                      │
│  Trong do:                                                          │
│  • η = hieu suat khe (thuong 0.55-0.65)                            │
│  • D = duong kinh dia (m)                                          │
│  • λ = buoc song = c / f (m)                                       │
│                                                                      │
│  DON GIAN (η ≈ 0.55):                                              │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  G (dBi) ≈ 20 log₁₀(D) + 20 log₁₀(f_GHz) + 17.8     │         │
│  │                                                        │         │
│  │  Gap doi duong kinh: +6 dB                           │         │
│  │  Gap doi tan so: +6 dB                               │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  VI DU DO LOI:                                                      │
│  ┌────────────────────────────────────────────────────┐            │
│  │                                                     │            │
│  │  Duong kinh │ 4 GHz │ 12 GHz │ 30 GHz             │            │
│  │  ───────────┼───────┼────────┼────────             │            │
│  │  0.6 m      │ 26 dBi│ 35 dBi │ 43 dBi             │            │
│  │  1.2 m      │ 32 dBi│ 41 dBi │ 49 dBi             │            │
│  │  3.0 m      │ 40 dBi│ 49 dBi │ 57 dBi             │            │
│  │  10 m       │ 50 dBi│ 59 dBi │ 67 dBi             │            │
│  │                                                     │            │
│  └────────────────────────────────────────────────────┘            │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## Ap dung

**Cac phep tinh EIRP:**

```python
import numpy as np

def cong_suat_sang_dbw(watts):
    """Chuyen watts sang dBW."""
    return 10 * np.log10(watts)

def dbw_sang_watts(dbw):
    """Chuyen dBW sang watts."""
    return 10 ** (dbw / 10)

def do_loi_parabol_dbi(duong_kinh_m, tan_so_ghz, hieu_suat=0.55):
    """
    Tinh do loi anten parabol tinh bang dBi.
    """
    buoc_song_m = 0.3 / tan_so_ghz  # c = 3e8 m/s
    do_loi_tuyen_tinh = hieu_suat * (np.pi * duong_kinh_m / buoc_song_m) ** 2
    return 10 * np.log10(do_loi_tuyen_tinh)

def tinh_eirp(hpa_watts, ton_hao_cap_db, ton_hao_tiep_db, do_loi_anten_dbi):
    """
    Tinh EIRP tinh bang dBW.
    """
    p_hpa_dbw = cong_suat_sang_dbw(hpa_watts)
    eirp = p_hpa_dbw - ton_hao_cap_db - ton_hao_tiep_db + do_loi_anten_dbi
    return eirp

# Vi du tinh toan
print("Tinh toan EIRP")
print("=" * 60)

# Duong len tram mat dat
print("\n1. Duong len Ku-band Tram mat dat (14 GHz)")
print("-" * 60)

cong_suat_hpa = 400  # watts
ton_hao_cap = 1.5  # dB
ton_hao_tiep = 0.5  # dB
duong_kinh_dia = 3.0  # met
tan_so = 14.0  # GHz

do_loi_anten = do_loi_parabol_dbi(duong_kinh_dia, tan_so, hieu_suat=0.55)
eirp = tinh_eirp(cong_suat_hpa, ton_hao_cap, ton_hao_tiep, do_loi_anten)

print(f"  Cong suat HPA:  {cong_suat_hpa} W = {cong_suat_sang_dbw(cong_suat_hpa):.1f} dBW")
print(f"  Ton hao cap:    -{ton_hao_cap} dB")
print(f"  Ton hao tiep:   -{ton_hao_tiep} dB")
print(f"  Duong kinh dia: {duong_kinh_dia} m")
print(f"  Do loi anten:   {do_loi_anten:.1f} dBi")
print(f"  ────────────────────────")
print(f"  EIRP:           {eirp:.1f} dBW")
print(f"  Tuong duong:    {dbw_sang_watts(eirp)/1e6:.1f} MW dang huong")

# Bo phat dap ve tinh
print("\n2. Bo phat dap C-band Ve tinh (4 GHz)")
print("-" * 60)

cs_vetinh = 10  # watts (TWTA)
ton_hao_vt = 0.3  # dB
dia_vt = 2.0  # met
ts_vt = 4.0  # GHz

do_loi_vt = do_loi_parabol_dbi(dia_vt, ts_vt, hieu_suat=0.6)
eirp_vt = tinh_eirp(cs_vetinh, 0, ton_hao_vt, do_loi_vt)

print(f"  Cong suat TWTA: {cs_vetinh} W = {cong_suat_sang_dbw(cs_vetinh):.1f} dBW")
print(f"  Ton hao:        -{ton_hao_vt} dB")
print(f"  Anten:          dia {dia_vt} m")
print(f"  Do loi anten:   {do_loi_vt:.1f} dBi")
print(f"  ────────────────────────")
print(f"  EIRP:           {eirp_vt:.1f} dBW")

# Dau cuoi di dong
print("\n3. Dau cuoi VSAT Ka-band (30 GHz)")
print("-" * 60)

cs_vsat = 2  # watts
ton_hao_vsat = 0.5  # dB
dia_vsat = 0.75  # met
ts_vsat = 30.0  # GHz

do_loi_vsat = do_loi_parabol_dbi(dia_vsat, ts_vsat, hieu_suat=0.55)
eirp_vsat = tinh_eirp(cs_vsat, 0, ton_hao_vsat, do_loi_vsat)

print(f"  Cong suat BUC:  {cs_vsat} W = {cong_suat_sang_dbw(cs_vsat):.1f} dBW")
print(f"  Ton hao:        -{ton_hao_vsat} dB")
print(f"  Anten:          dia {dia_vsat} m")
print(f"  Do loi anten:   {do_loi_vsat:.1f} dBi")
print(f"  ────────────────────────")
print(f"  EIRP:           {eirp_vsat:.1f} dBW")

# Bang do loi theo duong kinh
print("\n4. Bang Tham chieu Do loi Anten (η=0.55)")
print("-" * 60)
print(f"  {'Duong kinh':<10} {'4 GHz':>10} {'12 GHz':>10} {'30 GHz':>10}")
print("-" * 60)

for d in [0.6, 1.2, 2.4, 5.0, 10.0]:
    g4 = do_loi_parabol_dbi(d, 4)
    g12 = do_loi_parabol_dbi(d, 12)
    g30 = do_loi_parabol_dbi(d, 30)
    print(f"  {d:<10.1f} {g4:>8.1f} dBi {g12:>8.1f} dBi {g30:>8.1f} dBi")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Quen ton hao.** Cap, dau noi, bo tiep—moi dB giua HPA va anten giam EIRP.

2. **Sai hieu suat.** Dia parabol hieu suat 55-65%, khong phai 100%. Dung gia tri thuc te.

3. **Nham lan dBW va dBm.** Sai 30 dB! Luon kiem tra don vi trong ngan sach lien ket.

4. **Bo qua backoff.** HPA hoat dong duoi bao hoa; tinh backoff dau vao/ra.

5. **Sai tan so.** Do loi anten thay doi theo tan so; dung bang dung.

6. **Dinh vs ria do phu.** EIRP ve tinh thay doi theo do phu; dung gia tri EOC cho du tru.
