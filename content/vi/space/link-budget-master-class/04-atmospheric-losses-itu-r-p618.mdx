## Tai sao dieu nay quan trong

Ton hao duong truyen khong gian tu do gia dinh chan khong rong giua may phat va may thu. Nhung tin hieu ve tinh phai xuyen qua khi quyen Trai dat—mot hon hop khi va nuoc hap thu va tan xa nang luong. Cho Ka-band tro len, suy giam mua rieng co the vuot 20 dB. Hieu ton hao khi quyen la thiet yeu de thiet ke lien ket ve tinh dang tin cay, chon du tru phu hop, va trien khai ky thuat giam suy giam.

### Suy giam khi

```
Hap thu Khi P.676 (Troi quang):

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  Tong khi: A_khi = A_oxy + A_hoi_nuoc                              │
│                                                                      │
│  Don gian hoa cho duong nghieng:                                   │
│                                                                      │
│  A_khi ≈ (γ_o × h_o + γ_w × h_w) / sin(θ)                         │
│                                                                      │
│  Trong do:                                                          │
│  γ_o = suy giam rieng oxy (dB/km)                                  │
│  γ_w = suy giam rieng hoi nuoc (dB/km)                             │
│  h_o = do cao hieu dung oxy (~6 km)                                │
│  h_w = do cao hieu dung hoi nuoc (~2 km)                           │
│  θ = goc nang                                                       │
│                                                                      │
│  GIA TRI DIEN HINH TAI 30 GHz, θ=30°:                              │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  Oxy:        ~0.2 dB                                  │         │
│  │  Hoi nuoc:   ~0.5-1.0 dB (thay doi theo do am)       │         │
│  │  Tong:       ~0.7-1.2 dB                             │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  Ty le 1/sin(θ): goc nang thap = suy giam nhieu hon               │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## Ap dung

**Uoc luong ton hao khi quyen:**

```python
import numpy as np

def suy_giam_khi_don_gian(tan_so_ghz, goc_nang_do, do_am_pct=50):
    """
    Suy giam khi don gian hoa dua tren xap xi P.676.
    """
    el_rad = np.radians(goc_nang_do)
    
    # Suy giam rieng xap xi (dB/km) tai muc nuoc bien
    if tan_so_ghz < 10:
        gamma_o = 0.005
        gamma_w = 0.001 * do_am_pct / 50
    elif tan_so_ghz < 20:
        gamma_o = 0.007 + 0.001 * (tan_so_ghz - 10)
        gamma_w = 0.01 + 0.005 * (tan_so_ghz - 10) * do_am_pct / 50
    elif tan_so_ghz < 40:
        gamma_o = 0.02 + 0.002 * (tan_so_ghz - 20)
        gamma_w = 0.05 + 0.01 * (tan_so_ghz - 20) * do_am_pct / 50
    else:
        gamma_o = 0.1 + 0.05 * (tan_so_ghz - 40)
        gamma_w = 0.15 * do_am_pct / 50
    
    # Do cao hieu dung
    h_o = 6.0  # km
    h_w = 2.0  # km
    
    # Suy giam thien dinh
    a_zenith = gamma_o * h_o + gamma_w * h_w
    
    # He so duong nghieng
    a_tong = a_zenith / np.sin(el_rad)
    
    return a_tong

def suy_giam_rieng_mua(tan_so_ghz, luong_mua_mmhr, phan_cuc='H'):
    """
    Tinh suy giam rieng tu luong mua su dung mo hinh P.838.
    """
    he_so = {
        4: (0.00015, 1.13, 0.00014, 1.08),
        12: (0.023, 1.13, 0.019, 1.09),
        18: (0.051, 1.06, 0.045, 1.02),
        20: (0.073, 1.02, 0.064, 0.98),
        30: (0.19, 0.94, 0.17, 0.90),
        40: (0.35, 0.88, 0.31, 0.85),
    }
    
    tan_so_list = list(he_so.keys())
    tan_so_gan_nhat = min(tan_so_list, key=lambda x: abs(x - tan_so_ghz))
    k_h, a_h, k_v, a_v = he_so[tan_so_gan_nhat]
    
    if phan_cuc.upper() == 'H':
        gamma_r = k_h * (luong_mua_mmhr ** a_h)
    else:
        gamma_r = k_v * (luong_mua_mmhr ** a_v)
    
    return gamma_r

def suy_giam_mua_p618(tan_so_ghz, goc_nang_do, luong_mua_mmhr, 
                      do_cao_mua_km=4, do_cao_tram_km=0):
    """
    Tinh suy giam mua P.618 don gian hoa.
    """
    el_rad = np.radians(goc_nang_do)
    
    if goc_nang_do >= 5:
        L_s = (do_cao_mua_km - do_cao_tram_km) / np.sin(el_rad)
    else:
        L_s = 2 * (do_cao_mua_km - do_cao_tram_km) / (np.sin(el_rad) + 0.1)
    
    L_g = L_s * np.cos(el_rad)
    
    gamma_r = suy_giam_rieng_mua(tan_so_ghz, luong_mua_mmhr)
    
    r = 1 / (1 + 0.78 * np.sqrt(L_g * gamma_r / tan_so_ghz) - 
             0.38 * (1 - np.exp(-2 * L_g)))
    
    L_hieu_dung = L_s * r
    
    A_mua = gamma_r * L_hieu_dung
    
    return A_mua, L_s, gamma_r

def quy_doi_suy_giam_mua(a_001, phan_tram_muc_tieu):
    """
    Quy doi suy giam tu 0.01% sang phan tram khac.
    """
    if phan_tram_muc_tieu >= 1:
        beta = 0.33
    elif phan_tram_muc_tieu >= 0.1:
        beta = 0.41
    else:
        beta = 0.55
    
    a_p = a_001 * (phan_tram_muc_tieu / 0.01) ** (-beta)
    return a_p

# Vi du tinh toan
print("Ton hao Khi quyen cho Lien ket Ve tinh")
print("=" * 60)

# Ton hao khi troi quang
print("\n1. Suy giam Khi (Troi quang)")
print("-" * 60)

bang = [
    ("C-band", 6),
    ("Ku-band", 14),
    ("Ka-band", 30),
    ("V-band", 50),
]

goc_nang = [10, 30, 60, 90]

print(f"  {'Bang':<12}", end="")
for el in goc_nang:
    print(f"{el}°".center(10), end="")
print()
print("-" * 60)

for ten, tan_so in bang:
    print(f"  {ten:<12}", end="")
    for el in goc_nang:
        suy_giam = suy_giam_khi_don_gian(tan_so, el)
        print(f"{suy_giam:.2f} dB".center(10), end="")
    print()

# Suy giam mua
print("\n2. Suy giam Mua tai 30 GHz, goc nang 30°")
print("-" * 60)

luong_mua = [1, 5, 10, 25, 50, 100]

print(f"  Luong mua    Suy giam rieng    Suy giam duong")
print(f"  (mm/hr)      (dB/km)           (dB)")
print("-" * 60)

for lm in luong_mua:
    a_mua, l_s, gamma = suy_giam_mua_p618(30, 30, lm)
    print(f"  {lm:<12} {gamma:<16.2f} {a_mua:.1f}")

# Suy giam theo san sang
print("\n3. Yeu cau Du tru Mua (Ka-band, Khi hau Nhiet doi)")
print("-" * 60)

a_001, _, _ = suy_giam_mua_p618(30, 30, 60)

phan_tram = [1.0, 0.5, 0.1, 0.05, 0.01]

print(f"  {'San sang':<14} {'Gian doan/nam':<14} {'Du tru Mua':<12}")
print("-" * 60)

for p in phan_tram:
    san_sang = 100 - p
    gian_doan_gio = p * 365.25 * 24 / 100
    du_tru = quy_doi_suy_giam_mua(a_001, p)
    print(f"  {san_sang:.2f}%{' ':8} {gian_doan_gio:>6.1f} gio{' ':4} {du_tru:.1f} dB")

# Ngan sach khi quyen hoan chinh
print("\n4. Vi du Ngan sach Khi quyen Hoan chinh")
print("-" * 60)

tan_so = 30  # GHz
goc_nang = 30  # do
luong_mua = 25  # mm/hr cho san sang 99.9%

khi = suy_giam_khi_don_gian(tan_so, goc_nang)
mua, _, _ = suy_giam_mua_p618(tan_so, goc_nang, luong_mua)
nhap_nhay = 0.5  # Dien hinh cho Ka-band

print(f"  Kich ban: Ka-band ({tan_so} GHz) tai goc nang {goc_nang}°")
print(f"  Luong mua thiet ke: {luong_mua} mm/hr (san sang 99.9%)")
print()
print(f"  Hap thu khi:        {khi:.2f} dB")
print(f"  Suy giam mua:       {mua:.1f} dB")
print(f"  Suy giam nhap nhay: {nhap_nhay:.1f} dB")
print("-" * 60)
tong = khi + mua + nhap_nhay
print(f"  Tong khi quyen:     {tong:.1f} dB")
print()
print(f"  Luu y: Van hanh troi quang chi dung {khi:.1f} dB du tru")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Dung gia tri thien dinh cho goc nang thap.** Duong khi quyen ty le 1/sin(θ). Tai goc nang 10°, duong dai ~6× thien dinh.

2. **Bo qua bien doi do cao mua.** Do cao mua thay doi tu 2 km (cuc) den 5 km (nhiet doi). Sai h_R → sai do dai duong.

3. **Mot luong mua cho phu song toan cau.** R₀.₀₁ thay doi 10× giua vung cuc va nhiet doi. Dung ban do ITU dia phuong.

4. **Quen he so giam duong.** Mua khong lap day toan bo duong deu. Ap dung he so giam hoac ket qua bi quan qua.

5. **Thiet ke du tru tinh.** Lien ket Ka-band can ACM; du tru co dinh hoac lang phi dung luong hoac hong khi mua.

6. **Tan so duong len vs duong xuong.** Duong len thuong tan so cao hon—thuong 2-3 dB ton hao mua nhieu hon duong xuong.
