## Tai sao dieu nay quan trong

Phan cuc la huong cua dien truong khi no lan truyen. Neu may phat gui phan cuc tron phai va may thu mong doi phan cuc tron trai, ban mat gan nhu toan bo tin hieu—co the 20+ dB. Ngay ca voi phan cuc khop, anten khong hoan hao va xoay Faraday trong tang dien ly gay ton hao. Hieu ton hao phan cuc giup ban chon so do phan cuc phu hop, dac ta chat luong anten, va lap ngan sach cho suy giam thuc te.

```

│                                                                      │
│  Tron hoan hao:    AR = 1 (0 dB)                                   │

│  │                                                        │         │
```

## Ap dung

**Tinh toan ton hao phan cuc:**

```python
import numpy as np

def lenh_phan_cuc_tuyen_tinh_db(goc_do):
    """
    Ton hao phan cuc cho lenh tuyen tinh-tuyen tinh.
    
    L = 20 log|cos(θ)|
    """
    if goc_do >= 90:
        return float('inf')  # Truc giao = mat vo han
    
    goc_rad = np.radians(goc_do)
    ton_hao_db = -20 * np.log10(np.abs(np.cos(goc_rad)))
    return ton_hao_db

def ar_db_sang_tuyen_tinh(ar_db):
    """Chuyen ty so truc tu dB sang tuyen tinh."""
    return 10 ** (ar_db / 20)

def ar_tuyen_tinh_sang_db(ar_tuyen_tinh):
    """Chuyen ty so truc tu tuyen tinh sang dB."""
    return 20 * np.log10(ar_tuyen_tinh)

def ton_hao_pol_cp_cung_chieu(ar1_db, ar2_db):
    """
    Ton hao phan cuc cho hai anten CP cung chieu.
    """
    ar1 = ar_db_sang_tuyen_tinh(ar1_db)
    ar2 = ar_db_sang_tuyen_tinh(ar2_db)
    
    tu_so = 1 + (ar1 * ar2) ** 2
    mau_so = 2 * ar1 * ar2
    
    ton_hao_db = 10 * np.log10(tu_so / mau_so)
    return ton_hao_db

def ton_hao_pol_cp_nguoc_chieu(ar1_db, ar2_db):
    """
    Ton hao phan cuc cho hai anten CP nguoc chieu.
    """
    ar1 = ar_db_sang_tuyen_tinh(ar1_db)
    ar2 = ar_db_sang_tuyen_tinh(ar2_db)
    
    tu_so = (ar1 * ar2 + 1) ** 2
    mau_so = (ar1 * ar2 - 1) ** 2
    
    cach_ly_db = 10 * np.log10(tu_so / mau_so)
    return cach_ly_db

def ton_hao_tuyen_tinh_sang_tron():
    """Tuyen tinh sang tron luon mat 3 dB."""
    return 3.0

# Vi du tinh toan
print("Phan tich Ton hao Phan cuc")
print("=" * 60)

# Lenh tuyen tinh
print("\n1. Lenh Phan cuc Tuyen tinh-Tuyen tinh")
print("-" * 60)

goc = [0, 5, 10, 15, 30, 45, 60, 75, 85]

print(f"  {'Goc nghieng':<14} {'Ton hao (dB)':<12}")
print("-" * 60)

for g in goc:
    ton_hao = lenh_phan_cuc_tuyen_tinh_db(g)
    print(f"  {g}°{' ':10} {ton_hao:.2f}")

print(f"\n  Luu y: Nghieng 45° = mat 3 dB (nua cong suat cho moi pol)")

# Phan cuc tron voi ty so truc
print("\n2. Ton hao Phan cuc Tron (Cung Chieu)")
print("-" * 60)

print(f"  TX AR (dB)    RX AR (dB)    Ton hao (dB)")
print("-" * 60)

for ar_tx in [1, 2, 3]:
    for ar_rx in [1, 2, 3]:
        ton_hao = ton_hao_pol_cp_cung_chieu(ar_tx, ar_rx)
        print(f"  {ar_tx:<12} {ar_rx:<12} {ton_hao:.3f}")

# Cach ly cheo phan cuc
print("\n3. Cach ly Cheo Phan cuc (Nguoc Chieu)")
print("-" * 60)

print(f"  TX AR (dB)    RX AR (dB)    Cach ly (dB)")
print("-" * 60)

for ar_tx in [1, 2, 3, 6]:
    for ar_rx in [1, 2, 3, 6]:
        cach_ly = ton_hao_pol_cp_nguoc_chieu(ar_tx, ar_rx)
        print(f"  {ar_tx:<12} {ar_rx:<12} {cach_ly:.1f}")

# So sanh he thong
print("\n4. Ton hao Phan cuc theo Kich ban")
print("-" * 60)

kich_ban = [
    ("CP khop hoan hao (AR=0dB)", "cp_cung", 0, 0),
    ("CP khop dien hinh (AR=2dB)", "cp_cung", 2, 2),
    ("CP khop te (AR=6dB)", "cp_cung", 6, 6),
    ("Tuyen tinh lenh 10°", "tuyen_tinh", 10, 0),
    ("Tuyen tinh lenh 20°", "tuyen_tinh", 20, 0),
    ("Tuyen tinh sang Tron", "ttin_tron", 0, 0),
    ("CP nguoc (AR=2dB)", "cp_nguoc", 2, 2),
]

print(f"  {'Kich ban':<35} {'Ton hao/Cach ly':<15}")
print("-" * 60)

for ten, loai, ar1, ar2 in kich_ban:
    if loai == "cp_cung":
        ton_hao = ton_hao_pol_cp_cung_chieu(ar1, ar2)
        print(f"  {ten:<35} {ton_hao:.2f} dB mat")
    elif loai == "cp_nguoc":
        cach_ly = ton_hao_pol_cp_nguoc_chieu(ar1, ar2)
        print(f"  {ten:<35} {cach_ly:.1f} dB cach ly")
    elif loai == "tuyen_tinh":
        ton_hao = lenh_phan_cuc_tuyen_tinh_db(ar1)
        print(f"  {ten:<35} {ton_hao:.2f} dB mat")
    else:
        ton_hao = ton_hao_tuyen_tinh_sang_tron()
        print(f"  {ten:<35} {ton_hao:.1f} dB mat")

# Phan bo ngan sach lien ket
print("\n5. Ngan sach Ton hao Phan cuc Dien hinh")
print("-" * 60)

ngan_sach = [
    ("GEO DTH (Ku LP)", "0.1-0.3 dB", "Can chinh co dinh, anten tot"),
    ("GEO VSAT (Ku LP)", "0.2-0.5 dB", "Sai so can chinh thu cong"),
    ("LEO di dong (L/S CP)", "0.2-0.5 dB", "Sai so AR"),
    ("Gateway Ka-band (CP)", "0.1-0.3 dB", "Bo cap lieu chat luong cao"),
    ("GPS tieu dung (CP)", "0.5-1.0 dB", "AR anten gia re"),
]

print(f"  {'Ung dung':<25} {'Ngan sach':<12} {'Ghi chu':<30}")
print("-" * 60)

for ud, ns, gc in ngan_sach:
    print(f"  {ud:<25} {ns:<12} {gc:<30}")
```

## Cam bay

1. **Nham lan ton hao voi cach ly.** Ton hao la suy giam cung phan cuc; cach ly la tu choi phan cuc truc giao. Ca hai deu quan trong nhung khac nhau.

2. **Bo qua xoay Faraday tai L/S-band.** Duoi C-band, tin hieu LP co the xoay 10-50°. Dung CP cho di dong/LEO.

3. **Gia dinh CP hoan hao.** Anten thuc co AR = 1-6 dB. Lap ngan sach 0.1-0.5 dB cho lenh CP.

4. **Quen khoi phan cuc mua.** Mua lon tai Ka-band lam giam XPI 5-10 dB. Lap ngan sach cho no.

5. **Can chinh LP cho VSAT.** Can chinh thu cong boi nguoi lap dat sai so ±5-10°. Lap ngan sach 0.2-0.5 dB.

6. **Gia dinh may thu GPS.** Anten GPS tieu dung co AR = 3-6 dB. Lap ngan sach tuong ung.
