## Tai sao dieu nay quan trong

Pho ve tinh la bat dong san quy gia. Bang tan L, S, C, va X (1-12 GHz) la ngua keo cua cac he thong ve tinh truyen thong—GPS, satcom di dong, radar thoi tiet, va truyen thong quan su deu song o day. Cac tan so thap hon nay xuyen qua mua, tan la, va toa nha tot hon cac bang cao hon, lam cho chung can thiet cho ung dung di dong va hang hai. Hieu dac tinh va phan bo cua chung la nen tang cho bat ky ky su satcom nao.

## Ap dung

**May tinh so sanh bang tan so:**

```python
import numpy as np

C = 299792458  # Toc do anh sang (m/s)

def wavelength_m(frequency_ghz):
    """Tinh buoc song tu tan so."""
    return C / (frequency_ghz * 1e9)

def wavelength_cm(frequency_ghz):
    """Tinh buoc song bang centimet."""
    return wavelength_m(frequency_ghz) * 100

def fspl_db(frequency_ghz, distance_km):
    """Ton that khong gian tu do tinh bang dB."""
    return 20 * np.log10(distance_km) + 20 * np.log10(frequency_ghz) + 92.45

def antenna_diameter_for_gain(gain_db, frequency_ghz, efficiency=0.55):
    """Tinh duong kinh dia can cho tang ich cu the."""
    lambda_m = wavelength_m(frequency_ghz)
    gain_linear = 10 ** (gain_db / 10)
    d_squared = gain_linear * (lambda_m ** 2) / (efficiency * np.pi ** 2)
    return np.sqrt(d_squared)

def rain_attenuation_simplified(frequency_ghz, rain_rate_mm_hr, path_length_km=5):
    """Uoc tinh suy giam mua don gian (xap xi ITU-R P.838)."""
    if frequency_ghz < 2:
        return rain_rate_mm_hr * 0.001 * path_length_km
    elif frequency_ghz < 8:
        return rain_rate_mm_hr * 0.01 * path_length_km
    else:
        return rain_rate_mm_hr * 0.03 * path_length_km

# Dinh nghia bang
bands = [
    ("Bang L", 1.5, 1.0, 2.0),
    ("Bang S", 2.5, 2.0, 4.0),
    ("Bang C", 6.0, 4.0, 8.0),
    ("Bang X", 10.0, 8.0, 12.0),
    ("Bang Ku", 14.0, 12.0, 18.0),
    ("Bang Ka", 30.0, 26.0, 40.0),
]

print("So sanh Bang Tan so Ve tinh")
print("=" * 75)

# Thuoc tinh co ban
print("\n1. Buoc song va Thuoc tinh Co ban")
print("-" * 75)
print(f"{'Bang':<10} {'Trung tam':>10} {'Buoc song':>12} {'FSPL @1000km':>15}")
print(f"{'':<10} {'(GHz)':>10} {'(cm)':>12} {'(dB)':>15}")
print("-" * 75)

for name, center, low, high in bands:
    wl = wavelength_cm(center)
    fspl = fspl_db(center, 1000)
    print(f"{name:<10} {center:>10.1f} {wl:>12.2f} {fspl:>15.1f}")

# So sanh kich thuoc anten
print("\n2. Duong kinh Anten cho Tang ich 30 dBi")
print("-" * 75)
print(f"{'Bang':<10} {'Tan so':>10} {'Duong kinh':>12} {'Tuong doi':>12}")
print(f"{'':<10} {'(GHz)':>10} {'(m)':>12} {'kich thuoc':>12}")
print("-" * 75)

ref_size = antenna_diameter_for_gain(30, 6.0)
for name, center, low, high in bands:
    diameter = antenna_diameter_for_gain(30, center)
    relative = diameter / ref_size
    print(f"{name:<10} {center:>10.1f} {diameter:>12.3f} {relative:>12.2f}x")

# Suy giam mua
print("\n3. Suy giam Mua (25 mm/hr, duong 5 km)")
print("-" * 75)
print(f"{'Bang':<10} {'Tan so':>10} {'Suy giam':>15} {'Anh huong':>20}")
print(f"{'':<10} {'(GHz)':>10} {'(dB)':>15} {'':>20}")
print("-" * 75)

for name, center, low, high in bands:
    atten = rain_attenuation_simplified(center, 25)
    if atten < 0.5:
        impact = "Khong dang ke"
    elif atten < 2:
        impact = "Nhe"
    elif atten < 5:
        impact = "Trung binh"
    elif atten < 10:
        impact = "Dang ke"
    else:
        impact = "Nghiem trong"
    print(f"{name:<10} {center:>10.1f} {atten:>15.2f} {impact:>20}")

# Tom tat ung dung
print("\n4. Ung dung Chinh theo Bang")
print("-" * 75)
applications = {
    "Bang L": "GPS, Iridium, Inmarsat, satcom di dong",
    "Bang S": "Radar thoi tiet, NASA TDRSS, Sirius XM",
    "Bang C": "FSS truyen thong, phan phoi cap, SNG",
    "Bang X": "Quan su, anh SAR, radar bien",
    "Bang Ku": "DTH TV, VSAT, thu thap tin tuc",
    "Bang Ka": "Bang rong HTS, lien ket feeder Starlink",
}

for name, center, low, high in bands:
    apps = applications.get(name, "Khac nhau")
    print(f"{name:<10}: {apps}")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Nham lan chu cai bang.** Dinh nghia IEEE va ITU khac nhau; luon chi dinh dai tan so.

2. **Nhieu 5G.** Bang C 3.7-3.98 GHz hien chia se voi 5G o nhieu nuoc.

3. **Du tru mua.** Khong dung gia dinh du tru mua bang C cho thiet ke bang Ka.

4. **Kich thuoc anten.** Anten di dong bang L co tang ich rat thap; ngan sach lien ket anh huong.

5. **Bien doi quy dinh.** Phan bo bang khac nhau theo vung ITU (Chau My, Chau Au, Chau A-TBD).

6. **Nhieu mat dat.** WiFi/lo vi song bang S, 5G bang C, radar bang X.

7. **Gia dinh cu.** "Bang C chong mua" la tuong doi; van fade trong bao nhiet doi nang.
