## Tai sao dieu nay quan trong

Ve tinh thong luong cao (HTS) can bang thong—rat nhieu. Bang Ku, Ka, Q, V, va W (12-110 GHz) cung cap phan bo pho khong lo ma cac bang thap hon don gian khong the cung cap. Starlink, OneWeb, Viasat, va SES O3b mPOWER deu hoat dong chu yeu trong bang Ka. Hieu cac tan so cao hon nay—loi the dung luong va thach thuc fade mua cua chung—la can thiet de thiet ke he thong bang rong ve tinh hien dai.

## Ap dung

**So sanh bang tan so cao va may tinh fade mua:**

```python
import numpy as np

C = 299792458  # Toc do anh sang (m/s)

def wavelength_cm(frequency_ghz):
    """Tinh buoc song bang centimet."""
    return (C / (frequency_ghz * 1e9)) * 100

def fspl_db(frequency_ghz, distance_km):
    """Ton that khong gian tu do tinh bang dB."""
    return 20 * np.log10(distance_km) + 20 * np.log10(frequency_ghz) + 92.45

def rain_attenuation_itu_r838(frequency_ghz, rain_rate_mm_hr, 
                               elevation_deg=30, polarization='H'):
    """
    Mo hinh suy giam mua ITU-R P.838 (don gian hoa).
    Tra ve suy giam cu the tinh bang dB/km.
    """
    f = frequency_ghz
    
    if f < 2.9:
        k = 0.0000387 * f ** 0.912
        alpha = 0.912
    elif f < 54:
        k = 0.0000998 * f ** 1.276
        alpha = 1.021 + 0.0048 * f
    else:
        k = 0.0002 * f ** 1.175
        alpha = 1.1
    
    gamma_r = k * (rain_rate_mm_hr ** alpha)
    return gamma_r

def effective_path_length_km(rain_rate_mm_hr, elevation_deg, altitude_km=5):
    """
    Chieu dai duong hieu qua qua mua ITU-R P.618 (don gian hoa).
    """
    h_r = 4.0  # km, do cao mua dien hinh vi do trung
    L_s = (h_r - altitude_km * 0) / np.sin(np.radians(elevation_deg))
    L_s = max(L_s, 0)
    L_g = L_s * np.cos(np.radians(elevation_deg))
    r = 1 / (1 + 0.045 * L_g)
    return L_s * r

def total_rain_fade_db(frequency_ghz, rain_rate_mm_hr, elevation_deg=30):
    """Tong fade mua cho duong xieng."""
    gamma = rain_attenuation_itu_r838(frequency_ghz, rain_rate_mm_hr)
    L_eff = effective_path_length_km(rain_rate_mm_hr, elevation_deg)
    return gamma * L_eff

# Dinh nghia bang
hf_bands = [
    ("Bang Ku", 14.0, 12.0, 18.0),
    ("Bang Ka DL", 20.0, 17.7, 20.2),
    ("Bang Ka UL", 28.5, 27.5, 30.0),
    ("Bang Q", 42.0, 33.0, 50.0),
    ("Bang V", 55.0, 40.0, 75.0),
    ("Bang W", 85.0, 75.0, 110.0),
]

print("Phan tich Bang Ve tinh Tan so Cao")
print("=" * 75)

# Thuoc tinh co ban
print("\n1. So sanh Thuoc tinh Bang")
print("-" * 75)
print(f"{'Bang':<12} {'Tam':>8} {'λ':>10} {'FSPL @36Mm':>12} {'FSPL @550km':>12}")
print(f"{'':<12} {'(GHz)':>8} {'(cm)':>10} {'GEO (dB)':>12} {'LEO (dB)':>12}")
print("-" * 75)

for name, center, low, high in hf_bands:
    wl = wavelength_cm(center)
    fspl_geo = fspl_db(center, 36000)
    fspl_leo = fspl_db(center, 550)
    print(f"{name:<12} {center:>8.1f} {wl:>10.3f} {fspl_geo:>12.1f} {fspl_leo:>12.1f}")

# So sanh fade mua
print("\n2. So sanh Fade Mua (goc nang 30°)")
print("-" * 75)
print(f"{'Bang':<12} {'Mua nhe':>12} {'Vua phai':>12} {'Nang':>12} {'Nhiet doi':>12}")
print(f"{'':<12} {'5 mm/hr':>12} {'15 mm/hr':>12} {'25 mm/hr':>12} {'50 mm/hr':>12}")
print("-" * 75)

for name, center, low, high in hf_bands:
    fade_5 = total_rain_fade_db(center, 5)
    fade_15 = total_rain_fade_db(center, 15)
    fade_25 = total_rain_fade_db(center, 25)
    fade_50 = total_rain_fade_db(center, 50)
    print(f"{name:<12} {fade_5:>12.1f} {fade_15:>12.1f} {fade_25:>12.1f} {fade_50:>12.1f}")

# Dung luong ACM theo mua
print("\n3. Dung luong ACM theo Mua (vi du bang Ka 28 GHz)")
print("-" * 75)

modcods = [
    ("32APSK 9/10", 4.40, 14.0),
    ("16APSK 3/4", 3.00, 10.2),
    ("8PSK 2/3", 2.00, 7.8),
    ("QPSK 1/2", 1.00, 3.0),
    ("QPSK 1/4", 0.50, -0.3),
]

symbol_rate_msps = 250
clear_sky_cn = 20

print(f"Toc do symbol: {symbol_rate_msps} Msym/s")
print(f"C/N troi quang: {clear_sky_cn} dB")
print("-" * 75)
print(f"{'Luong mua':>12} {'Fade':>8} {'C/N hqu':>10} {'ModCod':>15} {'Thong luong':>12}")
print(f"{'(mm/hr)':>12} {'(dB)':>8} {'(dB)':>10} {'':>15} {'(Mbps)':>12}")
print("-" * 75)

for rain_rate in [0, 5, 10, 15, 25, 40]:
    fade = total_rain_fade_db(28.0, rain_rate)
    effective_cn = clear_sky_cn - fade
    
    best_modcod = None
    best_rate = 0
    for modcod_name, bits_per_sym, required_cn in modcods:
        if effective_cn >= required_cn:
            rate = bits_per_sym * symbol_rate_msps
            if rate > best_rate:
                best_rate = rate
                best_modcod = modcod_name
    
    if best_modcod is None:
        best_modcod = "MAT LIEN KET"
        best_rate = 0
    
    print(f"{rain_rate:>12} {fade:>8.1f} {effective_cn:>10.1f} {best_modcod:>15} {best_rate:>12.0f}")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Bo qua fade mua.** Bang Ka co the mat 20+ dB trong mua to; thiet ke cho 99.5% do kha dung can du tru dang ke.

2. **Gia dinh thong luong co dinh.** ACM co nghia thong luong thay doi theo thoi tiet; lap ke hoach dung luong phai tinh den thong ke mua.

3. **Nut that co chai gateway.** Lien ket feeder Q/V can da dang dia diem hoac gateway tro thanh diem loi duy nhat.

4. **Goc nang.** Goc nang thap = duong xieng dai hon qua mua = nhieu fade hon. Cac vi tri nhiet doi bi anh huong gap doi.

5. **Hap thu khi quyen.** Hap thu oxy 60 GHz (bang V) tao "cua so" va "tuong" trong pho.

6. **Gia dinh kich thuoc thiet bi.** Tan so cao hon can tang ich anten cao hon; "dia cung kich thuoc" khong co nghia hieu suat nhu nhau.

7. **Nhieu.** Bang Ka ngay cang dong duc; phoi hop voi cac nha van hanh khac la can thiet.
