## Tai sao dieu nay quan trong

Moi truyen dan ve tinh su dung pho da duoc phan bo boi Lien minh Vien thong Quoc te (ITU). Neu khong co phoi hop ITU, cac ve tinh se gay nhieu cho nhau va cho cac he thong mat dat. Hieu khung phan bo cua ITU—cac vung, loai dich vu, ghi chu, va quy trinh phoi hop—la can thiet cho bat ky ai phong hoac van hanh he thong ve tinh. Lam sai pho co the dung toan bo du an cua ban.

## Ap dung

**Tra cuu va phan tich phan bo ITU:**

```python
# Co so du lieu phan bo Quy dinh Vo tuyen ITU (don gian hoa)

ITU_ALLOCATIONS = {
    # Bang L
    "1525-1559": {
        "vung1": ["VE TINH DI DONG (↓)", "5.341", "5.351A"],
        "vung2": ["VE TINH DI DONG (↓)", "5.341"],
        "vung3": ["VE TINH DI DONG (↓)", "5.341"],
        "dich_vu": ["Inmarsat", "Thuraya"],
    },
    "1559-1610": {
        "vung1": ["RNSS", "RNSS HANG KHONG"],
        "vung2": ["RNSS", "RNSS HANG KHONG"],
        "vung3": ["RNSS", "RNSS HANG KHONG"],
        "dich_vu": ["GPS L1", "Galileo E1", "GLONASS L1"],
    },
    
    # Bang C
    "3700-4200": {
        "vung1": ["VE TINH CO DINH (↓)", "CO DINH"],
        "vung2": ["VE TINH CO DINH (↓)", "CO DINH", "5.C1 (5G)"],
        "vung3": ["VE TINH CO DINH (↓)", "CO DINH"],
        "dich_vu": ["Phan phoi TV bang C", "Backhaul teleport"],
    },
    
    # Bang Ka
    "17700-20200": {
        "vung1": ["FSS (↓)", "MSS (↓)"],
        "vung2": ["FSS (↓)", "MSS (↓)"],
        "vung3": ["FSS (↓)", "MSS (↓)"],
        "dich_vu": ["Downlink HTS", "Starlink", "O3b"],
    },
    "27500-30000": {
        "vung1": ["FSS (↑)", "MSS (↑)"],
        "vung2": ["FSS (↑)", "MSS (↑)"],
        "vung3": ["FSS (↑)", "MSS (↑)"],
        "dich_vu": ["Uplink HTS", "Starlink", "O3b"],
    },
}

def tra_cuu_phan_bo(freq_mhz):
    """Tim phan bo ITU cho tan so cho truoc."""
    for band, data in ITU_ALLOCATIONS.items():
        low, high = band.split("-")
        if float(low) <= freq_mhz <= float(high):
            return band, data
    return None, None

def in_bang_phan_bo():
    """In tom tat phan bo day du."""
    print("Quy dinh Vo tuyen ITU — Tom tat Phan bo Ve tinh")
    print("=" * 80)
    
    for band, data in ITU_ALLOCATIONS.items():
        low, high = band.split("-")
        freq_range = f"{low}-{high} MHz"
        if float(low) >= 1000:
            freq_range = f"{float(low)/1000:.1f}-{float(high)/1000:.1f} GHz"
        
        print(f"\n{freq_range}")
        print("-" * 40)
        print(f"  Vung 1: {', '.join(data.get('vung1', []))}")
        print(f"  Vung 2: {', '.join(data.get('vung2', []))}")
        print(f"  Vung 3: {', '.join(data.get('vung3', []))}")
        print(f"  Dich vu: {', '.join(data.get('dich_vu', []))}")

# May tinh thoi gian bieu nop ho so
from datetime import datetime, timedelta

def thoi_gian_bieu_nop_ho_so(ngay_phong_muc_tieu):
    """Tinh thoi han nop ho so ITU."""
    phong = datetime.strptime(ngay_phong_muc_tieu, "%Y-%m-%d")
    
    thoi_han = {
        "Nop API": phong - timedelta(days=7*365),
        "Nop CR": phong - timedelta(days=5*365),
        "Hoan tat phoi hop": phong - timedelta(days=2*365),
        "Thong bao": phong - timedelta(days=1*365),
        "Phong (BIU)": phong,
        "Thoi han BIU": phong + timedelta(days=90),
    }
    
    print(f"Ngay phong muc tieu: {ngay_phong_muc_tieu}")
    print("-" * 40)
    for moc, ngay in thoi_han.items():
        print(f"  {moc:25} {ngay.strftime('%Y-%m-%d')}")

in_bang_phan_bo()
print("\n" + "=" * 80)
print("\nMay tinh Thoi gian bieu Nop ho so ITU:")
print("-" * 40)
thoi_gian_bieu_nop_ho_so("2028-06-01")
```

**Danh sach kiem tra phoi hop:**

| Buoc | Thoi han | Hanh dong chinh |
|------|----------|-----------------|
| API | T-7 nam | Nop voi co quan quoc gia, cong bo trong IFIC |
| CR | T-5 nam | Chi tiet ky thuat, kich hoat phoi hop |
| Phoi hop | T-2 nam | Thoa thuan song phuong voi cac ben bi anh huong |
| Thong bao | T-1 nam | Yeu cau ghi MIFR |
| BIU | T+90 ngay | Chung minh su dung thuc te cac tan so |

<section class="lesson-pitfalls">

## Cam bay

1. **Bo lo thoi han nop ho so.** Cua so BIU 7 nam la nghiem ngat; bo lo no va mat ho so.

2. **Nham lan so cap vs thu cap.** Dich vu thu cap KHONG co quyen bao ve; viet hoa quan trong.

3. **Khac biet theo vung.** Cung tan so co the co quy tac khac nhau o cac vung ITU khac nhau.

4. **Phuc tap ghi chu.** Hang nghin ghi chu tao ngoai le cu the theo quoc gia; doc can than.

5. **Danh gia thap phoi hop.** Cung GEO dong duc; mong doi 50+ thoa thuan phoi hop cho cac bang pho bien.

6. **Rui ro ve tinh giay.** Mot so thuc the nop ho so dau co de ban quyen phoi hop; xac minh do tin cay doi tac.

7. **Tang quoc gia.** Phan bo ITU van phai duoc cap phep boi moi co quan quoc gia (FCC, Ofcom, v.v.).
