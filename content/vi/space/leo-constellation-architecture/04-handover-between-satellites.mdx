## Tai sao dieu nay quan trong

Ve tinh LEO o do cao 550 km bay qua trong khoang 5-10 phut. De duy tri dich vu lien tuc, thiet bi nguoi dung va gateway phai chuyen doi muot ma tu ve tinh nay sang ve tinh khac—thuong la moi vai phut. Qua trinh chuyen giao nay khac can ban voi handoff di dong: ca hai dau cuoi deu di chuyen (ve tinh 7.5 km/s, Trai Dat quay), va thoi gian co the du doan tu co hoc quy dao. Hieu cac chien luoc chuyen giao la can thiet de thiet ke dich vu LEO do tre thap, kha dung cao.

## Ap dung

**May tinh thoi gian chuyen giao:**

```python
import numpy as np
from math import radians, degrees, sin, cos, asin, sqrt, atan2

R_EARTH = 6371  # km
MU = 398600.4418  # km³/s²

def orbital_velocity_km_s(altitude_km):
    """Van toc quy dao tai do cao cho truoc."""
    r = R_EARTH + altitude_km
    return sqrt(MU / r)

def orbital_period_min(altitude_km):
    """Chu ky quy dao tinh bang phut."""
    r = R_EARTH + altitude_km
    return 2 * np.pi * sqrt(r**3 / MU) / 60

def visibility_duration_min(altitude_km, min_elevation_deg):
    """Thoi gian ve tinh hien thi tren goc ngoc toi thieu."""
    r = R_EARTH + altitude_km
    el_rad = radians(min_elevation_deg)
    rho = np.arcsin(R_EARTH / r * np.cos(el_rad)) - el_rad
    arc_fraction = 2 * rho / (2 * np.pi)
    T = orbital_period_min(altitude_km)
    return T * arc_fraction

def handovers_per_hour(altitude_km, min_elevation_deg, sats_in_view=2.5):
    """Uoc tinh tan suat chuyen giao."""
    visibility = visibility_duration_min(altitude_km, min_elevation_deg)
    return 60 / visibility * sats_in_view

def elevation_vs_time(altitude_km, max_elevation_deg, time_array_min):
    """Goc ngoc trong qua trinh bay qua ve tinh (don gian)."""
    pass_duration = visibility_duration_min(altitude_km, 5)
    t_peak = pass_duration / 2
    
    elevations = []
    for t in time_array_min:
        el = max_elevation_deg * (1 - ((t - t_peak) / t_peak)**2)
        elevations.append(max(0, el))
    return elevations

def handover_decision(
    current_el_deg,
    next_sat_el_deg,
    current_snr_db,
    threshold_el_deg=25,
    threshold_snr_db=5,
    hysteresis_db=2
):
    """Quyet dinh co nen chuyen giao khong."""
    reasons = []
    
    if current_el_deg < threshold_el_deg:
        reasons.append(f"goc ngoc {current_el_deg:.1f}° < {threshold_el_deg}°")
    
    if current_snr_db < threshold_snr_db:
        reasons.append(f"SNR {current_snr_db:.1f} dB < {threshold_snr_db} dB")
    
    if next_sat_el_deg > current_el_deg + hysteresis_db:
        reasons.append(f"sat moi ngoc {next_sat_el_deg:.1f}° > hien tai + tre")
    
    should_handover = len(reasons) > 0
    return should_handover, reasons
```

## Cam bay

1. **Anten co khi.** Khong the tao-truoc-ngat; chuyen giao mat 30-60 giay.

2. **Hieu ung ping-pong.** Tre khong dieu chinh tot gay chuyen nhanh giua hai ve tinh.

3. **Tre dinh tuyen ISL.** Chuyen giao tai thiet bi khong giup neu duong ISL mat thoi gian hoi tu.

4. **Chi phi xac thuc.** Xac thuc lai day du moi chuyen giao lang phi thoi gian; dung ve phien.

5. **Mat can bang tai.** Tat ca nguoi dung trong o chuyen giao dong thoi → dam dong len ve tinh moi.

6. **Nhay cam TCP.** Tang dot RTT lon kich hoat lui kiem soat tac nghen; dung QUIC neu co the.

7. **Truong hop bien hang hai.** Tau o vi do cuc thay it ve tinh hon; co the co khoang chuyen giao.
