## Tai sao dieu nay quan trong

Cac chom sao sieu lon nhu Starlink (12,000+ ve tinh), OneWeb (648), va Kuiper (3,236) dang dinh hinh lai ket noi toan cau. Khac voi cac he thong GEO voi 3 ve tinh phu Trai Dat, chom sao LEO can hang ngan tau vu tru hoat dong phoi hop. Hieu thiet ke chom sao sieu lon—cac lop quy dao, hinh hoc phu song, va mo rong dung luong—la can thiet cho bat ky ai xay dung hoac tich hop voi mang ve tinh the he moi.

## Ap dung

**May tinh dinh co chom sao:**

```python
import numpy as np

# Hang so
R_EARTH = 6371  # km
MU_EARTH = 398600.4418  # km³/s²

def coverage_radius_km(altitude_km, min_elevation_deg):
    """Tinh ban kinh phu song cho do cao va ngoc toi thieu."""
    r = R_EARTH + altitude_km
    el_rad = np.radians(min_elevation_deg)
    
    # Goc trung tam tu ve tinh den mep phu song
    rho = np.arcsin(R_EARTH / r * np.cos(el_rad)) - el_rad
    
    return R_EARTH * rho

def satellites_for_coverage(altitude_km, min_elevation_deg, overlap_factor=1.2):
    """Ve tinh toi thieu can cho phu song lien tuc."""
    coverage_radius = coverage_radius_km(altitude_km, min_elevation_deg)
    
    # Dien tich be mat Trai Dat
    earth_area = 4 * np.pi * R_EARTH**2
    
    # Dien tich phu song moi ve tinh (xap xi tron)
    sat_area = np.pi * coverage_radius**2
    
    # Tinh den chong lap va hinh hoc cau
    n_sats = earth_area / sat_area * overlap_factor * 1.5  # Hieu chinh cau
    
    return int(np.ceil(n_sats))

def orbital_period_min(altitude_km):
    """Chu ky quy dao tinh bang phut."""
    r = R_EARTH + altitude_km
    T = 2 * np.pi * np.sqrt(r**3 / MU_EARTH)
    return T / 60

def latency_ms(altitude_km, elevation_deg):
    """Do tre truyen mot chieu tinh bang mili giay."""
    r = R_EARTH + altitude_km
    el_rad = np.radians(elevation_deg)
    
    # Tam nghieng
    slant = r * np.sqrt(1 - (R_EARTH/r * np.cos(el_rad))**2) - R_EARTH * np.sin(el_rad)
    
    # Thoi gian khu hoi
    return slant / 299.792 * 2  # RTT tinh bang ms

def walker_constellation(inclination_deg, total_sats, n_planes, phasing):
    """Tham so chom sao Walker."""
    sats_per_plane = total_sats // n_planes
    plane_spacing_deg = 360 / n_planes
    sat_spacing_deg = 360 / sats_per_plane
    phase_offset_deg = phasing * sat_spacing_deg / n_planes
    
    return {
        'inclination_deg': inclination_deg,
        'total_sats': total_sats,
        'n_planes': n_planes,
        'sats_per_plane': sats_per_plane,
        'plane_spacing_deg': plane_spacing_deg,
        'sat_spacing_deg': sat_spacing_deg,
        'phase_offset_deg': phase_offset_deg,
        'notation': f"{inclination_deg}°:{total_sats}/{n_planes}/{phasing}"
    }

# Phan tich
print("Phan tich Thiet ke Chom sao Sieu lon")
print("=" * 60)

# Phan tich phu song cho cac do cao khac nhau
print("\n1. Phan tich Phu song theo Do cao")
print("-" * 60)
altitudes = [340, 550, 1200, 35786]
min_el = 25

print(f"  {'Do cao':>10} {'Phu song':>12} {'Ve tinh':>10} {'Chu ky':>10} {'RTT':>8}")
print(f"  {'(km)':>10} {'bk (km)':>12} {'toi thieu':>10} {'(phut)':>10} {'(ms)':>8}")
print("-" * 60)
for alt in altitudes:
    cov = coverage_radius_km(alt, min_el)
    n_sats = satellites_for_coverage(alt, min_el)
    period = orbital_period_min(alt)
    rtt = latency_ms(alt, min_el)
    print(f"  {alt:>10} {cov:>12.0f} {n_sats:>10} {period:>10.1f} {rtt:>8.1f}")

# Phan tich chom sao Starlink
print("\n2. Phan tich Lop Starlink")
print("-" * 60)
shells = [
    (550, 53.0, 72, 22, "Lop chinh"),
    (540, 53.2, 72, 22, "Du phong"),
    (570, 70.0, 36, 20, "Vi do cao"),
    (560, 97.6, 6, 58, "Cuc SSO"),
]

total_sats = 0
print(f"  {'Do cao':>6} {'Nghieng':>7} {'MP':>5} {'V/MP':>5} {'Tong':>7} {'Muc dich':>12}")
print("-" * 60)
for alt, inc, planes, spp, purpose in shells:
    total = planes * spp
    total_sats += total
    print(f"  {alt:>6} {inc:>6.1f}° {planes:>5} {spp:>5} {total:>7} {purpose:>12}")
print("-" * 60)
print(f"  {'Tong':>31} {total_sats:>7}")

# Ky hieu Walker
print("\n3. Vi du Chom sao Walker")
print("-" * 60)
constellations = [
    ("Starlink chinh", 53, 1584, 72, 1),
    ("OneWeb", 87.9, 648, 18, 1),

    print(f"    Ve tinh/MP: {walker['sats_per_plane']} (cach {walker['sat_spacing_deg']:.1f}°)")

# Uoc tinh dung luong
print("\n4. Uoc tinh Dung luong (lop Starlink)")
print("-" * 60)
```

<section class="lesson-pitfalls">

## Cam bay

1. **Phu song ≠ dung luong.** Co ve tinh phia tren khong co nghia la bang thong kha dung.

2. **Khoang trong vi do.** Nghieng 53° de lai vung cuc khong duoc phu neu khong co lop bo sung.

3. **Tac nghen tai giao cat.** Duong mat dat hoi tu tai vi do nghieng—rui ro va cham tang.

4. **Rang buoc quy dinh.** Dang ky ITU, giam thieu manh vo, va phoi hop pho gioi han trien khai.

5. **Nut co chai phong.** Ngay ca voi ten lua tai su dung, trien khai mat nhieu nam cho hang ngan ve tinh.

6. **Luc can khi quyen.** Quy dao thap hon can duy tri tram thuong xuyen hoac chap nhan tuoi tho ngan hon.

7. **O nhiem anh sang.** Chom sao lon anh huong thien van; giam thieu them chi phi.
