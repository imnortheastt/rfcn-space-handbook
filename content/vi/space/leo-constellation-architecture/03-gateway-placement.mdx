## Tai sao dieu nay quan trong

Ngay ca voi chom sao trang bi ISL, luu luong cuoi cung phai den internet mat dat. Vi tri gateway xac dinh dung luong he thong, do tre den may chu noi dung, tuan thu quy dinh, va kha nang phuc hoi hoat dong. Mang gateway dat khong tot co the gay tac nghen cho mot chom sao co kha nang. Hieu cac danh doi—truy cap cap quang, da dang thoi tiet, phe duyet quy dinh, va hinh hoc phu song—la can thiet cho nha van hanh va tich hop chom sao.

## Ap dung

**Bo toi uu vi tri gateway:**

```python
import numpy as np
from math import radians, sin, cos, asin, sqrt

R_EARTH = 6371  # km

def haversine_km(lat1, lon1, lat2, lon2):
    """Khoang cach duong tron lon giua hai diem."""
    lat1, lon1, lat2, lon2 = map(radians, [lat1, lon1, lat2, lon2])
    dlat = lat2 - lat1
    dlon = lon2 - lon1
    a = sin(dlat/2)**2 + cos(lat1)*cos(lat2)*sin(dlon/2)**2
    return 2 * R_EARTH * asin(sqrt(a))

def satellite_visibility_duration_min(altitude_km, min_elevation_deg):
    """Thoi gian xap xi ve tinh LEO hien thi."""
    r = R_EARTH + altitude_km
    el_rad = radians(min_elevation_deg)
    rho = np.arcsin(R_EARTH / r * np.cos(el_rad)) - el_rad
    arc_fraction = 2 * rho / (2 * np.pi)
    T = 2 * np.pi * np.sqrt(r**3 / 398600.4418) / 60
    return T * arc_fraction

def gateway_coverage_radius_km(altitude_km, min_elevation_deg):
    """Ban kinh phu song cua tram gateway."""
    r = R_EARTH + altitude_km
    el_rad = radians(min_elevation_deg)
    rho = np.arcsin(R_EARTH / r * np.cos(el_rad)) - el_rad
    return R_EARTH * rho

def rain_fade_margin_db(rain_rate_mm_hr, frequency_ghz, elevation_deg, path_length_km=5):
    """Uoc tinh fade mua (mo hinh ITU-R don gian)."""
    k = 0.0001 * frequency_ghz**1.5
    alpha = 1.0 + 0.01 * frequency_ghz
    gamma = k * (rain_rate_mm_hr ** alpha)
    L_eff = path_length_km / sin(radians(elevation_deg))
    return gamma * L_eff

def gateway_site_score(
    fiber_distance_km,
    ixp_distance_km,
    rain_zone,
    latitude,
    constellation_inclination=53
):
    """Cham diem vi tri gateway tiem nang (cao hon = tot hon)."""
    score = 100
    score -= min(30, fiber_distance_km * 0.5)
    score -= min(20, ixp_distance_km * 0.2)
    rain_penalties = {'A': 0, 'K': 10, 'P': 25}
    score -= rain_penalties.get(rain_zone, 15)
    lat_diff = abs(latitude) - constellation_inclination
    if lat_diff < 10:
        score += 10
    elif lat_diff > 30:
        score -= 10
    return max(0, score)

# Phan tich gateway vi du
print("Phan tich Bo tri Gateway LEO")
print("=" * 70)

sites = [
    ("Boca Chica, TX", 25.9, -97.4, 10, 50, 'K'),
    ("Redmond, WA", 47.7, -122.1, 5, 10, 'K'),
    ("Fucino, Italy", 42.0, 13.6, 20, 30, 'A'),
    ("Warkworth, NZ", -36.4, 174.7, 30, 80, 'K'),
    ("São Paulo, BR", -23.5, -46.6, 5, 15, 'P'),
    ("Pune, India", 18.5, 73.9, 10, 40, 'P'),
    ("Nairobi, Kenya", -1.3, 36.8, 50, 100, 'K'),
]

print(f"\n{'Tram':<20} {'Vi do':>6} {'Quang':>8} {'IXP':>8} {'Mua':>6} {'Diem':>7}")
print("-" * 70)

for name, lat, lon, fiber_km, ixp_km, rain in sites:
    score = gateway_site_score(fiber_km, ixp_km, rain, lat)
    print(f"{name:<20} {lat:>6.1f}° {fiber_km:>6}km {ixp_km:>6}km {rain:>6} {score:>7.1f}")

# Phan tich phu song
print("\n" + "=" * 70)
print("Tham so Phu song Gateway (LEO 550 km)")
print("-" * 70)

altitude = 550
for min_el in [10, 25, 40]:
    radius = gateway_coverage_radius_km(altitude, min_el)
    duration = satellite_visibility_duration_min(altitude, min_el)
    print(f"Ngoc toi thieu {min_el}°: bk phu song {radius:.0f} km, thoi gian bay {duration:.1f} phut")

# Lap ke hoach dung luong
print("\n" + "=" * 70)
print("Lap ke hoach Dung luong Gateway")
print("-" * 70)

antennas_per_site = 8
throughput_per_antenna_gbps = 15
sites_count = 30
user_avg_mbps = 100
overbooking_ratio = 20

total_gateway_capacity_gbps = antennas_per_site * throughput_per_antenna_gbps * sites_count
max_concurrent_users_gbps = total_gateway_capacity_gbps * 1000 / user_avg_mbps
max_subscribers = max_concurrent_users_gbps * overbooking_ratio

print(f"Anten moi tram: {antennas_per_site}")
print(f"Thong luong moi anten: {throughput_per_antenna_gbps} Gbps")
print(f"So tram gateway: {sites_count}")
print(f"Tong dung luong gateway: {total_gateway_capacity_gbps} Gbps")
print(f"ND dong thoi toi da @ {user_avg_mbps} Mbps: {max_concurrent_users_gbps:,.0f}")
print(f"Thue bao toi da ({overbooking_ratio}:1): {max_subscribers/1e6:.1f}M")

# Vi du fade mua
print("\n" + "=" * 70)
print("Phan tich Fade Mua (Ka-band 20 GHz, ngoc 30°)")
print("-" * 70)

rain_rates = [
    ("Mua nhe (Vung A)", 8),
    ("Mua vua (Vung K)", 42),
    ("Mua to (Vung P)", 145),
]

for desc, rate in rain_rates:
    fade = rain_fade_margin_db(rate, 20, 30)
    print(f"{desc}: {rate} mm/hr → {fade:.1f} dB fade")
```

## Cam bay

1. **Doan cuoi cap quang.** Gateway khong co backhaul cap quang la nut co chai.

2. **Cham tre quy dinh.** Cap phep tan so co the mat 1-3 nam o mot so quoc gia.

3. **Fade mua.** Lien ket Ka-band mat 10-25 dB trong mua to—can da dang vi tri.

4. **Khoang chuyen giao.** Khong du anten, xay ra mat ket noi ngan khi chuyen ve tinh.

5. **Tieu thu dien lon.** Tram gateway su dung 200-500 kW; vi tri xa co the thieu dung luong.

6. **Sai lech vi do.** Gateway o xich dao thay it lan bay hon tai vi do nghieng chom sao.

7. **Diem loi don.** Mat gateway khu vuc co the mat dien vung lon neu khong co ISL du phong.
