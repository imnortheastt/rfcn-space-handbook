## Tai sao dieu nay quan trong

Khac voi he thong GEO voi dau chan co dinh, dung luong chom sao LEO la dong—no thay doi theo vi do, thoi gian trong ngay, va vi tri ve tinh tuc thoi. ## Ap dung

**May tinh lap ke hoach dung luong:**

```python
import numpy as np

def satellite_capacity_gbps(
    bandwidth_ghz=3.5,
    spectral_efficiency_bps_hz=3.0,
    overhead_fraction=0.15
):
    """Dung luong thong luong tho ve tinh."""
    raw = bandwidth_ghz * spectral_efficiency_bps_hz
    usable = raw * (1 - overhead_fraction)
    return usable

def users_per_satellite(
    sat_capacity_gbps,
    avg_user_speed_mbps,
    concurrent_fraction=0.15
):
    """So thue bao toi da mot ve tinh co the phuc vu."""
    concurrent_capacity_mbps = sat_capacity_gbps * 1000
    concurrent_users = concurrent_capacity_mbps / avg_user_speed_mbps
    total_subscribers = concurrent_users / concurrent_fraction
    return int(total_subscribers)

def system_capacity(
    n_satellites,
    sat_capacity_gbps,
    active_fraction=0.5
):
    """Tong dung luong he thong tren vung dan cu."""
    return n_satellites * sat_capacity_gbps * active_fraction

def congestion_probability(
    subscribers_in_cell,
    sat_capacity_gbps,
    avg_session_mbps=25,
    peak_concurrent_fraction=0.25
):
    """Uoc tinh xac suat tac nghen trong gio cao diem."""
    peak_demand_gbps = (subscribers_in_cell * peak_concurrent_fraction * 
                        avg_session_mbps / 1000)
    
    if peak_demand_gbps < sat_capacity_gbps * 0.7:
        return "Thap (< 10%)"
    elif peak_demand_gbps < sat_capacity_gbps * 0.9:
        return "Trung binh (10-30%)"
    elif peak_demand_gbps < sat_capacity_gbps:
        return "Cao (30-60%)"
    else:
        return "Rat cao (> 60%)"

def latitude_density_factor(latitude_deg, inclination_deg=53):
    """Mat do ve tinh tuong doi tai vi do cho truoc."""
    if abs(latitude_deg) > inclination_deg:
        return max(0.1, 1 - (abs(latitude_deg) - inclination_deg) / 30)
    else:
        return 0.5 + 0.5 * np.sin(np.pi * abs(latitude_deg) / (2 * inclination_deg))

# Phan tich
print("Phan tich Dung luong Chom sao LEO")
print("=" * 70)

# Dung luong moi ve tinh
sat_cap = satellite_capacity_gbps()
print(f"\n1. Dung luong Moi Ve tinh")
print("-" * 70)
print(f"  Bang thong: 3.5 GHz")
print(f"  Hieu qua pho: 3.0 bps/Hz")
print(f"  Dung luong su dung: {sat_cap:.1f} Gbps")

# Dung luong he thong
print(f"\n2. Dung luong He thong (4000 ve tinh)")
print("-" * 70)
n_sats = 4000
sys_cap = system_capacity(n_sats, sat_cap)
print(f"  Tong ve tinh: {n_sats}")
print(f"  Hoat dong tren vung dan cu: ~50%")
print(f"  Dung luong he thong: {sys_cap:.0f} Gbps = {sys_cap/1000:.1f} Tbps")

# Uoc tinh thue bao
print(f"\n3. Uoc tinh Dung luong Thue bao")
print("-" * 70)
speeds = [100, 50, 25]
for speed in speeds:
    users = users_per_satellite(sat_cap, speed)
    total = users * n_sats * 0.5
    print(f"  @ {speed} Mbps trung binh: {users:,} moi sat, {total/1e6:.1f}M toan he thong")

# Phan tich tac nghen theo vi tri
print(f"\n4. Phan tich Tac nghen theo Mat do O")
print("-" * 70)
print(f"{'Thue bao/o':<20} {'Nhu cau dinh':<15} {'Rui ro tac nghen':<20}")
print("-" * 70)
for subs in [100, 300, 500, 1000, 2000]:
    demand = subs * 0.25 * 25 / 1000
    risk = congestion_probability(subs, sat_cap)
    print(f"{subs:<20} {demand:.1f} Gbps{'':<7} {risk:<20}")

# Phan tich vi do
print(f"\n5. Dung luong theo Vi do (nghieng 53°)")
print("-" * 70)
print(f"{'Vi do':<12} {'Mat do VT':<15} {'DL hieu qua':<15}")
print("-" * 70)
for lat in [0, 20, 35, 45, 53, 60, 70]:
    density = latitude_density_factor(lat)
    eff_cap = sat_cap * density
    print(f"{lat}°{'':<10} {density:.2f}x{'':<12} {eff_cap:.1f} Gbps/sat")

# Mo phong thoi gian trong ngay
print(f"\n6. Bien doi Nhu cau (Dong My, dan cu dien hinh)")
print("-" * 70)
hours = [0, 3, 6, 9, 12, 15, 18, 21]
demand_factors = [0.2, 0.1, 0.15, 0.4, 0.5, 0.6, 0.9, 1.0]

print(f"{'Gio':<10} {'Nhu cau':<15} {'Trang thai o':<20}")
print("-" * 70)
for hour, factor in zip(hours, demand_factors):
    demand_gbps = 500 * factor * 0.25 * 25 / 1000
    status = congestion_probability(int(500 * factor), sat_cap)
    bar = "█" * int(factor * 20)
    print(f"{hour:02d}:00{'':<5} {bar:<20} {status:<20}")

# Lap ke hoach SLA
print(f"\n7. Huong dan Lap ke hoach SLA")
print("-" * 70)
print("""

  ───────────────────┼─────────────┼─────────────┼──────────
  Toc do cam ket     │ 25 Mbps     │ 100 Mbps    │ 10 Mbps
  Toc do dinh        │ 250 Mbps    │ 500 Mbps    │ 100 Mbps
  Kha dung           │ 99%         │ 99.5%       │ 99%
  Do tre (dien hinh) │ < 50 ms     │ < 40 ms     │ < 100 ms
  Uu tien khi        │ Binh thuong │ Cao         │ Dam bao
  tac nghen          │             │             │
""")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Marketing vs thuc te.** "Len den 500 Mbps" la dinh, khong phai trung binh khi tac nghen.

2. **Gia dinh overbooking.** 20:1 hoat dong cho nguoi dung binh thuong; gamer/streamer pha vo mo hinh.

3. **Bo qua dia ly.** Ban cho vung do thi day dac ma khong lap ke hoach dung luong.

4. **Mu gio cao diem.** Dung luong trong tot luc 3 AM; 8 PM moi la bai kiem tra thuc.

5. **Hieu ung bien o.** Nguoi dung tai bien o co toc do kem hon trung tam o.

6. **Anh huong thoi tiet.** Fade mua giam dung luong hieu qua; phai lap ke hoach cho kha dung 99%.

7. **Tac nghen backhaul.** Dung luong gateway co the gioi han truoc khi dung luong ve tinh.
