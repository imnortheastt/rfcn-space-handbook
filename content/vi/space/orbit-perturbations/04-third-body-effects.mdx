## Tai sao dieu nay quan trong

Mat Trang va Mat Troi khong chi de tao thuy trieu. Luc hap dan cua chung len ve tinh gay nhieu loan quy dao dang ke, dac biet cho cac nhiem vu GEO va do cao lon. Khong co hieu chinh, ve tinh GEO troi 0.75°/nam ve do nghieng—can delta-v dang ke de doi pho.

## Ap dung

**Tinh toan vat the thu ba:**

```python
import numpy as np

# Hang so
MU_EARTH = 398600.4418  # km³/s²
MU_MOON = 4902.8        # km³/s²
MU_SUN = 1.327e11       # km³/s²
R_MOON = 384400         # km (khoang cach trung binh)
R_SUN = 149597870.7     # km (1 AU)

def third_body_accel(r_sat, r_third_body, mu_third):
    """
    Tinh gia toc nhieu loan vat the thu ba
    
    r_sat: vector vi tri ve tinh (tam Trai Dat) [km]
    r_third_body: vi tri vat the thu ba (tam Trai Dat) [km]
    mu_third: tham so hap dan vat the thu ba [km³/s²]
    
    Tra ve: vector gia toc [km/s²]
    """
    # Vector tu ve tinh den vat the thu ba
    r_rel = r_third_body - r_sat
    r_rel_mag = np.linalg.norm(r_rel)
    
    # Vector tu Trai Dat den vat the thu ba
    R_mag = np.linalg.norm(r_third_body)
    
    # Gia toc nhieu loan
    a = mu_third * (r_rel / r_rel_mag**3 - r_third_body / R_mag**3)
    
    return a

def inclination_drift_rate(i_deg):
    """
    Toc do troi do nghieng GEO gan dung do mat trang-mat troi
    
    Tra ve: di/dt theo do/nam
    """
    i = np.radians(i_deg)
    # Mo hinh don gian - toc do thuc thay doi theo RAAN
    return 0.85  # do/nam trung binh

def ns_stationkeeping_dv(inclination_tolerance_deg, years=1):
    """
    Uoc tinh delta-v duy tri vi tri B-N
    
    Tra ve: delta-v theo m/s
    """
    drift_rate = 0.85  # do/nam
    if inclination_tolerance_deg < drift_rate * years:
        # Can doi pho toan bo troi
        dv_per_year = 50  # m/s/nam (dien hinh)
        return dv_per_year * years
    else:
        return 0

# Vi du: Nhieu loan tai GEO
r_geo = np.array([42164, 0, 0])  # km, tren truc x
r_moon = np.array([R_MOON, 0, 0])  # don gian: Mat Trang tren truc x

a_moon = third_body_accel(r_geo, r_moon, MU_MOON)
a_sun = third_body_accel(r_geo, np.array([R_SUN, 0, 0]), MU_SUN)

print("Nhieu loan Vat the Thu ba tai GEO:")
print("-" * 50)
print(f"Gia toc Mat Trang: {np.linalg.norm(a_moon):.2e} km/s²")
print(f"Gia toc Mat Troi:  {np.linalg.norm(a_sun):.2e} km/s²")
print(f"Ti le Trang/Troi:  {np.linalg.norm(a_moon)/np.linalg.norm(a_sun):.1f}")
print()

# Ngan sach duy tri vi tri
print("Ngan sach Duy tri Vi tri GEO (nhiem vu 15 nam):")
print("-" * 50)
ns_dv = ns_stationkeeping_dv(0.05, 15)
ew_dv = 4 * 15  # ~4 m/s/nam trung binh
print(f"B-N (do nghieng): {ns_dv:.0f} m/s")
print(f"D-T (kinh do):    {ew_dv:.0f} m/s")
print(f"Tong:             {ns_dv + ew_dv:.0f} m/s")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Vi tri Mat Trang quan trong.** Khoang cach Mat Trang thay doi ±7%; nhieu loan thay doi ±20%.

2. **Khong chi do lon.** Huong so voi quy dao xac dinh phan tu nao thay doi.

3. **Thang thoi gian dai.** Hieu ung do nghieng mat vai thang de bieu hien; khong the thay trong mot vong.

4. **Ket hop voi J2.** De du doan chinh xac, phai bao gom J2 + mat trang-mat troi cung nhau.

5. **Hoang dao vs xich dao.** Quy dao Mat Trang nam trong mat phang hoang dao, khong phai xich dao Trai Dat.

6. **GEO nghieng.** Mot so nha dieu hanh chap nhan do nghieng 5-10° de tiet kiem delta-v B-N.

7. **Quy dao xu ly.** Kozai-Lidov co the gay tai nhap quy dao nghia trang nghieng cao.
