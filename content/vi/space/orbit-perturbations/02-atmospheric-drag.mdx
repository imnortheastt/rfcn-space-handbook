## Tai sao dieu nay quan trong

## Ap dung

**Tinh toan luc can:**

```python
import numpy as np

# Hang so
MU = 398600.4418  # km³/s²
R_E = 6378.137    # km

def atmospheric_density(alt_km, F107=150):
    """
    Mo hinh khi quyen ham mu don gian
    
    alt_km: do cao tinh bang km
    F107: chi so thong luong mat troi (70=min, 250=max)
    
    Tra ve: mat do tinh bang kg/m³
    """
    # Mat do tham chieu va do cao thang (mo hinh don gian)
    h0 = 400  # do cao tham chieu km
    rho0 = 3.7e-12 * (F107/150)**1.5  # kg/m³ tai 400 km
    H = 60 + 0.1*(F107-70)  # do cao thang km
    
    return rho0 * np.exp(-(alt_km - h0)/H)

def decay_rate(a_km, B, F107=150):
    """
    Tinh toc do suy giam ban truc lon
    
    a_km: ban truc lon tinh bang km
    B: he so ballistic tinh bang kg/m²
    F107: chi so thong luong mat troi
    
    Tra ve: da/dt tinh bang km/ngay
    """
    alt = a_km - R_E
    rho = atmospheric_density(alt, F107)
    
    # Chuyen sang SI de tinh toan
    a_m = a_km * 1000
    mu_m = MU * 1e9  # m³/s²
    
    # da/dt = -pi * a * rho * sqrt(mu*a) / B
    da_dt = -np.pi * a_m * rho * np.sqrt(mu_m * a_m) / B
    
    # Chuyen sang km/ngay
    return da_dt * 86400 / 1000

def orbital_lifetime(a0_km, B, F107=150, dt_days=1):
    """
    Uoc tinh tuoi tho quy dao qua tich phan so
    
    Tra ve: tuoi tho tinh bang ngay
    """
    a = a0_km
    t = 0
    
    while a > R_E + 120:  # Dung tai 120 km (tai nhap)
        da = decay_rate(a, B, F107) * dt_days
        a += da
        t += dt_days
        
        if t > 100000:  # Gioi han 274 nam
            return float('inf')
    
    return t

# Vi du: Tham so ISS
B_iss = 150  # kg/m²
a_iss = R_E + 400

print("Phan tich Suy giam Quy dao ISS:")
print("-" * 50)
for F107 in [70, 150, 250]:
    da_dt = decay_rate(a_iss, B_iss, F107)
    life = orbital_lifetime(a_iss, B_iss, F107)
    print(f"F10.7 = {F107}:")
    print(f"  Toc do suy giam: {da_dt:.3f} km/ngay")
    print(f"  Tuoi tho: {life:.0f} ngay")
    print()

# So sanh CubeSat
B_cubesat = 25  # kg/m²
print(f"CubeSat (B={B_cubesat} kg/m²) tai 400 km:")
life_cs = orbital_lifetime(a_iss, B_cubesat, 150)
print(f"  Tuoi tho: {life_cs:.0f} ngay (vs ISS: {orbital_lifetime(a_iss, B_iss, 150):.0f} ngay)")
```

**Toc do suy giam LEO dien hinh:**

| Ve tinh | Do cao | B (kg/m²) | da/dt | Tan suat day |
|---------|--------|-----------|-------|--------------|
| ISS | 400 km | 150 | 0.05-0.2 km/ngay | Hang thang |
| Starlink | 550 km | ~20 | 0.001 km/ngay | Lien tuc |
| CubeSat | 400 km | 25 | 0.3-1 km/ngay | Khong |

## Cam bay

1. **Bat dinh mat do.** Hoat dong mat troi co the thay doi mat do 30× o cung do cao.

2. **Do nhay ham mu.** Sai so nho do cao → sai so lon luc can.

3. **Tu the quan trong.** C_D×A thay doi neu ve tinh quay vs on dinh.

4. **Khong chi LEO.** Can diem GTO di qua khi quyen; luc can anh huong chu ky.

5. **Nong vs can.** Nong tai nhap bat dau ~120 km; luc can dang ke tu 600 km.

6. **Do cao thang thay doi.** H tang theo do cao; mo hinh mot-H la gan dung.

7. **Bien doi ngay/dem.** Mat do tai 400 km co the thay doi 2× giua mat ngay va dem.
