## Tai sao dieu nay quan trong

Moi ve tinh ban theo doi truc tuyen deu su dung SGP4. Do la thuat toan bien bo Phan tu Hai dong (TLE) thanh du doan vi tri. Space-Track.org, Celestrak, va moi nguoi theo doi ve tinh nghiep du deu dua vao thuat toan 50 nam tuoi nay—hieu diem manh va han che cua no la thiet yeu cho bat ky ai lam viec voi du lieu quy dao.

## Ap dung

**Su dung SGP4 trong Python:**

```python
from sgp4.api import Satrec, jday
from sgp4 import exporter
import numpy as np

# TLE vi du (ISS)
line1 = "1 25544U 98067A   24001.50000000  .00001234  00000-0  12345-4 0  9999"
line2 = "2 25544  51.6400 123.4567 0006789  12.3456 123.4567 15.50000000123456"

# Tao doi tuong ve tinh
satellite = Satrec.twoline2rv(line1, line2)

# Lan truyen den thoi gian cu the
# jday(nam, thang, ngay, gio, phut, giay)
jd, fr = jday(2024, 1, 1, 12, 0, 0)

# Lay vi tri va van toc trong TEME
error, r, v = satellite.sgp4(jd, fr)

if error == 0:
    print(f"Vi tri (TEME): {r} km")
    print(f"Van toc (TEME): {v} km/s")
else:
    print(f"Ma loi SGP4: {error}")

# Lan truyen qua khoang thoi gian
def propagate_tle(sat, start_jd, duration_hours, step_min=1):
    """Lan truyen TLE va tra ve vi tri"""
    positions = []
    t = 0
    while t < duration_hours * 60:
        jd = start_jd + t / 1440  # Chuyen phut sang ngay
        e, r, v = sat.sgp4(jd, 0.0)
        if e == 0:
            positions.append(r)
        t += step_min
    return np.array(positions)

# Chuyen TEME sang ECEF (don gian hoa)
def teme_to_ecef(r_teme, gmst_rad):
    """Xoay TEME sang ECEF dung GMST"""
    c, s = np.cos(gmst_rad), np.sin(gmst_rad)
    R = np.array([[c, s, 0], [-s, c, 0], [0, 0, 1]])
    return R @ r_teme

# Tinh GMST (don gian hoa)
def gmst(jd):
    """Gio Thien van Trung binh Greenwich (radian)"""
    T = (jd - 2451545.0) / 36525.0
    gmst_sec = 67310.54841 + (876600*3600 + 8640184.812866)*T + 0.093104*T**2
    return np.radians((gmst_sec % 86400) / 240)  # Chuyen sang radian

print("\nVi du lan truyen:")
jd_start, _ = jday(2024, 1, 1, 0, 0, 0)
positions = propagate_tle(satellite, jd_start, 2)  # 2 gio
print(f"Tao {len(positions)} diem vi tri")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Epoch TLE quan trong.** Do chinh xac giam nhanh; dung TLE moi.

2. **Khong hop le sau dieu dong.** TLE khong hop le sau bat ky dieu dong nao; cho cap nhat.

3. **He TEME.** Dau ra la TEME, khong phai J2000 hay ECEF; chuyen khi can.

4. **Hieu sai B*.** B* khong phai he so ballistic; dung dung cho vat ly.

5. **Nguong khong gian sau.** Chu ky > 225 phut kich hoat SDP4; thuat toan khac.

6. **Loi checksum.** Xac nhan checksum TLE truoc khi su dung.

7. **Trung binh vs tiep xuc.** Phan tu TLE la phan tu trung binh, khong phai tiep xuc.

8. **Hoat dong mat troi.** Bat dinh luc can tu hoat dong mat troi chu dao sai so LEO.
