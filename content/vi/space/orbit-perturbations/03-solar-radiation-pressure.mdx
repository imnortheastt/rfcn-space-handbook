## Tai sao dieu nay quan trong

Anh sang mat troi day. Tai 1 AU, Mat Troi tac dung ~4.5 μN/m² len bat ky be mat nao. Cho ve tinh do cao lon noi luc can bien mat, ap suat buc xa mat troi (SRP) tro thanh luc phi hap dan chu dao—quan trong cho duy tri vi tri GEO va la co so cho buom mat troi.

## Ap dung

**Tinh toan SRP:**

```python
import numpy as np

# Hang so
P_SR = 4.56e-6  # N/m² tai 1 AU
C_LIGHT = 299792458  # m/s
SOLAR_FLUX = 1361  # W/m² tai 1 AU

def srp_acceleration(area, mass, C_R=1.4, r_AU=1.0):
    """
    Tinh gia toc SRP
    
    area: dien tich mat cat ngang [m²]
    mass: khoi luong ve tinh [kg]
    C_R: he so phan xa (1.0-2.0)
    r_AU: khoang cach tu Mat Troi tinh bang AU
    
    Tra ve: gia toc [m/s²]
    """
    return P_SR * C_R * area / (mass * r_AU**2)

def srp_force(area, C_R=1.4, r_AU=1.0):
    """
    Tinh luc SRP
    
    Tra ve: luc [N]
    """
    return P_SR * area * C_R / r_AU**2

def characteristic_acceleration(sigma):
    """
    Gia toc dac trung buom mat troi
    
    sigma: mat do dien tich [kg/m²]
    
    Tra ve: gia toc [m/s²]
    """
    return 2 * P_SR / sigma

# Vi du: Ve tinh vien thong GEO
mass = 2000  # kg
area = 50    # m² (tam pin mat troi)
C_R = 1.3

a_srp = srp_acceleration(area, mass, C_R)
print("SRP Ve tinh GEO:")
print(f"  Ti le A/m: {area/mass:.4f} m²/kg")
print(f"  Gia toc SRP: {a_srp:.2e} m/s²")
print(f"  Delta-v moi nam: {a_srp * 365.25 * 86400:.1f} m/s")
print()

# Vi du: Buom mat troi
print("So sanh Buom Mat troi:")
for name, sigma in [("IKAROS", 0.5), ("LightSail 2", 0.1), ("Tien tien", 0.01)]:
    a_c = characteristic_acceleration(sigma)
    print(f"  {name}: σ={sigma} kg/m², a_c={a_c*1000:.3f} mm/s²")
print()

# So sanh SRP vs luc can tai cac do cao khac nhau
print("SRP vs Luc can Chu dao:")
print("-" * 50)
for alt in [400, 600, 800, 1000, 35786]:
    if alt < 1000:
        drag_approx = 1e-5 * np.exp(-(alt-400)/100)
    else:
        drag_approx = 1e-15
    
    srp = a_srp
    ratio = srp / drag_approx if drag_approx > 0 else float('inf')
    dominant = "Luc can" if ratio < 1 else "SRP"
    print(f"  {alt:>6} km: Ti le SRP/Can = {ratio:.1e}, chu dao: {dominant}")
```

<section class="lesson-pitfalls">

## Cam bay

1. **Dien tich bien doi.** Tam pin mat troi theo doi Mat Troi; dien tich hieu dung thay doi theo tu the.

2. **Bat dinh C_R.** Tinh chat be mat suy thoai; C_R thay doi theo tuoi tho nhiem vu.

3. **Mo hinh bong.** Bo qua ban bong gay sai so du doan.

4. **Tu che bong.** Cac thanh phan ve tinh che bong nhau; hinh hoc phuc tap.

5. **Albedo Trai Dat.** Trai Dat phan xa anh sang mat troi; them vao SRP tai do cao thap.

6. **Phat lai buc xa nhiet.** Be mat nong phat photon IR; hieu ung giong Yarkovsky.

7. **Giam theo 1/r².** SRP tai Sao Hoa (1.5 AU) chi 44% cua Trai Dat.
