## Tai sao dieu nay quan trong

Trong khi RF ISL la cong nghe da duoc chung minh, chung bi gioi han bang thong boi tinh kha dung pho va kich thuoc anten. Lien ket lien ve tinh quang (laser) hoat dong tai buoc song hong ngoai noi bang thong gan nhu khong gioi han va anten thu nho xuong kinh vien vong co centimeters. Mang luoi laser cua Starlink chung minh rang ISL quang cho phep ket noi do tre thap toan cau ma khong phu thuoc tram mat dat. Hieu vat ly va ky thuat ISL quang la quan trong cho thiet ke chom sao the he tiep theo.

## Ap dung

**May tinh ngan sach lien ket ISL quang:**

```python
import numpy as np

# Hang so
C = 299792458  # m/s
H_PLANCK = 6.62607015e-34  # J·s

def optical_fspl_db(distance_km, wavelength_nm):
    """Ton hao khong gian tu do cho buoc song quang."""
    d_m = distance_km * 1000
    lambda_m = wavelength_nm * 1e-9
    return 20 * np.log10(4 * np.pi * d_m / lambda_m)

def telescope_gain_db(aperture_m, wavelength_nm, efficiency=0.8):
    """Tang ich kinh vien vong (gioi han nhieu xa)."""
    lambda_m = wavelength_nm * 1e-9
    area = np.pi * (aperture_m / 2) ** 2
    gain = efficiency * (np.pi * aperture_m / lambda_m) ** 2
    return 10 * np.log10(gain)

def beam_divergence_urad(aperture_m, wavelength_nm):
    """Phan ky buc xa goc day tinh bang microradians."""
    lambda_m = wavelength_nm * 1e-9
    theta = 2.44 * lambda_m / aperture_m
    return theta * 1e6  # Chuyen doi sang μrad

def beam_footprint_m(range_km, divergence_urad):
    """Duong kinh buc xa tai tam muc tieu."""
    return range_km * 1000 * divergence_urad * 1e-6

def photons_per_bit(rx_power_dbm, data_rate_gbps, wavelength_nm):
    """Tinh so photon nhan duoc moi bit."""
    rx_power_w = 10 ** ((rx_power_dbm - 30) / 10)
    lambda_m = wavelength_nm * 1e-9
    photon_energy = H_PLANCK * C / lambda_m
    photons_per_sec = rx_power_w / photon_energy
    bits_per_sec = data_rate_gbps * 1e9
    return photons_per_sec / bits_per_sec

def optical_isl_link_budget(tx_power_dbm, tx_aperture_m, rx_aperture_m,
                            wavelength_nm, range_km, 
                            tx_loss_db=2, rx_loss_db=2, pointing_loss_db=3):
    """Ngan sach lien ket ISL quang day du."""
    # Tang ich
    tx_gain = telescope_gain_db(tx_aperture_m, wavelength_nm)
    rx_gain = telescope_gain_db(rx_aperture_m, wavelength_nm)
    
    # Ton hao duong truyen
    fspl = optical_fspl_db(range_km, wavelength_nm)
    
    # Cong suat nhan
    rx_power = (tx_power_dbm + tx_gain - tx_loss_db 
                - fspl + rx_gain - rx_loss_db - pointing_loss_db)
    
    # Tham so buc xa
    divergence = beam_divergence_urad(tx_aperture_m, wavelength_nm)
    footprint = beam_footprint_m(range_km, divergence)
    pointing_req = divergence / 10  # μrad
    
    return {
        'tx_power_dbm': tx_power_dbm,
        'tx_gain_dbi': tx_gain,
        'rx_gain_dbi': rx_gain,
        'fspl_db': fspl,
        'rx_power_dbm': rx_power,
        'divergence_urad': divergence,
        'footprint_m': footprint,
        'pointing_req_urad': pointing_req,
        'pointing_req_arcsec': pointing_req * 0.206,
    }

def detector_sensitivity_dbm(modulation, data_rate_gbps):
    """Do nhay bo tach gan dung cho dieu che cho truoc."""
    # Do nhay theo photons/bit, sau do chuyen doi sang dBm
    ppb = {
        'OOK': 100,       # On-off keying
        'DPSK': 30,       # Differential PSK
        'BPSK_ket_hop': 10,   # BPSK ket hop
        'QPSK_ket_hop': 15,   # QPSK ket hop
    }
    
    if modulation not in ppb:
        return None
    
    wavelength_nm = 1550
    lambda_m = wavelength_nm * 1e-9
    photon_energy = H_PLANCK * C / lambda_m
    
    power_w = ppb[modulation] * photon_energy * data_rate_gbps * 1e9
    return 10 * np.log10(power_w) + 30  # dBm

# Phan tich
print("Phan tich Ngan sach Lien ket ISL Quang")
print("=" * 60)

# ISL quang lop Starlink
print("\n1. ISL Quang LEO (lop Starlink)")
print("-" * 60)
budget = optical_isl_link_budget(
    tx_power_dbm=37,         # 5 W quang
    tx_aperture_m=0.08,      # kinh vien vong 8 cm
    rx_aperture_m=0.08,
    wavelength_nm=1550,      # Bang C vien thong
    range_km=2000,
)

for k, v in budget.items():
    if 'db' in k.lower():
        print(f"  {k:20}: {v:+.1f} dB")
    elif 'urad' in k.lower():
        print(f"  {k:20}: {v:.1f} μrad")
    elif 'arcsec' in k.lower():
        print(f"  {k:20}: {v:.1f} giay cung")
    elif '_m' in k.lower() and 'dbm' not in k.lower():
        print(f"  {k:20}: {v:.1f} m")
    else:
        print(f"  {k:20}: {v:.1f}")

# Kiem tra bien do cho cac dieu che khac nhau

    margin = budget['rx_power_dbm'] - sens
    print(f"  {mod:15} {sens:>10.1f} dBm {margin:>8.1f} dB")

# Photon moi bit
```

## Cam bay

1. **Chi huong la tat ca.** Loi chi huong 5 μrad mat 3 dB; 20 μrad mat hau het tin hieu.

2. **Thoi gian thu.** Tim ve tinh doi tac mat 10-60 giay; lap ke hoach cho khoang trong chuyen giao.

3. **Choi loa Mat Troi.** Khi Mat Troi gan duong ngam, bo tach bao hoa; co the can cat lien ket.

4. **Nhieu nen.** Phan xa Trai Dat, phan xa Mat Troi tu muc tieu co the tang san nhieu.

5. **Quan ly nhiet.** Laser sinh nhiet; van hanh on dinh can dieu khien nhiet chu dong.

6. **Lua chon buoc song quan trong.** 1550 nm co nen mat troi thap hon nhung 1064 nm co laser cong suat cao hon.

7. **Danh doi ket hop vs khong ket hop.** Tach song ket hop can laser do rong vach khop — them phuc tap.
