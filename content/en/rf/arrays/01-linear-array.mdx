<section class="lesson-hook">

## Why linear arrays?

A single antenna element has limited directivity. To get a narrow beam and high gain, you need multiple elements working together. The linear array — elements arranged in a line — is the simplest multi-element configuration.

Linear arrays are everywhere: radar antennas, cellular base stations, phased arrays, and radio telescopes. Understanding the linear array is the foundation for all array antenna theory.

</section>

<section class="lesson-intuition">

## Intuition

Each element radiates its own wave. When the waves from all elements arrive at a distant point, they add constructively or destructively depending on their relative phases.

If you arrange elements along a line with proper spacing, the waves add in phase in certain directions (creating a strong beam) and cancel in others (creating nulls). The more elements, the narrower the beam.

Think of it like synchronized swimmers: when they move in unison, the combined effect is dramatic. When out of sync, it's chaos.

</section>

<section class="lesson-mechanism">

## Mechanism

### Uniform linear array (ULA)

$N$ identical elements spaced uniformly at distance $d$ along the z-axis.

The total electric field in far-field:

$$E_{total}(\theta) = E_{element}(\theta) \times AF(\theta)$$

where $AF$ is the array factor:

$$AF(\theta) = \sum_{n=0}^{N-1} w_n \, e^{jnkd\cos\theta}$$

For uniform amplitude ($w_n = 1$):

$$AF(\theta) = \frac{\sin\left(\frac{N\psi}{2}\right)}{\sin\left(\frac{\psi}{2}\right)}$$

where $\psi = kd\cos\theta + \beta$ and $\beta$ is the progressive phase shift between elements.

### Broadside array

All elements fed in phase ($\beta = 0$). Maximum radiation perpendicular to array axis.

Peak at $\theta = 90°$ (broadside direction).

### Endfire array

Progressive phase shift $\beta = -kd$. Maximum radiation along array axis.

Peak at $\theta = 0°$ or $\theta = 180°$ (endfire direction).

### Beamwidth

For broadside ULA:

$$\theta_{HPBW} \approx 0.886 \frac{\lambda}{Nd}$$

More elements or wider spacing → narrower beam.

### Grating lobes

If element spacing $d > \lambda$, additional main lobes (grating lobes) appear:

$$\cos\theta_{grating} = \cos\theta_{main} \pm \frac{m\lambda}{d}$$

To avoid grating lobes in all visible space:

$$d \leq \frac{\lambda}{1 + |\sin\theta_{scan}|}$$

For broadside ($\theta_{scan} = 0$): $d \leq \lambda$.

For ±90° scan: $d \leq \lambda/2$.

### Directivity

For broadside ULA with $d = \lambda/2$:

$$D_{array} \approx 2N$$

or in dB: $D_{dB} \approx 10\log_{10}(2N)$.

### Sidelobe levels

Uniform amplitude: first sidelobe at −13.3 dB.

Amplitude taper (e.g., Taylor, Chebyshev) reduces sidelobes at cost of slightly wider main beam.

</section>

<section class="lesson-apply">

## Apply

**Example: 8-element broadside array**

$N = 8$ elements, $d = \lambda/2$, uniform amplitude.

Beamwidth: $\theta_{HPBW} \approx 0.886 \times \lambda / (8 \times \lambda/2) = 12.6°$

Directivity: $D \approx 2 \times 8 = 16 = 12$ dBi (array factor only).

First sidelobe: −13.3 dB.

**Example: Endfire array**

Same 8 elements with $\beta = -kd = -\pi$ (for endfire).

Beam points along array axis. Beamwidth wider than broadside (~25°). Superdirectivity possible with closer spacing but at cost of efficiency.

**Example: Grating lobe check**

10 GHz array with $d = 2$ cm.

$\lambda = 3$ cm, so $d/\lambda = 0.67$.

For broadside: $d < \lambda$ satisfied, no grating lobes.

For ±60° scan: need $d < \lambda/(1 + \sin 60°) = 0.54\lambda$. Our $d = 0.67\lambda$ → grating lobe enters visible space at extreme scan.

**Example: Cellular sector antenna**

65° horizontal beamwidth requires approximately:

$$N \approx 0.886 \times 360° / 65° \times (d/\lambda) \approx 5 \text{ elements at } d = \lambda/2$$

Combined with vertical stacking for vertical pattern control.

</section>

<section class="lesson-pitfalls">

## Pitfalls

1. **Element pattern ignored.** Array factor × element pattern = total pattern. Element nulls can suppress array peaks.

2. **Grating lobe surprise.** Wide spacing works at broadside but creates grating lobes when scanned. Always check scan range.

3. **Mutual coupling.** Real elements interact. Close spacing changes element impedance and pattern. Simple ULA theory ignores this.

4. **Edge effects.** End elements see different environments than center elements. Non-uniform excitation results.

5. **Superdirectivity limits.** Endfire with $d < \lambda/4$ can achieve high directivity but with narrow bandwidth and high sensitivity.

6. **Sidelobe control.** Uniform excitation means −13 dB sidelobes. Lower sidelobes require amplitude taper, reducing gain.

</section>
