## Why this matters

Some locations in space let a small object orbit the Sun with the same period as Earth—despite being at different distances. These Lagrange points arise from the balance of gravitational and centrifugal forces in the Sun-Earth system. JWST sits at L2 (1.5 million km from Earth), space weather satellites watch from L1, and L4/L5 trap asteroid clouds. Understanding these points opens doors to deep-space mission design.

### Why L1 and L2 exist

```

│  ┌───────────────────────────────────────────────────────┐         │

│  │  • Farther (1.01 AU): T > 1 year                     │         │
│  │                                                        │         │

│  │                                                        │         │

│                                                                      │
│  THE ANSWER — Earth's gravity helps:                                │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  AT L1 (between Sun and Earth):                       │         │
│  │                                                        │         │
│  │  ←──── Sun gravity    Earth gravity ────→             │         │
```

## Apply

**Lagrange point calculations:**

```python
import numpy as np

# Constants
AU = 149597870.7  # km
M_SUN = 1.989e30  # kg
M_EARTH = 5.972e24  # kg
G = 6.674e-11  # m³/(kg·s²)

def lagrange_l1_l2_distance():
    """Calculate L1 and L2 distances from Earth."""
    # Mass ratio
    mu = M_EARTH / (M_SUN + M_EARTH)
    
    # Hill sphere approximation for L1, L2
    # r_L ≈ R × (μ/3)^(1/3)
    r_hill = AU * (mu / 3) ** (1/3)
    
    # More precise: solve cubic equation
    # For L1: (1-μ)/(x-μ)² - μ/(x-1+μ)² = x (normalized)
    # Approximate: r_L1 ≈ R × (μ/3)^(1/3) × (1 - μ/3 - μ²/9)
    # r_L2 ≈ R × (μ/3)^(1/3) × (1 + μ/3 - μ²/9)
    
    correction_l1 = 1 - mu/3 - mu**2/9
    correction_l2 = 1 + mu/3 - mu**2/9
    
    r_l1 = r_hill * correction_l1
    r_l2 = r_hill * correction_l2
    
    return {
        'mass_ratio_mu': mu,
        'hill_sphere_km': r_hill,
        'l1_distance_km': r_l1,
        'l2_distance_km': r_l2,
        'l1_distance_lunar': r_l1 / 384400,
        'l2_distance_lunar': r_l2 / 384400
    }

def l4_l5_positions():
    """Calculate L4 and L5 positions."""
    # L4 and L5 form equilateral triangles with Sun and Earth
    # At 60° ahead and behind Earth in its orbit
    
    r_earth = AU
    angle_l4 = 60  # degrees ahead
    angle_l5 = -60  # degrees behind (or 300°)
    
    # In Earth's orbital frame
    l4_x = r_earth * np.cos(np.radians(angle_l4))
    l4_y = r_earth * np.sin(np.radians(angle_l4))
    
    l5_x = r_earth * np.cos(np.radians(angle_l5))
    l5_y = r_earth * np.sin(np.radians(angle_l5))
    
    # Distance from Earth
    dist_l4 = np.sqrt((l4_x - r_earth)**2 + l4_y**2)
    dist_l5 = np.sqrt((l5_x - r_earth)**2 + l5_y**2)
    
    return {
        'l4_x_au': l4_x / AU,
        'l4_y_au': l4_y / AU,
        'l5_x_au': l5_x / AU,
        'l5_y_au': l5_y / AU,
        'distance_from_earth_km': dist_l4,
        'distance_from_earth_au': dist_l4 / AU
    }

def stability_timescale():
    """Calculate instability timescale at L2."""
    # Linearized dynamics near L2 show exponential divergence
    # Characteristic time τ ≈ T_orbit / (2π × √(9μ/2))
    
    mu = M_EARTH / (M_SUN + M_EARTH)
    T_orbit_days = 365.25
    
    # Approximate e-folding time
    tau = T_orbit_days / (2 * np.pi * np.sqrt(9 * mu / 2))
    
    return {
        'e_folding_time_days': tau,
        'doubling_time_days': tau * np.log(2),
        'station_keeping_interval_weeks': tau / 7 * 0.5  # Conservative
    }

def transfer_delta_v():
    """Calculate delta-V for Earth to L2 transfer."""
    # From LEO (400 km) to L2 halo orbit
    # Approximate as trans-lunar injection variant
    
    r_leo = 6371 + 400  # km
    v_leo = np.sqrt(G * M_EARTH / (r_leo * 1000)) / 1000  # km/s
    
    # Escape velocity from LEO
    v_escape = v_leo * np.sqrt(2)
    
    # L2 transfer is slightly less than lunar transfer
    # because L2 is at Earth's gravitational influence boundary
    
    # Trans-L2 injection delta-V
    v_tli_l2 = 3.2  # km/s (empirical from missions)
    
    # Departure burn from LEO
    delta_v_departure = v_tli_l2
    
    # Mid-course corrections
    delta_v_corrections = 0.05  # km/s
    
    # L2 insertion
    delta_v_insertion = 0.02  # km/s (into halo orbit)
    
    return {
        'leo_velocity_km_s': v_leo,
        'escape_velocity_km_s': v_escape,
        'departure_delta_v_km_s': delta_v_departure,
        'corrections_km_s': delta_v_corrections,
        'insertion_km_s': delta_v_insertion,
        'total_km_s': delta_v_departure + delta_v_corrections + delta_v_insertion
    }

# Calculate and display
print("Lagrange Point Analysis (Sun-Earth System)")
print("=" * 60)

# L1/L2 distances
l12 = lagrange_l1_l2_distance()
print(f"\nL1 and L2 Distances:")
print(f"  Mass ratio (Earth/Sun+Earth): {l12['mass_ratio_mu']:.2e}")
print(f"  L1 distance from Earth: {l12['l1_distance_km']/1e6:.2f} million km")
print(f"  L2 distance from Earth: {l12['l2_distance_km']/1e6:.2f} million km")
print(f"  L1 in lunar distances: {l12['l1_distance_lunar']:.1f} LD")
print(f"  L2 in lunar distances: {l12['l2_distance_lunar']:.1f} LD")

# L4/L5
print("\n" + "-" * 60)
l45 = l4_l5_positions()
print(f"L4 and L5 Positions:")
print(f"  L4: 60° ahead of Earth in orbit")
print(f"  L5: 60° behind Earth in orbit")
print(f"  Distance from Earth: {l45['distance_from_earth_km']/1e6:.1f} million km")
print(f"  Distance from Earth: {l45['distance_from_earth_au']:.2f} AU")

# Stability
print("\n" + "-" * 60)
stab = stability_timescale()
print(f"L2 Stability:")
print(f"  E-folding time: {stab['e_folding_time_days']:.1f} days")
print(f"  Station-keeping interval: ~{stab['station_keeping_interval_weeks']:.0f} weeks")

# Transfer
print("\n" + "-" * 60)
dv = transfer_delta_v()
print(f"Transfer from LEO to L2:")
print(f"  Departure delta-V: {dv['departure_delta_v_km_s']:.1f} km/s")
print(f"  Mid-course + insertion: {dv['corrections_km_s'] + dv['insertion_km_s']:.2f} km/s")
print(f"  Total: ~{dv['total_km_s']:.1f} km/s")
print(f"  Compare to Moon: ~3.9 km/s (L2 is easier!)")
```

| L3 | ~300 million km | Unstable | Not used |
| L4 | 150 million km | Stable | Trojans (natural) |
| L5 | 150 million km | Stable | Trojans (natural) |

</section>

<section class="lesson-pitfalls">

## Pitfalls

1. **L1/L2/L3 are unstable.** Objects drift away exponentially; station-keeping is mandatory.

2. **"At L2" doesn't mean at the point.** JWST orbits around L2 in a halo orbit, not at the exact location.

3. **Communication delay.** L2 is ~5 light-seconds from Earth; not real-time control.

4. **No rescue possible.** L2 is 4× farther than the Moon; servicing missions are impractical.

5. **Eclipse risk in Lissajous orbits.** Some paths cross the Sun-Earth line, causing power loss.

6. **L4/L5 stability requires mass ratio.** Only works if primary/secondary mass ratio > 25.

7. **Fuel still needed.** Even stable L4/L5 have perturbations from other planets requiring occasional correction.

</section>
