## Why this matters

Geostationary orbit (GEO) revolutionized telecommunications: park a satellite at 35,786 km, and it appears stationary from Earth. Fixed dishes point at it forever—no tracking needed. This single insight enabled global TV broadcast, VSAT networks, and weather satellites. Understanding GEO versus GSO, orbital slots, and station-keeping is essential for satcom engineering.

### Station-keeping

```
GEO Station-Keeping:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  PERTURBATIONS (forces that push satellite off position):           │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  1. NORTH-SOUTH DRIFT (inclination change)            │         │
│  │     • Sun and Moon gravity pull satellite out of      │         │
│  │       equatorial plane                                │         │
│  │     • Rate: ~0.75-0.95°/year                         │         │
│  │     • Without correction: satellite oscillates N-S    │         │
│  │                                                        │         │
│  │  2. EAST-WEST DRIFT (longitude change)                │         │
│  │     • Earth's gravitational "bumps" (tesseral harmonics)│        │
│  │     • Two stable points: 75°E and 105°W              │         │
│  │     • Two unstable points: 162°E and 12°W            │         │
│  │     • Satellites drift toward stable points          │         │
│  │                                                        │         │
│  │  3. SOLAR RADIATION PRESSURE                          │         │
│  │     • Photons push on solar panels                   │         │
│  │     • Affects eccentricity (orbit becomes elliptical)│         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  STATION-KEEPING MANEUVERS:                                         │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  Maneuver Type     │ Frequency    │ Delta-V/year      │         │
│  │  ──────────────────┼──────────────┼───────────────────│         │
│  │  North-South (N-S) │ Every 2 weeks│ ~45-50 m/s        │         │
│  │  East-West (E-W)   │ Every 2 weeks│ ~2-6 m/s          │         │
│  │  Eccentricity ctrl │ Varies       │ ~1-2 m/s          │         │
│  │                                                        │         │
│  │  Total: ~50 m/s/year for full station-keeping        │         │
│  │                                                        │         │
│  │  N-S is 90% of the fuel budget!                      │         │
│  │  Some operators accept "inclined GEO" to save fuel   │         │
│  │  → Figure-8 ground track, needs tracking antenna     │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  END OF LIFE:                                                       │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  When fuel runs out:                                  │         │
│  │  1. Raise orbit ~300 km to "graveyard orbit"         │         │
│  │  2. Passivate (vent tanks, discharge batteries)      │         │
│  │  3. Satellite drifts, no longer usable               │         │
│  │                                                        │         │
│  │  GEO graveyard: above 36,000 km, debris stays forever│         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## Apply

**GEO orbital calculations:**

```python
import numpy as np

# Constants
MU_EARTH = 398600.4418  # km³/s²
R_EARTH = 6371  # km
C = 299792.458  # km/s
SIDEREAL_DAY_S = 86164.1  # seconds

def geo_altitude():
    """Calculate GEO altitude from first principles."""
    # For geosynchronous orbit: T = 2π√(a³/μ)
    # Solve for a: a = (μ × T² / 4π²)^(1/3)
    
    a_cubed = MU_EARTH * SIDEREAL_DAY_S**2 / (4 * np.pi**2)
    a = a_cubed ** (1/3)  # Semi-major axis (from Earth center)
    altitude = a - R_EARTH
    
    return {
        'semi_major_axis_km': a,
        'altitude_km': altitude,
        'orbital_period_hours': SIDEREAL_DAY_S / 3600
    }

def geo_coverage(min_elevation_deg=5):
    """Calculate GEO coverage geometry."""
    h = 35786  # km
    eps = np.radians(min_elevation_deg)
    
    # Maximum latitude visible
    # At horizon (0° elevation), max latitude ≈ arccos(R/(R+h))
    max_lat_theoretical = np.degrees(np.arccos(R_EARTH / (R_EARTH + h)))
    
    # With minimum elevation constraint
    rho = np.arcsin(R_EARTH * np.cos(eps) / (R_EARTH + h))
    max_lat_practical = 90 - np.degrees(eps) - np.degrees(rho)
    
    # Coverage longitude range (approximation for nadir point)
    coverage_half_angle = max_lat_practical
    
    # Fraction of Earth visible
    coverage_fraction = (1 - np.cos(np.radians(max_lat_practical))) / 2
    
    return {
        'max_latitude_theoretical': max_lat_theoretical,
        'max_latitude_at_min_elev': max_lat_practical,
        'coverage_fraction': coverage_fraction,
        'coverage_percent': coverage_fraction * 100
    }

def geo_latency(elevation_deg=30):
    """Calculate GEO link latency."""
    h = 35786  # km
    eps = np.radians(elevation_deg)
    
    # Slant range
    rho = np.arcsin(R_EARTH * np.cos(eps) / (R_EARTH + h))
    slant_range = (R_EARTH + h) * np.sin(np.pi/2 - eps - rho) / np.cos(eps)
    
    # Latency
    one_way_ms = slant_range / C * 1000
    
    return {
        'slant_range_km': slant_range,
        'one_way_ms': one_way_ms,
        'round_trip_ms': 2 * one_way_ms,
        'user_to_user_ms': 4 * one_way_ms  # Up + down + up + down
    }

def station_keeping_budget():
    """Estimate GEO station-keeping delta-V requirements."""
    # Typical values for GEO station-keeping
    return {
        'north_south_m_s_year': 50,  # Inclination control
        'east_west_m_s_year': 3,     # Longitude control
        'eccentricity_m_s_year': 2,  # Eccentricity control
        'total_m_s_year': 55,
        'typical_lifetime_years': 15,
        'total_delta_v_m_s': 55 * 15
    }

# Calculate GEO parameters
print("GEO Orbital Analysis")
print("=" * 60)

geo = geo_altitude()
print(f"\nGEO Altitude (calculated):")
print(f"  Semi-major axis: {geo['semi_major_axis_km']:.1f} km")
print(f"  Altitude: {geo['altitude_km']:.1f} km")
print(f"  Orbital period: {geo['orbital_period_hours']:.3f} hours")

# Coverage analysis
print("\n" + "-" * 60)
print("Coverage Analysis:")
coverage = geo_coverage(5)
print(f"  Max latitude (theoretical): {coverage['max_latitude_theoretical']:.1f}°")
print(f"  Max latitude (5° min elev): {coverage['max_latitude_at_min_elev']:.1f}°")
print(f"  Coverage per satellite: {coverage['coverage_percent']:.1f}%")
print(f"  Satellites for global: 3 (except polar)")

# Latency analysis
print("\n" + "-" * 60)
print("Latency Analysis:")
for elev in [90, 45, 10, 5]:
    lat = geo_latency(elev)
    print(f"  At {elev}° elevation:")
    print(f"    Slant range: {lat['slant_range_km']:.0f} km")
    print(f"    One-way: {lat['one_way_ms']:.1f} ms")
    print(f"    RTT (hub-terminal): {lat['round_trip_ms']:.1f} ms")

# Station-keeping
print("\n" + "-" * 60)
print("Station-Keeping Budget:")
sk = station_keeping_budget()
print(f"  N-S control: {sk['north_south_m_s_year']} m/s/year")
print(f"  E-W control: {sk['east_west_m_s_year']} m/s/year")
print(f"  Total: {sk['total_m_s_year']} m/s/year")
print(f"  15-year lifetime: {sk['total_delta_v_m_s']} m/s total")
```

<section class="lesson-pitfalls">

## Pitfalls

1. **GEO ≠ GSO.** GEO is a subset of GSO; inclined GSO needs tracking antennas.

2. **No polar coverage.** Above ~81° latitude, GEO satellites are below the horizon.

3. **Slot scarcity.** Prime GEO slots are limited; ITU coordination takes years.

4. **Latency penalty.** 550+ ms RTT makes GEO unsuitable for interactive applications.

5. **Station-keeping fuel.** N-S maneuvers consume 90% of fuel; limits satellite life.

6. **Eclipse seasons.** Twice yearly, GEO satellites enter Earth's shadow for ~72 minutes.

7. **Graveyard orbit.** End-of-life satellites must be moved up; debris persists forever.
