## Why this matters

When a smartphone connects via Starlink while roaming in another country, who pays whom? 5G NTN introduces complex multi-party billing: satellite operators, terrestrial MNOs, visited networks, and service providers all need compensation. Understanding NTN charging architecture is essential for operators building hybrid terrestrial-satellite networks.

## Apply

**NTN charging calculations:**

```python
from dataclasses import dataclass
from datetime import datetime, timedelta
from typing import Optional
import json

@dataclass
class NTNCDRecord:
    """NTN Charging Data Record."""
    record_id: str
    supi: str
    session_start: datetime
    session_end: datetime
    data_volume_ul_kb: int
    data_volume_dl_kb: int
    satellite_id: str
    ntn_cell_id: str
    access_type: str  # NR-NTN-LEO, NR-NTN-GEO, etc.
    service_area: str
    qos_class: int
    home_plmn: str
    visited_plmn: Optional[str] = None

def rate_ntn_session(cdr: NTNCDRecord, rate_plan: dict) -> dict:
    """Calculate charges for an NTN session."""
    
    total_kb = cdr.data_volume_ul_kb + cdr.data_volume_dl_kb
    total_mb = total_kb / 1024
    
    # Base rate
    base_rate = rate_plan.get('base_rate_per_mb', 0.01)
    base_charge = total_mb * base_rate
    
    # NTN surcharge based on access type
    ntn_surcharges = {
        'NR-NTN-LEO': 0.005,  # LEO has lower surcharge (lower latency)
        'NR-NTN-MEO': 0.008,
        'NR-NTN-GEO': 0.010,  # GEO higher (scarce capacity)
    }
    ntn_surcharge = ntn_surcharges.get(cdr.access_type, 0.010)
    ntn_charge = total_mb * ntn_surcharge
    
    # Peak hour multiplier
    hour = cdr.session_start.hour
    peak_hours = rate_plan.get('peak_hours', [8, 9, 10, 17, 18, 19, 20])
    peak_multiplier = 1.5 if hour in peak_hours else 1.0
    
    # QoS premium
    qos_premiums = {
        1: 0.0,    # Best effort
        2: 0.10,   # Conversational voice
        3: 0.05,   # Streaming video
        4: 0.02,   # Interactive
    }
    qos_premium = qos_premiums.get(cdr.qos_class, 0.0)
    
    # Calculate total
    subtotal = (base_charge + ntn_charge) * peak_multiplier
    total_charge = subtotal * (1 + qos_premium)
    
    return {
        'record_id': cdr.record_id,
        'total_mb': round(total_mb, 2),
        'base_charge': round(base_charge, 4),
        'ntn_surcharge': round(ntn_charge, 4),
        'peak_multiplier': peak_multiplier,
        'qos_premium_pct': qos_premium * 100,
        'total_charge': round(total_charge, 4),
        'currency': rate_plan.get('currency', 'USD')
    }

def generate_tap_record(cdr: NTNCDRecord, iot_rate: float) -> dict:
    """Generate TAP record for roaming settlement."""
    
    total_kb = cdr.data_volume_ul_kb + cdr.data_volume_dl_kb
    duration = (cdr.session_end - cdr.session_start).total_seconds()
    
    charge = (total_kb / 1024) * iot_rate
    tax = charge * 0.10  # 10% tax example
    
    return {
        'tap_version': '3.12',
        'sender': cdr.visited_plmn or 'NTNOP001',
        'recipient': cdr.home_plmn,
        'file_sequence': 1,
        'call_records': [{
            'imsi': cdr.supi.replace('imsi-', ''),
            'call_type': 'GPRS',
            'gprs_service_used': {
                'data_volume_incoming_kb': cdr.data_volume_dl_kb,
                'data_volume_outgoing_kb': cdr.data_volume_ul_kb,
                'charging_id': cdr.record_id,
                'pdp_type': 'IPv4',
            },
            'call_timestamp': cdr.session_start.isoformat(),
            'duration_seconds': int(duration),
            'charge': {
                'amount': round(charge, 4),
                'currency': 'USD',
                'exchange_rate': 1.0,
            },
            'tax': round(tax, 4),
            # NTN extensions (proposed)
            'ntn_info': {
                'satellite_id': cdr.satellite_id,
                'access_type': cdr.access_type,
                'service_area': cdr.service_area,
            }
        }],
        'total_charge': round(charge + tax, 4)
    }

def calculate_wholesale_cost(
    capacity_mbps: float,
    hours_used: float,
    wholesale_rate_per_mbps_hour: float
) -> dict:
    """Calculate wholesale satellite capacity cost."""
    
    capacity_cost = capacity_mbps * hours_used * wholesale_rate_per_mbps_hour
    
    # Fixed costs (gateway, ground segment)
    gateway_cost_per_hour = 50.0  # Example
    ground_segment_cost = hours_used * gateway_cost_per_hour
    
    total = capacity_cost + ground_segment_cost
    
    return {
        'capacity_mbps': capacity_mbps,
        'hours_used': hours_used,
        'capacity_cost': round(capacity_cost, 2),
        'ground_segment_cost': round(ground_segment_cost, 2),
        'total_wholesale_cost': round(total, 2)
    }

# Example usage

print("=" * 60)

    record_id="CDR-2026-06-03-00001",
    supi="imsi-310260123456789",
```

<section class="lesson-pitfalls">

## Pitfalls

1. **Latency impacts online charging.** Credit checks take longer via satellite; use larger quota grants.

2. **Satellite ID mapping.** CDRs must correctly identify which satellite served the session for proper cost allocation.

3. **Coverage beam boundaries.** Session may span multiple beams; determine which to charge.

4. **Handover between operators.** If user hands over from terrestrial to NTN mid-session, split the CDR correctly.

5. **Currency and taxation.** NTN crosses borders; determine applicable taxes and currency conversion.

6. **Prepaid balance depletion.** In remote areas with no terrestrial backup, ensure graceful service degradation.

7. **Wholesale vs retail margin.** MNOs must price NTN services to cover wholesale costs while remaining competitive.
