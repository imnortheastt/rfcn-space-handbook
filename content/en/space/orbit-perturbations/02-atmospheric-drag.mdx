## Why this matters

Even at 400 km altitude, there's enough atmosphere to slow satellites by meters per second per year. Without periodic boosts, ISS would reenter within months. Understanding drag is essential for LEO mission planning and debris tracking.

### Drag force

```
Drag Force Equation:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│            1                                             │
│  F_drag = ─ × ρ × v² × C_D × A                          │
│            2                                             │
│                                                          │
│  Where:                                                 │
│  ρ   = atmospheric density [kg/m³]                      │
│  v   = orbital velocity [m/s]                           │
│  C_D = drag coefficient (typically 2.0-2.5)             │
│  A   = cross-sectional area [m²]                        │
│                                                          │
│  ─────────────────────────────────────────────────────  │
│                                                          │
│  BALLISTIC COEFFICIENT (B):                             │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │       m                                         │    │
│  │  B = ──────    [kg/m²]                          │    │
│  │      C_D × A                                    │    │
│  │                                                  │    │
│  │  Higher B = less drag effect = longer lifetime  │    │
│  │  Lower B  = more drag effect = faster decay     │    │
│  │                                                  │    │
│  │  Typical values:                                │    │
```

## Apply

**Drag calculations:**

```python
import numpy as np

# Constants
MU = 398600.4418  # km³/s²
R_E = 6378.137    # km

def atmospheric_density(alt_km, F107=150):
    """
    Simplified exponential atmosphere model
    
    alt_km: altitude in km
    F107: solar flux index (70=min, 250=max)
    
    Returns: density in kg/m³
    """
    # Reference density and scale height (simplified model)
    h0 = 400  # reference altitude km
    rho0 = 3.7e-12 * (F107/150)**1.5  # kg/m³ at 400 km
    H = 60 + 0.1*(F107-70)  # scale height km
    
    return rho0 * np.exp(-(alt_km - h0)/H)

def decay_rate(a_km, B, F107=150):
    """
    Compute semi-major axis decay rate
    
    a_km: semi-major axis in km
    B: ballistic coefficient in kg/m²
    F107: solar flux index
    
    Returns: da/dt in km/day
    """
    alt = a_km - R_E
    rho = atmospheric_density(alt, F107)
    
    # Convert to SI for calculation
    a_m = a_km * 1000
    mu_m = MU * 1e9  # m³/s²
    
    # da/dt = -pi * a * rho * sqrt(mu*a) / B
    da_dt = -np.pi * a_m * rho * np.sqrt(mu_m * a_m) / B
    
    # Convert to km/day
    return da_dt * 86400 / 1000

def orbital_lifetime(a0_km, B, F107=150, dt_days=1):
    """
    Estimate orbital lifetime via numerical integration
    
    Returns: lifetime in days
    """
    a = a0_km
    t = 0
    
    while a > R_E + 120:  # Stop at 120 km (reentry)
        da = decay_rate(a, B, F107) * dt_days
        a += da
        t += dt_days
        
        if t > 100000:  # Cap at 274 years
            return float('inf')
    
    return t

# Example: ISS parameters
B_iss = 150  # kg/m²
a_iss = R_E + 400

print("ISS Orbital Decay Analysis:")
print("-" * 50)
for F107 in [70, 150, 250]:
    da_dt = decay_rate(a_iss, B_iss, F107)
    life = orbital_lifetime(a_iss, B_iss, F107)
    print(f"F10.7 = {F107}:")
    print(f"  Decay rate: {da_dt:.3f} km/day")
    print(f"  Lifetime: {life:.0f} days")
    print()

# CubeSat comparison
B_cubesat = 25  # kg/m²
print(f"CubeSat (B={B_cubesat} kg/m²) at 400 km:")
life_cs = orbital_lifetime(a_iss, B_cubesat, 150)
print(f"  Lifetime: {life_cs:.0f} days (vs ISS: {orbital_lifetime(a_iss, B_iss, 150):.0f} days)")
```

**Typical LEO decay rates:**

| Satellite | Altitude | B (kg/m²) | da/dt | Reboost Freq |
|-----------|----------|-----------|-------|--------------|
| ISS | 400 km | 150 | 0.05-0.2 km/day | Monthly |
| Starlink | 550 km | ~20 | 0.001 km/day | Continuous |
| CubeSat | 400 km | 25 | 0.3-1 km/day | None |

<section class="lesson-pitfalls">

## Pitfalls

1. **Density uncertainty.** Solar activity can change density 30× at same altitude.

2. **Exponential sensitivity.** Small altitude error → large drag error.

3. **Attitude matters.** C_D×A changes if satellite tumbles vs stabilized.

4. **Not just LEO.** GTO perigee passes through atmosphere; drag affects period.

5. **Heating vs drag.** Reentry heating starts ~120 km; drag significant from 600 km.

6. **Scale height varies.** H increases with altitude; single-H model is approximate.

7. **Day/night variation.** Density at 400 km can vary 2× between day and night side.
