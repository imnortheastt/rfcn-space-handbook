## Why this matters

The Moon and Sun aren't just for tides. Their gravitational pull on satellites causes significant orbital perturbations, especially for GEO and high-altitude missions. Without correction, GEO satellites drift 0.75°/year in inclination—requiring substantial delta-v to counteract.

## Apply

**Third-body calculations:**

```python
import numpy as np

# Constants
MU_EARTH = 398600.4418  # km³/s²
MU_MOON = 4902.8        # km³/s²
MU_SUN = 1.327e11       # km³/s²
R_MOON = 384400         # km (mean distance)
R_SUN = 149597870.7     # km (1 AU)

def third_body_accel(r_sat, r_third_body, mu_third):
    """
    Compute third-body perturbation acceleration
    
    r_sat: satellite position vector (Earth-centered) [km]
    r_third_body: third body position (Earth-centered) [km]
    mu_third: gravitational parameter of third body [km³/s²]
    
    Returns: acceleration vector [km/s²]
    """
    # Vector from satellite to third body
    r_rel = r_third_body - r_sat
    r_rel_mag = np.linalg.norm(r_rel)
    
    # Vector from Earth to third body
    R_mag = np.linalg.norm(r_third_body)
    
    # Perturbation acceleration
    a = mu_third * (r_rel / r_rel_mag**3 - r_third_body / R_mag**3)
    
    return a

def inclination_drift_rate(i_deg):
    """
    Approximate GEO inclination drift rate due to lunisolar
    
    Returns: di/dt in degrees/year
    """
    i = np.radians(i_deg)
    # Simplified model - actual rate varies with RAAN
    return 0.85  # deg/year average

def ns_stationkeeping_dv(inclination_tolerance_deg, years=1):
    """
    Estimate N-S stationkeeping delta-v
    
    Returns: delta-v in m/s
    """
    drift_rate = 0.85  # deg/year
    if inclination_tolerance_deg < drift_rate * years:
        # Need to counteract full drift
        dv_per_year = 50  # m/s/year (typical)
        return dv_per_year * years
    else:
        return 0

# Example: Perturbation at GEO
r_geo = np.array([42164, 0, 0])  # km, on x-axis
r_moon = np.array([R_MOON, 0, 0])  # simplified: Moon on x-axis

a_moon = third_body_accel(r_geo, r_moon, MU_MOON)
a_sun = third_body_accel(r_geo, np.array([R_SUN, 0, 0]), MU_SUN)

print("Third-Body Perturbations at GEO:")
print("-" * 50)
print(f"Moon acceleration: {np.linalg.norm(a_moon):.2e} km/s²")
print(f"Sun acceleration:  {np.linalg.norm(a_sun):.2e} km/s²")
print(f"Ratio Moon/Sun:    {np.linalg.norm(a_moon)/np.linalg.norm(a_sun):.1f}")
print()

# Stationkeeping budget
print("GEO Stationkeeping Budget (15-year mission):")
print("-" * 50)
ns_dv = ns_stationkeeping_dv(0.05, 15)
ew_dv = 4 * 15  # ~4 m/s/year average
print(f"N-S (inclination): {ns_dv:.0f} m/s")
print(f"E-W (longitude):   {ew_dv:.0f} m/s")
print(f"Total:             {ns_dv + ew_dv:.0f} m/s")
```

<section class="lesson-pitfalls">

## Pitfalls

1. **Moon position matters.** Lunar distance varies ±7%; perturbation varies ±20%.

2. **Not just magnitude.** Direction relative to orbit determines which elements change.

3. **Long timescales.** Inclination effects take months to manifest; can't see in one orbit.

4. **Coupled with J2.** For accurate predictions, must include J2 + lunisolar together.

5. **Ecliptic vs equatorial.** Moon orbit is in ecliptic plane, not Earth's equator.

6. **Inclined GEO.** Some operators accept 5-10° inclination to save N-S delta-v.

7. **Disposal orbits.** Kozai-Lidov can cause reentry of high-inclination graveyard orbits.
