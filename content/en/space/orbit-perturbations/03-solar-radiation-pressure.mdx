## Why this matters

Sunlight pushes. At 1 AU, the Sun exerts ~4.5 μN/m² on any surface. For high-altitude satellites where drag vanishes, solar radiation pressure (SRP) becomes the dominant non-gravitational force—critical for GEO stationkeeping and the basis for solar sailing.

## Apply

**SRP calculations:**

```python
import numpy as np

# Constants
P_SR = 4.56e-6  # N/m² at 1 AU
C_LIGHT = 299792458  # m/s
SOLAR_FLUX = 1361  # W/m² at 1 AU

def srp_acceleration(area, mass, C_R=1.4, r_AU=1.0):
    """
    Compute SRP acceleration
    
    area: cross-sectional area [m²]
    mass: satellite mass [kg]
    C_R: reflectivity coefficient (1.0-2.0)
    r_AU: distance from Sun in AU
    
    Returns: acceleration [m/s²]
    """
    return P_SR * C_R * area / (mass * r_AU**2)

def srp_force(area, C_R=1.4, r_AU=1.0):
    """
    Compute SRP force
    
    Returns: force [N]
    """
    return P_SR * area * C_R / r_AU**2

def characteristic_acceleration(sigma):
    """
    Solar sail characteristic acceleration
    
    sigma: areal density [kg/m²]
    
    Returns: acceleration [m/s²]
    """
    return 2 * P_SR / sigma

# Example: GEO communication satellite
mass = 2000  # kg
area = 50    # m² (solar panels)
C_R = 1.3

a_srp = srp_acceleration(area, mass, C_R)
print("GEO Satellite SRP:")
print(f"  A/m ratio: {area/mass:.4f} m²/kg")
print(f"  SRP acceleration: {a_srp:.2e} m/s²")
print(f"  Delta-v per year: {a_srp * 365.25 * 86400:.1f} m/s")
print()

# Example: Solar sail
print("Solar Sail Comparison:")
for name, sigma in [("IKAROS", 0.5), ("LightSail 2", 0.1), ("Advanced", 0.01)]:
    a_c = characteristic_acceleration(sigma)
    print(f"  {name}: σ={sigma} kg/m², a_c={a_c*1000:.3f} mm/s²")
print()

# Compare SRP vs drag at different altitudes
print("SRP vs Drag Dominance:")
print("-" * 50)
for alt in [400, 600, 800, 1000, 35786]:
    if alt < 1000:
        drag_approx = 1e-5 * np.exp(-(alt-400)/100)
    else:
        drag_approx = 1e-15
    
    srp = a_srp
    ratio = srp / drag_approx if drag_approx > 0 else float('inf')
    dominant = "Drag" if ratio < 1 else "SRP"
    print(f"  {alt:>6} km: SRP/Drag ratio = {ratio:.1e}, dominant: {dominant}")
```

<section class="lesson-pitfalls">

## Pitfalls

1. **Variable area.** Solar panels track Sun; effective area changes with attitude.

2. **C_R uncertainty.** Surface properties degrade; C_R changes over mission life.

3. **Shadow modeling.** Neglecting penumbra introduces prediction errors.

4. **Self-shadowing.** Satellite components shadow each other; complex geometry.

5. **Earth albedo.** Earth reflects sunlight; adds to SRP at low altitudes.

6. **Thermal re-radiation.** Hot surfaces emit IR photons; Yarkovsky-like effect.

7. **1/r² falloff.** SRP at Mars (1.5 AU) is only 44% of Earth's.
