## Why this matters

Mega-constellations like Starlink (12,000+ satellites), OneWeb (648), and Kuiper (3,236) are reshaping global connectivity. Unlike GEO systems with 3 satellites covering Earth, LEO constellations require thousands of spacecraft working in concert. Understanding mega-constellation design—orbital shells, coverage geometry, and capacity scaling—is essential for anyone building or integrating with next-generation satellite networks.

## Apply

**Constellation sizing calculator:**

```python
import numpy as np

# Constants
R_EARTH = 6371  # km
MU_EARTH = 398600.4418  # km³/s²

def coverage_radius_km(altitude_km, min_elevation_deg):
    """Calculate coverage radius for given altitude and min elevation."""
    r = R_EARTH + altitude_km
    el_rad = np.radians(min_elevation_deg)
    
    # Central angle from satellite to edge of coverage
    rho = np.arcsin(R_EARTH / r * np.cos(el_rad)) - el_rad
    
    return R_EARTH * rho

def satellites_for_coverage(altitude_km, min_elevation_deg, overlap_factor=1.2):
    """Minimum satellites needed for continuous coverage."""
    coverage_radius = coverage_radius_km(altitude_km, min_elevation_deg)
    
    # Earth surface area
    earth_area = 4 * np.pi * R_EARTH**2
    
    # Coverage area per satellite (circular approximation)
    sat_area = np.pi * coverage_radius**2
    
    # Account for overlap and spherical geometry
    n_sats = earth_area / sat_area * overlap_factor * 1.5  # Spherical correction
    
    return int(np.ceil(n_sats))

def orbital_period_min(altitude_km):
    """Orbital period in minutes."""
    r = R_EARTH + altitude_km
    T = 2 * np.pi * np.sqrt(r**3 / MU_EARTH)
    return T / 60

def latency_ms(altitude_km, elevation_deg):
    """One-way propagation delay in milliseconds."""
    r = R_EARTH + altitude_km
    el_rad = np.radians(elevation_deg)
    
    # Slant range
    slant = r * np.sqrt(1 - (R_EARTH/r * np.cos(el_rad))**2) - R_EARTH * np.sin(el_rad)
    
    # Round trip time
    return slant / 299.792 * 2  # RTT in ms

def walker_constellation(inclination_deg, total_sats, n_planes, phasing):
    """Walker constellation parameters."""
    sats_per_plane = total_sats // n_planes
    plane_spacing_deg = 360 / n_planes
    sat_spacing_deg = 360 / sats_per_plane
    phase_offset_deg = phasing * sat_spacing_deg / n_planes
    
    return {
        'inclination_deg': inclination_deg,
        'total_sats': total_sats,
        'n_planes': n_planes,
        'sats_per_plane': sats_per_plane,
        'plane_spacing_deg': plane_spacing_deg,
        'sat_spacing_deg': sat_spacing_deg,
        'phase_offset_deg': phase_offset_deg,
        'notation': f"{inclination_deg}°:{total_sats}/{n_planes}/{phasing}"
    }

# Analysis
print("Mega-Constellation Design Analysis")
print("=" * 60)

# Coverage analysis for different altitudes
print("\n1. Coverage Analysis by Altitude")
print("-" * 60)
altitudes = [340, 550, 1200, 35786]
min_el = 25

print(f"  {'Altitude':>10} {'Coverage':>12} {'Min Sats':>10} {'Period':>10} {'RTT':>8}")
print(f"  {'(km)':>10} {'radius (km)':>12} {'(approx)':>10} {'(min)':>10} {'(ms)':>8}")
print("-" * 60)
for alt in altitudes:
    cov = coverage_radius_km(alt, min_el)
    n_sats = satellites_for_coverage(alt, min_el)
    period = orbital_period_min(alt)
    rtt = latency_ms(alt, min_el)
    orbit_type = "LEO" if alt < 2000 else "MEO" if alt < 20000 else "GEO"
    print(f"  {alt:>10} {cov:>12.0f} {n_sats:>10} {period:>10.1f} {rtt:>8.1f}")

# Starlink constellation analysis
print("\n2. Starlink Shell Analysis")
print("-" * 60)
shells = [
    (550, 53.0, 72, 22, "Main shell"),
    (540, 53.2, 72, 22, "Backup"),

    (560, 97.6, 6, 58, "Polar SSO"),

total_sats = 0
print(f"  {'Alt':>6} {'Inc':>6} {'Planes':>7} {'S/P':>5} {'Total':>7} {'Purpose':>12}")

for alt, inc, planes, spp, purpose in shells:
    total = planes * spp

# Walker notation
```

<section class="lesson-pitfalls">

## Pitfalls

1. **Coverage ≠ capacity.** Having satellites overhead doesn't mean bandwidth is available.

2. **Latitude gaps.** 53° inclination leaves polar regions uncovered without additional shells.

3. **Congestion at crossings.** Ground tracks converge at inclination latitude—collision risk increases.

4. **Regulatory constraints.** ITU filing, debris mitigation, and spectrum coordination limit deployment.

5. **Launch bottleneck.** Even with reusable rockets, deployment takes years for thousands of satellites.

6. **Atmospheric drag.** Lower orbits need frequent station-keeping or accept shorter lifetimes.

7. **Light pollution.** Large constellations impact astronomy; mitigation adds cost.
