## Why this matters

Even with ISL-equipped constellations, traffic must eventually reach the terrestrial internet. Gateway placement determines system capacity, latency to content servers, regulatory compliance, and operational resilience. A poorly placed gateway network can bottleneck an otherwise capable constellation. Understanding the tradeoffs—fiber access, weather diversity, regulatory approval, and coverage geometry—is essential for constellation operators and integrators.

## Apply

**Gateway placement optimizer:**

```python
import numpy as np
from math import radians, sin, cos, asin, sqrt

R_EARTH = 6371  # km

def haversine_km(lat1, lon1, lat2, lon2):
    """Great circle distance between two points."""
    lat1, lon1, lat2, lon2 = map(radians, [lat1, lon1, lat2, lon2])
    dlat = lat2 - lat1
    dlon = lon2 - lon1
    a = sin(dlat/2)**2 + cos(lat1)*cos(lat2)*sin(dlon/2)**2
    return 2 * R_EARTH * asin(sqrt(a))

def satellite_visibility_duration_min(altitude_km, min_elevation_deg):
    """Approximate time a LEO satellite is visible."""
    r = R_EARTH + altitude_km
    el_rad = radians(min_elevation_deg)
    
    # Central angle for visibility
    rho = np.arcsin(R_EARTH / r * np.cos(el_rad)) - el_rad
    
    # Arc length as fraction of orbit
    arc_fraction = 2 * rho / (2 * np.pi)
    
    # Orbital period
    T = 2 * np.pi * np.sqrt(r**3 / 398600.4418) / 60  # minutes
    
    return T * arc_fraction

def gateway_coverage_radius_km(altitude_km, min_elevation_deg):
    """Coverage radius of a gateway site."""
    r = R_EARTH + altitude_km
    el_rad = radians(min_elevation_deg)
    rho = np.arcsin(R_EARTH / r * np.cos(el_rad)) - el_rad
    return R_EARTH * rho

def rain_fade_margin_db(rain_rate_mm_hr, frequency_ghz, elevation_deg, path_length_km=5):
    """Estimate rain fade (simplified ITU-R model)."""
    # Simplified specific attenuation (dB/km)
    # Real calculation uses ITU-R P.838
    k = 0.0001 * frequency_ghz**1.5
    alpha = 1.0 + 0.01 * frequency_ghz
    gamma = k * (rain_rate_mm_hr ** alpha)
    
    # Effective path length through rain
    L_eff = path_length_km / sin(radians(elevation_deg))
    
    return gamma * L_eff

def gateway_site_score(
    fiber_distance_km,
    ixp_distance_km,
    rain_zone,
    latitude,
    constellation_inclination=53
):
    """Score a potential gateway site (higher = better)."""
    score = 100
    
    # Fiber penalty (0-30 points)
    score -= min(30, fiber_distance_km * 0.5)
    
    # IXP penalty (0-20 points)
    score -= min(20, ixp_distance_km * 0.2)
    
    # Rain zone penalty (A=0, K=10, P=25)
    rain_penalties = {'A': 0, 'K': 10, 'P': 25}
    score -= rain_penalties.get(rain_zone, 15)
    
    # Latitude bonus (closer to constellation inclination = more passes)
    lat_diff = abs(latitude) - constellation_inclination
    if lat_diff < 10:
        score += 10  # Optimal latitude band
    elif lat_diff > 30:
        score -= 10  # Poor visibility
    
    return max(0, score)

    ("São Paulo, BR", -23.5, -46.6, 5, 15, 'P'),

]

# Coverage analysis
print("\n" + "=" * 70)
print("Gateway Coverage Parameters (550 km LEO)")
print("-" * 70)

altitude = 550
for min_el in [10, 25, 40]:
    radius = gateway_coverage_radius_km(altitude, min_el)
    duration = satellite_visibility_duration_min(altitude, min_el)

# Capacity planning
print("\n" + "=" * 70)
print("Gateway Capacity Planning")
print("-" * 70)

antennas_per_site = 8
throughput_per_antenna_gbps = 15
sites_count = 30

print(f"Max subscribers ({overbooking_ratio}:1 overbooking): {max_subscribers/1e6:.1f}M")
```

<section class="lesson-pitfalls">

## Pitfalls

1. **Fiber last mile.** A gateway without fiber backhaul is a bottleneck.

2. **Regulatory delays.** Spectrum licensing can take 1-3 years in some countries.

3. **Rain fade.** Ka-band links lose 10-25 dB in heavy rain—need site diversity.

4. **Handoff gaps.** Without enough antennas, brief outages occur during satellite transitions.

5. **Power hungry.** A gateway site draws 200-500 kW; remote locations may lack capacity.

6. **Latitude mismatch.** Gateways at equator see fewer passes than at constellation inclination.

7. **Single points of failure.** Regional gateway outage can black out large areas without ISL backup.
