# MicroDragon Satellite

## Vietnam's Space Engineering Milestone

MicroDragon represents a fundamental shift: from buying satellites to building them. Vietnamese engineers trained at Japanese universities designed and integrated this 50kg microsatellite, launched in January 2019.

## Core Concepts

### Why Microsatellites?

Traditional satellites cost $100M+ and take 5-10 years to develop. Microsatellites (10-100 kg) offer:
- Development time: 2-4 years
- Cost: $5-15M (including launch)
- Technology training: Hands-on experience for engineers
- Risk tolerance: Acceptable loss for technology demonstration

### MicroDragon Key Facts

| Parameter | Value |
|-----------|-------|
| Launch Date | 18 January 2019 |
| Launch Vehicle | Epsilon-4 (JAXA) |
| Launch Site | Uchinoura, Japan |
| Mass | 50 kg |
| Dimensions | 50 x 50 x 50 cm |
| Orbit Type | Sun-Synchronous (SSO) |
| Altitude | 511 km |
| Inclination | 97.31° |
| LTDN | 10:30 local time |
| Design Life | 2 years |
| Developer | VNSC + Japanese universities |

### Training Program Background

MicroDragon emerged from the Space Engineering Education Satellite (SEES) program:
- 36 Vietnamese engineers trained at 5 Japanese universities
- Keio, Hokkaido, Tohoku, Tokyo, Kyushu universities
- Training period: 2013-2018
- Full satellite design, integration, and testing in Japan

### Attitude Determination and Control (ADCS)

| Component | Specification |
|-----------|---------------|
| Attitude Control | 3-axis stabilization |
| Actuators | 4 reaction wheels + 3 magnetorquers |
| Sensors | Star tracker, sun sensors, magnetometer, gyroscopes |
| Pointing Accuracy | < 0.1° (3σ) |
| Pointing Stability | < 0.01°/s |

### Multispectral Imaging Payload

The primary payload is a custom 6-band multispectral camera optimized for water quality assessment:

| Band | Wavelength (nm) | Target Parameter |
|------|-----------------|------------------|
| B1 | 405-425 | Violet - Dissolved organic matter |
| B2 | 433-453 | Blue - Chlorophyll absorption |
| B3 | 480-500 | Cyan - Suspended sediment |
| B4 | 545-565 | Green - Chlorophyll peak |
| B5 | 655-675 | Red - Chlorophyll absorption |
| B6 | 765-785 | NIR - Water/land discrimination |

**Optical Parameters:**
- Ground Sampling Distance (GSD): 78 m at nadir
- Swath Width: 98 km
- Focal Length: 280 mm
- Aperture: f/4.0
- Detector: CMOS, 1280 x 1024 pixels

## Orbit Analysis and Ground Coverage

### Sun-Synchronous Orbit Calculation

```python
import math

def microdragon_orbit_analysis():
    """
    Analyze MicroDragon's sun-synchronous orbit parameters.
    """
    # Orbital elements
    altitude_km = 511
    inclination_deg = 97.31
    ltdn_hours = 10.5  # 10:30 local time descending node
    
    # Earth parameters
    R_earth_km = 6371
    mu_earth = 398600.4418  # km^3/s^2
    J2 = 1.08263e-3
    
    # Calculate orbital period
    a_km = R_earth_km + altitude_km  # Semi-major axis
    period_sec = 2 * math.pi * math.sqrt(a_km**3 / mu_earth)
    period_min = period_sec / 60
    
    # Calculate orbital velocity
    v_km_s = math.sqrt(mu_earth / a_km)
    
    # Calculate ground track velocity
    ground_track_km_s = v_km_s * math.cos(math.radians(inclination_deg))
    
    # Nodal precession rate for SSO
    # dΩ/dt = -3/2 * J2 * (R_e/a)^2 * n * cos(i)
    n = math.sqrt(mu_earth / a_km**3)  # Mean motion (rad/s)
    precession_rad_s = -1.5 * J2 * (R_earth_km/a_km)**2 * n * math.cos(math.radians(inclination_deg))
    precession_deg_day = math.degrees(precession_rad_s) * 86400
    
    # SSO requires precession = 360°/365.25 days = 0.9856°/day
    required_precession = 360 / 365.25
    
    # Revisit time (approximate)
    orbits_per_day = 86400 / period_sec
    
    print("=== MicroDragon Orbit Analysis ===")
    print(f"Altitude:           {altitude_km:.1f} km")
    print(f"Inclination:        {inclination_deg:.2f}°")
    print(f"Semi-major axis:    {a_km:.1f} km")
    print(f"Orbital period:     {period_min:.2f} min ({period_sec:.1f} s)")
    print(f"Orbital velocity:   {v_km_s:.3f} km/s")
    print(f"Orbits per day:     {orbits_per_day:.2f}")
    print(f"")
    print(f"=== Sun-Synchronous Verification ===")
    print(f"Nodal precession:   {precession_deg_day:.4f}°/day")
    print(f"Required (SSO):     {required_precession:.4f}°/day")
    print(f"Difference:         {abs(precession_deg_day - required_precession):.4f}°/day")
    print(f"LTDN:               {ltdn_hours:.1f} hours (10:30 local)")
    
    # Ground track repeat
    # For 511 km, approximately 14-15 day repeat cycle
    print(f"")
    print(f"=== Coverage ===")
    print(f"Swath width:        98 km")
    print(f"GSD:                78 m")
    
    return period_min, orbits_per_day

if __name__ == "__main__":
    microdragon_orbit_analysis()
```

### Water Quality Retrieval Algorithm

```python
import numpy as np

def chlorophyll_a_estimation(band2_blue, band4_green, band5_red):
    """
    Estimate chlorophyll-a concentration from MicroDragon imagery.
    Uses Ocean Color Index (OCI) algorithm variant.
    
    Args:
        band2_blue: Blue band reflectance (433-453 nm)
        band4_green: Green band reflectance (545-565 nm)  
        band5_red: Red band reflectance (655-675 nm)
    
    Returns:
        Chlorophyll-a concentration in mg/m^3
    """
    # Band ratio approach (similar to OC3M algorithm)
    # R = log10(max(Rrs_blue/Rrs_green, Rrs_blue/Rrs_red))
    
    ratio_bg = band2_blue / band4_green
    ratio_br = band2_blue / band5_red
    
    R = np.log10(np.maximum(ratio_bg, ratio_br))
    
    # Polynomial coefficients (empirical, regional calibration needed)
    # Chl-a = 10^(a0 + a1*R + a2*R^2 + a3*R^3 + a4*R^4)
    a = [0.3272, -2.9940, 2.7218, -1.2259, -0.5683]
    
    chl_log = a[0] + a[1]*R + a[2]*R**2 + a[3]*R**3 + a[4]*R**4
    chl_a = 10 ** chl_log
    
    return chl_a

def turbidity_estimation(band3_cyan, band6_nir):
    """
    Estimate turbidity (NTU) from suspended sediment.
    
    Args:
        band3_cyan: Cyan band reflectance (480-500 nm)
        band6_nir: NIR band reflectance (765-785 nm)
    
    Returns:
        Turbidity in NTU
    """
    # Empirical relationship for coastal waters
    # Higher NIR/Cyan ratio indicates more suspended sediment
    ratio = band6_nir / band3_cyan
    
    # Linear model (needs regional calibration)
    turbidity_ntu = 50 * ratio + 5
    
    return turbidity_ntu

# Example usage
if __name__ == "__main__":
    # Simulated reflectance values (typical coastal water)
    b2 = 0.012  # Blue
    b3 = 0.015  # Cyan
    b4 = 0.018  # Green
    b5 = 0.008  # Red
    b6 = 0.004  # NIR
    
    chl = chlorophyll_a_estimation(b2, b4, b5)
    turb = turbidity_estimation(b3, b6)
    
    print(f"Estimated Chlorophyll-a: {chl:.2f} mg/m³")
    print(f"Estimated Turbidity: {turb:.1f} NTU")
```

## Common Pitfalls

### 1. Confusing MicroDragon with NanoDragon
- **MicroDragon** (2019): 50 kg, multispectral imaging, VNSC-developed
- **NanoDragon** (2021): 4 kg, AIS receiver, VNSC-developed
Different missions, different capabilities.

### 2. Atmospheric Correction Neglect
Raw MicroDragon imagery requires atmospheric correction before water quality retrieval. Aerosol optical depth in coastal Vietnam can cause 20-40% reflectance errors.

### 3. Cloud Coverage in Tropics
Vietnam experiences 60-80% cloud cover during monsoon season (May-October). Effective imaging windows are limited, requiring multi-day composite strategies.

### 4. Geometric Correction
78m GSD requires precise geolocation. Off-nadir pointing and attitude errors can cause 1-2 pixel (80-160m) misregistration without ground control points.

### 5. Calibration Drift
On-orbit radiometric calibration degrades over time. MicroDragon uses vicarious calibration via comparison with reference sites (desert, deep ocean).
