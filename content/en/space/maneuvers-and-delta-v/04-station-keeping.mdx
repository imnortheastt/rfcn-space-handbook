## Why this matters

Getting to orbit is half the battle—staying there is the other half. Satellites face continuous perturbations from gravitational anomalies, solar radiation pressure, and atmospheric drag. Station keeping consumes 10-50 m/s of delta-v per year, and this directly determines mission lifetime. Understanding these forces is essential for fuel budgeting and constellation management.

import numpy as np

def geo_ew_drift_rate(longitude_deg):
    """
    Estimate E-W drift acceleration due to Earth's triaxiality.
    
    Stable points: ~75°E and ~255°E (105°W)
    Max drift at ~15°E and ~165°E
    """
    # Simplified model
    # Drift toward nearest stable point
    stable_east = 75
    stable_west = 255
    
    # Distance to nearest stable point
    dist_east = abs(longitude_deg - stable_east)
    dist_west = abs(longitude_deg - stable_west)
    
    # Max drift rate ~0.002 deg/day² at worst locations
    max_accel = 0.002  # deg/day²
    
    if dist_east < dist_west:
        # Drifts toward 75°E
        accel = max_accel * np.sin(np.radians(2 * (longitude_deg - stable_east)))
    else:
        # Drifts toward 255°E
        accel = max_accel * np.sin(np.radians(2 * (longitude_deg - stable_west)))
    
    return accel

def geo_ns_delta_v_per_year(target_inclination_deg=0.05):
    """
    Annual delta-v for N-S station keeping.
    
    Inclination growth: ~0.85 deg/year
    Must counteract with plane change burns
    """
    inclination_growth_per_year = 0.85  # degrees
    v_geo = 3.07  # km/s
    
    # Required plane change per year
    delta_i_rad = np.radians(inclination_growth_per_year)
    
    # Delta-v for plane change
    delta_v = 2 * v_geo * np.sin(delta_i_rad / 2)
    
    return delta_v * 1000  # m/s

def leo_drag_delta_v(altitude_km, area_to_mass_ratio, years=1):
    """
    Estimate delta-v needed to counteract drag.
    
    Args:
        altitude_km: Orbital altitude
        area_to_mass_ratio: A/m in m²/kg
        years: Duration
    
    Returns:
        Delta-v in m/s
    """
    # Simplified drag model
    # Density varies exponentially with altitude
    rho_400 = 3e-12  # kg/m³ at 400 km (average solar activity)
    scale_height = 60  # km
    
    rho = rho_400 * np.exp(-(altitude_km - 400) / scale_height)
    
    # Orbital velocity
    v = 7.7  # km/s approximate for LEO
    
    # Drag acceleration (m/s²)
    C_D = 2.2
    a_drag = 0.5 * rho * (v * 1000)**2 * C_D * area_to_mass_ratio
    
    # Delta-v per year
    seconds_per_year = 365.25 * 24 * 3600
    delta_v = a_drag * seconds_per_year * years
    
    return delta_v

<section class="lesson-pitfalls">

## Pitfalls

1. **Underestimating N-S costs.** N-S control dominates GEO budget; E-W is almost free by comparison.

2. **Solar cycle ignorance.** Solar max can 10× LEO drag; budget for worst case.

3. **Electric propulsion duty cycle.** EP thrusters run hours per day—plan power and thermal accordingly.

4. **Inclined GEO savings.** If ground antennas can track, skip N-S control and save 90% of SK budget.

5. **Collision avoidance.** Maneuvers for debris avoidance come from the same fuel budget.

6. **End-of-life reserve.** Must save fuel for graveyard orbit or de-orbit—don't deplete station-keeping budget.
