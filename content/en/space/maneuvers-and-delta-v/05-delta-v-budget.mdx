## Why this matters

A delta-v budget is the financial statement of spaceflight—it tells you whether your mission is feasible and how long it will last. Every maneuver, from orbit raising to collision avoidance, draws from this finite account. Miss a line item, and you strand your satellite. Overestimate margins, and you waste payload capacity. Accurate budgeting separates successful missions from expensive failures.

### The Tsiolkovsky equation

```
Converting Delta-V to Propellant Mass:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  TSIOLKOVSKY ROCKET EQUATION:                                       │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │              Δv = v_e × ln(m_initial / m_final)       │         │
│  │                                                        │         │
│  │  Where:                                               │         │
│  │  • Δv = velocity change required                     │         │
│  │  • v_e = exhaust velocity = Isp × g₀                 │         │
│  │  • m_initial = mass before burn (wet mass)           │         │
│  │  • m_final = mass after burn (dry mass)              │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  REARRANGED FOR PROPELLANT:                                         │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  Mass ratio: MR = exp(Δv / v_e)                       │         │
│  │                                                        │         │
│  │  Propellant fraction: 1 - 1/MR                        │         │
│  │                                                        │         │
│  │  Propellant mass: m_prop = m_dry × (MR - 1)          │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  THE TYRANNY OF THE ROCKET EQUATION:                               │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  • Exponential growth with Δv                        │         │
│  │  • Doubling Δv MORE than doubles propellant          │         │
│  │  • High Isp is critical for high-Δv missions         │         │
│  │                                                        │         │
│  │  Example (Isp = 320s):                               │         │
│  │  Δv = 3 km/s → 61% propellant                        │         │
│  │  Δv = 6 km/s → 85% propellant                        │         │
│  │  Δv = 9 km/s → 95% propellant                        │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Budget margin philosophy

```
Margin Allocation Strategy:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  MARGIN TYPES:                                                      │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  Type           │ Purpose              │ Typical     │         │
│  │  ───────────────┼──────────────────────┼────────────  │         │
│  │  Navigation     │ Maneuver errors      │ 2-5%        │         │
│  │  Propulsion     │ Isp uncertainty      │ 1-3%        │         │
│  │  Contingency    │ Unknown unknowns     │ 5-10%       │         │
│  │  Extended life  │ Beyond design life   │ 0-20%       │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  MARGIN STACKING METHODS:                                           │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  Method 1: Additive (conservative)                   │         │
│  │    Total margin = nav + prop + contingency = 10%     │         │
│  │                                                        │         │
│  │  Method 2: RSS (root sum square)                     │         │
│  │    Total margin = √(3² + 2² + 5²) = 6%              │         │
│  │    (assumes independent errors)                      │         │
│  │                                                        │         │
│  │  Method 3: Per-phase margins                         │         │
│  │    Different margins for different phases            │         │
│  │    Orbit acquisition: 5%                             │         │
│  │    Operations: 10%                                   │         │
│  │    EOL: 3%                                           │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  GOLDEN RULE: Never use all your margin early!                     │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## Apply

**Delta-v budget calculator:**

```python
import numpy as np

def tsiolkovsky(delta_v_m_s, isp_s, dry_mass_kg):
    """
    Calculate propellant mass using Tsiolkovsky equation.
    
    Returns:
        Dictionary with mass ratio, propellant mass, wet mass
    """
    g0 = 9.81  # m/s²
    v_exhaust = isp_s * g0  # m/s
    
    mass_ratio = np.exp(delta_v_m_s / v_exhaust)
    propellant = dry_mass_kg * (mass_ratio - 1)
    wet_mass = dry_mass_kg + propellant
    prop_fraction = propellant / wet_mass
    
    return {
        "mass_ratio": mass_ratio,
        "propellant_kg": propellant,
        "wet_mass_kg": wet_mass,
        "prop_fraction": prop_fraction,
    }

def inverse_tsiolkovsky(propellant_kg, isp_s, dry_mass_kg):
    """Calculate available delta-v from propellant mass."""
    g0 = 9.81
    v_exhaust = isp_s * g0
    
    wet_mass = dry_mass_kg + propellant_kg
    mass_ratio = wet_mass / dry_mass_kg
    delta_v = v_exhaust * np.log(mass_ratio)
    
    return delta_v

def build_budget(items, margin_percent=10):
    """
    Build a delta-v budget from line items.
    
    Args:
        items: List of (name, delta_v) tuples
        margin_percent: Overall margin to add
    
    Returns:
        Budget dictionary
    """
    subtotal = sum(dv for _, dv in items)
    margin = subtotal * margin_percent / 100
    total = subtotal + margin
    
    return {
        "items": items,
        "subtotal": subtotal,
        "margin_percent": margin_percent,
        "margin": margin,
        "total": total,
    }
```

<section class="lesson-pitfalls">

## Pitfalls

1. **Forgetting de-orbit.** Regulatory requirement; must reserve fuel or use passive methods.

2. **Underestimating N-S SK.** 50 m/s/year dominates GEO budget; consider inclined-orbit operation.

3. **Ignoring collision avoidance.** Debris environment worsening; budget 1-5 m/s/year.

4. **Using all margin early.** Unexpected maneuvers happen; preserve margin throughout mission.

5. **Wrong Isp for operations.** EP Isp varies with power; use realistic operational values.

6. **Forgetting momentum dumping.** Reaction wheels saturate; thrusters needed for desaturation.

7. **Launch vehicle dispersion.** GTO delivery varies; budget for worst-case injection.
