## Why this matters

The ground track is the path traced by the sub-satellite point on Earth's surface—the spot directly beneath the satellite. Understanding ground tracks tells you which regions a satellite can observe, when it will pass over a ground station, and why certain orbits provide global coverage while others don't. It's the bridge between orbital mechanics and practical mission planning.

### Why ground tracks drift westward

```
Earth Rotation Effect:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  Earth rotates 360° in ~24 hours = 15°/hour                        │
│                                                                      │
│  LEO satellite (90-min period):                                     │
│  • In 90 min, Earth rotates 22.5°                                  │
│  • Ascending node shifts 22.5° WEST each orbit                     │
│                                                                      │
│  ┌────────────────────────────────────────────────────┐            │
│  │                                                     │            │
│  │  Orbit 1:  ascending at 0°                         │            │
│  │  Orbit 2:  ascending at -22.5° (337.5°E)          │            │
│  │  Orbit 3:  ascending at -45° (315°E)              │            │
│  │  ...                                               │            │
│  │  Orbit 16: back near starting longitude           │            │
│  │                                                     │            │
│  └────────────────────────────────────────────────────┘            │
│                                                                      │
│  Higher altitude = longer period = more westward drift per orbit   │
│                                                                      │
│  GEO: period = 24 hours → drift = 360° → appears stationary!       │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## How it works

### Ground track algorithm

```
Computing Ground Track Step-by-Step:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  INPUT: Orbital elements or TLE, time span, time step               │
│                                                                      │
│  FOR each time t:                                                   │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  STEP 1: Propagate orbit                              │         │
│  │    r_ECI, v_ECI = propagator(t)                       │         │
│  │    (Kepler, SGP4, or numerical)                       │         │
│  │                                                        │         │
│  │  STEP 2: ECI to ECEF                                  │         │
│  │    θ = GMST(t)                                        │         │
│  │    r_ECEF = R_z(θ) × r_ECI                           │         │
│  │                                                        │         │
│  │  STEP 3: ECEF to Geodetic                             │         │
│  │    λ = atan2(y, x)                     ← longitude   │         │
│  │    φ = iterative_latitude(x, y, z)     ← latitude    │         │
│  │    h = altitude (for reference)                       │         │
│  │                                                        │         │
│  │  OUTPUT: (λ, φ) is the SSP at time t                 │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  Connect all (λ, φ) points → ground track                          │
│                                                                      │
│  HANDLING WRAP-AROUND:                                              │
│  When longitude crosses ±180°, break the line to avoid             │
│  drawing across the map.                                            │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Repeating ground tracks

```
Repeat Ground Track Condition:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  A ground track repeats when:                                       │
│                                                                      │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  N × T_orbit = D × T_sidereal                         │         │
│  │                                                        │         │
│  │  Where:                                               │         │
│  │  N = number of orbits                                 │         │
│  │  D = number of days                                   │         │
│  │  T_orbit = orbital period                             │         │
│  │  T_sidereal = sidereal day = 86164.1 s               │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  EXAMPLES:                                                          │
│  ┌────────────────────────────────────────────────────┐            │
│  │                                                     │            │

│  │             (233 orbits in 16 days)                │            │

│  │                                                     │            │
│  └────────────────────────────────────────────────────┘            │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## Apply

**Ground track computation:**

```python
import numpy as np
from datetime import datetime, timedelta

# WGS84 constants
WGS84_A = 6378137.0
WGS84_E2 = 0.00669437999014
MU = 3.986004418e14  # m³/s²

def gmst_rad(dt):
    """Greenwich Mean Sidereal Time in radians."""
    jd = (dt - datetime(2000, 1, 1, 12, 0, 0)).total_seconds() / 86400.0 + 2451545.0
    T = (jd - 2451545.0) / 36525.0
    gmst_deg = 280.46061837 + 360.98564736629 * (jd - 2451545.0) + 0.000387933 * T**2
    return np.radians(gmst_deg % 360.0)

def kepler_to_eci(a, e, i, raan, argp, nu):
    """
    Convert Keplerian elements to ECI position.
    
    Args:
        a: semi-major axis (m)
        e: eccentricity
        i: inclination (rad)
        raan: right ascension of ascending node (rad)
        argp: argument of perigee (rad)
        nu: true anomaly (rad)
    
    Returns:
        [x, y, z] in ECI frame (m)
    """
    # Distance from focus
    r = a * (1 - e**2) / (1 + e * np.cos(nu))
    
    # Position in orbital plane
    x_orb = r * np.cos(nu)
    y_orb = r * np.sin(nu)
    
    # Rotation matrices
    cos_raan, sin_raan = np.cos(raan), np.sin(raan)
    cos_argp, sin_argp = np.cos(argp), np.sin(argp)
    cos_i, sin_i = np.cos(i), np.sin(i)
    
    # Combined rotation
    x = (cos_raan * cos_argp - sin_raan * sin_argp * cos_i) * x_orb + \
        (-cos_raan * sin_argp - sin_raan * cos_argp * cos_i) * y_orb
    y = (sin_raan * cos_argp + cos_raan * sin_argp * cos_i) * x_orb + \
        (-sin_raan * sin_argp + cos_raan * cos_argp * cos_i) * y_orb
    z = (sin_argp * sin_i) * x_orb + (cos_argp * sin_i) * y_orb
    
    return np.array([x, y, z])

def propagate_kepler(a, e, i, raan, argp, M0, t_since):
    """
    Simple Keplerian propagation (no perturbations).
    
    Args:
        t_since: time since epoch (seconds)
    
    Returns:
        ECI position [x, y, z] (m)
    """
    # Mean motion
    n = np.sqrt(MU / a**3)
    
    # Mean anomaly at time t
    M = M0 + n * t_since
    
    # Solve Kepler's equation (Newton-Raphson)
    E = M
    for _ in range(10):
        E = E - (E - e * np.sin(E) - M) / (1 - e * np.cos(E))
    
    # True anomaly
    nu = 2 * np.arctan2(np.sqrt(1 + e) * np.sin(E / 2),
                         np.sqrt(1 - e) * np.cos(E / 2))
    
    return kepler_to_eci(a, e, i, raan, argp, nu)

def eci_to_geodetic(r_eci, dt):
    """Convert ECI to geodetic coordinates."""
    # ECI to ECEF
    theta = gmst_rad(dt)
    c, s = np.cos(theta), np.sin(theta)
    x_ecef = c * r_eci[0] + s * r_eci[1]
    y_ecef = -s * r_eci[0] + c * r_eci[1]
    z_ecef = r_eci[2]
    
    # ECEF to geodetic
    lon = np.arctan2(y_ecef, x_ecef)
    p = np.sqrt(x_ecef**2 + y_ecef**2)
    lat = np.arctan2(z_ecef, p)
    
    for _ in range(5):
        N = WGS84_A / np.sqrt(1 - WGS84_E2 * np.sin(lat)**2)
        lat = np.arctan2(z_ecef + WGS84_E2 * N * np.sin(lat), p)
    
    return np.degrees(lon), np.degrees(lat)

def compute_ground_track(a, e, i, raan, argp, M0, epoch, duration_hours, dt_minutes=1):
    """
    Compute ground track for given orbital elements.
    
    Returns:
        List of (lon, lat) tuples
    """
    track = []
    n_points = int(duration_hours * 60 / dt_minutes)
    
    for k in range(n_points + 1):
        t_since = k * dt_minutes * 60  # seconds
        current_time = epoch + timedelta(seconds=t_since)
        
        r_eci = propagate_kepler(a, e, i, raan, argp, M0, t_since)
        lon, lat = eci_to_geodetic(r_eci, current_time)
        
        track.append((lon, lat))
    
    return track

# Example: ISS-like orbit
print("Ground Track Computation")
print("=" * 60)

# ISS orbital elements (approximate)
a_iss = 6778e3  # 400 km altitude
e_iss = 0.0001
i_iss = np.radians(51.6)  # 51.6° inclination
raan_iss = np.radians(0)
argp_iss = np.radians(0)
M0_iss = np.radians(0)

epoch = datetime(2026, 6, 3, 0, 0, 0)
duration = 3  # hours (about 2 orbits)

track = compute_ground_track(a_iss, e_iss, i_iss, raan_iss, argp_iss, M0_iss, epoch, duration, dt_minutes=2)

print(f"\nISS-like orbit (a={a_iss/1e3:.0f} km, i={np.degrees(i_iss):.1f}°)")
print(f"Duration: {duration} hours ({len(track)} points)")
print(f"\nSample points (every 30 min):")
print(f"{'Time (min)':<12} {'Longitude':<12} {'Latitude':<12}")
print("-" * 40)

for k in range(0, len(track), 15):
    lon, lat = track[k]
    t_min = k * 2
    print(f"{t_min:<12} {lon:>10.2f}° {lat:>10.2f}°")

# Orbital period
T = 2 * np.pi * np.sqrt(a_iss**3 / MU)
print(f"\nOrbital period: {T/60:.1f} min")
print(f"Earth rotation in one orbit: {360 * T / 86164.1:.1f}° westward shift")

# GEO example
print(f"\n{'='*60}")
print("GEO satellite (stationary point)")
a_geo = 42164e3
e_geo = 0.0
i_geo = np.radians(0)

track_geo = compute_ground_track(a_geo, e_geo, i_geo, 0, 0, 0, epoch, 24, dt_minutes=60)

print(f"\nGEO ground track over 24 hours:")
for k, (lon, lat) in enumerate(track_geo[:5]):
    print(f"  t={k} hr: lon={lon:.2f}°, lat={lat:.2f}°")
print(f"  (remains at same longitude)")
```

| Orbit type | Period | Westward shift/orbit | Coverage |
|------------|--------|---------------------|----------|
| LEO (400 km) | 93 min | 23° | Global over days |
| MEO (GPS) | 12 hr | 180° | Repeat every 2 orbits |
| GEO | 24 hr | 360° | Fixed point |
| Molniya | 12 hr | 180° | High dwell at apogee |

## Pitfalls

1. **Forgetting Earth rotation.** ECI position is meaningless without GMST conversion.

2. **Longitude wrap-around.** Don't draw a line from +179° to -179°; handle the discontinuity.

3. **Using solar day.** Ground tracks repeat with sidereal day (86164 s), not solar day (86400 s).

4. **Ignoring perturbations.** J2 causes RAAN drift ~0.5-10°/day for LEO; affects long-term tracks.

5. **Spherical Earth assumption.** Acceptable for visualization but introduces ~20 km error in SSP.

6. **Confusing altitude and radius.** Altitude is above surface; orbital radius includes Earth's radius.
