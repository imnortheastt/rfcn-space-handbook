## Why this matters

Deep space communication pushes RF engineering to its limits. At Mars distance (225 million km), a 20W transmitter on the spacecraft produces a signal that arrives at Earth weaker than the thermal noise floor—yet we still decode it. This walkthrough shows how 70-meter dishes, cryogenic amplifiers, and extreme coding make it possible.

## How it works

### Downlink budget (Spacecraft → Earth)

```
Mars-Earth Downlink Budget:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  Parameter                           Value      Unit    Notes       │
│  ═══════════════════════════════════════════════════════════════   │
│                                                                      │
│  TRANSMIT SIDE (Spacecraft):                                        │
│  ────────────────────────────────────────────────────────────────  │
│  Tx power (35W TWTA)                 +15.4      dBW                │
│  Circuit/waveguide loss              -0.5       dB                 │
│  High-gain antenna (2.5m)            +40.0      dBi                │
│  Pointing loss                       -0.5       dB                 │
│  ────────────────────────────────────────────────────────────────  │

│                                                                      │
│  PATH:                                                              │
│  ────────────────────────────────────────────────────────────────  │
│  Frequency                           8.4        GHz    X-band DL   │
│  Distance                            225M       km     1.5 AU      │
│  Free space loss                    -279.5      dB     Enormous!   │
│  Atmospheric loss                    -0.1       dB     Minimal     │
│  ────────────────────────────────────────────────────────────────  │
│  Total path loss                    -279.6      dB                 │
│                                                                      │

│  ────────────────────────────────────────────────────────────────  │

│  Pointing loss                       -0.1       dB                 │

│  G/T                                 +59.4      dB/K   World-class │

│                                                                      │
│  LINK CALCULATION:                                                  │
│  ────────────────────────────────────────────────────────────────  │
│  Boltzmann constant                 +228.6      dBW/K/Hz          │
```

## Apply

**Complete deep space link budget:**

```python
import numpy as np

def deep_space_link_budget():
    """
    X-band deep space link budget (Mars distance).
    """
    # Distance
    distance_au = 1.5  # Average Earth-Mars
    distance_km = distance_au * 149597870.7  # km per AU
    light_time_s = distance_km / 299792.458
    light_time_min = light_time_s / 60
    
    # Downlink frequency
    freq_ghz_dl = 8.4  # X-band downlink
    
    # Free space path loss
    fspl_dl = 92.45 + 20 * np.log10(freq_ghz_dl * distance_km)
    
    # Spacecraft (transmit)
    p_tx_w = 35
    p_tx_dbw = 10 * np.log10(p_tx_w)
    g_tx_dbi = 40.0  # 2.5m HGA
    circuit_loss = 0.5
    pointing_loss_sc = 0.5
    eirp_dbw = p_tx_dbw + g_tx_dbi - circuit_loss - pointing_loss_sc
    
    # Path losses
    atm_loss = 0.1
    total_loss = fspl_dl + atm_loss
    
    # DSN 70m (receive)
    g_rx_dbi = 74.0
    efficiency_loss = 0.5
    pointing_loss_dsn = 0.1
    t_sys_k = 25.0  # Cryogenic maser
    g_t_dbk = g_rx_dbi - efficiency_loss - pointing_loss_dsn - 10 * np.log10(t_sys_k)
    
    # Link budget
    k_dbw = -228.6
    c_n0_dbhz = eirp_dbw - total_loss + g_t_dbk - k_dbw
    
    print("Deep Space X-band Downlink Budget")
    print("=" * 65)
    print(f"\n--- GEOMETRY ---")
    print(f"Distance:        {distance_au} AU = {distance_km/1e6:.1f} million km")
    print(f"Light time:      {light_time_min:.1f} minutes one-way")
    
    print(f"\n--- TRANSMIT (Spacecraft) ---")
    print(f"Tx power:        {p_tx_w} W = {p_tx_dbw:.1f} dBW")
    print(f"Antenna gain:    {g_tx_dbi:.1f} dBi (2.5m HGA)")
    print(f"Losses:          {circuit_loss + pointing_loss_sc:.1f} dB")
    print(f"EIRP:            {eirp_dbw:.1f} dBW")
    
    print(f"\n--- PATH ---")
    print(f"Frequency:       {freq_ghz_dl} GHz")
    print(f"Free space loss: {fspl_dl:.1f} dB")
    print(f"Atmospheric:     {atm_loss} dB")
    print(f"Total loss:      {total_loss:.1f} dB")
    
    print(f"\n--- RECEIVE (DSN 70m) ---")
    print(f"Antenna gain:    {g_rx_dbi:.1f} dBi")
    print(f"System temp:     {t_sys_k:.0f} K (cryogenic)")
    print(f"G/T:             {g_t_dbk:.1f} dB/K")
    
    print(f"\n--- RESULT ---")
    print(f"C/N₀:            {c_n0_dbhz:.1f} dBHz")
    
    # Data rate calculation
    print(f"\n--- DATA RATE ANALYSIS ---")
    eb_n0_required = {
        'Uncoded BPSK': 10.5,
        'Convolutional (r=1/2)': 5.0,
        'Turbo code (r=1/3)': 1.0,
        'LDPC (r=1/2)': 1.5
    }
    
    for coding, eb_n0 in eb_n0_required.items():
        max_rate = 10 ** ((c_n0_dbhz - eb_n0) / 10)
        print(f"  {coding:25} Eb/N₀={eb_n0} dB → {max_rate/1000:.1f} kbps")
    
    # Distance sensitivity
    print(f"\n--- DISTANCE SENSITIVITY ---")
    distances_au = [0.5, 1.0, 1.5, 2.0, 2.5]
    for d_au in distances_au:
        d_km = d_au * 149597870.7
        fspl = 92.45 + 20 * np.log10(freq_ghz_dl * d_km)
        c_n0 = eirp_dbw - fspl - atm_loss + g_t_dbk - k_dbw
        rate = 10 ** ((c_n0 - 1.0) / 10)  # Turbo coded
        print(f"  {d_au:.1f} AU: Path loss = {fspl:.1f} dB, "
              f"C/N₀ = {c_n0:.1f} dBHz, Rate = {rate/1000:.1f} kbps")
    
    return c_n0_dbhz

c_n0 = deep_space_link_budget()

# Antenna comparison
print("\n" + "=" * 65)
print("DSN ANTENNA COMPARISON")
print("-" * 65)
antennas = [
    ('DSN-70', 70, 74),
    ('DSN-34', 34, 68),
    ('DSN-26', 26, 65),
    ('ESA 35m', 35, 68.5),
]
base_cn0 = 62.8  # With 70m
for name, diam, gain in antennas:
    delta = gain - 74  # vs 70m
    cn0 = base_cn0 + delta
    rate = 10 ** ((cn0 - 1.0) / 10)
    print(f"  {name:12} ({diam}m): G={gain} dBi, "
          f"C/N₀={cn0:.1f} dBHz, Rate={rate/1000:.1f} kbps")
```

<section class="lesson-pitfalls">

## Pitfalls

1. **Distance variation.** Mars ranges 0.4-2.7 AU; budget for worst case (conjunction).

2. **Solar conjunction.** When Mars is behind Sun, link is blocked for ~2 weeks.

3. **Doppler is significant.** ±30 km/s relative velocity = large frequency shift.

4. **Pointing is critical.** 0.01° error on HGA = significant loss; use star trackers.

5. **One-way light time.** 4-24 minutes; no interactive commanding.

6. **DSN scheduling.** 70m dishes are oversubscribed; plan months ahead.

7. **Emergency modes.** Must work with low-gain omni; design for safe mode comms.
