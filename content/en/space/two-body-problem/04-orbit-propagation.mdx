## Why this matters

"Where will this satellite be in 2 hours?" This fundamental question drives all space operations—from ground station scheduling to collision avoidance. Orbit propagation transforms orbital elements and time into position predictions.

## Intuition

### Propagation overview

```
Orbit Propagation: From Elements to Position

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  INPUT:                                                 │
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                    │  │
│  │  Orbital elements at epoch t₀:                    │  │
│  │  (a, e, i, Ω, ω, M₀)                              │  │
│  │                                                    │  │
│  │  Target time: t                                   │  │
│  │                                                    │  │
│  └───────────────────────────────────────────────────┘  │
│                                                          │
│  PROCESS:                                               │
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                    │  │
│  │  1. Compute time elapsed: Δt = t - t₀             │  │
│  │                                                    │  │
│  │  2. Update mean anomaly:                          │  │
│  │     M = M₀ + n × Δt                               │  │
│  │     where n = √(μ/a³) (mean motion)               │  │
│  │                                                    │  │
│  │  3. Solve Kepler's equation:                      │  │
│  │     M → E (eccentric anomaly)                     │  │
│  │                                                    │  │
│  │  4. Convert to true anomaly:                      │  │
│  │     E → ν                                         │  │
│  │                                                    │  │
│  │  5. Compute position in orbital plane             │  │
│  │                                                    │  │
│  │  6. Rotate to ECI using i, Ω, ω                   │  │
│  │                                                    │  │
│  └───────────────────────────────────────────────────┘  │
│                                                          │
│  OUTPUT:                                                │
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                    │  │
│  │  Position vector: r⃗ = [x, y, z]                   │  │
│  │  Velocity vector: v⃗ = [vx, vy, vz]                │  │
│  │                                                    │  │
│  └───────────────────────────────────────────────────┘  │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### Two-body vs perturbed

```
Propagation Approaches:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  TWO-BODY (KEPLERIAN)                                   │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  Assumptions:                                   │    │
│  │  • Point mass central body                      │    │
│  │  • No other forces (drag, solar, J2...)         │    │
│  │  • Orbit is fixed ellipse                       │    │
│  │                                                  │    │
│  │  Elements that stay CONSTANT:                   │    │
│  │  a, e, i, Ω, ω                                  │    │
│  │                                                  │    │
│  │  Element that changes:                          │    │
│  │  M = M₀ + n×t (linear with time)                │    │
│  │                                                  │    │
│  │  Accuracy: Good for hours, degrades over days   │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  PERTURBED (REAL WORLD)                                 │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  Perturbations that change elements:            │    │
│  │  • J2 (Earth oblateness): Ω, ω drift            │    │
│  │  • Drag (LEO): a, e decay                       │    │
│  │  • Solar/lunar: all elements                    │    │
│  │  • Solar radiation pressure: e, ω              │    │
│  │                                                  │    │
│  │  Methods:                                       │    │
│  │  • Analytical (SGP4): fast, approximate         │    │
│  │  • Numerical integration: accurate, slower      │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## How it works

### Analytical two-body propagation

```
Step-by-Step Two-Body Propagation:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  STEP 1: COMPUTE MEAN MOTION                            │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  n = √(μ/a³)    [rad/s]                         │    │
│  │                                                  │    │
│  │  For Earth: μ = 398600.4418 km³/s²              │    │
│  │                                                  │    │
│  │  Example: a = 7000 km                           │    │
│  │  n = √(398600/7000³) = 0.001078 rad/s           │    │
│  │  Period T = 2π/n = 5828 s ≈ 97 min              │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  STEP 2: UPDATE MEAN ANOMALY                            │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  M = M₀ + n × Δt                                │    │
│  │                                                  │    │
│  │  Keep M in [0, 2π]:                             │    │
│  │  M = M mod 2π                                   │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  STEP 3: SOLVE KEPLER'S EQUATION                        │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  M = E - e×sin(E)                               │    │
│  │                                                  │    │
│  │  Newton-Raphson:                                │    │
│  │  E_{n+1} = E_n - (E_n - e×sin(E_n) - M)         │    │
│  │                    ─────────────────────        │    │
│  │                      (1 - e×cos(E_n))           │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  STEP 4: COMPUTE TRUE ANOMALY                           │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  tan(ν/2) = √((1+e)/(1-e)) × tan(E/2)           │    │
│  │                                                  │    │
│  │  Use atan2 for correct quadrant                 │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### Position and velocity computation

```
From Anomaly to State Vector:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  STEP 5: POSITION IN PERIFOCAL FRAME                    │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  Compute radius:                                │    │
│  │  r = a(1 - e²) / (1 + e×cos(ν))                 │    │
│  │                                                  │    │
│  │  Position in perifocal (P, Q, W) frame:         │    │
│  │  r_pqw = [r×cos(ν), r×sin(ν), 0]                │    │
│  │                                                  │    │
│  │  P: points to periapsis                         │    │
│  │  Q: perpendicular, in orbital plane             │    │
│  │  W: normal to orbital plane                     │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  STEP 6: VELOCITY IN PERIFOCAL FRAME                    │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  p = a(1 - e²)     (semi-latus rectum)          │    │
│  │                                                  │    │
│  │  v_pqw = √(μ/p) × [-sin(ν), e + cos(ν), 0]      │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  STEP 7: ROTATE TO ECI                                  │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  Rotation matrix:                               │    │
│  │  R = R_z(-Ω) × R_x(-i) × R_z(-ω)                │    │
│  │                                                  │    │
│  │  r_eci = R × r_pqw                              │    │
│  │  v_eci = R × v_pqw                              │    │
│  │                                                  │    │
│  │  R_z(θ) = [cos(θ)  -sin(θ)  0]                  │    │
│  │           [sin(θ)   cos(θ)  0]                  │    │
│  │           [0        0       1]                  │    │
│  │                                                  │    │
│  │  R_x(θ) = [1   0        0     ]                 │    │
│  │           [0   cos(θ)  -sin(θ)]                 │    │
│  │           [0   sin(θ)   cos(θ)]                 │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

Numerical Integration Overview:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  EQUATIONS OF MOTION:                                   │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │

## Apply

**Complete two-body propagator:**

```python
import numpy as np

def propagate_kepler(coe, t0, t, mu=398600.4418):
    """
    Two-body orbit propagation
    
    coe: dict with a, e, i, RAAN, omega, M0 (angles in radians)
    t0: epoch time (seconds)
    t: target time (seconds)
    mu: gravitational parameter (km³/s²)
    
    Returns: r_eci, v_eci (position and velocity in ECI)
    """
    a = coe['a']
    e = coe['e']
    i = coe['i']
    RAAN = coe['RAAN']
    omega = coe['omega']
    M0 = coe['M0']
    
    # Mean motion
    n = np.sqrt(mu / a**3)
    
    # Mean anomaly at time t
    dt = t - t0
    M = M0 + n * dt
    M = M % (2 * np.pi)
    
    # Solve Kepler's equation
    E = M  # Initial guess
    for _ in range(50):
        E_new = E - (E - e*np.sin(E) - M) / (1 - e*np.cos(E))
        if abs(E_new - E) < 1e-12:
            break
        E = E_new
    
    # True anomaly
    nu = 2 * np.arctan2(
        np.sqrt(1+e) * np.sin(E/2),
        np.sqrt(1-e) * np.cos(E/2)
    )
    
    # Position in perifocal frame
    p = a * (1 - e**2)
    r = p / (1 + e * np.cos(nu))
    r_pqw = np.array([r*np.cos(nu), r*np.sin(nu), 0])
    
    # Velocity in perifocal frame
    v_pqw = np.sqrt(mu/p) * np.array([-np.sin(nu), e+np.cos(nu), 0])
    
    # Rotation matrices
    def Rz(angle):
        c, s = np.cos(angle), np.sin(angle)
        return np.array([[c, -s, 0], [s, c, 0], [0, 0, 1]])
    
    def Rx(angle):
        c, s = np.cos(angle), np.sin(angle)
        return np.array([[1, 0, 0], [0, c, -s], [0, s, c]])
    
    # Combined rotation: PQW -> ECI
    R = Rz(-RAAN) @ Rx(-i) @ Rz(-omega)
    
    r_eci = R @ r_pqw
    v_eci = R @ v_pqw
    
    return r_eci, v_eci

# Example: ISS-like orbit
coe = {
    'a': 6778.0,       # km
    'e': 0.0006,
    'i': np.radians(51.6),
    'RAAN': np.radians(45.0),
    'omega': np.radians(30.0),
    'M0': np.radians(0.0)
}

r, v = propagate_kepler(coe, t0=0, t=3600)  # 1 hour later
print(f"Position: {r} km")
print(f"Velocity: {v} km/s")
```

| Method | Accuracy | Speed | Use Case |

| Two-body | ~1 km/orbit | Very fast | Quick estimates |
| SGP4 | ~1 km (LEO) | Fast | TLE-based tracking |
| HPOP | meters | Slow | Precise prediction |

<section class="lesson-pitfalls">

## Pitfalls

1. **Units consistency.** μ in km³/s² requires a in km, gives v in km/s.

2. **Angle conventions.** radians for calculations, degrees for display.

3. **Kepler convergence.** High eccentricity needs more iterations or better initial guess.

4. **Quadrant errors.** Use atan2, not atan, for anomaly conversions.

5. **Rotation order.** Matrix multiplication order matters: R_z(-Ω) × R_x(-i) × R_z(-ω).

6. **Two-body limitations.** Accuracy degrades rapidly without perturbations.

7. **Time systems.** Ensure consistent time scale (UTC, TT, etc.).

8. **Reference frames.** ECI vs ECEF vs other frames—know which you're using.
