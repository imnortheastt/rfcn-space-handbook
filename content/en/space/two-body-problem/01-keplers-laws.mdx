## Why this matters

Kepler's laws are the foundation of orbital mechanics. Published in the early 1600s, they describe how satellites, planets, and spacecraft move through space. Every orbit prediction, every satellite maneuver, and every interplanetary trajectory builds on these three elegant laws.

## Intuition

### The three laws

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  LAW 1: LAW OF ELLIPSES                                 │
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                    │  │
│  │  "The orbit of a planet is an ellipse with the    │  │
│  │   Sun at one focus."                              │  │
│  │                                                    │  │
│  │            ┌──────────────────┐                   │  │
│  │           /                    \                  │  │
│  │          │   *         ☉       │                  │  │

│  │            └──────────────────┘                   │  │
│  │                 empty focus                       │  │
│  │                                                    │  │

│  │                                                    │  │
│  └───────────────────────────────────────────────────┘  │
│                                                          │
│  LAW 2: LAW OF EQUAL AREAS                              │
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                    │  │

│  │         periapsis (fast)                          │  │

│  │             /█\                                   │  │
│  │            / █ \      Same area,                  │  │

│  │  Consequence: faster at periapsis, slower at      │  │

│  │                                                    │  │

│                                                          │

│  ┌───────────────────────────────────────────────────┐  │

│             ○──────┼────☉────┼──────○                   │

## How it works

### Deriving orbital period

```
From Kepler's Third Law to Orbital Period:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  DERIVATION:                                            │
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                    │  │
│  │  Start with:  T² = (4π²/μ) × a³                   │  │
│  │                                                    │  │
│  │  Solve for T: T = 2π × √(a³/μ)                    │  │
│  │                                                    │  │
│  │  For Earth (μ = 398600 km³/s²):                   │  │
│  │                                                    │  │
│  │  T = 2π × √(a³/398600)  [seconds]                 │  │
│  │                                                    │  │
│  └───────────────────────────────────────────────────┘  │
│                                                          │
│  COMMON ORBITS:                                         │
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                    │  │
│  │  Orbit          a (km)      T (period)            │  │
│  │  ─────────────────────────────────────            │  │
│  │  ISS (LEO)      6,778       92.6 min              │  │
│  │  GPS (MEO)      26,560      11.97 hr              │  │
│  │  GEO            42,164      23.93 hr              │  │
│  │  Moon           384,400     27.3 days             │  │
│  │                                                    │  │
│  └───────────────────────────────────────────────────┘  │
│                                                          │
│  SPECIAL CASE: GEOSTATIONARY                            │
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                    │  │
│  │  Required: T = 86164 s (sidereal day)             │  │
│  │                                                    │  │
│  │  Solve: a = ∛(μT²/4π²)                            │  │
│  │           = ∛(398600 × 86164²/4π²)                │  │
│  │           = 42,164 km (from Earth center)         │  │
│  │                                                    │  │
│  │  Altitude = 42,164 - 6,378 = 35,786 km            │  │
│  │                                                    │  │
│  └───────────────────────────────────────────────────┘  │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### Velocity at any point

```
Vis-Viva Equation (velocity from position):

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  v² = μ × (2/r - 1/a)                                   │
│                                                          │
│  where:                                                 │
│    v = velocity at radius r                             │
│    μ = gravitational parameter (GM)                     │
│    r = current radius from center                       │
│    a = semi-major axis                                  │
│                                                          │
│  SPECIAL CASES:                                         │
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                    │  │
│  │  Circular orbit (a = r):                          │  │
│  │  v² = μ/r  →  v = √(μ/r)                          │  │
│  │                                                    │  │
│  │  Periapsis (r = rp):                              │  │
│  │  vp = √(μ × (2/rp - 1/a))  [maximum velocity]     │  │
│  │                                                    │  │
│  │  Apoapsis (r = ra):                               │  │
│  │  va = √(μ × (2/ra - 1/a))  [minimum velocity]     │  │
│  │                                                    │  │
│  │  Escape velocity (a → ∞):                         │  │
│  │  vesc = √(2μ/r)                                   │  │
│  │                                                    │  │
│  └───────────────────────────────────────────────────┘  │
│                                                          │
│  EXAMPLE: LEO at 400 km altitude                        │
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                    │  │
│  │  r = 6378 + 400 = 6778 km                         │  │
│  │  v = √(398600/6778) = 7.67 km/s                   │  │
│  │                                                    │  │
│  │  vesc = √(2 × 398600/6778) = 10.85 km/s           │  │
│  │                                                    │  │
│  │  Δv to escape = 10.85 - 7.67 = 3.18 km/s          │  │
│  │                                                    │  │
│  └───────────────────────────────────────────────────┘  │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### Angular momentum

```
Conservation of Angular Momentum (Law 2):

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  SPECIFIC ANGULAR MOMENTUM:                             │
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                    │  │
│  │  h = r × v  (cross product)                       │  │
│  │                                                    │  │
│  │  For orbit in a plane:                            │  │
│  │  h = r × v_perp  (scalar magnitude)               │  │
│  │                                                    │  │
│  │  h is CONSTANT throughout the orbit               │  │
│  │  (this is Kepler's 2nd Law in action)             │  │
│  │                                                    │  │
│  └───────────────────────────────────────────────────┘  │
│                                                          │
│  AT PERIAPSIS AND APOAPSIS:                             │
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                    │  │
│  │  At these points, velocity is perpendicular:      │  │
│  │                                                    │  │
│  │  h = rp × vp = ra × va                            │  │
│  │                                                    │  │
│  │  Therefore:                                       │  │
│  │  vp/va = ra/rp                                    │  │
│  │                                                    │  │
│  │  Faster at periapsis, slower at apoapsis          │  │
│  │                                                    │  │
│  └───────────────────────────────────────────────────┘  │
│                                                          │
│  EXAMPLE: GTO                                           │
│  ┌───────────────────────────────────────────────────┐  │
│  │                                                    │  │
│  │  rp = 6,578 km (200 km altitude)                  │  │
│  │  ra = 42,164 km (GEO altitude)                    │  │
│  │                                                    │  │
│  │  vp/va = 42,164/6,578 = 6.4                       │  │
│  │                                                    │  │
│  │  vp ≈ 10.2 km/s                                   │  │
│  │  va ≈ 1.6 km/s                                    │  │
│  │                                                    │  │
│  └───────────────────────────────────────────────────┘  │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## Apply

**Quick calculations:**

```python
import math

# Constants
mu_earth = 398600.4418  # km³/s² (Earth)
R_earth = 6378.137      # km (Earth radius)

def orbital_period(a_km):
    """Period in seconds from semi-major axis in km"""
    return 2 * math.pi * math.sqrt(a_km**3 / mu_earth)

def circular_velocity(r_km):
    """Circular velocity at radius r_km"""
    return math.sqrt(mu_earth / r_km)

def escape_velocity(r_km):
    """Escape velocity at radius r_km"""
    return math.sqrt(2 * mu_earth / r_km)

# Examples
leo_alt = 400  # km
leo_r = R_earth + leo_alt

print(f"LEO period: {orbital_period(leo_r)/60:.1f} min")
print(f"LEO velocity: {circular_velocity(leo_r):.2f} km/s")
print(f"Escape velocity: {escape_velocity(leo_r):.2f} km/s")
```

**Common reference values:**

| Orbit | Altitude (km) | Semi-major axis (km) | Period | Velocity (km/s) |
|-------|---------------|---------------------|--------|-----------------|
| ISS | 400 | 6,778 | 92.6 min | 7.67 |
| GPS | 20,180 | 26,560 | 12 hr | 3.87 |
| GEO | 35,786 | 42,164 | 24 hr | 3.07 |

<section class="lesson-pitfalls">

## Pitfalls

1. **Mixing radius and altitude.** Always use radius from Earth center, not altitude above surface.

2. **Units mismatch.** μ in km³/s² requires distances in km, velocities in km/s.

3. **Forgetting sidereal vs solar day.** GEO uses sidereal day (86164s), not solar day (86400s).

4. **Assuming circular orbits.** Most orbits are elliptical. Use vis-viva for general case.

5. **Ignoring perturbations.** Kepler's laws are ideal two-body. Real orbits have J2, drag, etc.

6. **Sign conventions.** Angular momentum direction matters for 3D problems.

7. **Escape vs orbit.** Escape velocity is √2 times circular velocity at same radius.

8. **Period depends on a only.** Period is independent of eccentricity for same semi-major axis.
