## Why this matters

The Starlink "Dishy" terminal can track multiple LEO satellites zooming overhead at 7.5 km/sвҖ”with no moving parts visible. This magic is phased array technology: hundreds or thousands of tiny antenna elements that steer the beam electronically by adjusting phase. Understanding phased arrays is essential for designing modern satellite terminals, from consumer flat panels to enterprise-grade electronically steered antennas (ESAs).

```python
import numpy as np

C = 299792458  # Speed of light (m/s)

def wavelength_m(frequency_ghz):
    """Calculate wavelength from frequency."""
    return C / (frequency_ghz * 1e9)

def array_gain_db(n_elements, element_gain_dbi=5, efficiency=0.7):
    """
    Calculate phased array gain.
    
    Array gain = N Г— element_gain Г— efficiency
    In dB: G_array = 10*log10(N) + G_element + 10*log10(О·)
    """
    return 10 * np.log10(n_elements) + element_gain_dbi + 10 * np.log10(efficiency)

def scan_loss_db(theta_deg):
    """
    Calculate gain loss due to scan angle.
    
    Projected aperture reduces as cos(Оё).
    """
    theta_rad = np.radians(theta_deg)
    return 10 * np.log10(np.cos(theta_rad))

def max_element_spacing_m(frequency_ghz, max_scan_deg):
    """
    Calculate maximum element spacing to avoid grating lobes.
    
    d вүӨ О» / (1 + sin(Оё_max))
    """
    lambda_m = wavelength_m(frequency_ghz)
    theta_rad = np.radians(max_scan_deg)
    return lambda_m / (1 + np.sin(theta_rad))

def phase_shift_deg(element_index, element_spacing_m, frequency_ghz, scan_angle_deg):
    """
    Calculate required phase shift for element n.
    
    ПҶ_n = n Г— (2ПҖ/О») Г— d Г— sin(Оё) [radians]
    """
    lambda_m = wavelength_m(frequency_ghz)
    theta_rad = np.radians(scan_angle_deg)
    k = 2 * np.pi / lambda_m
    phase_rad = element_index * k * element_spacing_m * np.sin(theta_rad)
    return np.degrees(phase_rad) % 360

def beamwidth_deg(n_elements_linear, element_spacing_m, frequency_ghz):
    """
    Approximate 3dB beamwidth for linear array.
    
    Оё_3dB вүҲ 0.886 Г— О» / (N Г— d)
    """
    lambda_m = wavelength_m(frequency_ghz)
    aperture = n_elements_linear * element_spacing_m
    return np.degrees(0.886 * lambda_m / aperture)
```

<section class="lesson-pitfalls">

## Pitfalls

1. **Scan loss ignorance.** At 60В° from zenith, you lose 3 dBвҖ”factor this into link budgets.

2. **Grating lobes.** Element spacing > О»/2 at wide scan angles creates unwanted beams.

3. **Thermal limits.** High-power ESAs generate significant heat; thermal design is critical.

4. **Cost assumptions.** Consumer ESAs took 20+ years to become affordable; enterprise still expensive.

5. **Polarization.** Phased arrays can do both circular polarizations but need proper calibration.

6. **Mutual coupling.** Elements affect each other; real performance differs from ideal calculations.

7. **Regulatory limits.** EIRP masks limit off-axis emissions to protect adjacent satellites.
