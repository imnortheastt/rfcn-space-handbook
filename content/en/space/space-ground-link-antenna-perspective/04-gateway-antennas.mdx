## Why this matters

Gateway earth stations are the backbone of satellite communications—massive antenna farms that aggregate traffic from millions of user terminals. Whether you're designing HTS constellations, planning ground segment capacity, or understanding satellite network architecture, gateways are where space meets terrestrial infrastructure. A single gateway can handle 10-100+ Gbps of throughput.

## Apply

**Gateway link budget and capacity calculations:**

```python
import numpy as np

def gateway_eirp_dbw(power_w, antenna_gain_dbi, losses_db=2.0):
    """
    Calculate gateway EIRP.
    
    Args:
        power_w: HPA output power in watts
        antenna_gain_dbi: Antenna gain in dBi
        losses_db: Feed and waveguide losses
    
    Returns:
        EIRP in dBW
    """
    power_dbw = 10 * np.log10(power_w)
    return power_dbw + antenna_gain_dbi - losses_db

def gateway_gt_dbk(antenna_gain_dbi, system_temp_k, losses_db=1.0):
    """
    Calculate gateway G/T.
    """
    noise_temp_db = 10 * np.log10(system_temp_k)
    return antenna_gain_dbi - losses_db - noise_temp_db

def parabolic_gain_dbi(diameter_m, freq_ghz, efficiency=0.65):
    """
    Calculate parabolic antenna gain.
    """
    wavelength = 0.3 / freq_ghz  # meters
    area = np.pi * (diameter_m / 2) ** 2
    gain = efficiency * (4 * np.pi * area) / (wavelength ** 2)
    return 10 * np.log10(gain)

def gateway_capacity_gbps(bandwidth_mhz, spectral_efficiency, num_antennas, duty_cycle=1.0):
    """
    Calculate total gateway throughput capacity.
    """
    capacity_per_antenna = bandwidth_mhz * spectral_efficiency / 1000  # Gbps
    return capacity_per_antenna * num_antennas * duty_cycle

def diversity_availability(single_site_avail, num_sites, correlation=0.1):
    """
    Estimate availability with gateway diversity.
    
    Args:
        single_site_avail: Single site availability (e.g., 0.99)
        num_sites: Number of diverse sites
        correlation: Rain correlation between sites (0-1)
    
    Returns:
        Combined availability
    """
    # Simplified model: independent sites + correlation factor
    single_unavail = 1 - single_site_avail
    # Correlated unavailability model
    combined_unavail = single_unavail ** num_sites * (1 + correlation * (num_sites - 1))
    return 1 - combined_unavail
```

<section class="lesson-pitfalls">

## Pitfalls

1. **Rain fade underestimation.** V-band gateways can see 30+ dB fade; budget margin and diversity carefully.

2. **Fiber dependency.** Gateway without fiber = stranded capacity. Verify multiple diverse fiber paths.

3. **Interference to/from other systems.** ITU coordination required; gateway EIRP density limits apply.

4. **Power and cooling.** Large HPAs generate significant heat; plan for 2-3× power for cooling.

5. **LEO tracking complexity.** NGSO gateways need multiple tracking antennas and fast handoff logic.

6. **Regulatory delays.** Gateway licensing can take 6-18 months; start early.

7. **Scaling vs redundancy.** N+M redundancy (e.g., 8+2) needed for maintenance without outage.
