## Why this matters

Voyager 1 and 2 visited four planets with fuel budgets that should have reached only one. Cassini used four gravity assists to reach Saturn. New Horizons flew by Jupiter to cut travel time to Pluto by years. Gravity assists are the only way to reach distant destinations with reasonable mass budgets—and they're free.

## How it works

### Turn angle calculation

```
Hyperbolic Deflection:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  GEOMETRY:                                              │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │            Asymptote in                         │    │
│  │               ╲                                 │    │
│  │                ╲   δ = turn angle               │    │
│  │                 ╲ ╱                              │    │
│  │         r_p ──▶ ●🪐                             │    │
│  │                 ╱ ╲                              │    │
│  │                ╱   ╲                             │    │
│  │               ╱     Asymptote out               │    │
│  │                                                  │    │
│  │  sin(δ/2) = 1/e                                 │    │
│  │                                                  │    │
│  │  e = 1 + r_p × V∞² / μ                         │    │
│  │                                                  │    │
│  │  e = eccentricity of hyperbola                 │    │
│  │  r_p = periapsis distance                       │    │
│  │  V∞ = hyperbolic excess velocity                │    │
│  │  μ = planet gravitational parameter            │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  KEY INSIGHT:                                           │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  Closer flyby (smaller r_p) → larger turn      │    │
│  │  Slower approach (smaller V∞) → larger turn    │    │
│  │  Bigger planet (larger μ) → larger turn        │    │
│  │                                                  │    │
│  │  Maximum turn: pass just above atmosphere      │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### Velocity change

```
Computing ΔV from Gravity Assist:

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  VECTOR DIAGRAM:                                        │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │        V∞_in                                    │    │
│  │           ╲                                     │    │
│  │            ╲                                    │    │
│  │         δ  ╲│                                   │    │
│  │             ●───────▶ V_planet                  │    │
│  │             │╲                                  │    │

│  │             ▼  V∞_out                          │    │
│  │                                                  │    │
│  │  ΔV_helio = V∞_out - V∞_in (vector subtraction)│    │
│  │                                                  │    │
│  │  |ΔV| = 2 × V∞ × sin(δ/2)                      │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  GAIN vs LOSE ENERGY:                                   │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  TRAILING EDGE FLYBY (speed up):                │    │
│  │                                                  │    │
│  │          V_planet ────▶                         │    │
│  │                      🪐                          │    │
│  │                 ◁────╱                          │    │
│  │              S/C   ╱                            │    │
│  │                   ╱                              │    │
│  │                                                  │    │
│  │  Pass behind planet (in its orbit direction)   │    │
│  │  V∞ rotates to align with V_planet → gain      │    │
│  │                                                  │    │
│  │  ──────────────────────────────────────────    │    │
│  │                                                  │    │
│  │  LEADING EDGE FLYBY (slow down):                │    │
│  │                                                  │    │
│  │          V_planet ────▶                         │    │
│  │                      🪐                          │    │
│  │                        ╲                        │    │
│  │                   S/C ──▷                       │    │
│  │                                                  │    │
│  │  Pass in front of planet                       │    │
│  │  V∞ rotates opposite to V_planet → lose       │    │
│  │  (Useful for inner planet missions!)           │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
```

## Apply

**Gravity assist calculations:**

```python
import numpy as np

def turn_angle(v_inf, r_periapsis, mu_planet):
    """
    Calculate hyperbolic turn angle.
    
    v_inf: hyperbolic excess velocity (km/s)
    r_periapsis: closest approach distance (km)
    mu_planet: gravitational parameter (km³/s²)
    
    Returns: turn angle in degrees
    """
    # Eccentricity
    e = 1 + r_periapsis * v_inf**2 / mu_planet
    
    # Turn angle
    delta_rad = 2 * np.arcsin(1 / e)
    return np.degrees(delta_rad)

def gravity_assist_dv(v_inf, turn_angle_deg):
    """
    Calculate heliocentric velocity change magnitude.
    
    Returns: |ΔV| in km/s
    """
    delta_rad = np.radians(turn_angle_deg)
    return 2 * v_inf * np.sin(delta_rad / 2)

def max_possible_turn(v_inf, r_min, mu_planet):
    """
    Maximum turn angle given minimum periapsis.
    """
    return turn_angle(v_inf, r_min, mu_planet)

# Planetary parameters (km³/s², km, km/s)
PLANETS = {
    'Venus': {'mu': 324859, 'r_min': 6151, 'v_orbit': 35.0},
    'Earth': {'mu': 398600, 'r_min': 6578, 'v_orbit': 29.8},
    'Mars':  {'mu': 42828,  'r_min': 3590, 'v_orbit': 24.1},
    'Jupiter': {'mu': 126686534, 'r_min': 75000, 'v_orbit': 13.1},
    'Saturn': {'mu': 37931187, 'r_min': 65000, 'v_orbit': 9.7},
}

print("Gravity Assist Analysis:")
print("=" * 60)

# Example: Voyager at Jupiter
v_inf_jupiter = 10.0  # km/s (typical for Jupiter flyby)
r_p_jupiter = 350000  # km (Voyager 1 was ~349,000 km)

delta = turn_angle(v_inf_jupiter, r_p_jupiter, PLANETS['Jupiter']['mu'])
dv = gravity_assist_dv(v_inf_jupiter, delta)

print(f"\nVoyager-like Jupiter flyby:")
print(f"  V_infinity: {v_inf_jupiter} km/s")
print(f"  Periapsis: {r_p_jupiter:,} km (~5 Jupiter radii)")
print(f"  Turn angle: {delta:.1f}°")
print(f"  Heliocentric ΔV: {dv:.1f} km/s")

# Compare all planets at same V_inf
print(f"\nMaximum ΔV at V∞ = 8 km/s:")
print("-" * 60)
v_inf_test = 8.0

for name, p in PLANETS.items():
    delta = max_possible_turn(v_inf_test, p['r_min'], p['mu'])
    dv = gravity_assist_dv(v_inf_test, delta)
    print(f"  {name:10} Turn: {delta:5.1f}°  ΔV: {dv:5.1f} km/s")

# Multiple flybys example
print(f"\nCassini-style VVEJGA trajectory:")
print("  Venus flyby 1: ~2 km/s gain")
print("  Venus flyby 2: ~2 km/s gain")  
print("  Earth flyby:   ~4 km/s gain")
print("  Jupiter flyby: ~3 km/s gain")
print("  Total free ΔV: ~11 km/s (vs ~3 km/s fuel budget)")
```

## Pitfalls

1. **Timing is everything.** Planet must be in right position; opportunities are rare.

2. **Plane change penalty.** Changing inclination during flyby costs "turn angle budget."

3. **Not always faster.** Indirect route with assists can take longer than direct.

4. **V∞ constraint.** Can only turn what you bring; need initial energy to reach planet.

5. **Aiming is critical.** Small B-plane errors become large trajectory errors.

6. **Radiation hazard.** Jupiter flyby exposes spacecraft to intense radiation.

7. **Moon interactions.** Large moons can perturb the flyby trajectory.
