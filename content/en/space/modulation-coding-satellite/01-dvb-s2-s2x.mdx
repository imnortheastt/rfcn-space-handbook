## Why this matters

If you've ever watched satellite TV, received weather imagery, or downloaded data from a GEO communications satellite, you've used DVB-S2. It's the industry standard for satellite broadcast since 2005, and its successor DVB-S2X (2014) pushes efficiency even further. These standards define everything from modulation schemes to frame structures to error correction—they're the "TCP/IP of satellite broadcasting." Understanding DVB-S2/S2X is essential for anyone working with commercial satellite communications.

Adaptive Coding and Modulation (ACM):

│  │                                                        │         │
│  │  Terminal A: 32APSK 5/6 → 4.0 bps/Hz (high SNR)      │         │

│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  HOW ACM WORKS:                                                     │
│                                                                      │
│  1. Each terminal measures received SNR                            │
│  2. Terminal reports SNR via return channel                        │
│  3. Hub selects optimal MODCOD for each terminal's frames          │
│  4. MODCOD signaled in PL header (receiver self-decodes)           │
│  5. When rain clears, terminal C gets upgraded to higher MODCOD   │
│                                                                      │
│  BENEFIT: No "design for worst case" wasted capacity!              │
│  VCM (Variable Coding & Modulation): Similar but without feedback  │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘

## Apply

**MODCOD selection and capacity calculation:**

```python
import numpy as np

# DVB-S2 MODCOD table (subset)
MODCODS = [
    {"name": "QPSK 1/4",    "mod": "QPSK",   "rate": 0.25, "eff": 0.49, "es_n0_db": -2.35},
    {"name": "QPSK 1/3",    "mod": "QPSK",   "rate": 0.33, "eff": 0.66, "es_n0_db": -1.24},
    {"name": "QPSK 1/2",    "mod": "QPSK",   "rate": 0.50, "eff": 0.99, "es_n0_db": 1.00},
    {"name": "QPSK 2/3",    "mod": "QPSK",   "rate": 0.67, "eff": 1.32, "es_n0_db": 3.10},
    {"name": "QPSK 3/4",    "mod": "QPSK",   "rate": 0.75, "eff": 1.49, "es_n0_db": 4.03},
    {"name": "QPSK 5/6",    "mod": "QPSK",   "rate": 0.83, "eff": 1.65, "es_n0_db": 4.68},
    {"name": "8PSK 2/3",    "mod": "8PSK",   "rate": 0.67, "eff": 1.98, "es_n0_db": 6.62},
    {"name": "8PSK 3/4",    "mod": "8PSK",   "rate": 0.75, "eff": 2.23, "es_n0_db": 7.91},
    {"name": "16APSK 2/3",  "mod": "16APSK", "rate": 0.67, "eff": 2.64, "es_n0_db": 9.04},
    {"name": "16APSK 3/4",  "mod": "16APSK", "rate": 0.75, "eff": 2.97, "es_n0_db": 10.21},
    {"name": "16APSK 5/6",  "mod": "16APSK", "rate": 0.83, "eff": 3.30, "es_n0_db": 11.03},
    {"name": "32APSK 3/4",  "mod": "32APSK", "rate": 0.75, "eff": 3.70, "es_n0_db": 12.17},
    {"name": "32APSK 5/6",  "mod": "32APSK", "rate": 0.83, "eff": 4.12, "es_n0_db": 13.05},
    {"name": "32APSK 8/9",  "mod": "32APSK", "rate": 0.89, "eff": 4.40, "es_n0_db": 13.64},
]

def select_modcod(available_es_n0_db, margin_db=1.0):
    """Select best MODCOD for given Es/N0 with margin."""
    usable = available_es_n0_db - margin_db
    best = None
    for mc in MODCODS:
        if mc["es_n0_db"] <= usable:
            if best is None or mc["eff"] > best["eff"]:
                best = mc
    return best

def calculate_data_rate(symbol_rate_msps, modcod, rolloff=0.2):
    """Calculate data rate for given symbol rate and MODCOD."""
    return symbol_rate_msps * modcod["eff"]  # Mbps

print("DVB-S2 MODCOD Analysis")
print("=" * 60)

print("\n1. MODCOD Selection by SNR")
print("-" * 60)

test_snrs = [-2, 0, 3, 6, 9, 12, 14]

print(f"  {'Es/N0':>8} {'Selected MODCOD':<16} {'Efficiency':>12}")
print("-" * 60)

for snr in test_snrs:
    mc = select_modcod(snr)
    if mc:
        print(f"  {snr:>6.1f} dB  {mc['name']:<16} {mc['eff']:>8.2f} bps/Hz")
    else:
        print(f"  {snr:>6.1f} dB  {'(no MODCOD)':<<16}")

# Transponder capacity calculation
print("\n2. Transponder Capacity Analysis")
print("-" * 60)

transponder_bw_mhz = 36  # Typical Ku-band transponder
rolloffs = [0.35, 0.25, 0.20, 0.15, 0.10, 0.05]

print(f"  Transponder bandwidth: {transponder_bw_mhz} MHz")
print()
print(f"  {'Roll-off':>8} {'Symbol Rate':>14} {'Capacity (32APSK 5/6)':>22}")
print("-" * 60)

modcod_32apsk_56 = MODCODS[-3]  # 32APSK 5/6

for ro in rolloffs:
    symbol_rate = transponder_bw_mhz / (1 + ro)
    capacity = calculate_data_rate(symbol_rate, modcod_32apsk_56)
    dvb = "DVB-S2" if ro >= 0.20 else "DVB-S2X"
    print(f"  {ro*100:>5.0f}%    {symbol_rate:>10.2f} Msps    {capacity:>8.1f} Mbps ({dvb})")

# ACM scenario
print("\n3. ACM Capacity Improvement")
print("-" * 60)

# Assume 10 terminals with varying conditions
terminals = [
    {"id": 1, "snr": 14, "time_pct": 10},  # Excellent conditions
    {"id": 2, "snr": 12, "time_pct": 20},
    {"id": 3, "snr": 10, "time_pct": 30},
    {"id": 4, "snr": 8, "time_pct": 25},
    {"id": 5, "snr": 5, "time_pct": 15},  # Rain fade
]

# CCM: design for worst case
worst_snr = min(t["snr"] for t in terminals)
ccm_modcod = select_modcod(worst_snr)

# ACM: adapt per terminal
acm_avg_eff = 0
total_time = 0

for t in terminals:
    mc = select_modcod(t["snr"])
    acm_avg_eff += mc["eff"] * t["time_pct"] / 100
    total_time += t["time_pct"]

print(f"  CCM (worst-case design): {ccm_modcod['name']}")
print(f"    Fixed efficiency: {ccm_modcod['eff']:.2f} bps/Hz")
print()
print(f"  ACM (adaptive):")
print(f"    Average efficiency: {acm_avg_eff:.2f} bps/Hz")
print()

improvement = (acm_avg_eff / ccm_modcod['eff'] - 1) * 100
print(f"  ACM improvement: +{improvement:.1f}%")

# Frame timing
print("\n4. Frame Timing")
print("-" * 60)

symbol_rate_msps = 30  # 30 Msps typical

normal_frame_symbols = 33282  # With pilots
short_frame_symbols = 8370

normal_frame_us = normal_frame_symbols / symbol_rate_msps
short_frame_us = short_frame_symbols / symbol_rate_msps

print(f"  Symbol rate: {symbol_rate_msps} Msps")
print()
print(f"  Normal frame: {normal_frame_symbols} symbols = {normal_frame_us:.0f} µs = {normal_frame_us/1000:.2f} ms")
print(f"  Short frame:  {short_frame_symbols} symbols = {short_frame_us:.0f} µs = {short_frame_us/1000:.2f} ms")
print()
print(f"  Short frames useful for: low-latency voice, interactive services")
```

<section class="lesson-pitfalls">

## Pitfalls

1. **Confusing DVB-S2 with DVB-S2X.** S2X requires compatible equipment. Most legacy receivers only support S2.

2. **Using high MODCODs without link margin.** 32APSK 8/9 is efficient but needs ~14 dB Es/N0 plus margin.

3. **Ignoring roll-off in bandwidth calculations.** Symbol rate = bandwidth / (1 + roll-off). Small roll-off = more capacity.

4. **Forgetting HPA backoff for high-order APSK.** 32APSK needs 2-3 dB output backoff, reducing effective power.

5. **ACM without proper return channel.** Need reliable SNR reporting; stale measurements cause MODCOD mismatches.

6. **Mixing frame sizes mid-stream.** Receivers need consistent frame type within a Physical Layer Pipe (PLP).
