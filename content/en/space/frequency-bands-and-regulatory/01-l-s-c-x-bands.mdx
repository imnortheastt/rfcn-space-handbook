## Why this matters

Satellite spectrum is precious real estate. L, S, C, and X bands (1-12 GHz) are the workhorses of legacy satellite systems—GPS, mobile satcom, weather radar, and military communications all live here. These lower frequencies penetrate rain, foliage, and buildings better than higher bands, making them essential for mobile and maritime applications. Understanding their characteristics and allocations is foundational for any satcom engineer.

## Apply

**Frequency band comparison calculator:**

```python
import numpy as np

C = 299792458  # Speed of light (m/s)

def wavelength_m(frequency_ghz):
    """Calculate wavelength from frequency."""
    return C / (frequency_ghz * 1e9)

def wavelength_cm(frequency_ghz):
    """Calculate wavelength in centimeters."""
    return wavelength_m(frequency_ghz) * 100

def fspl_db(frequency_ghz, distance_km):
    """Free space path loss in dB."""
    return 20 * np.log10(distance_km) + 20 * np.log10(frequency_ghz) + 92.45

def antenna_diameter_for_gain(gain_db, frequency_ghz, efficiency=0.55):
    """Calculate dish diameter needed for specific gain."""
    lambda_m = wavelength_m(frequency_ghz)
    # G = η × (π × D / λ)²
    gain_linear = 10 ** (gain_db / 10)
    d_squared = gain_linear * (lambda_m ** 2) / (efficiency * np.pi ** 2)
    return np.sqrt(d_squared)

def rain_attenuation_simplified(frequency_ghz, rain_rate_mm_hr, path_length_km=5):
    """Simplified rain attenuation estimate (ITU-R P.838 approximation)."""
    # Very simplified model - real calculation uses ITU-R P.838
    if frequency_ghz < 2:
        return rain_rate_mm_hr * 0.001 * path_length_km
    elif frequency_ghz < 8:
        return rain_rate_mm_hr * 0.01 * path_length_km
    else:
        return rain_rate_mm_hr * 0.03 * path_length_km

# Band definitions
bands = [
    ("L-band", 1.5, 1.0, 2.0),
    ("S-band", 2.5, 2.0, 4.0),
    ("C-band", 6.0, 4.0, 8.0),
    ("X-band", 10.0, 8.0, 12.0),
    ("Ku-band", 14.0, 12.0, 18.0),
    ("Ka-band", 30.0, 26.0, 40.0),
]

print("Satellite Frequency Band Comparison")
print("=" * 75)

# Basic properties
print("\n1. Wavelength and Basic Properties")
print("-" * 75)
print(f"{'Band':<10} {'Center':>10} {'Wavelength':>12} {'FSPL @1000km':>15}")
print(f"{'':<10} {'(GHz)':>10} {'(cm)':>12} {'(dB)':>15}")
print("-" * 75)

for name, center, low, high in bands:
    wl = wavelength_cm(center)
    fspl = fspl_db(center, 1000)
    print(f"{name:<10} {center:>10.1f} {wl:>12.2f} {fspl:>15.1f}")

# Antenna size comparison
print("\n2. Antenna Diameter for 30 dBi Gain")
print("-" * 75)
print(f"{'Band':<10} {'Frequency':>10} {'Diameter':>12} {'Relative':>12}")
print(f"{'':<10} {'(GHz)':>10} {'(m)':>12} {'size':>12}")
print("-" * 75)

ref_size = antenna_diameter_for_gain(30, 6.0)  # C-band reference
for name, center, low, high in bands:
    diameter = antenna_diameter_for_gain(30, center)
    relative = diameter / ref_size
    print(f"{name:<10} {center:>10.1f} {diameter:>12.3f} {relative:>12.2f}x")

# Rain attenuation
print("\n3. Rain Attenuation (25 mm/hr, 5 km path)")
print("-" * 75)
print(f"{'Band':<10} {'Frequency':>10} {'Attenuation':>15} {'Impact':>20}")
print(f"{'':<10} {'(GHz)':>10} {'(dB)':>15} {'':>20}")
print("-" * 75)

for name, center, low, high in bands:
    atten = rain_attenuation_simplified(center, 25)
    if atten < 0.5:
        impact = "Negligible"
    elif atten < 2:
        impact = "Minor"
    elif atten < 5:
        impact = "Moderate"
    elif atten < 10:
        impact = "Significant"
    else:
        impact = "Severe"
    print(f"{name:<10} {center:>10.1f} {atten:>15.2f} {impact:>20}")

# Application summary
print("\n4. Primary Applications by Band")
print("-" * 75)
applications = {
    "L-band": "GPS, Iridium, Inmarsat, mobile satcom",
    "S-band": "Weather radar, NASA TDRSS, Sirius XM",
    "C-band": "Traditional FSS, cable distribution, SNG",
    "X-band": "Military, SAR imaging, marine radar",
    "Ku-band": "DTH TV, VSAT, news gathering",
    "Ka-band": "HTS broadband, Starlink feeder links",
}

for name, center, low, high in bands:
    apps = applications.get(name, "Various")
    print(f"{name:<10}: {apps}")

# Link budget example
print("\n5. Link Budget Comparison (GEO at 36,000 km)")
print("-" * 75)
print(f"{'Band':<10} {'FSPL':>12} {'Rain fade':>12} {'Total loss':>12}")
print(f"{'':<10} {'(dB)':>12} {'(dB)':>12} {'(dB)':>12}")
print("-" * 75)

for name, center, low, high in bands[:4]:  # L, S, C, X only
    fspl = fspl_db(center, 36000)
    rain = rain_attenuation_simplified(center, 25)
    total = fspl + rain
    print(f"{name:<10} {fspl:>12.1f} {rain:>12.1f} {total:>12.1f}")
```

**Band selection guide:**

| Requirement | Best Band | Reason |
|-------------|-----------|--------|
| Rain reliability (tropics) | L or C | Low rain fade |
| Mobile terminal | L or S | Omni antenna OK |
| High throughput | Ku or Ka | More bandwidth |
| Military secure | X | Dedicated allocation |
| Maritime | L or C | Rain, antenna size |

<section class="lesson-pitfalls">

## Pitfalls

1. **Band letter confusion.** IEEE and ITU definitions differ; always specify frequency range.

2. **5G interference.** C-band 3.7-3.98 GHz now shared with 5G in many countries.

3. **Rain margin.** Don't use C-band rain margin assumptions for Ka-band designs.

4. **Antenna size.** L-band mobile antennas have very low gain; link budget suffers.

5. **Regulatory variation.** Band allocations differ by ITU region (Americas, Europe, Asia-Pacific).

6. **Terrestrial interference.** S-band WiFi/microwave ovens, C-band 5G, X-band radar.

7. **Legacy assumptions.** "C-band is rain-proof" is relative; still fades in heavy tropical storms.
