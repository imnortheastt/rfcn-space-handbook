## Why this matters

Every satellite transmission uses spectrum that's been allocated by the International Telecommunication Union (ITU). Without ITU coordination, satellites would interfere with each other and with terrestrial systems. Understanding ITU's allocation framework—regions, service types, footnotes, and coordination procedures—is essential for anyone launching or operating satellite systems. Getting spectrum wrong can ground your entire project.

### Coordination procedures

```
ITU Coordination Process:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  SATELLITE FILING LIFECYCLE:                                        │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  1. API (Advance Publication Information)             │         │
│  │     ↓  Filed 2-7 years before launch                 │         │
│  │     ↓  Warns other operators of planned system        │         │
│  │                                                        │         │
│  │  2. Coordination Request (CR)                         │         │
│  │     ↓  Filed after API                               │         │
│  │     ↓  Detailed technical parameters                  │         │
│  │     ↓  Triggers coordination with affected parties    │         │
│  │                                                        │         │
│  │  3. Coordination Agreements                           │         │
│  │     ↓  Bilateral negotiations with other operators   │         │
│  │     ↓  May require operational constraints            │         │
│  │     ↓  Power limits, pointing restrictions, etc.     │         │
│  │                                                        │         │
│  │  4. Notification                                      │         │
│  │     ↓  Request recording in MIFR                     │         │
│  │     ↓  International recognition of spectrum use     │         │
│  │                                                        │         │
│  │  5. Bringing into Use (BIU)                          │         │
│  │     ↓  Must launch within 7 years of filing          │         │
│  │     ↓  Must actually use the frequencies             │         │
│  │                                                        │         │
│  │  6. MIFR Recording                                    │         │
│  │        Master International Frequency Register       │         │
│  │        International protection established          │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  TIMELINE EXAMPLE:                                                  │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  Year 0        Year 2-4        Year 5-7     Year 7   │         │
│  │  │             │               │            │        │         │
│  │  API ───────► CR ───────► Notify ───────► Launch    │         │
│  │  filed        filed        filed          (BIU)     │         │
│  │               │                                      │         │
│  │               └── Coordination negotiations ──────►  │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
```

```python
def lookup_allocation(freq_mhz):
    """Find ITU allocation for a given frequency."""
    for band, data in ITU_ALLOCATIONS.items():
        low, high = band.split("-")
        if float(low) <= freq_mhz <= float(high):
            return band, data
    return None, None

def print_allocation_table():
    """Print full allocation summary."""
    print("ITU Radio Regulations — Satellite Allocations Summary")
    print("=" * 80)
    
    for band, data in ITU_ALLOCATIONS.items():
        low, high = band.split("-")
        freq_range = f"{low}-{high} MHz"
        if float(low) >= 1000:
            freq_range = f"{float(low)/1000:.1f}-{float(high)/1000:.1f} GHz"
        
        print(f"\n{freq_range}")
        print("-" * 40)
        print(f"  Region 1: {', '.join(data['region1'])}")

def check_coordination_needed(freq_mhz, service_type, region):
    """Check if coordination is likely needed."""
    band, data = lookup_allocation(freq_mhz)
    if not band:
        return "Frequency not in common satellite allocations"
    
    region_key = f"region{region}"
    allocations = data.get(region_key, [])

    is_primary = any(service_upper in alloc for alloc in allocations)

        "notes": [
            "File API 2-7 years before launch",
            "Coordinate with affected administrations",
            "Negotiate with existing operators in same arc",
        ]
    }

from datetime import datetime, timedelta

def filing_timeline(target_launch_date):
    """Calculate ITU filing deadlines."""
    launch = datetime.strptime(target_launch_date, "%Y-%m-%d")
```

<section class="lesson-pitfalls">

## Pitfalls

1. **Missing filing deadlines.** 7-year BIU window is strict; miss it and lose the filing.

2. **Primary vs secondary confusion.** Secondary services have NO protection rights; capitalization matters.

3. **Regional differences.** Same frequency can have different rules in different ITU regions.

4. **Footnote complexity.** Thousands of footnotes create country-specific exceptions; read carefully.

5. **Coordination underestimation.** GEO arc is crowded; expect 50+ coordination agreements for popular bands.

6. **Paper satellite risk.** Some entities file speculatively to sell coordination rights; verify counterparty credibility.

7. **National layer.** ITU allocations must still be licensed by each national administration (FCC, Ofcom, etc.).
