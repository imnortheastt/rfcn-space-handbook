## Why this matters

Traditional GEO satellites covered entire continents with a single wide beam—low capacity, high latency workaround. HTS satellites use dozens to hundreds of spot beams to concentrate power and reuse spectrum, delivering 10-100× more capacity from the same orbital slot.

## Apply

**Spot beam calculations:**

```python
import numpy as np

def spot_beam_diameter(altitude_km, beam_angle_deg, off_nadir_deg=0):
    """
    Calculate spot beam footprint diameter
    
    altitude_km: satellite altitude
    beam_angle_deg: half-power beamwidth (degrees)
    off_nadir_deg: pointing angle from nadir
    
    Returns: diameter in km (at nadir equivalent)
    """
    h = altitude_km
    theta = np.radians(beam_angle_deg)
    
    # Nadir beam
    diameter = 2 * h * np.tan(theta / 2)
    
    # Off-nadir correction (simplified)
    if off_nadir_deg > 0:
        off_nadir = np.radians(off_nadir_deg)
        diameter *= 1 / np.cos(off_nadir)
    
    return diameter

def antenna_beamwidth(frequency_ghz, diameter_m):
    """
    Estimate antenna beamwidth
    
    Returns: half-power beamwidth in degrees
    """
    c = 3e8  # m/s
    wavelength = c / (frequency_ghz * 1e9)
    theta_rad = 70 * wavelength / diameter_m
    return np.degrees(theta_rad)

def frequency_reuse_capacity(total_bw_mhz, num_colors, num_beams):
    """
    Calculate effective bandwidth with frequency reuse
    
    Returns: effective bandwidth in MHz
    """
    bw_per_beam = total_bw_mhz / num_colors
    return bw_per_beam * num_beams

def hts_capacity_estimate(bw_per_beam_mhz, spectral_eff, num_beams):
    """
    Estimate total HTS capacity
    
    spectral_eff: bits/s/Hz (typically 2-5 for HTS)
    
    Returns: capacity in Gbps
    """
    capacity_per_beam = bw_per_beam_mhz * spectral_eff / 1000  # Gbps
    return capacity_per_beam * num_beams

# Example: Ka-band HTS
print("Ka-Band HTS Spot Beam Design:")
print("-" * 50)

freq = 20  # GHz (Ka downlink)
ant_dia = 2.5  # meters

beamwidth = antenna_beamwidth(freq, ant_dia)
print(f"Antenna diameter: {ant_dia} m")
print(f"Ka-band ({freq} GHz) beamwidth: {beamwidth:.2f}°")

diameter = spot_beam_diameter(35786, beamwidth)
print(f"Spot beam diameter (nadir): {diameter:.0f} km")

# Capacity calculation
total_bw = 500  # MHz available
num_colors = 4
num_beams = 100

eff_bw = frequency_reuse_capacity(total_bw, num_colors, num_beams)
print(f"\nFrequency reuse:")
print(f"  {total_bw} MHz / {num_colors} colors × {num_beams} beams = {eff_bw/1000:.1f} GHz effective")

capacity = hts_capacity_estimate(total_bw/num_colors, 3.5, num_beams)
print(f"  Capacity @ 3.5 bits/Hz: {capacity:.0f} Gbps total")
```

## Pitfalls

1. **Edge-of-beam performance.** Users at beam edge see 3dB less power; design margin needed.

2. **Adjacent beam interference.** Same-frequency beams can interfere if too close.

3. **Rain fade.** Ka-band spot beams suffer significant rain attenuation.

4. **Beam pointing accuracy.** Small pointing errors move the beam hundreds of km.

5. **Traffic imbalance.** Some beams may be congested while others idle.

6. **Antenna complexity.** More beams = more feeds, larger reflector, higher cost.

7. **Thermal design.** High-power amplifiers in compact payload = thermal challenges.
