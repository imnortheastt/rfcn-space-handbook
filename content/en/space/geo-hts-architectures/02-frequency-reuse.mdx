## Why this matters

Spectrum is expensive and scarce. A traditional satellite uses each frequency once. HTS satellites reuse the same frequencies across many spot beams—turning 500 MHz of spectrum into 5+ GHz of effective bandwidth. The technique is identical to cellular networks but operates at orbital scale.

## How it works

### Co-channel interference

```
C/I Ratio (Carrier-to-Interference):

┌─────────────────────────────────────────────────────────┐
│                                                          │
│  TARGET BEAM:                                           │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │        Same-frequency interferers               │    │
│  │                                                  │    │
│  │           [B]        [B]                        │    │
│  │             ╲         ╱                          │    │
│  │       [B]────[A]────[B]   ← Target beam A       │    │
│  │             ╱         ╲                          │    │
│  │           [B]        [B]                        │    │
│  │                                                  │    │
│  │  C/I = Power_desired / Σ Power_interferers      │    │
│  │                                                  │    │
│  │  For 4-color pattern:                           │    │
│  │  First-tier interferers at distance D (center   │    │
│  │  to center of nearest same-color beam)          │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
│  ISOLATION MECHANISMS:                                  │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                  │    │
│  │  1. SPATIAL SEPARATION:                         │    │
│  │     Same-color beams physically distant         │    │
│  │     Path loss + antenna sidelobe rolloff        │    │
│  │                                                  │    │
│  │  2. POLARIZATION:                               │    │
│  │     RHCP vs LHCP adds ~25-30 dB isolation       │    │
│  │     When used in combination with frequency     │    │
│  │                                                  │    │
│  │  3. ANTENNA PATTERN:                            │    │
│  │     Beam sidelobes typically -20 to -30 dB      │    │
│  │     Further reduces interference                │    │
│  │                                                  │    │
│  │  Target C/I: typically 15-25 dB                 │    │
│  │                                                  │    │
│  └─────────────────────────────────────────────────┘    │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## Apply

**Frequency reuse calculations:**

```python
import numpy as np

def frequency_reuse_capacity(system_bw_mhz, num_colors, num_beams):
    """
    Calculate effective bandwidth with frequency reuse
    
    Returns: dict with capacity metrics
    """
    bw_per_beam = system_bw_mhz / num_colors
    effective_bw = bw_per_beam * num_beams
    reuse_gain = num_beams / num_colors
    
    return {
        'bw_per_beam_mhz': bw_per_beam,
        'effective_bw_mhz': effective_bw,
        'reuse_gain': reuse_gain
    }

def carrier_to_interference(antenna_gain_db, sidelobe_level_db, 
                            reuse_distance_beamwidths, num_interferers=6):
    """
    Estimate C/I for hexagonal pattern
    
    reuse_distance_beamwidths: distance to nearest same-color beam
                               in units of beamwidth
    """
    # Simplified: interference from sidelobes of co-channel beams
    # At distance D, antenna pattern rolloff provides isolation
    
    # First-tier interferers: typically 6 in hexagonal grid
    # Each at distance D, contributing through sidelobe level
    
    # Path loss difference (simplified, assumes flat earth)
    # Power falls as 1/distance^2 in free space
    distance_loss_db = 20 * np.log10(reuse_distance_beamwidths)
    
    # C/I per interferer = main beam gain - sidelobe - distance factor
    single_ci = antenna_gain_db - sidelobe_level_db - distance_loss_db
    
    # Multiple interferers (power sum)
    aggregate_ci = single_ci - 10 * np.log10(num_interferers)
    
    return aggregate_ci

def polarization_isolation(cross_pol_db=25):
    """
    Additional isolation from orthogonal polarization
    
    Typical values: 25-30 dB for circular polarization
    """
    return cross_pol_db

# Example: Ka-band HTS design
print("Frequency Reuse Analysis:")
print("=" * 50)

# System parameters
system_bw = 500  # MHz
num_beams = 100

for n_colors in [7, 4, 3]:
    result = frequency_reuse_capacity(system_bw, n_colors, num_beams)
    print(f"\nN={n_colors} reuse pattern:")
    print(f"  Bandwidth per beam: {result['bw_per_beam_mhz']:.1f} MHz")
    print(f"  Effective bandwidth: {result['effective_bw_mhz']/1000:.1f} GHz")
    print(f"  Reuse gain: {result['reuse_gain']:.1f}×")

# C/I estimation
print("\n" + "=" * 50)
print("C/I Estimation (4-color pattern):")

# Typical antenna parameters
main_beam_gain = 45  # dBi
sidelobe_level = -25  # dB relative to peak
reuse_distance = 2.0  # beam widths to nearest co-channel

ci = carrier_to_interference(main_beam_gain, sidelobe_level, 
                             reuse_distance, num_interferers=6)
print(f"  C/I (spatial only): {ci:.1f} dB")

# With polarization discrimination
ci_with_pol = ci + polarization_isolation(25)
print(f"  C/I (with pol isolation): {ci_with_pol:.1f} dB")
```

**Reuse pattern comparison:**

| Pattern | Colors | Reuse Gain | C/I Margin | Complexity |
|---------|--------|------------|------------|------------|
| N=7 | 7 | Low | High (~25dB) | Simple |
| N=4 | 4 | Medium | Good (~18dB) | Standard |
| N=3 | 3 | High | Lower (~15dB) | Moderate |
| N=1 | 1 | Maximum | Requires IC | Complex |

<section class="lesson-pitfalls">

## Pitfalls

1. **Ignoring sidelobes.** Antenna sidelobe levels directly impact achievable C/I.

2. **Edge beam distortion.** Beams at coverage edge have different geometry—may need different reuse.

3. **Rain scatter.** Precipitation can scatter signals between beams, degrading isolation.

4. **Cross-polarization.** Real antennas have imperfect cross-pol; don't assume ideal 30dB.

5. **Traffic hotspots.** Same reuse everywhere ignores that demand varies spatially.

6. **Interference aggregation.** Multiple same-color beams all contribute—don't consider just one.

7. **Gateway backhaul.** Gateway links also need frequency planning; can't ignore feeder side.
