## Why this matters

Lunar Gateway is humanity's first space station beyond low Earth orbit—a permanent outpost orbiting the Moon that will serve as a staging point for lunar surface missions and eventually Mars. For RF engineers, Gateway presents unique challenges: a Near Rectilinear Halo Orbit that changes Earth visibility constantly, link budgets spanning 400,000 km, and a communications architecture that must support crewed operations, robotics, and science payloads. Understanding Gateway's design reveals how cislunar communications differs from familiar GEO/LEO systems.

```

┌─────────────────────────────────────────────────────────────────────┐

│                                                                      │
```

## Apply

**Gateway link budget estimation:**

```python
import numpy as np

def fspl_db(distance_km, freq_ghz):
    """Free Space Path Loss in dB."""
    return 20 * np.log10(distance_km) + 20 * np.log10(freq_ghz) + 92.45

def parabolic_gain_dbi(diameter_m, freq_ghz, efficiency=0.55):
    """Parabolic antenna gain in dBi."""
    wavelength_m = 0.3 / freq_ghz
    gain_linear = efficiency * (np.pi * diameter_m / wavelength_m) ** 2
    return 10 * np.log10(gain_linear)

def link_budget_cn0(eirp_dbw, distance_km, freq_ghz, g_over_t_dbk, other_losses_db=0):
    """Calculate C/N0 for a space link."""
    k_dbw = -228.6  # Boltzmann constant
    path_loss = fspl_db(distance_km, freq_ghz)
    cn0 = eirp_dbw - path_loss - other_losses_db + g_over_t_dbk - k_dbw
    return cn0, path_loss

# Gateway parameters
print("Lunar Gateway Link Budget Analysis")
print("=" * 60)

# Distances
earth_moon_avg = 384400  # km
nrho_perilune = 3000  # km from Moon
nrho_apolune = 70000  # km from Moon

# Gateway distances from Earth (approximate)
gateway_min = earth_moon_avg - nrho_apolune  # ~314,000 km (Moon between Earth and Gateway)
gateway_max = earth_moon_avg + nrho_apolune  # ~454,000 km (Gateway beyond Moon)
gateway_avg = earth_moon_avg  # ~384,000 km

# Frequency bands
ka_band = 32  # GHz (return link)
ka_fwd = 26   # GHz (forward link)

# DSN parameters
dsn_34m_gain = parabolic_gain_dbi(34, ka_band, 0.60)
dsn_70m_gain = parabolic_gain_dbi(70, ka_band, 0.55)
dsn_g_over_t = 52  # dB/K (typical Ka-band 34m BWG)

# Gateway parameters
gateway_dish = 2.0  # m (approximate)
gateway_gain = parabolic_gain_dbi(gateway_dish, ka_band)
gateway_tx_power = 20  # W
gateway_tx_power_dbw = 10 * np.log10(gateway_tx_power)
gateway_eirp = gateway_tx_power_dbw + gateway_gain

print("\n1. Distance Comparison")
print("-" * 60)

distances = [

print(f"  34m BWG antenna:")

print(f"")

print("-" * 60)

other_losses = 2.0  # dB (pointing, atmospheric, etc.)

for name, dist in [("Min distance", gateway_min), ("Avg distance", gateway_avg), ("Max distance", gateway_max)]:
    cn0, fspl = link_budget_cn0(gateway_eirp, dist, ka_band, dsn_g_over_t, other_losses)

print("\n5. Achievable Data Rates")
print("-" * 60)

# Assume QPSK with rate 1/2 coding: Eb/N0 ≈ 3 dB required
# C/N0 = Eb/N0 + 10 log(Rb)
# Rb = 10^((C/N0 - Eb/N0)/10)

ebn0_required = 3.0  # dB (QPSK rate 1/2)

for name, dist in [("Min distance", gateway_min), ("Max distance", gateway_max)]:
    cn0, _ = link_budget_cn0(gateway_eirp, dist, ka_band, dsn_g_over_t, other_losses)
    rb_bps = 10 ** ((cn0 - ebn0_required) / 10)
    rb_mbps = rb_bps / 1e6
    print(f"  {name}: Up to {rb_mbps:.0f} Mbps")

# Comparison with GEO
print("\n6. Comparison with GEO Commercial")
print("-" * 60)

geo_dist = 36000
geo_eirp = 52  # dBW (typical satellite)
geo_gt = 27  # dB/K (typical ground station)

cn0_gateway, _ = link_budget_cn0(gateway_eirp, gateway_avg, ka_band, dsn_g_over_t, other_losses)
cn0_geo, _ = link_budget_cn0(geo_eirp, geo_dist, ka_band, geo_gt, 1.0)

print(f"  GEO commercial: C/N₀ = {cn0_geo:.1f} dBHz")
print(f"  Gateway:        C/N₀ = {cn0_gateway:.1f} dBHz")
print(f"  Difference:     {cn0_gateway - cn0_geo:.1f} dB")
print(f"")

print(f"  much higher than commercial ({geo_gt} dB/K).")
```

<section class="lesson-pitfalls">

## Pitfalls

1. **Assuming constant distance.** NRHO varies from 314,000 to 454,000 km from Earth. Budget for worst case.

2. **Ignoring lunar occultation.** ~14% of orbit has no Earth line-of-sight. Plan for store-and-forward.

3. **Expecting GEO data rates.** 21 dB more path loss means much lower throughput without DSN.

4. **Forgetting light-time delay.** 1.0-1.5 second round-trip means no real-time control.

5. **Single-string thinking.** Gateway needs S-band backup; Ka-band can fail.

6. **Orbital period confusion.** 6.5-day orbit means scheduling is different from GEO or LEO.
