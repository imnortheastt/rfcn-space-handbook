## Why this matters

FDMA wastes bandwidth when traffic is bursty — each user occupies their frequency slice even when silent. TDMA fixes this: all users share the entire transponder bandwidth but take turns transmitting in assigned time slots. When you're not talking, someone else uses your slot. Result: near-100% transponder utilization instead of 10%. This is why TDMA dominates modern satellite communications, from Intelsat trunking to DVB-RCS2 broadband.

## How it works

```
TDMA Synchronization Challenge:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │

│  Stations at different distances have different delays             │
│                                                                      │
│                        GEO Satellite                                │
│                            ▲                                        │
│                           /│\                                       │

│                         /  │  \                                     │
│                        /   │   \                                    │
│                       /    │    \                                   │

│               Station A  Station B  Station C                       │
│               (under sat) (edge)   (far edge)                       │

│  WITHOUT CORRECTION:                                                │

│  │                       ↑               ↑                      │  │
│  │                   COLLISION!      COLLISION!                 │  │

│                                                                      │
│  SOLUTION: Each station pre-advances its burst                     │
│                                                                      │

│                                                                      │
│  WITH CORRECTION:                                                   │
│  ┌──────────────────────────────────────────────────────────────┐  │

│  │                      ↑           ↑           ↑               │  │
│  │                   No overlap! Properly aligned!              │  │
│  └──────────────────────────────────────────────────────────────┘  │
```

## Apply

**TDMA frame and capacity calculator:**

```python
import numpy as np

def tdma_frame_efficiency(frame_duration_us, num_bursts, guard_time_us, 
                          preamble_bits, symbol_rate_msps):
    """
    Calculate TDMA frame efficiency.
    
    Args:
        frame_duration_us: Total frame duration (microseconds)
        num_bursts: Number of traffic bursts per frame
        guard_time_us: Guard time per burst (microseconds)
        preamble_bits: Preamble + UW bits per burst
        symbol_rate_msps: Symbol rate (Msps)
    
    Returns:
        Dict with efficiency metrics
    """
    guard_overhead_us = num_bursts * guard_time_us
    
    preamble_duration_us = preamble_bits / symbol_rate_msps
    preamble_overhead_us = num_bursts * preamble_duration_us
    
    total_overhead_us = guard_overhead_us + preamble_overhead_us
    
    traffic_time_us = frame_duration_us - total_overhead_us
    efficiency = traffic_time_us / frame_duration_us
    
    return {
        "frame_duration_us": frame_duration_us,
        "guard_overhead_us": guard_overhead_us,
        "preamble_overhead_us": preamble_overhead_us,
        "traffic_time_us": traffic_time_us,
        "efficiency": efficiency
    }

def tdma_capacity(transponder_bw_mhz, symbol_rate_msps, bits_per_symbol,
                  code_rate, frame_efficiency):
    """
    Calculate TDMA system throughput.
    """
    gross_rate_mbps = symbol_rate_msps * bits_per_symbol
    
    coded_rate_mbps = gross_rate_mbps * code_rate
    
    net_rate_mbps = coded_rate_mbps * frame_efficiency
    
    return {
        "gross_rate_mbps": gross_rate_mbps,
        "coded_rate_mbps": coded_rate_mbps,
        "net_rate_mbps": net_rate_mbps
    }

def timing_advance_calculation(station_distance_km, sat_altitude_km=35786):
    """
    Calculate required timing advance for TDMA synchronization.
    """
    c = 299792.458  # km/s
    
    round_trip_delay_ms = 2 * (sat_altitude_km + station_distance_km) / c * 1000
    
    reference_delay_ms = 2 * sat_altitude_km / c * 1000
    
    timing_advance_us = (round_trip_delay_ms - reference_delay_ms) * 1000
    
    return {
        "round_trip_delay_ms": round_trip_delay_ms,
        "timing_advance_us": timing_advance_us
    }

def compare_fdma_tdma(num_users, activity_factor, transponder_bw_mhz):
    """
    Compare FDMA vs TDMA capacity for bursty traffic.
    """
    fdma_bw_per_user = transponder_bw_mhz / num_users
    fdma_effective_bw = fdma_bw_per_user * activity_factor
    fdma_total_efficiency = activity_factor
    
    tdma_overhead = 0.05  # 5% typical
    tdma_efficiency = 1 - tdma_overhead
    
    capacity_ratio = tdma_efficiency / fdma_total_efficiency
    
    return {
        "fdma_efficiency": fdma_total_efficiency,
        "tdma_efficiency": tdma_efficiency,
        "tdma_advantage_ratio": capacity_ratio
    }
```

| Round-trip delay | 2 × Distance / c |
| TDMA advantage | (1 - overhead) / Activity_factor |

<section class="lesson-pitfalls">

## Pitfalls

1. **Ignoring timing complexity.** TDMA requires microsecond-accurate synchronization. Each station needs unique timing advance based on distance. GPS helps enormously.

2. **Underestimating preamble overhead.** Each burst needs carrier recovery, bit timing recovery, and unique word. For short bursts, this overhead dominates.

3. **Forgetting rain fade effects.** Rain attenuates signal AND changes propagation delay. TDMA must dynamically adjust timing.

4. **Assuming TDMA always wins.** For continuous traffic (video broadcast), FDMA's simplicity may be preferable. TDMA wins for bursty, interactive traffic.

5. **Guard time too short.** Timing uncertainty, oscillator drift, and Doppler all add up. Guard times of 1-5 μs are typical for GEO.
