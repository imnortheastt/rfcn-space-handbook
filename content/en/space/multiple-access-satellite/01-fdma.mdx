## Why this matters

FDMA is the oldest and simplest multiple access scheme. Every user gets their own frequency slice — like separate radio stations on AM/FM. It's still the dominant method for traditional satellite TV (one carrier per channel) and maritime VSAT. Understanding FDMA is the foundation for understanding why we invented TDMA, CDMA, and MF-TDMA to overcome its limitations.

## How it works

### Transponder allocation

```
Satellite Transponder FDMA Architecture:

┌─────────────────────────────────────────────────────────────────────┐

│                             │                       │               │
```

### Guard bands and efficiency

```
Why Guard Bands Are Necessary:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  IDEAL (no guard bands):                                            │
│  ┌────────────────┬────────────────┬────────────────┐              │
│  │    Carrier 1   │    Carrier 2   │    Carrier 3   │              │
│  │    6 MHz       │    6 MHz       │    6 MHz       │              │
│  └────────────────┴────────────────┴────────────────┘              │
│                                                                      │
│  REALITY (with guard bands):                                        │
│  ┌──────────┐ ░░ ┌──────────┐ ░░ ┌──────────┐ ░░                  │
│  │Carrier 1 │    │Carrier 2 │    │Carrier 3 │                      │
│  │ 5.4 MHz  │0.6 │ 5.4 MHz  │0.6 │ 5.4 MHz  │0.6                   │
│  └──────────┘    └──────────┘    └──────────┘                      │
│                                                                      │
│  Guard bands needed because:                                        │
│  • Oscillator drift (± kHz)                                        │
│  • Doppler shift from satellite motion                             │
│  • Filter roll-off (non-ideal rectangular filters)                 │
│  • Intermodulation products from transponder                       │
│                                                                      │
│  EFFICIENCY CALCULATION:                                            │
│  Usable BW: 5.4 × 3 = 16.2 MHz                                     │
│  Total BW:  6.0 × 3 = 18.0 MHz                                     │
│  Efficiency: 16.2/18.0 = 90%                                       │
│                                                                      │
│  (Typical FDMA efficiency: 85-92%)                                 │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Intermodulation distortion

```
The Intermodulation Problem:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  When multiple carriers pass through a non-linear amplifier:       │
│                                                                      │
│  Input: f1 and f2                                                   │
│                                                                      │
│  Output: f1, f2, AND:                                               │
│    2f1 - f2  (3rd order IM)  ← falls in-band!                      │
│    2f2 - f1  (3rd order IM)  ← falls in-band!                      │
│    3f1 - 2f2 (5th order IM)                                        │
│    ...                                                              │
│                                                                      │
│  EXAMPLE:                                                           │
│  f1 = 14.000 GHz, f2 = 14.010 GHz                                  │
│  IM product: 2(14.000) - 14.010 = 13.990 GHz                       │
│  IM product: 2(14.010) - 14.000 = 14.020 GHz                       │
│                                                                      │
│  These IM products interfere with other carriers!                  │
│                                                                      │
│  SOLUTION: Back-off the amplifier (run below saturation)           │
│                                                                      │
│     Output                                                          │
│     Power ▲                                                         │
│           │            ╭────── Saturation                          │
│           │          ╭─╯                                            │
│           │        ╭─╯                                              │
│           │      ╭─╯                                                │
│           │    ╭─╯   ← Operating point (backed off 3-6 dB)        │
│           │  ╭─╯                                                    │
│           │╭─╯                                                      │
│           └──────────────────────────────────→ Input Power         │
│                                                                      │
│  TRADE-OFF: Less IM distortion, but less total output power       │
│             (wastes expensive transponder capacity)                │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## Apply

**FDMA capacity and efficiency calculator:**

```python
import numpy as np

def fdma_capacity(transponder_bw_mhz, carrier_bw_mhz, guard_band_mhz):
    """
    Calculate FDMA system capacity.
    
    Args:
        transponder_bw_mhz: Total transponder bandwidth (MHz)
        carrier_bw_mhz: Bandwidth per carrier (MHz)
        guard_band_mhz: Guard band between carriers (MHz)
    
    Returns:
        Dict with capacity metrics
    """
    effective_carrier_bw = carrier_bw_mhz + guard_band_mhz
    num_carriers = int(transponder_bw_mhz / effective_carrier_bw)
    usable_bw = num_carriers * carrier_bw_mhz
    efficiency = usable_bw / transponder_bw_mhz
    
    return {
        "num_carriers": num_carriers,
        "usable_bw_mhz": usable_bw,
        "wasted_bw_mhz": transponder_bw_mhz - usable_bw,
        "efficiency": efficiency
    }

def carrier_to_intermod_ratio(input_backoff_db, output_backoff_db):
    """
    Estimate C/IM ratio based on amplifier backoff.
    
    Rule of thumb: C/IM improves ~2 dB per dB of output backoff
    """
    baseline_cim = 15  # dB at saturation
    cim_improvement = 2 * output_backoff_db
    return baseline_cim + cim_improvement

def link_budget_fdma(eirp_dbw, path_loss_db, gt_dbk, carrier_bw_hz, 
                     backoff_db, c_im_db, c_n_uplink_db):
    """
    Calculate FDMA link budget including intermod effects.
    """
    k_dbw = -228.6  # Boltzmann constant in dBW/K/Hz
    
    c_n0_downlink = eirp_dbw - path_loss_db + gt_dbk - k_dbw - backoff_db
    
    c_n_downlink = c_n0_downlink - 10 * np.log10(carrier_bw_hz)
    
    c_n_uplink_linear = 10**(c_n_uplink_db/10)
    c_n_downlink_linear = 10**(c_n_downlink/10)
    c_im_linear = 10**(c_im_db/10)
    
    c_n_total = 1 / (1/c_n_uplink_linear + 1/c_n_downlink_linear + 1/c_im_linear)
    c_n_total_db = 10 * np.log10(c_n_total)
    
    return {
        "c_n0_downlink_dbhz": c_n0_downlink,
        "c_n_downlink_db": c_n_downlink,
        "c_n_total_db": c_n_total_db
    }
```

**Key formulas:**

| Metric | Formula |
|--------|---------|
| Number of carriers | floor(Transponder_BW / (Carrier_BW + Guard)) |
| Spectral efficiency | Usable_BW / Total_BW |
| C/IM ratio | ~15 dB + 2 × Output_backoff (rule of thumb) |
| Total C/N | 1 / (1/C/N_up + 1/C/N_down + 1/C/IM) |

<section class="lesson-pitfalls">

## Pitfalls

1. **Ignoring intermodulation.** Multiple carriers through one amplifier create IM products. You must back off the amplifier, losing 3-6 dB of capacity.

2. **Forgetting guard bands.** Real filters aren't rectangular. Budget 5-10% for guard bands or you'll get adjacent channel interference.

3. **Assuming linear amplifiers.** Satellite TWTAs are non-linear near saturation. This changes everything for multi-carrier FDMA.

4. **Assigning full bandwidth to bursty users.** FDMA assigns bandwidth permanently. If traffic is bursty, 90% of your transponder sits idle.

5. **Mixing carrier sizes carelessly.** Different-sized carriers have different backoff requirements. The largest carrier dominates intermod.
