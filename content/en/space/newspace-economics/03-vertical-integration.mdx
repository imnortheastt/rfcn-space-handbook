## Why this matters

ULA buys engines from Aerojet Rocketdyne, fairings from RUAG, and avionics from Honeywell — each vendor adds margins and lead times. SpaceX builds Merlin engines, Raptor engines, fairings, avionics, flight computers, and even office furniture in-house. The result: SpaceX's cost per kg to orbit is 5x lower than ULA's. Starlink takes this further — SpaceX builds the satellites, launches them on their own rockets, and sells internet directly to consumers. No middlemen. Understanding vertical integration tells you which space companies have structural cost advantages and which are stuck paying supplier markups forever.

import numpy as np

def margin_stacking(base_cost, margins):
    """
    Calculate final price with stacked vendor margins.
    
    Args:
        base_cost: Raw production cost
        margins: List of margin percentages at each tier [0.25, 0.20, 0.15]
    
    Returns:
        Final price after all margins applied
    """
    price = base_cost
    for margin in margins:
        price = price * (1 + margin)
    return price

def integration_savings(outsourced_cost, margin_layers, internal_overhead=0.10):
    """
    Calculate savings from vertical integration.
    
    Args:
        outsourced_cost: What you'd pay vendors
        margin_layers: Number of vendor tiers
        internal_overhead: Additional cost of doing it yourself
    
    Returns:
        Dict with cost comparison
    """
    avg_margin = 0.20  # Typical aerospace vendor margin
    
    estimated_base = outsourced_cost / ((1 + avg_margin) ** margin_layers)
    internal_cost = estimated_base * (1 + internal_overhead)
    savings = outsourced_cost - internal_cost
    
    return {
        "outsourced": outsourced_cost,
        "estimated_base": estimated_base,
        "internal_cost": internal_cost,
        "savings": savings,
        "savings_pct": (savings / outsourced_cost) * 100
    }

def launch_cost_comparison(payload_kg, internal_marginal, external_price):
    """
    Compare internal vs external launch costs.
    """
    internal_per_kg = internal_marginal / payload_kg
    external_per_kg = external_price / payload_kg
    
    return {
        "internal_total": internal_marginal,
        "external_total": external_price,
        "internal_per_kg": internal_per_kg,
        "external_per_kg": external_per_kg,
        "cost_ratio": external_price / internal_marginal
    }

def constellation_integration_advantage(num_satellites, internal_sat_cost, external_sat_cost,
                                         internal_launch_cost, external_launch_cost,
                                         sats_per_launch):
    """
    Calculate total cost advantage of vertical integration for a constellation.
    """
    num_launches = np.ceil(num_satellites / sats_per_launch)
    
    internal_total = (num_satellites * internal_sat_cost) + (num_launches * internal_launch_cost)
    external_total = (num_satellites * external_sat_cost) + (num_launches * external_launch_cost)
    
    return {
        "internal_total": internal_total,
        "external_total": external_total,
        "savings": external_total - internal_total,
        "savings_pct": ((external_total - internal_total) / external_total) * 100,
        "cost_ratio": external_total / internal_total
    }

## Pitfalls

1. **Underestimating NRE.** Building in-house requires upfront R&D, tooling, and facilities. Only pays off at volume.

2. **Integrating non-strategic components.** Making fasteners in-house wastes engineering on commodity items.

3. **Ignoring opportunity cost.** Engineers building desks aren't building rockets. Measure total productivity.

4. **Assuming vendor margins are pure profit.** Vendors have expertise you'd have to build. Factor in learning costs.

5. **Neglecting supply chain risk.** Integration reduces external risk but concentrates internal risk. One factory fire stops everything.

6. **Over-integrating early.** Startups should buy until volume justifies making. Don't build factories before product-market fit.
