## Why this matters

In 2010, launching 1 kg to LEO cost $54,000 on a Delta IV. In 2024, it costs $1,500 on Falcon 9. That 36x reduction unlocked the entire NewSpace economy: Starlink, Planet, OneWeb, and hundreds of smallsat startups exist because launch became affordable. Understanding cost-to-orbit trends tells you which business models are now viable, which will become viable with Starship, and why legacy aerospace is scrambling to compete.

## How it works

### The reusability equation

```
Cost Breakdown of a Launch:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  EXPENDABLE ROCKET (traditional):                                   │
│                                                                      │
│  Cost per launch = Hardware + Operations + Range + Insurance       │
│                  = $150M    + $30M       + $5M   + $15M            │
│                  = $200M (all thrown away)                          │
│                                                                      │
│  For 22,800 kg to LEO: $200M / 22,800 kg = $8,772/kg               │
│                                                                      │
│  ─────────────────────────────────────────────────────────────────  │
│                                                                      │
│  REUSABLE ROCKET (Falcon 9 Block 5):                               │
│                                                                      │
│  First stage: $35M amortized over ~20 flights = $1.75M/flight     │
│  Second stage: $12M (still expendable)                             │
│  Operations: $7M (streamlined)                                     │
│  Refurbishment: $2M/flight                                         │
│  Total: ~$27M                                                       │
│                                                                      │
│  For 22,800 kg to LEO: $27M / 22,800 kg = $1,184/kg               │
│                                                                      │
│  ─────────────────────────────────────────────────────────────────  │
│                                                                      │
│  STARSHIP (full reusability target):                               │
│                                                                      │
│  Super Heavy: $100M / 100+ flights = $1M/flight                   │
│  Starship: $50M / 100+ flights = $0.5M/flight                     │
│  Propellant: $1M (mostly methane)                                  │
│  Operations: $0.5M (airplane-like turnaround)                      │
│  Total: ~$3M                                                        │
│                                                                      │
│  For 150,000 kg to LEO: $3M / 150,000 kg = $20/kg                 │
│                                                                      │
│  (SpaceX public target: $200/kg including margins)                 │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### The Starlink effect on pricing

```
SpaceX Internal Economics:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  FALCON 9 PRODUCTION:                                               │
│                                                                      │
│  • Manufacturing rate: ~1 booster/month (2024)                     │
│  • Each booster flies 15-20+ times                                 │
│  • Marginal cost per flight: ~$15-20M (internal estimate)          │
│  • External price: $67M → healthy margin                           │
│                                                                      │
│  STARLINK INTERNAL LAUNCHES:                                        │
│                                                                      │
│  • SpaceX pays internal marginal cost (~$15M)                      │
│  • 60 Starlink v1 sats × 260 kg = 15,600 kg                        │
│  • Internal $/kg: $15M / 15,600 kg ≈ $960/kg                       │
│                                                                      │
│  This explains why Starlink works economically:                    │
│  • SpaceX doesn't pay $67M/launch — they pay $15M                 │
│  • Competitors buying launch pay 4-5x more per satellite          │
│  • Vertical integration = massive cost advantage                   │
│                                                                      │
│  RIDESHARE IMPLICATIONS:                                            │
│                                                                      │
│  • Transporter missions: ~$5,000-6,000/kg for small sats          │
│  • SpaceX fills unused capacity → nearly free marginal revenue    │
│  • Undercuts every small launch provider on price                  │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

**Calculate your launch economics:**

```python
import numpy as np

# Launch vehicle database (2024 data)
VEHICLES = {
    "Falcon 9": {"leo_kg": 22800, "price_m": 67, "reuse": True},
    "Falcon Heavy": {"leo_kg": 63800, "price_m": 97, "reuse": True},
    "Starship": {"leo_kg": 150000, "price_m": 3, "reuse": True},  # Target
    "Atlas V": {"leo_kg": 18850, "price_m": 110, "reuse": False},
    "Vulcan": {"leo_kg": 27200, "price_m": 110, "reuse": False},
    "Ariane 6": {"leo_kg": 21600, "price_m": 82, "reuse": False},
    "New Glenn": {"leo_kg": 45000, "price_m": 70, "reuse": True},
    "Electron": {"leo_kg": 300, "price_m": 7.5, "reuse": True},
}

def cost_per_kg(vehicle):
    """Calculate $/kg to LEO."""
    v = VEHICLES[vehicle]
    return (v["price_m"] * 1e6) / v["leo_kg"]

def launch_cost_for_payload(payload_kg, vehicle):
    """Calculate total launch cost for a given payload."""
    v = VEHICLES[vehicle]
    if payload_kg > v["leo_kg"]:
        return None  # Payload too heavy
    return v["price_m"] * 1e6

def rideshare_cost(payload_kg, base_rate=5500):
    """Estimate rideshare cost (SpaceX Transporter typical rates)."""
    return payload_kg * base_rate

def constellation_launch_cost(sat_mass_kg, num_sats, vehicle, sats_per_launch=None):
    """Calculate total launch cost for a constellation."""
    v = VEHICLES[vehicle]
    if sats_per_launch is None:
        sats_per_launch = int(v["leo_kg"] / sat_mass_kg)
    
    num_launches = np.ceil(num_sats / sats_per_launch)
    total_cost = num_launches * v["price_m"] * 1e6
    cost_per_sat = total_cost / num_sats
    
    return {
        "num_launches": int(num_launches),
        "sats_per_launch": sats_per_launch,
        "total_cost_m": total_cost / 1e6,
        "cost_per_sat_m": cost_per_sat / 1e6,
    }
```

**Key metrics:**

| Metric | Formula |
|--------|---------|
| Cost per kg | Launch price / LEO capacity |
| Constellation launches | ceil(num_sats / (LEO_cap / sat_mass)) |
| Total launch cost | Launches × price_per_launch |
| Break-even flight count | Vehicle cost / (price - marginal cost) |

## Pitfalls

1. **Comparing list prices.** SpaceX internal cost is ~$15M/flight; external price is $67M. Starlink pays internal.

2. **Ignoring rideshare.** A 100 kg cubesat doesn't need $67M — it pays ~$500k on Transporter.

3. **Forgetting payload fairings.** Heavy sats may be volume-limited, not mass-limited. $/kg misleads.

4. **Assuming linear scaling.** Starship at $200/kg doesn't mean 100 kg costs $20k — there's minimum pricing.

5. **Neglecting integration costs.** Launch is 30-50% of total mission cost. Ground systems and ops add up.

6. **Extrapolating Starship too early.** Production Starship is years away. Plan on Falcon 9 prices for 2024-2026.
