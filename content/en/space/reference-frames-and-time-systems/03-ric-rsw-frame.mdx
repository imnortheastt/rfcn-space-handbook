## Why this matters

When two satellites approach each other — for docking, collision avoidance, or formation flying — their absolute positions in ECI don't give intuitive information. "Target is 500 km ahead" makes immediate sense; "target is at [X, Y, Z] in J2000" requires calculation. Local orbital frames like RIC (Radial-In-track-Cross-track) and RSW (Radial-Along-track-Normal) center on one spacecraft and orient axes along its orbital motion. These frames make relative dynamics intuitive, maneuver planning straightforward, and conjunction screening meaningful.

## How it works

### ECI to RIC transformation

```
Converting from ECI to RIC:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  Given: Target state (r_t, v_t) and Chaser state (r_c, v_c) in ECI │
│                                                                      │
│  STEP 1: Build RIC rotation matrix from Target state               │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  R̂ = r_t / |r_t|           (radial unit vector)      │         │
│  │  ĥ = (r_t × v_t) / |r_t × v_t|  (orbit normal)       │         │
│  │  Î = ĥ × R̂                  (in-track unit vector)   │         │
│  │                                                        │         │
│  │  Rotation matrix ECI→RIC:                             │         │
│  │                                                        │         │
│  │       ┌         ┐                                     │         │
│  │       │ R̂ᵀ      │                                     │         │
│  │  M =  │ Îᵀ      │   (rows are unit vectors)          │         │
│  │       │ Ĉᵀ      │                                     │         │
│  │       └         ┘                                     │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  STEP 2: Transform relative position                               │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  Δr_eci = r_c - r_t        (ECI relative position)   │         │
│  │  Δr_ric = M · Δr_eci       (RIC relative position)   │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  STEP 3: Transform relative velocity (includes rotation)          │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  ω = h / |r_t|²           (orbital angular rate)     │         │
│  │  Ω = ω · Ĉ                (angular velocity vector)  │         │
│  │                                                        │         │
│  │  Δv_eci = v_c - v_t                                   │         │
│  │  Δv_ric = M · Δv_eci - Ω × Δr_ric                    │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Relative motion dynamics (CW equations)

```
Clohessy-Wiltshire Equations (Circular Reference Orbit):

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  For small relative separations in circular orbit:                 │
│                                                                      │
│  ẍ = 3n²x + 2nẏ           (radial)                                │
│  ÿ = -2nẋ                  (in-track)                              │
│  z̈ = -n²z                  (cross-track)                          │
│                                                                      │
│  Where n = √(μ/a³) = mean motion (rad/s)                           │
│                                                                      │
│  KEY INSIGHTS:                                                      │
│  ┌───────────────────────────────────────────────────────┐         │
│  │                                                        │         │
│  │  1. Cross-track (z) is DECOUPLED                      │         │
│  │     → Simple harmonic oscillation at orbital period   │         │
│  │     → A cross-track burn only affects cross-track     │         │
│  │                                                        │         │
│  │  2. Radial and in-track are COUPLED                   │         │
│  │     → Radial displacement drifts in-track             │         │
│  │     → In-track velocity creates radial oscillation    │         │
│  │                                                        │         │
│  │  3. Natural motion forms ELLIPSES                     │         │
│  │     → 2:1 in-track to radial ratio                   │         │
│  │     → Free relative orbit = stadium shaped           │         │
│  │                                                        │         │
│  │  4. Drift rate: Δẏ ≈ -1.5n · Δx                      │         │
│  │     → 1 km below = drifts ~2.3 km/orbit ahead        │         │
│  │     → (for LEO with 90-min period)                   │         │
│  │                                                        │         │
│  └───────────────────────────────────────────────────────┘         │
│                                                                      │
│  NATURAL DRIFT VISUALIZATION:                                       │
│                                                                      │
│        y (in-track)                                                 │
│            ↑                                                        │
│            │    ╭──────────────╮                                    │
│            │   ╱                ╲                                   │
│            │  │                  │ ← 2:1 ellipse                   │
│            │  │     Target       │                                  │
│  ──────────┼──│────── 🛰️ ───────│──────────→ x (radial)            │
│            │  │                  │                                  │
│            │   ╲                ╱                                   │
│            │    ╰──────────────╯                                    │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## Apply

**RIC frame computations:**

```python
import numpy as np

MU_EARTH = 398600.4418  # km³/s²

def eci_to_ric(r_target, v_target, r_chaser, v_chaser):
    """Convert chaser state from ECI to RIC relative to target."""
    # Compute RIC unit vectors
    r_mag = np.linalg.norm(r_target)
    
    r_hat = r_target / r_mag                    # Radial (R)
    h = np.cross(r_target, v_target)            # Angular momentum
    h_mag = np.linalg.norm(h)
    c_hat = h / h_mag                           # Cross-track (C)
    i_hat = np.cross(c_hat, r_hat)              # In-track (I)
    
    # Rotation matrix ECI -> RIC
    R_eci_to_ric = np.array([r_hat, i_hat, c_hat])
    
    # Relative position in ECI
    delta_r_eci = r_chaser - r_target
    
    # Transform to RIC
    delta_r_ric = R_eci_to_ric @ delta_r_eci
    
    # Relative velocity (accounting for frame rotation)
    omega = h_mag / r_mag**2  # Orbital angular rate
    omega_vec = omega * c_hat
    
    delta_v_eci = v_chaser - v_target
    delta_v_ric = R_eci_to_ric @ delta_v_eci - np.cross(omega_vec, delta_r_ric)
    
    return delta_r_ric, delta_v_ric, R_eci_to_ric

def cw_propagate(r0_ric, v0_ric, t, n):
    """Propagate relative state using Clohessy-Wiltshire equations."""
    x0, y0, z0 = r0_ric
    vx0, vy0, vz0 = v0_ric
    
    nt = n * t
    cos_nt = np.cos(nt)
    sin_nt = np.sin(nt)
    
    # Position
    x = (4 - 3*cos_nt)*x0 + sin_nt/n*vx0 + 2/n*(1 - cos_nt)*vy0
    y = 6*(sin_nt - nt)*x0 + y0 - 2/n*(1 - cos_nt)*vx0 + (4*sin_nt - 3*nt)/n*vy0
    z = z0*cos_nt + vz0/n*sin_nt
    
    # Velocity
    vx = 3*n*sin_nt*x0 + cos_nt*vx0 + 2*sin_nt*vy0
    vy = 6*n*(cos_nt - 1)*x0 + 2*sin_nt*vx0 + (4*cos_nt - 3)*vy0
    vz = -z0*n*sin_nt + vz0*cos_nt
    
    return np.array([x, y, z]), np.array([vx, vy, vz])

def drift_rate(delta_r_radial, n):
    """Calculate in-track drift rate due to radial offset."""
    return -1.5 * n * delta_r_radial

def miss_distance(r_ric):
    """Calculate miss distance (total relative distance)."""
    return np.linalg.norm(r_ric)

print("RIC/RSW Local Orbital Frame Examples")
print("=" * 70)

# Example: ISS and visiting vehicle
print("\n1. ECI to RIC Conversion")
print("-" * 70)

# ISS (target) in ECI
r_iss = np.array([-2436.45, -2436.45, 6891.04])  # km
v_iss = np.array([-5.654, 4.830, 2.312])          # km/s

# Visiting vehicle (chaser) - 2 km behind, 100 m below
r_chaser = np.array([-2434.45, -2434.45, 6889.04])  # km (approximately)
v_chaser = np.array([-5.654, 4.830, 2.312])          # km/s (matching velocity)

delta_r, delta_v, R_mat = eci_to_ric(r_iss, v_iss, r_chaser, r_chaser)

print("  Target (ISS) state:")
print(f"    Position: [{r_iss[0]:.2f}, {r_iss[1]:.2f}, {r_iss[2]:.2f}] km")
print(f"    Velocity: [{v_iss[0]:.3f}, {v_iss[1]:.3f}, {v_iss[2]:.3f}] km/s")

print("\n  Chaser relative state in RIC:")
print(f"    Radial (R):      {delta_r[0]*1000:.1f} m")
print(f"    In-track (I):    {delta_r[1]*1000:.1f} m")
print(f"    Cross-track (C): {delta_r[2]*1000:.1f} m")
print(f"    Miss distance:   {miss_distance(delta_r)*1000:.1f} m")

print(f"    TCA miss distance: {tca_miss_distance} m")
```

**Key RIC relationships:**

| Concept | Formula/Value |
|---------|---------------|
| Drift from radial offset | ẏ ≈ -1.5n·x |
| Natural relative orbit shape | 2:1 in-track:radial ellipse |
| Mean motion (LEO) | n ≈ 0.001 rad/s |
| Drift rate (1 km below, LEO) | ~2.3 km/orbit ahead |

</section>

<section class="lesson-pitfalls">

## Pitfalls

1. **Sign convention confusion.** RIC +R is zenith; LVLH +Z is often nadir. Always verify.

2. **Forgetting frame rotation.** RIC velocity includes frame rotation. Use full transformation.

3. **CW equations for elliptical orbits.** CW assumes circular reference orbit. Use Tschauner-Hempel for eccentric.

4. **Large separations.** CW is linearized for small Δr. For >10 km, use full nonlinear propagation.

5. **Mixing frames.** A position in RIC with velocity in RSW produces garbage. Keep everything in one frame.

6. **Ignoring J2 for long-term.** RIC ignores orbital perturbations. For hours-to-days, add J2 effects.
