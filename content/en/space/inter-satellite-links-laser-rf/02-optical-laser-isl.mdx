## Why this matters

While RF ISLs are proven technology, they're bandwidth-limited by spectrum availability and antenna size. Optical (laser) inter-satellite links operate at infrared wavelengths where bandwidth is virtually unlimited and antennas shrink to centimeter-scale telescopes. Starlink's laser mesh demonstrates that optical ISL enables global low-latency connectivity without ground station dependency. Understanding optical ISL physics and engineering is critical for next-generation constellation design.

## Apply

**Optical ISL link budget calculator:**

```python
import numpy as np

# Constants
C = 299792458  # m/s
H_PLANCK = 6.62607015e-34  # J·s

def optical_fspl_db(distance_km, wavelength_nm):
    """Free space path loss for optical wavelength."""
    d_m = distance_km * 1000
    lambda_m = wavelength_nm * 1e-9
    return 20 * np.log10(4 * np.pi * d_m / lambda_m)

def telescope_gain_db(aperture_m, wavelength_nm, efficiency=0.8):
    """Telescope gain (diffraction-limited)."""
    lambda_m = wavelength_nm * 1e-9
    area = np.pi * (aperture_m / 2) ** 2
    gain = efficiency * (np.pi * aperture_m / lambda_m) ** 2
    return 10 * np.log10(gain)

def beam_divergence_urad(aperture_m, wavelength_nm):
    """Full-angle beam divergence in microradians."""
    lambda_m = wavelength_nm * 1e-9
    theta = 2.44 * lambda_m / aperture_m
    return theta * 1e6  # Convert to μrad

def beam_footprint_m(range_km, divergence_urad):
    """Beam diameter at target range."""
    return range_km * 1000 * divergence_urad * 1e-6

def photons_per_bit(rx_power_dbm, data_rate_gbps, wavelength_nm):
    """Calculate received photons per bit."""
    rx_power_w = 10 ** ((rx_power_dbm - 30) / 10)
    lambda_m = wavelength_nm * 1e-9
    photon_energy = H_PLANCK * C / lambda_m
    photons_per_sec = rx_power_w / photon_energy
    bits_per_sec = data_rate_gbps * 1e9
    return photons_per_sec / bits_per_sec

def optical_isl_link_budget(tx_power_dbm, tx_aperture_m, rx_aperture_m,
                            wavelength_nm, range_km, 
                            tx_loss_db=2, rx_loss_db=2, pointing_loss_db=3):
    """Complete optical ISL link budget."""
    # Gains
    tx_gain = telescope_gain_db(tx_aperture_m, wavelength_nm)
    rx_gain = telescope_gain_db(rx_aperture_m, wavelength_nm)
    
    # Path loss
    fspl = optical_fspl_db(range_km, wavelength_nm)
    
    # Received power
    rx_power = (tx_power_dbm + tx_gain - tx_loss_db 
                - fspl + rx_gain - rx_loss_db - pointing_loss_db)
    
    # Beam parameters
    divergence = beam_divergence_urad(tx_aperture_m, wavelength_nm)
    footprint = beam_footprint_m(range_km, divergence)
    pointing_req = divergence / 10  # μrad
    
    return {
        'tx_power_dbm': tx_power_dbm,
        'tx_gain_dbi': tx_gain,
        'rx_gain_dbi': rx_gain,
        'fspl_db': fspl,
        'rx_power_dbm': rx_power,
        'divergence_urad': divergence,
        'footprint_m': footprint,
        'pointing_req_urad': pointing_req,
        'pointing_req_arcsec': pointing_req * 0.206,
    }

def detector_sensitivity_dbm(modulation, data_rate_gbps):
    """Approximate detector sensitivity for given modulation."""
    # Sensitivity in photons/bit, then convert to dBm
    ppb = {
        'OOK': 100,       # On-off keying
        'DPSK': 30,       # Differential PSK
        'BPSK_coherent': 10,  # Coherent BPSK
        'QPSK_coherent': 15,  # Coherent QPSK
    }
    
    if modulation not in ppb:
        return None
    
    wavelength_nm = 1550
    lambda_m = wavelength_nm * 1e-9
    photon_energy = H_PLANCK * C / lambda_m
    
    power_w = ppb[modulation] * photon_energy * data_rate_gbps * 1e9
    return 10 * np.log10(power_w) + 30  # dBm

# Analysis
print("Optical ISL Link Budget Analysis")
print("=" * 60)

# Starlink-class optical ISL
print("\n1. LEO Optical ISL (Starlink-class)")
print("-" * 60)
budget = optical_isl_link_budget(
    tx_power_dbm=37,         # 5 W optical
    tx_aperture_m=0.08,      # 8 cm telescope
    rx_aperture_m=0.08,
    wavelength_nm=1550,      # Telecom C-band
    range_km=2000,
)

for k, v in budget.items():
    if 'db' in k.lower():
        print(f"  {k:20}: {v:+.1f} dB")
    elif 'urad' in k.lower():
        print(f"  {k:20}: {v:.1f} μrad")
    elif 'arcsec' in k.lower():
        print(f"  {k:20}: {v:.1f} arcsec")
    elif '_m' in k.lower() and 'dbm' not in k.lower():
        print(f"  {k:20}: {v:.1f} m")
    else:
        print(f"  {k:20}: {v:.1f}")

# Check margin for different modulations
print("\n  Modulation Analysis (10 Gbps):")
print(f"  {'Modulation':15} {'Sensitivity':>12} {'Margin':>10}")
for mod in ['OOK', 'DPSK', 'BPSK_coherent', 'QPSK_coherent']:

    margin = budget['rx_power_dbm'] - sens

# Photons per bit

    elif 'arcsec' in k.lower():
        print(f"  {k:20}: {v:.1f} arcsec")
```

<section class="lesson-pitfalls">

## Pitfalls

1. **Pointing is everything.** A 5 μrad pointing error loses 3 dB; 20 μrad loses most of your signal.

2. **Acquisition time.** Finding the partner satellite takes 10-60 seconds; plan for handover gaps.

3. **Sun blinding.** When Sun is near line of sight, detector saturates; may need to break link.

4. **Background noise.** Earth albedo, Sun reflection from target can raise noise floor.

5. **Thermal management.** Laser generates heat; stable operation needs active thermal control.

6. **Wavelength choice matters.** 1550 nm has lower solar background but 1064 nm has higher power lasers.

7. **Coherent vs incoherent tradeoff.** Coherent detection needs matched linewidth lasers — adds complexity.
