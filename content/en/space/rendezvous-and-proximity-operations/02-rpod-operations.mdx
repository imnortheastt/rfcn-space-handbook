## Why this matters

Every spacecraft dockingвҖ”Crew Dragon to ISS, Soyuz to station, on-orbit servicing vehicle to client satelliteвҖ”follows a carefully choreographed sequence of phases. From the initial phasing burn hundreds of kilometers away to the final centimeter-level alignment at capture, RPOD operations demand precise navigation, robust abort capability, and strict procedural discipline. Understanding these phases is essential for mission planning and flight operations.

**RPOD phase timing and delta-V:**

```python
import numpy as np

# Constants
MU_EARTH = 398600.4418  # kmВі/sВІ

def compute_orbital_params(altitude_km):
    """Compute basic orbital parameters."""
    r = 6371 + altitude_km
    v = np.sqrt(MU_EARTH / r)
    T = 2 * np.pi * np.sqrt(r**3 / MU_EARTH)
    n = 2 * np.pi / T
    return {
        'radius_km': r,
        'velocity_km_s': v,
        'period_s': T,
        'period_min': T / 60,
        'mean_motion_rad_s': n
    }

def phasing_maneuver(phase_angle_deg, target_altitude_km, num_orbits=1):
    """Calculate delta-V for phasing to reduce phase angle."""
    # Phase angle: how far behind (positive) or ahead (negative)
    r_target = 6371 + target_altitude_km
    T_target = 2 * np.pi * np.sqrt(r_target**3 / MU_EARTH)
    
    # Phasing orbit period
    phase_angle_rad = np.radians(phase_angle_deg)
    phase_to_cover = 2 * np.pi * num_orbits - phase_angle_rad
    T_phasing = T_target * phase_to_cover / (2 * np.pi * num_orbits)
    
    # Semi-major axis of phasing orbit
    a_phasing = (MU_EARTH * (T_phasing / (2 * np.pi))**2) ** (1/3)
    
    # Delta-V (Hohmann-like)
    v_target = np.sqrt(MU_EARTH / r_target)
    v_phasing_apo = np.sqrt(MU_EARTH * (2/r_target - 1/a_phasing))
    
    dv_enter = abs(v_phasing_apo - v_target)
    dv_exit = dv_enter  # Symmetric for return
    
    return {
        'phase_angle_deg': phase_angle_deg,
        'num_orbits': num_orbits,
        'phasing_period_min': T_phasing / 60,
        'target_period_min': T_target / 60,
        'delta_v_total_m_s': (dv_enter + dv_exit) * 1000,
        'phasing_altitude_km': (2 * a_phasing - r_target) - 6371
    }

def approach_delta_v(ranges_km, closure_rates_m_s):
    """Estimate delta-V for approach maneuvers."""
    # Simplified: assume each range change requires accel + decel
    total_dv = 0
    maneuvers = []
    
    for i in range(len(ranges_km) - 1):
        # Accelerate to closure rate, then decelerate to slower rate
        if i < len(closure_rates_m_s) - 1:
            dv = abs(closure_rates_m_s[i] - closure_rates_m_s[i+1])
        else:
            dv = closure_rates_m_s[i]
        
        total_dv += dv * 2  # Accel + decel
        maneuvers.append({
            'from_km': ranges_km[i],
            'to_km': ranges_km[i+1],
            'closure_m_s': closure_rates_m_s[i],
            'dv_m_s': dv * 2
        })
    
    return total_dv, maneuvers

def capture_requirements():
    """Typical capture interface requirements."""
    return {
        'max_closing_velocity_m_s': 0.10,  # 10 cm/s
        'max_lateral_velocity_m_s': 0.05,  # 5 cm/s
        'max_angular_rate_deg_s': 0.5,
        'max_lateral_misalignment_m': 0.15,  # 15 cm
        'max_angular_misalignment_deg': 5.0,
        'capture_envelope_diameter_m': 0.30,  # 30 cm
    }

# Analysis

print("=" * 60)

# Target orbit

orbit = compute_orbital_params(altitude)
print(f"\nTarget Orbit (ISS):")
print(f"  Altitude: {altitude} km")
print(f"  Velocity: {orbit['velocity_km_s']:.3f} km/s")
print(f"  Period: {orbit['period_min']:.1f} min")
```

<section class="lesson-pitfalls">

## Pitfalls

1. **Sensor handoff gaps.** Transition between sensors can cause navigation discontinuities; overlap coverage.

2. **Lighting constraints.** Optical sensors fail in shadow; plan approach during orbital day.

3. **Communication blackouts.** Loss of ground contact during critical phases requires onboard autonomy.

4. **Abort fuel reserves.** CAM must always be available; don't consume margins approaching.

5. **Target attitude uncertainty.** Tumbling or off-nominal target attitude complicates capture.

6. **Thruster plume impingement.** RCS firings can damage target solar arrays or sensors.

7. **Relative GPS multipath.** Near large structures, GPS accuracy degrades from reflections.
