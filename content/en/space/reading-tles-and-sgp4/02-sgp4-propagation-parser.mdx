## Why this matters

A TLE is useless without a propagator. SGP4 (Simplified General Perturbations 4) is THE algorithm for converting TLEs into satellite positions at any time. Every ground station controller, collision avoidance system, and satellite tracker uses it. The algorithm is publicly available, battle-tested since the 1970s, and has known accuracy limits — typically 1 km/day of error growth for LEO satellites.

## Intuition

### What SGP4 does

```
SGP4: TLE → Position at Any Time

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  INPUT:                                                             │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  TLE (from Space-Track/CelesTrak)                             │ │
│  │  + Target time (e.g., "2024-01-01 15:30:00 UTC")              │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                         │                                           │
│                         ▼                                           │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                    SGP4 ALGORITHM                              │ │
│  │                                                                 │ │
│  │  1. Parse TLE elements                                        │ │
│  │  2. Compute time since epoch (minutes)                        │ │
│  │  3. Apply perturbation models:                                │ │
│  │     - Secular terms (J2 drag effect over time)                │ │
│  │     - Periodic terms (short/long period oscillations)         │ │
│  │  4. Handle special cases (deep space vs near-earth)           │ │
│  │  5. Convert mean elements to osculating position/velocity     │ │
│  │                                                                 │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                         │                                           │
│                         ▼                                           │
│  OUTPUT:                                                            │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  Position: X, Y, Z in TEME (True Equator Mean Equinox) [km]   │ │
│  │  Velocity: Vx, Vy, Vz in TEME [km/s]                          │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Perturbation models

```
What SGP4 Accounts For:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  EARTH OBLATENESS (J2, J3, J4):                                    │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  Earth bulges at equator → non-spherical gravity              │ │
│  │                                                                 │ │
│  │  Effects on orbit:                                             │ │
│  │    • RAAN drifts westward (nodal precession)                   │ │
│  │    • Argument of perigee rotates                               │ │
│  │    • Mean motion changes slightly                              │ │
│  │                                                                 │ │
│  │  J2 is dominant (~1000x stronger than J3, J4)                  │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                      │
│  ATMOSPHERIC DRAG (B*):                                            │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  For LEO satellites, atmosphere slows the satellite           │ │
│  │                                                                 │ │
│  │  B* = (Cd × A × ρ) / (2 × m) ≈ drag coefficient               │ │
│  │                                                                 │ │
│  │  Effects:                                                       │ │
│  │    • Semi-major axis shrinks                                   │ │
│  │    • Orbit circularizes                                        │ │
│  │    • Eventually re-entry                                       │ │
│  │                                                                 │ │
│  │  Highly variable with solar activity!                          │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                      │
│  WHAT SGP4 DOES NOT MODEL:                                         │
│  • Maneuvers (TLEs go stale after burns)                           │
│  • Solar radiation pressure (significant for large area/mass)      │
│  • High-fidelity atmospheric density variations                    │
│  • General relativity effects                                      │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

### Algorithm flow

```
SGP4 Algorithm Steps (Simplified):

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  1. INITIALIZATION (once per TLE):                                 │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  • Parse TLE into mean elements                                │ │
│  │  • Compute constants from J2, J3, J4 harmonics                 │ │
│  │  • Determine near-earth vs deep-space mode                     │ │
│  │  • Pre-compute secular rate coefficients                       │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                         │                                           │
│                         ▼                                           │
│  2. PROPAGATION (for each target time):                            │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  Δt = target_time - epoch_time (in minutes)                    │ │
│  │                                                                 │ │
│  │  a. Apply secular terms (linear drift):                        │ │
│  │     M = M₀ + n·Δt + ṅ·Δt²/2                                    │ │
│  │     Ω = Ω₀ + Ω̇·Δt                                              │ │
│  │     ω = ω₀ + ω̇·Δt                                              │ │
│  │                                                                 │ │
│  │  b. Apply periodic terms (oscillations):                       │ │
│  │     Short period: ~orbit period                                │ │
│  │     Long period: ~months                                       │ │
│  │                                                                 │ │
│  │  c. Convert mean → osculating elements                        │ │
│  │                                                                 │ │
│  │  d. Kepler's equation: M → E → true anomaly                   │ │
│  │                                                                 │ │
│  │  e. Elements → position/velocity in TEME                      │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                         │                                           │
│                         ▼                                           │
│  3. OUTPUT: r = [x, y, z] km,  v = [vx, vy, vz] km/s              │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

## Apply

**SGP4 propagation with Python (using sgp4 library):**

```python
from sgp4.api import Satrec, jday
from sgp4.api import WGS84
from datetime import datetime, timedelta
import math

def propagate_tle(line1, line2, target_time):
    """
    Propagate TLE to target time using SGP4.
    
    Returns position (km) and velocity (km/s) in TEME.
    """
    satellite = Satrec.twoline2rv(line1, line2, WGS84)
    
    jd, fr = jday(target_time.year, target_time.month, target_time.day,
                  target_time.hour, target_time.minute, target_time.second)
    
    error_code, position, velocity = satellite.sgp4(jd, fr)
    
    if error_code != 0:
        errors = {
            1: "mean elements, ecc >= 1.0 or ecc < -0.001",
            2: "mean motion less than 0.0",
            3: "pert elements, ecc < 0.0 or ecc > 1.0",
            4: "semi-latus rectum < 0.0",
            5: "epoch elements are sub-orbital",
            6: "satellite has decayed"
        }
        raise ValueError(f"SGP4 error {error_code}: {errors.get(error_code, 'unknown')}")
    
    return position, velocity

def teme_to_lla(position, time):
    """
    Convert TEME position to latitude, longitude, altitude.
    Simplified conversion (ignores nutation).
    """
    x, y, z = position
    
    gmst = greenwich_sidereal_time(time)
    
    x_ecef = x * math.cos(gmst) + y * math.sin(gmst)
    y_ecef = -x * math.sin(gmst) + y * math.cos(gmst)
    z_ecef = z
    
    a = 6378.137
    b = 6356.752
    e2 = 1 - (b/a)**2
    
    lon = math.atan2(y_ecef, x_ecef)
    
    p = math.sqrt(x_ecef**2 + y_ecef**2)
    lat = math.atan2(z_ecef, p * (1 - e2))
    
    for _ in range(5):
        N = a / math.sqrt(1 - e2 * math.sin(lat)**2)
        lat = math.atan2(z_ecef + e2 * N * math.sin(lat), p)
    
    N = a / math.sqrt(1 - e2 * math.sin(lat)**2)
    alt = p / math.cos(lat) - N
    
    return math.degrees(lat), math.degrees(lon), alt

def greenwich_sidereal_time(time):
    """Compute GMST in radians."""
    jd = (time - datetime(2000, 1, 1, 12)).total_seconds() / 86400 + 2451545.0
    T = (jd - 2451545.0) / 36525.0
    
    gmst_deg = 280.46061837 + 360.98564736629 * (jd - 2451545.0) + \
               0.000387933 * T**2 - T**3 / 38710000.0
    gmst_deg = gmst_deg % 360
    
    return math.radians(gmst_deg)

def compute_ground_track(line1, line2, start_time, duration_minutes, step_minutes=1):
    """
    Compute satellite ground track over time.
    """
    track = []
    current = start_time
    end = start_time + timedelta(minutes=duration_minutes)
    
    while current <= end:
        try:
            pos, vel = propagate_tle(line1, line2, current)
            lat, lon, alt = teme_to_lla(pos, current)
            
            track.append({
                "time": current.isoformat(),
                "lat": lat,
                "lon": lon,
                "alt_km": alt,
                "speed_km_s": math.sqrt(vel[0]**2 + vel[1]**2 + vel[2]**2)
            })
        except ValueError as e:
            print(f"Propagation error at {current}: {e}")
            break
        
        current += timedelta(minutes=step_minutes)
    
    return track

def predict_passes(line1, line2, observer_lat, observer_lon, observer_alt,
                   start_time, duration_hours, min_elevation=10):
    """
    Predict satellite passes over an observer location.
    """
    passes = []
    current = start_time
    end = start_time + timedelta(hours=duration_hours)
    step = timedelta(seconds=30)
    
    in_pass = False
    pass_start = None
    max_el = 0
    max_el_time = None
    
    while current <= end:
        try:
            pos, vel = propagate_tle(line1, line2, current)
            az, el = compute_look_angles(pos, current, 
                                        observer_lat, observer_lon, observer_alt)
            
            if el > min_elevation and not in_pass:
                in_pass = True
                pass_start = current
                max_el = el
                max_el_time = current
            elif el > min_elevation and in_pass:
                if el > max_el:
                    max_el = el
                    max_el_time = current
            elif el <= min_elevation and in_pass:
                passes.append({
                    "rise": pass_start.isoformat(),
                    "set": current.isoformat(),
                    "max_elevation": max_el,
                    "max_elevation_time": max_el_time.isoformat(),
                    "duration_min": (current - pass_start).total_seconds() / 60
                })
                in_pass = False
                pass_start = None
                max_el = 0
                
        except ValueError:
            pass
        
        current += step
    
    return passes

def compute_look_angles(sat_pos_teme, time, obs_lat, obs_lon, obs_alt):
    """
    Compute azimuth and elevation from observer to satellite.
    """
    gmst = greenwich_sidereal_time(time)
    
    lat_rad = math.radians(obs_lat)
    lon_rad = math.radians(obs_lon)
    
    a = 6378.137
    e2 = 0.00669437999014
    N = a / math.sqrt(1 - e2 * math.sin(lat_rad)**2)
    
    obs_ecef = [
        (N + obs_alt/1000) * math.cos(lat_rad) * math.cos(lon_rad),
        (N + obs_alt/1000) * math.cos(lat_rad) * math.sin(lon_rad),
        (N * (1 - e2) + obs_alt/1000) * math.sin(lat_rad)
    ]
    
    sat_ecef = [
        sat_pos_teme[0] * math.cos(gmst) + sat_pos_teme[1] * math.sin(gmst),
        -sat_pos_teme[0] * math.sin(gmst) + sat_pos_teme[1] * math.cos(gmst),
        sat_pos_teme[2]
    ]
    
    range_vec = [sat_ecef[i] - obs_ecef[i] for i in range(3)]
    
    south = [-math.cos(lat_rad) * math.cos(lon_rad),
             -math.cos(lat_rad) * math.sin(lon_rad),
             -math.sin(lat_rad)]
    east = [-math.sin(lon_rad), math.cos(lon_rad), 0]
    up = [math.cos(lat_rad) * math.cos(lon_rad),
          math.cos(lat_rad) * math.sin(lon_rad),
          math.sin(lat_rad)]
    
    s = sum(range_vec[i] * south[i] for i in range(3))
    e = sum(range_vec[i] * east[i] for i in range(3))
    u = sum(range_vec[i] * up[i] for i in range(3))
    
    range_mag = math.sqrt(s**2 + e**2 + u**2)
    el = math.degrees(math.asin(u / range_mag))
    az = math.degrees(math.atan2(e, -s)) % 360
    
    return az, el

print("SGP4 Propagation Demo")
print("=" * 70)

line1 = "1 25544U 98067A   24001.50000000  .00016717  00000-0  10270-3 0  9993"
line2 = "2 25544  51.6400 247.4627 0006703 130.5360 325.0288 15.72125391463537"

print("-" * 70)

print(f"  Target time: {target.isoformat()} UTC")
print(f"\n  Position (TEME):")
print(f"    X: {pos[0]:>12.3f} km")
print(f"    Y: {pos[1]:>12.3f} km")
print(f"    Z: {pos[2]:>12.3f} km")
print(f"\n  Velocity (TEME):")
print(f"    Vx: {vel[0]:>11.6f} km/s")
print(f"    Vy: {vel[1]:>11.6f} km/s")
print(f"    Vz: {vel[2]:>11.6f} km/s")

lat, lon, alt = teme_to_lla(pos, target)
print(f"\n  Ground position:")
print(f"    Latitude:  {lat:>8.4f}°")
print(f"    Longitude: {lon:>8.4f}°")
print(f"    Altitude:  {alt:>8.2f} km")

speed = math.sqrt(vel[0]**2 + vel[1]**2 + vel[2]**2)
print(f"\n  Speed: {speed:.3f} km/s ({speed * 3600:.0f} km/h)")

print("\n2. Ground Track (90 minute orbit)")
print("-" * 70)

track = compute_ground_track(line1, line2, target, 90, step_minutes=15)
print(f"\n  {'Time':>20} {'Lat':>10} {'Lon':>10} {'Alt (km)':>10}")
print(f"  {'-'*54}")
for pt in track:
    t = pt['time'].split('T')[1][:8]
    print(f"  {t:>20} {pt['lat']:>10.2f} {pt['lon']:>10.2f} {pt['alt_km']:>10.2f}")

print("\n3. Accuracy Considerations")
print("-" * 70)
print("""
  ┌──────────────────────────────────────────────────────────────────┐
  │  TLE Age        │ Typical Error │ Use Case                      │
  │  ───────────────┼───────────────┼─────────────────────────────  │
  │  < 24 hours     │ 1-2 km        │ Tracking, antenna pointing    │
  │  1-3 days       │ 3-5 km        │ Coarse predictions            │
  │  > 1 week       │ 10+ km        │ Educational only              │
  │  After maneuver │ INVALID       │ Wait for new TLE              │
  └──────────────────────────────────────────────────────────────────┘
  
  Always use the freshest TLE available!
""")
```

**Key formulas:**

| Parameter | Formula |
|-----------|---------|
| Time since epoch | Δt = t - t_epoch (minutes) |
| Mean motion | n = 2π / T (rad/min) |
| GMST | θ = 280.46° + 360.985647°×JD + ... |
| Elevation | el = arcsin(u / range) |
| Azimuth | az = atan2(e, -s) |

<section class="lesson-pitfalls">

## Pitfalls

1. **Using TLEs with wrong propagator.** TLEs are ONLY valid with SGP4/SDP4. Using Keplerian or numerical propagators gives wrong results.

2. **Ignoring coordinate frame.** SGP4 outputs TEME, not ECEF or ECI. You must convert for ground tracks or star observations.

3. **Propagating too far from epoch.** Error grows ~1 km/day. A week-old TLE can be 10+ km off. Always get fresh TLEs.

4. **Forgetting satellite maneuvers.** After a thruster burn, the TLE is invalid. Watch for new TLEs after maneuvers.

5. **Mixing up time systems.** SGP4 uses UTC. Mixing in GPS time or TAI without conversion adds seconds of error (7+ km for ISS).
