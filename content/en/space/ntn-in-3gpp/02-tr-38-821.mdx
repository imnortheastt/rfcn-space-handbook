## Why this matters

TR 38.821 is where theory meets engineering. While TR 38.811 identified the challenges of satellite-based 5G, TR 38.821 defines the actual solutions: how to extend timing advance for 600+ km distances, how to modify HARQ for 500+ ms round-trips, and how to handle beam handovers when satellites zip across the sky. This document laid the groundwork for the normative NTN specifications in Release 17.

## How it works

### Timing advance solution

```
TR 38.821 Timing Advance Architecture:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  TOTAL TA = TA-Common + TA-Offset + TA-UE                          │
│                                                                      │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                                 │ │
│  │  1. TA-Common (broadcast in SIB):                              │ │
│  │     • Reference delay from satellite nadir point              │ │
│  │     • Shared by all UEs in the beam                            │ │
│  │     • Updated as satellite moves                               │ │
│  │     • Granularity: 4.072 μs (16 × Tc × 64)                    │ │
│  │                                                                 │ │
│  │  2. TA-Offset (UE-computed from GNSS + ephemeris):            │ │
│  │     • Difference between UE position and reference point      │ │
│  │     • UE knows its lat/lon from GNSS                          │ │
│  │     • UE knows satellite position from ephemeris              │ │
│  │     • UE computes slant range difference → TA-Offset          │ │
│  │                                                                 │ │
│  │  3. TA-UE (standard NR TA adjustment from MAC-CE):            │ │
│  │     • Fine-tuning from network timing measurements            │ │
│  │     • Same as terrestrial 5G NR                               │ │
│  │     • Corrects residual errors                                │ │
│  │                                                                 │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                      │
│  EXAMPLE (LEO-600, edge of beam):                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                                 │ │
│  │  TA-Common (nadir):      4.0 ms                                │ │
│  │  TA-Offset (edge):       +2.5 ms                               │ │
│  │  TA-UE (fine adjust):    +0.1 ms                               │ │
│  │  ─────────────────────────────                                 │ │
│  │  Total TA:               6.6 ms                                │ │
│  │                                                                 │ │
│  │  Standard NR could only express 2 ms — SOLVED!                │ │
│  │                                                                 │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

│  │                                                                 │ │
│  │  For GEO (RTT ~540 ms):                                        │ │
│  │    • HARQ retransmission adds another 540 ms                   │ │

│  │                                                                 │ │

│  │                                                                 │ │

### Doppler compensation

```
TR 38.821 Doppler Pre-Compensation:

┌─────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  CHALLENGE: LEO Doppler can exceed subcarrier spacing              │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                                 │ │
│  │  LEO-600 at 2 GHz:  ±36 kHz Doppler                           │ │
│  │  15 kHz SCS:        ICI destroys signal (>100% of SCS)        │ │
│  │  30 kHz SCS:        Still >100% of SCS                        │ │
│  │  60 kHz SCS:        ~60% of SCS (marginal)                    │ │
│  │                                                                 │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                      │
│  SOLUTION: UE pre-compensates before transmission                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                                 │ │
│  │  1. UE receives satellite ephemeris in SIB19                   │ │
│  │  2. UE knows its position from GNSS                            │ │
│  │  3. UE computes Doppler: f_d = (v · r̂) / c × f_c             │ │
│  │  4. UE shifts its transmit frequency by -f_d                   │ │
│  │  5. Signal arrives at satellite with near-zero Doppler        │ │
│  │                                                                 │ │
│  │  ┌───────────────────────────────────────────────────────┐    │ │
│  │  │   UE TX at (f_c - f_d)                                 │    │ │
│  │  │         │                                              │    │ │
│  │  │         ▼  (+f_d from motion)                         │    │ │
│  │  │   Satellite RX at ≈ f_c                               │    │ │
│  │  │                                                        │    │ │
│  │  │   Residual Doppler: typically < 1 kHz                 │    │ │
│  │  │   (within fine AFC range of receiver)                  │    │ │
│  │  └───────────────────────────────────────────────────────┘    │ │
│  │                                                                 │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                      │
│  DOWNLINK: Network compensates at gateway transmitter              │
│  UPLINK:   UE compensates using ephemeris + GNSS                   │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```

│  │                                                                 │ │
│  │  LEO-600 at 51° inclination:                                   │ │
│  │    • Ground speed: ~7 km/s                                     │ │
│  │    • Cell diameter: ~500 km                                    │ │
│  │    • Cell dwell time: ~70 seconds                              │ │
│  │    • Handover every ~1 minute                                  │ │
│  │                                                                 │ │
│  │  Compare to terrestrial: handover every 10+ minutes           │ │
│  │                                                                 │ │
│  └───────────────────────────────────────────────────────────────┘ │

│  │     │ TA1 │ TA2 │ TA3 │ TA4 │  ← Fixed grid on Earth        │ │

│  │                │ Satellite beam sweeps across                │ │

│  │                                                                 │ │
│  │  Benefit: Fewer TAU procedures (UE stays in same TA)         │ │

│  │                                                                 │ │

│  │  2. UE monitors conditions (position, signal quality)          │ │
│  │  3. When condition met, UE executes HO autonomously           │ │
│  │  4. No round-trip delay for HO command                        │ │
│  │                                                                 │ │
│  │  Critical for LEO where HO decision + execution must be fast │ │

import math
from datetime import datetime, timedelta

def compute_ta_components(ue_lat, ue_lon, sat_lat, sat_lon, sat_alt_km, 
                          ref_lat, ref_lon):
    """
    Compute TA-Common and TA-Offset per TR 38.821.
    """
    def slant_range(lat1, lon1, lat2, lon2, alt):
        R = 6371
        lat1, lon1, lat2, lon2 = map(math.radians, [lat1, lon1, lat2, lon2])
        
        dlat = lat2 - lat1
        dlon = lon2 - lon1
        a = math.sin(dlat/2)**2 + math.cos(lat1) * math.cos(lat2) * math.sin(dlon/2)**2
        c = 2 * math.asin(math.sqrt(a))
        ground_dist = R * c
        
        el = math.atan2(alt - ground_dist * math.tan(0), ground_dist)
        slant = math.sqrt(ground_dist**2 + alt**2 + 2 * R * alt)
        
        return slant
    
    c = 299792.458
    
    ref_slant = slant_range(ref_lat, ref_lon, sat_lat, sat_lon, sat_alt_km)
    ta_common_ms = (2 * ref_slant / c) * 1000
    
    ue_slant = slant_range(ue_lat, ue_lon, sat_lat, sat_lon, sat_alt_km)
    ta_offset_ms = (2 * (ue_slant - ref_slant) / c) * 1000
    
    total_ta_ms = ta_common_ms + ta_offset_ms
    
    return {
        "ref_slant_km": ref_slant,
        "ue_slant_km": ue_slant,
        "ta_common_ms": ta_common_ms,
        "ta_offset_ms": ta_offset_ms,
        "total_ta_ms": total_ta_ms
    }

def compute_doppler_compensation(ue_lat, ue_lon, sat_velocity_km_s, 
                                  sat_heading_deg, freq_ghz):
    """
    Compute Doppler shift and required compensation.
    """
    c = 299792.458
    
    heading_rad = math.radians(sat_heading_deg)
    vel_north = sat_velocity_km_s * math.cos(heading_rad)
    vel_east = sat_velocity_km_s * math.sin(heading_rad)
    
    doppler_hz = (sat_velocity_km_s / c) * freq_ghz * 1e9 * math.cos(math.radians(30))
    
    compensation_hz = -doppler_hz
    
    return {
        "doppler_hz": doppler_hz,
        "compensation_hz": compensation_hz,
        "doppler_khz": doppler_hz / 1000,
        "relative_to_15khz_scs": abs(doppler_hz) / 15000 * 100,
        "relative_to_30khz_scs": abs(doppler_hz) / 30000 * 100
    }

def harq_process_calculation(rtt_ms, slot_duration_ms, target_throughput_mbps):
    """
    Calculate HARQ process requirements for NTN.
    """
    slots_in_flight = math.ceil(rtt_ms / slot_duration_ms)
    
    min_processes = slots_in_flight + 1
    
    standard_processes = 16
    extended_processes = 32
    
    if min_processes <= standard_processes:
        recommended = standard_processes
        mode = "Standard NR"
    elif min_processes <= extended_processes:
        recommended = extended_processes
        mode = "NTN Extended"
    else:
        recommended = None
        mode = "HARQ Disabled"
    
    return {
        "rtt_ms": rtt_ms,
        "slots_in_flight": slots_in_flight,
        "min_processes_needed": min_processes,
        "recommended_processes": recommended,
        "mode": mode
    }

def mobility_analysis(sat_velocity_km_s, cell_diameter_km, ho_execution_ms):
    """
    Analyze mobility requirements for NTN.
    """
    cell_dwell_time_s = cell_diameter_km / sat_velocity_km_s
    
    ho_per_hour = 3600 / cell_dwell_time_s
    
    distance_during_ho = sat_velocity_km_s * (ho_execution_ms / 1000)
    
    return {
        "cell_dwell_time_s": cell_dwell_time_s,
        "handovers_per_hour": ho_per_hour,
        "distance_during_ho_km": distance_during_ho,
        "cho_recommended": ho_execution_ms > 100
    }

## Pitfalls

1. **Assuming UE has perfect GNSS.** Indoor or urban canyon UEs may have poor GNSS. Fallback mechanisms exist but degrade performance.

2. **Ignoring ephemeris staleness.** Satellite position changes — stale ephemeris causes timing/Doppler errors. SIB19 must be refreshed regularly.

3. **Forgetting feeder link timing.** For transparent payloads, gateway-to-satellite delay adds to total TA. Include both hops.

4. **Using wrong Doppler sign.** Doppler is positive when satellite approaches, negative when receding. Pre-compensation must use the opposite sign.

5. **Underestimating HO frequency.** LEO constellations may require 50+ handovers per hour. Signaling overhead is significant.
